# Logout

---

## 1. Finalidade do `docker logout`

O **`docker logout`** remove suas credenciais de um **registry** Docker do arquivo local (`~/.docker/config.json`), fazendo com que futuros comandos que dependam de autenticação (como `pull` ou `push`) exijam novo login.

---

## 2. Sintaxe básica

```bash
docker logout [SERVER]

```

- **`SERVER`** (opcional): host ou URL do registry do qual você deseja sair.
- Se omitido, usa o registry padrão configurado no daemon (normalmente o Docker Hub).

---

## 3. Parâmetros

| Parâmetro | Obrigatório | Descrição |
| --- | --- | --- |
| `SERVER` | Não | Registry de onde as credenciais serão removidas. |

---

## 4. Fluxo de funcionamento

1. **Identificação do registry**
    - Se você especificou `SERVER`, o comando mira naquele registro.
    - Caso contrário, atua sobre o padrão (Docker Hub).
2. **Remoção de credenciais**
    - Apaga a entrada correspondente em `~/.docker/config.json`.
3. **Resultado**
    - Imprime mensagem confirmando o logout, por exemplo:
        
        ```
        Removing login credentials for <SERVER>
        
        ```
        

---

## 5. Exemplos de uso

1. **Logout do Docker Hub (padrão)**
    
    ```bash
    docker logout
    # Removing login credentials for https://index.docker.io/v1/
    
    ```
    
2. **Logout de um registry privado**
    
    ```bash
    docker logout myregistry.example.com
    # Removing login credentials for myregistry.example.com
    
    ```
    
3. **Script para limpar múltiplos registros**
    
    ```bash
    for reg in index.docker.io mycorp.registry.com; do
      docker logout $reg
    done
    
    ```
    

---

## 6. Tabela de referência rápida

| Componente | Sintaxe | Descrição |
| --- | --- | --- |
| Comando base | `docker logout [SERVER]` | Remove credenciais armazenadas para o registry especificado. |
| Parâmetro | `SERVER` | Host/URL do registry; se omitido, usa o padrão (Docker Hub). |

---

Após o `logout`, sempre que você tentar `docker pull` ou `docker push` naquele registry, será solicitado novo login. Se precisar de mais detalhes, estou à disposição!
