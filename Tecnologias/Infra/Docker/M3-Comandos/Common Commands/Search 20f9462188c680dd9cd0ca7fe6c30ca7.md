# Search

---

## 1. O que faz o `docker search`

O **`docker search`** permite **procurar** imagens públicas no **Docker Hub** (ou em outro registry configurado) com base em um termo de busca. Ele retorna metadados básicos (nome, descrição, stars, oficial, automated) para te ajudar a escolher a imagem mais adequada.

---

## 2. Sintaxe básica

```bash
docker search [OPTIONS] TERM

```

- **`TERM`**: palavra-chave a ser buscada no nome ou na descrição das imagens (ex.: `nginx`, `node`, `postgres`).

---

## 3. Parâmetro

| Parâmetro | Obrigatório | Descrição |
| --- | --- | --- |
| `TERM` | Sim | Termo de pesquisa para filtrar imagens no Hub. |

---

## 4. Opções disponíveis

| Opção | Tipo | Padrão | Finalidade |
| --- | --- | --- | --- |
| `-f, --filter filter` | string | — | Filtra resultados por condição, ex.: `is-official=true`, `stars=100`, `is-automated=true`. |
| `--format string` | string | `table` | Usa **Go template** para formatar a saída (ex.: `--format "{{.Name}}\t{{.StarCount}}"`). |
| `--limit int` | integer | `25` | Número máximo de resultados retornados. |
| `--no-trunc` | boolean | `false` | Não trunca campos longos (descrição, nome), mostra texto completo. |

---

## 5. Exemplos de uso

1. **Pesquisar “nginx”**
    
    ```bash
    docker search nginx
    # NAME                 DESCRIPTION                                     STARS   OFFICIAL   AUTOMATED
    # nginx                Official build of Nginx.                        18500   [OK]
    # jwilder/nginx-proxy  Automated Nginx reverse proxy for docker con…   2120             [OK]
    
    ```
    
2. **Mostrar apenas imagens oficiais**
    
    ```bash
    docker search --filter is-official=true ubuntu
    # NAME      DESCRIPTION                                     STARS   OFFICIAL   AUTOMATED
    # ubuntu    Ubuntu is a Debian-based Linux operating sys…   14000   [OK]
    
    ```
    
3. **Limitar resultados a 5**
    
    ```bash
    docker search --limit 5 redis
    
    ```
    
4. **Formatar somente nome e número de estrelas**
    
    ```bash
    docker search --format "{{.Name}}: {{.StarCount}} stars" mysql
    # library/mysql: 8200 stars
    # bitnami/mysql: 3100 stars
    # ...
    
    ```
    
5. **Não truncar descrição**
    
    ```bash
    docker search --no-trunc alpine
    
    ```

## 6. Tabela de referência rápida

| Componente | Sintaxe | Descrição |
| --- | --- | --- |
| Comando base | `docker search [OPTIONS] TERM` | Procura imagens no Docker Hub por termo. |
| Parâmetro | `TERM` | Palavra-chave para buscar no nome/descrição. |
| Opção | `-f, --filter <condição>` | Filtra resultados (ex.: `stars=100`, `is-automated=true`). |
| Opção | `--format <template>` | Formata saída com Go template. |
| Opção | `--limit <n>` | Máximo de resultados (padrão 25). |
| Opção | `--no-trunc` | Mostra campos completos, sem truncar. |

Com isso, você dispõe de tudo para usar o **`docker search`** e encontrar rapidamente as imagens certas no Docker Hub. Se precisar de mais dicas ou filtros específicos, é só falar!
