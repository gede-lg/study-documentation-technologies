# Version

---

## 1. Finalidade do `docker version`

O **`docker version`** exibe informações sobre as versões do **cliente** e do **daemon** Docker que estão instalados e em execução no seu host. É útil para diagnósticos, compatibilidades de API e verificação de atualizações disponíveis.

---

## 2. Sintaxe básica

```bash
docker version [OPTIONS]

```

- **`docker version`**: comando para mostrar detalhes de versão.
- **`[OPTIONS]`**: flags que alteram o formato de saída.

---

## 3. Parâmetros

Este comando **não** recebe argumentos posicionais — apenas as opções listadas abaixo.

---

## 4. Opções disponíveis

| Opção | Tipo | Padrão | Descrição |
| --- | --- | --- | --- |
| `-f, --format string` | string | `plain` | Formata a saída usando um template Go ou em JSON. |
|  |  |  | - `json` → imprime toda a informação em JSON bruto. |
|  |  |  | - `TEMPLATE` → customiza via Go template (ex.: `--format '{{.Server.Version}}'`). |

---

## 5. Fluxo de funcionamento

1. **Detecção do cliente**
    - Exibe a versão e build do CLI Docker em uso.
2. **Conexão com o daemon**
    - Se o daemon estiver ativo, exibe também sua versão, API mínima e build.
    - Se não conseguir conectar (daemon parado), mostra apenas a parte do cliente e indica o erro de conexão.

---

## 6. Exemplos de uso

1. **Exibir versão de forma simples**
    
    ```bash
    docker version
    # Cliente:
    #  Version:           24.0.2
    #  API version:       1.42
    #  Go version:        go1.20.5
    # Daemon:
    #  Version:           24.0.2
    #  API version:       1.42 (mínima 1.12)
    
    ```
    
2. **Saída em JSON**
    
    ```bash
    docker version --format json
    # {
    #   "Client": { "Version": "24.0.2", "ApiVersion": "1.42", … },
    #   "Server": { "Version": "24.0.2", "ApiVersion": "1.42", … }
    # }
    
    ```
    
3. **Extrair apenas a versão do daemon**
    
    ```bash
    docker version --format '{{.Server.Version}}'
    # 24.0.2
    
    ```
    
4. **Mostrar versão do cliente**
    
    ```bash
    docker version --format '{{.Client.Version}}'
    # 24.0.2
    
    ```
