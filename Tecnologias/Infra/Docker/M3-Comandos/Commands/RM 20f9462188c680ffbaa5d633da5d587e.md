# RM

---

## 1. Finalidade do `docker rm`

O **`docker rm`** remove um ou mais **contêineres** do host Docker. Por padrão, só remove containers que **já estão parados**; para remover em execução, é necessário usar `--force`. Também pode limpar volumes anônimos associados e links de container.

---

## 2. Sintaxe básica

```bash
docker rm [OPTIONS] CONTAINER [CONTAINER...]

```

- **`CONTAINER`**: nome(s) ou ID(s) do(s) contêiner(es) a serem removidos. Pode passar múltiplos de uma vez.

---

## 3. Opções disponíveis

| Opção | Tipo | Padrão | Descrição |
| --- | --- | --- | --- |
| `-f, --force` | boolean | `false` | Força a remoção de contêineres em **execução** (envia SIGKILL antes de remover). |
| `-v, --volumes` | boolean | `false` | Remove **volumes anônimos** montados pelo contêiner. Não afeta volumes nomeados. |
| `-l, --link` | boolean | `false` | Remove apenas o **link** especificado, não o próprio contêiner (usar em contêineres vinculados). |

---

## 4. Exemplos de uso

1. **Remover um contêiner parado**
    
    ```bash
    docker rm my_container
    
    ```
    
2. **Remover múltiplos contêineres**
    
    ```bash
    docker rm web1 web2 db
    
    ```
    
3. **Forçar remoção de um contêiner em execução**
    
    ```bash
    docker rm --force running_container
    
    ```
    
4. **Remover contêiner e seus volumes anônimos**
    
    ```bash
    docker rm --volumes old_container
    
    ```
    
5. **Remover apenas o link entre contêineres**
    
    ```bash
    docker rm --link mylink
    
    ```
    

---

## 5. Tabela de referência rápida

| Componente | Sintaxe | Descrição |
| --- | --- | --- |
| Comando | `docker rm [OPTIONS] CONTAINER [CONTAINER...]` | Remove contêiner(es) do host Docker. |
| Opção | `-f, --force` | Força remoção mesmo se em execução. |
| Opção | `-v, --volumes` | Remove volumes anônimos associados. |
| Opção | `-l, --link` | Remove apenas o link especificado. |

---

Use **`docker rm`** para limpar contêineres que não são mais necessários e, opcionalmente, liberar também seus volumes órfãos.
