# Inspect

---

## 1. Finalidade do `docker inspect`

O **`docker inspect`** retorna informações detalhadas em baixo nível (JSON) sobre objetos Docker, como **containers**, **imagens**, **volumes**, **networks**, **nodes**, **services**, etc. É essencial para depuração e automação, pois permite extrair qualquer campo de metadata.

---

## 2. Sintaxe básica

```bash
docker inspect [OPTIONS] NAME|ID [NAME|ID...]

```

- **`NAME|ID`**: nome ou ID dos objetos a inspecionar (pode fornecer múltiplos).
    - Ex.: container (`my_container`), imagem (`ubuntu:20.04` ou seu ID), volume, network, serviço, node, etc.

---

## 3. Parâmetros

| Parâmetro | Obrigatório | Descrição |
| --- | --- | --- |
| `NAME | ID [...]` | Sim |

---

## 4. Opções disponíveis

| Opção | Tipo | Padrão | Descrição |
| --- | --- | --- | --- |
| `-f, --format string` | string | `json` | Formata a saída usando um **Go template** ou em **JSON**. |
- `json` → JSON bruto completo.
- `TEMPLATE` → p.ex.: `-format '{{.Config.Image}}'`. |
    
    | `-s, --size`            | boolean | `false`  | Exibe **tamanhos** de camadas e do writable layer para **containers**.                                          |
    
    | `--type string`         | string  | —        | Restringe inspeção a um **tipo** de objeto (`container`, `image`, `volume`, `network`, `service`, `node`, etc.). |
    

---

## 5. Exemplos de uso

1. **Inspecionar um container e ver todo o JSON**
    
    ```bash
    docker inspect my_container
    
    ```
    
2. **Extrair apenas o endereço IP do container (formato template)**
    
    ```bash
    docker inspect -f '{{range .NetworkSettings.Networks}}{{.IPAddress}}{{end}}' my_container
    
    ```
    
3. **Obter o comando CMD de uma imagem**
    
    ```bash
    docker inspect --type=image -f '{{.Config.Cmd}}' ubuntu:20.04
    
    ```
    
4. **Ver tamanhos de camadas de um container**
    
    ```bash
    docker inspect -s my_container
    # Exibe SizeRw e SizeRootFs
    
    ```
    
5. **Inspecionar múltiplos objetos de uma vez**
    
    ```bash
    docker inspect webapp db redis
    
    ```
