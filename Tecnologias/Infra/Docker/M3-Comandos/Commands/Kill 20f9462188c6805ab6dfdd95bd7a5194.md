# Kill

---

## 1. Finalidade do `docker kill`

O **`docker kill`** envia um **sinal** para um ou mais contêineres em execução, forçando-os a parar **imediatamente**. Ao contrário de `docker stop` (que tenta um encerramento gracioso via SIGTERM seguido de SIGKILL após timeout), o `kill` dispara o sinal especificado (por padrão SIGKILL) de forma instantânea.

---

## 2. Sintaxe básica

```bash
docker kill [OPTIONS] CONTAINER [CONTAINER...]

```

- **`CONTAINER`**: nome ou ID do(s) contêiner(es) a receberem o sinal. Você pode listar múltiplos contêineres em sequência.

---

## 3. Parâmetro

| Parâmetro | Obrigatório | Descrição |
| --- | --- | --- |
| `CONTAINER` | Sim | Nome(s) ou ID(s) do(s) contêiner(es) que serão “kill”ados. |

---

## 4. Opções disponíveis

| Opção | Tipo | Padrão | Descrição |
| --- | --- | --- | --- |
| `-s, --signal string` | string | `SIGKILL` | Sinal UNIX a enviar ao processo principal do contêiner (ex.: `SIGTERM`, `SIGINT`, `SIGQUIT`). |

---

## 5. Exemplos de uso

1. **Enviar SIGKILL (padrão) a um container**
    
    ```bash
    docker kill my_container
    
    ```
    
    Equivalente a `docker kill --signal SIGKILL my_container`.
    
2. **Enviar SIGTERM para um container**
    
    ```bash
    docker kill --signal SIGTERM my_container
    
    ```
    
    Tenta um encerramento mais gracioso antes de forçar.
    
3. **Matar múltiplos containers de uma vez**
    
    ```bash
    docker kill web_api db_cache worker
    
    ```
    
4. **Usar nome parcial ou ID abreviado**
    
    ```bash
    docker kill 8f3a1d2c
    
    ```
    

---

## 6. Tabela de referência rápida

| Componente | Sintaxe / Valor | Descrição |
| --- | --- | --- |
| Comando base | `docker kill [OPTIONS] CONTAINER...` | Envia sinal imediatamente para parar o(s) container(es). |
| Parâmetro | `CONTAINER` | Nome ou ID do container-alvo. |
| Opção | `-s, --signal <SIGNAL>` | Define o sinal UNIX a enviar (padrão `SIGKILL`). |

---

Use **`docker kill`** quando precisar interromper **imediatamente** um contêiner, escolhendo o sinal mais adequado para a sua situação.
