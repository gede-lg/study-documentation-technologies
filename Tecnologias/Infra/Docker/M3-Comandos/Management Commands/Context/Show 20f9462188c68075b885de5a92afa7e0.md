# Show

## Visão Geral

O comando `docker context show` imprime o **nome** do contexto Docker atualmente ativo no cliente. É útil para scripts e verificações rápidas de qual ambiente está em uso antes de executar outros comandos.

---

## Sintaxe

```bash
docker context show

```

---

## Parâmetros

Este comando **não** possui parâmetros ou opções adicionais:

- Ele simplesmente exibe o nome do contexto que está marcado com “*” em `docker context ls`.

---

## Exemplos de Uso

1. **Exibir o contexto ativo**
    
    ```bash
    $ docker context show
    default
    
    ```
    
    Retorna `default` se esse for o contexto em uso.
    
2. **Em scripts de shell**
    
    ```bash
    current_ctx=$(docker context show)
    echo "Contexto atual: $current_ctx"
    
    ```
    
    Armazena o nome do contexto em uma variável para uso posterior em automações.
    

---

> Dica:
> 
> 
> Combine com `docker context ls` para ver rapidamente todos os contextos disponíveis e identificar qual está ativo:
> 
> ```bash
> docker context ls
> echo "Ativo: $(docker context show)"
> 
> ```
