# LS

## Visão Geral

O comando `docker plugin ls` lista todos os plugins instalados no seu host Docker, mostrando informações como ID, nome, descrição e estado (habilitado ou não). É útil para verificar rapidamente quais extensões estão disponíveis e seu status atual. ([docs.docker.com](https://docs.docker.com/reference/cli/docker/plugin/ls/?utm_source=chatgpt.com))

---

## Sintaxe

```bash
docker plugin ls [OPÇÕES]

```

---

## Aliases

- `docker plugin list`

---

## Opções

| Opção | Descrição |
| --- | --- |
| `-f, --filter filter` | Filtra a saída por condição. Exemplos de filtros:• `enabled=true` ou `false`• `capability=volumedriver`• `capability=networkdriver` ([man.archlinux.org](https://man.archlinux.org/man/docker-plugin-ls.1.en?utm_source=chatgpt.com)) |
| `--format string` | Formata a saída usando um template Go:• `table`: exibe tabela (padrão)• `table TEMPLATE`• `json`• `TEMPLATE` |
| `--no-trunc` | Não trunca valores longos (IDs, nomes etc.), mostrando o texto completo. |
| `-q, --quiet` | Exibe apenas os **IDs** dos plugins, um por linha. |

---

## Exemplos de Uso

1. **Listar todos os plugins instalados**
    
    ```bash
    docker plugin ls
    
    ```
    
    Saída típica:
    
    ```
    ID                  NAME                           DESCRIPTION              ENABLED
    4f4b060cbeaa        rexray/s3fs:latest             REX-Ray S3 Script plugin true
    9a2b7d9b4e3f        traefik:latest                 Traefik edge router      false
    
    ```
    
2. **Exibir apenas plugins habilitados**
    
    ```bash
    docker plugin ls --filter enabled=true
    
    ```
    
3. **Filtrar por capacidade de volume driver**
    
    ```bash
    docker plugin ls --filter capability=volumedriver
    
    ```
    
4. **Mostrar somente IDs dos plugins**
    
    ```bash
    docker plugin ls -q
    
    ```
    
5. **Formatar como JSON para integração com scripts**
    
    ```bash
    docker plugin ls --format=json
    
    ```
    
6. **Tabela customizada com nome e status**
    
    ```bash
    docker plugin ls --format "table {{.Name}}\t{{.Enabled}}"
    
    ```
