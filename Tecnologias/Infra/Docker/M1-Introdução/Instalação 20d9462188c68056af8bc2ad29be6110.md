# Instalação

Seguir o passo a passo da pagina ensinando a instalar o docker engine:

[Ubuntu | Docker Docs](https://docs.docker.com/engine/install/ubuntu/)

após seguir todos os passos, rodar `systemctl status docker` para ver se o docker está ativo e rodando ( ele precisa estar ativo e rodando ).

Caso não esteja, rodar o comando `sudo systemctl start docker` para ativar o docker engine.

após isso, rodar os comandos:

```bash
## Comandos para habilitar a auto-execução do docker quando a maquina for reiniciada
sudo systemctl enable docker.socket
sudo systemctl enable docker.service

## Configurar o usuario normal para utilizar o docker

# captura o grupo do docker
getent group docker 
# Adiciona o usuario ao grupo do docker para poder utilizar o docker sem ser root
sudo usermod -a -G docker <nome_usuario>

## Atualizar os grupos da seção sem precisar dar reboot
newgrp docker 
```
