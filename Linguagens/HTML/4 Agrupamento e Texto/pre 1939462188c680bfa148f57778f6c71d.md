# <pre>

## 1. Introdução

A tag `<pre>` é um elemento do HTML5 utilizado para exibir blocos de texto pré-formatado. Ela mantém os espaços em branco, quebras de linha e tabulações exatamente como foram escritos no código-fonte, diferentemente de outros elementos de texto, como `<p>` e `<span>`, que normalmente ignoram espaços extras.

Essa funcionalidade é essencial para exibição de códigos-fonte, mensagens de terminal, ASCII art e qualquer outra saída de texto onde o formato original deve ser preservado.

## 3. Conteúdo Detalhado

### 3.1 Definição e Conceitos Fundamentais

A tag `<pre>` significa "preformatted text" (texto pré-formatado). Seu objetivo principal é preservar a formatação do texto exatamente como foi escrito dentro da tag, o que a torna ideal para exibir conteúdos que dependem da formatação exata, como:

- Código-fonte de programas;
- Dados tabulares simples sem o uso de tabelas;
- Comandos e saídas de terminal;
- Diagramas em ASCII.

### 3.2 Sintaxe e Estrutura

A sintaxe básica da tag `<pre>` é simples:

```html
<pre>
Texto pré-formatado mantido exatamente como escrito.
   Incluindo espaços, tabulações e quebras de linha.
</pre>

```

### 3.3 Componentes Principais

A `<pre>` pode conter qualquer tipo de texto, incluindo espaços e quebras de linha. Entretanto, normalmente é combinada com a tag `<code>` para destacar trechos de código:

```html
<pre>
<code>
function helloWorld() {
    console.log("Hello, World!");
}
</code>
</pre>

```

### 3.4 Propriedades e Atributos

A tag `<pre>` não possui atributos específicos além dos globais do HTML5, como:

- **`class`** – Para estilização com CSS.
- **`id`** – Para identificação única no documento.
- **`style`** – Para estilização inline.
- **`lang`** – Para indicar o idioma do conteúdo.

### Exemplo de estilização:

```html
<pre style="color: blue; background-color: lightgray; padding: 10px;">
Este é um texto pré-formatado estilizado.
</pre>

```

### 3.5 Uso Avançado

A tag `<pre>` pode ser combinada com CSS para manipulação avançada, como rolagem de blocos de código grandes:

```css
pre {
    white-space: pre-wrap; /* Permite quebras de linha automáticas */
    word-wrap: break-word; /* Evita que palavras longas quebrem o layout */
    overflow-x: auto; /* Adiciona barra de rolagem horizontal quando necessário */
}

```

E no HTML:

```html
<pre>
Este texto será quebrado automaticamente quando atingir o limite da tela.
</pre>

```

## 4. Exemplos Práticos

### Exemplo 1: Exibição de Código

```html
<pre>
<code>
def soma(a, b):
    return a + b

print(soma(2, 3))
</code>
</pre>

```

### Exemplo 2: Texto pré-formatado com ASCII Art

```html
<pre>
  *
 ***
*****
 ***
  *
</pre>

```

### Exemplo 3: Combinando `<pre>` com `<kbd>` para exibir atalhos de teclado

```html
<pre>
Pressione <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar o texto.
</pre>

```

---

## 5. Informações Adicionais

- O `<pre>` é frequentemente usado junto com `<code>` para exibir códigos-fonte corretamente.
- Pode ser estilizado para exibir fontes monoespaçadas, imitando editores de código.
- O uso da propriedade `white-space: pre-wrap;` é recomendado para evitar rolagem horizontal excessiva.
