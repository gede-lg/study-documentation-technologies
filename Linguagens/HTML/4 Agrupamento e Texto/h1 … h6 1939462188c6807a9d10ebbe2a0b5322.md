# <h1>…<h6>

## 1. Introdução

As tags `<h1>` até `<h6>` são utilizadas em HTML5 para definir títulos e subtítulos dentro de uma página web. Elas possuem um papel fundamental na estruturação do conteúdo, impactando tanto a acessibilidade quanto a otimização para mecanismos de busca (SEO). O uso correto dessas tags melhora a organização da informação, tornando o conteúdo mais compreensível para usuários e motores de busca.

## 3. Conteúdo Detalhado

### 3.1 Definição e conceitos fundamentais

As tags `<h1>` até `<h6>` são elementos semânticos utilizados para estruturar títulos e subtítulos. Elas são organizadas de forma hierárquica, onde:

- `<h1>` representa o título mais importante (de maior hierarquia)
- `<h2>` é um subtítulo do `<h1>`
- `<h3>` é um subtítulo do `<h2>` e assim por diante até `<h6>`

Elas ajudam na escaneabilidade do conteúdo, facilitando a navegação tanto para usuários humanos quanto para tecnologias assistivas.

### 3.2 Sintaxe e estrutura

A sintaxe das tags de cabeçalho é simples:

```html
<h1>Este é um título de nível 1</h1>
<h2>Este é um título de nível 2</h2>
<h3>Este é um título de nível 3</h3>
<h4>Este é um título de nível 4</h4>
<h5>Este é um título de nível 5</h5>
<h6>Este é um título de nível 6</h6>

```

A estrutura correta dessas tags é importante para garantir uma hierarquia lógica do conteúdo, facilitando a leitura e o ranqueamento nos mecanismos de busca.

### 3.3 Componentes principais

As tags `<h1>` até `<h6>` possuem apenas o conteúdo textual como componente principal. Elas não podem conter outros elementos em seu interior, exceto inline elements, como `<span>`.

### 3.4 Propriedades/Atributos específicos

Essas tags não possuem atributos específicos próprios além dos atributos globais do HTML5, como:

- `id`: Define um identificador único para o cabeçalho.
- `class`: Permite aplicar estilos CSS personalizados.
- `style`: Define estilos inline.
- `lang`: Especifica o idioma do conteúdo do cabeçalho.

Exemplo:

```html
<h1 id="principal" class="titulo" style="color: blue;">Título Principal</h1>

```

### 3.5 Uso avançado

### 3.5.1 SEO (Search Engine Optimization)

Os motores de busca atribuem grande importância às tags de cabeçalho, especialmente `<h1>`. Regras gerais:

- Use `<h1>` apenas uma vez por página para o título principal.
- Utilize `<h2>` para seções principais e `<h3>` para subseções.
- Evite pular níveis de hierarquia (exemplo: de `<h1>` diretamente para `<h4>`).

### 3.5.2 Acessibilidade

Ferramentas de leitura de tela usam essas tags para navegar entre seções da página. Manter uma estrutura coerente melhora a acessibilidade do conteúdo.

### 3.5.3 Customização com CSS

Os cabeçalhos podem ser estilizados para melhor apresentação:

```css
h1 {
    font-size: 2.5em;
    color: #333;
}
h2 {
    font-size: 2em;
    color: #555;
}

```

## 4. Exemplos práticos

### Exemplo básico

```html
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo de Cabeçalhos</title>
</head>
<body>
    <h1>Blog de Tecnologia</h1>
    <h2>Novidades</h2>
    <h3>Lançamento de Novos Processadores</h3>
    <h3>Avanços na Inteligência Artificial</h3>
    <h2>Tutoriais</h2>
    <h3>Como Configurar um Servidor Web</h3>
</body>
</html>

```

### Exemplo avançado com CSS

```html
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Estilização de Cabeçalhos</title>
    <style>
        h1 {
            font-size: 2.5em;
            color: navy;
            text-align: center;
        }
        h2 {
            font-size: 2em;
            color: darkred;
        }
        h3 {
            font-size: 1.5em;
            color: darkgreen;
        }
    </style>
</head>
<body>
    <h1>Meu Site</h1>
    <h2>Seção de Notícias</h2>
    <h3>Últimos Acontecimentos</h3>
</body>
</html>

```

## 5. Informações adicionais

- Evite usar cabeçalhos apenas para estilização. Se precisar de tamanhos diferentes para textos, use CSS.
- Sempre siga a ordem hierárquica para garantir acessibilidade e SEO eficiente.
- Ferramentas como o Google Lighthouse podem ajudar a avaliar a estrutura do HTML para otimização.
