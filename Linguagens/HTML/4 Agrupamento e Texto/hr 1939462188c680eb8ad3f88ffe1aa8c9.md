# <hr>

## 1. Introdução

A tag `<hr>` é um elemento do HTML5 utilizado para criar uma linha horizontal na página web. Ela tem como principal função a separação de conteúdo dentro de um documento HTML, funcionando como um divisor visual. Além do aspecto visual, a tag `<hr>` também tem um significado semântico, indicando uma mudança de contexto entre seções de conteúdo.

## 3. Definição e conceitos fundamentais

A tag `<hr>` (horizontal rule) representa uma divisão temática entre elementos de uma página. Em versões anteriores do HTML, ela era usada puramente para estilização, mas no HTML5 ganhou uma função semântica, indicando uma quebra entre blocos de conteúdo.

Exemplo de uso:

```html
<p>Primeiro parágrafo</p>
<hr>
<p>Segundo parágrafo</p>

```

## 4. Sintaxe e estrutura

A tag `<hr>` é um elemento vazio (ou seja, não precisa de uma tag de fechamento) e segue a sintaxe:

```html
<hr>

```

Ela pode ser estilizada com CSS para personalizar sua aparência.

## 5. Componentes principais

A `<hr>` é composta por um elemento auto-fechado e pode ser estilizada através de CSS. Ela não possui conteúdo interno, apenas um espaço de separação entre elementos na página.

## 6. Propriedades/Atributos

A tag `<hr>` não possui atributos próprios, mas pode ser personalizada via CSS. Alguns atributos HTML obsoletos como `size`, `width`, `noshade` e `align` foram descontinuados em versões modernas do HTML.

### Atributos aplicáveis via CSS

Os principais estilos que podem ser aplicados a `<hr>` são:

```css
hr {
  border: none;
  height: 2px;
  background-color: #333;
  margin: 20px 0;
}

```

## 7. Uso avançado

Além do uso padrão, a `<hr>` pode ser utilizada em conjunto com CSS para criar efeitos visuais personalizados:

### Exemplo de linha pontilhada:

```css
hr {
  border: none;
  border-top: 2px dashed gray;
}

```

### Exemplo de linha colorida e estilizada:

```css
hr {
  height: 5px;
  background: linear-gradient(to right, red, blue);
  border: none;
}

```

## 8. Exemplos práticos

### Exemplo 1: Linha básica

```html
<p>Seção 1</p>
<hr>
<p>Seção 2</p>

```

### Exemplo 2: Linha estilizada com espaçamento

```css
hr {
  width: 80%;
  height: 3px;
  background-color: black;
  margin: 40px auto;
}

```

```html
<p>Texto antes da linha</p>
<hr>
<p>Texto depois da linha</p>

```

## 9. Informações adicionais

- A `<hr>` pode ser utilizada para indicar mudanças de contexto sem adicionar elementos visuais desnecessários.
- A tag pode ser estilizada para combinar melhor com a identidade visual do site.
- Elementos como `<div>` e `<section>` podem substituir `<hr>` quando a semântica não for necessária.
