# <em> e <strong>

## 1. Introdução

No desenvolvimento web moderno, a semântica é um pilar fundamental para garantir que o conteúdo seja compreendido não apenas pelos navegadores, mas também por tecnologias assistivas e mecanismos de busca. As tags `<em>` e `<strong>` são exemplos de elementos semânticos do HTML5 que indicam, respectivamente, ênfase e importância em partes específicas do texto. Seu uso correto contribui para uma melhor acessibilidade, SEO (Search Engine Optimization) e clareza na estruturação do conteúdo.

## 3. Conteúdo Detalhado

### 3.1 Definição e Conceitos Fundamentais

- **`<em>` (ênfase):**
    - **Conceito:** Representa uma ênfase moderada no conteúdo, sugerindo uma mudança de entonação ou a necessidade de se dar uma atenção especial a essa parte do texto.
    - **Semântica:** Indica que o trecho tem uma inflexão ou destaque na leitura, mas sem atribuir um nível extremo de importância.
- **`<strong>` (forte ênfase):**
    - **Conceito:** Indica que o conteúdo possui uma importância extra, devendo receber atenção significativa.
    - **Semântica:** Transmite a ideia de que o trecho é crítico ou essencial dentro do contexto, indo além de uma simples alteração de entonação.

**Diferença chave:**

Enquanto `<em>` é utilizado para modificar a entonação e sinalizar ênfase leve, `<strong>` atribui uma relevância maior, funcionando como uma indicação de alerta para o conteúdo destacado.

---

### 3.2 Sintaxe e Estrutura

A sintaxe para ambos os elementos segue o padrão HTML, onde cada tag possui uma abertura e um fechamento, envolvendo o conteúdo que deseja destacar.

- **Sintaxe do `<em>`:**
    
    ```html
    <em>Texto enfatizado</em>
    
    ```
    
- **Sintaxe do `<strong>`:**
    
    ```html
    <strong>Texto de grande importância</strong>
    
    ```
    

**Exemplo básico:**

```html
<p>Este é um exemplo de <em>texto enfatizado</em> e de <strong>texto importante</strong>.</p>

```

---

### 3.3 Componentes Principais

- **Tag `<em>`:**
    - **Papel:** Destacar um trecho do texto com ênfase na leitura.
    - **Comportamento padrão:** Geralmente exibido em itálico pelos navegadores.
- **Tag `<strong>`:**
    - **Papel:** Destacar um trecho do texto como altamente relevante.
    - **Comportamento padrão:** Geralmente exibido em negrito pelos navegadores.

Ambos os elementos ajudam a construir uma hierarquia semântica, facilitando a compreensão do conteúdo para usuários e máquinas.

---

### 3.4 Propriedades/Atributos Específicos

- **Atributos Específicos:**
As tags `<em>` e `<strong>` **não possuem atributos próprios**. Elas aceitam apenas os atributos globais do HTML, tais como:
    - `class`
    - `id`
    - `style`
    - `title`
    - entre outros

**Exemplo com atributo global:**

```html
<p>Confira este <strong id="destaque">conteúdo importante</strong> com um identificador específico.</p>

```

---

### 3.5 Uso Avançado

- **Acessibilidade:**
    
    O uso adequado dessas tags melhora a experiência de usuários que utilizam leitores de tela, já que esses dispositivos conseguem interpretar a ênfase e a importância indicadas pelo `<em>` e `<strong>`.
    
- **SEO (Otimização para Mecanismos de Busca):**
    
    Elementos semânticos auxiliam os motores de busca a identificar e priorizar o conteúdo da página, potencialmente melhorando o ranqueamento do site.
    
- **Customização com CSS:**
    
    Embora os navegadores possuam estilos padrão para `<em>` (itálico) e `<strong>` (negrito), é possível personalizá-los conforme a identidade visual do projeto:
    
    ```css
    em {
      color: blue;
      font-style: italic;
    }
    strong {
      color: red;
      font-weight: bold;
    }
    
    ```
    
- **Aninhamento e Combinação:**
    
    Em alguns casos, pode ser útil aninhar `<em>` dentro de `<strong>` (ou vice-versa) para combinar ênfase e importância. Contudo, é fundamental que essa prática respeite a semântica do conteúdo:
    
    ```html
    <p>Este é um <strong>texto muito <em>importante</em></strong> para a mensagem que queremos transmitir.</p>
    
    ```

## 4. Exemplos Práticos

### Exemplo Básico

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Exemplo Básico de <em> e <strong></title>
  <style>
    /* Estilos básicos para customização */
    em {
      color: blue;
      font-style: italic;
    }
    strong {
      color: red;
      font-weight: bold;
    }
  </style>
</head>
<body>
  <p>Utilizando a tag <em> para enfatizar: Este é um <em>exemplo simples</em> de ênfase.</p>
  <p>Utilizando a tag <strong> para destacar: Este é um <strong>exemplo importante</strong> de destaque.</p>
</body>
</html>

```

### Exemplo Avançado

```html
<!DOCTYPE html>
<html lang="pt-BR">
```

## 5. Informações Adicionais

- **Acessibilidade:**
    
    O uso de `<em>` e `<strong>` permite que leitores de tela interpretem corretamente a hierarquia e a importância do conteúdo, melhorando a experiência dos usuários com deficiências visuais.
    
- **SEO e Semântica:**
    
    A utilização de elementos semânticos auxilia os mecanismos de busca na compreensão do conteúdo da página, o que pode influenciar positivamente o posicionamento em resultados de busca.
    
- **Boas Práticas:**
    - Utilize `<em>` e `<strong>` apenas quando o significado semântico do conteúdo realmente exigir uma ênfase ou destaque.
    - Evite usar essas tags apenas para aplicar estilos visuais; para isso, utilize CSS.
    - Sempre mantenha a clareza e a hierarquia semântica para facilitar a manutenção e a compreensão do código.
- **Customização:**
    
    Com CSS, é possível alterar a aparência padrão desses elementos para que se adequem à identidade visual do projeto, sem comprometer a semântica.
