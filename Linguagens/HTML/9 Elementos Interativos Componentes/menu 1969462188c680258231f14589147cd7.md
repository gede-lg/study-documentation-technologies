# <menu>

### 1. Introdução

O elemento `<menu>` foi reintroduzido no HTML5 com uma nova semântica e propósito. Originalmente, o `<menu>` era usado para criar menus de contexto, mas essa funcionalidade foi depreciada. No HTML5, o `<menu>` foi redefinido para representar uma lista de comandos ou ações que o usuário pode realizar. Ele é útil para criar menus de contexto personalizados, barras de ferramentas, ou listas de ações interativas. Apesar de sua utilidade, o `<menu>` ainda não é amplamente suportado em todos os navegadores, e seu uso pode exigir polyfills ou alternativas.

### 3. Conteúdo Detalhado

### Definição e conceitos fundamentais

O `<menu>` é um elemento HTML que representa uma lista de comandos ou ações. Ele pode ser usado para criar menus de contexto, barras de ferramentas, ou listas de ações interativas. O `<menu>` é frequentemente usado em conjunto com o elemento `<menuitem>`, que define itens individuais no menu. No entanto, o suporte para `<menuitem>` é limitado, e muitas vezes é necessário usar alternativas como `<button>` ou `<a>` dentro do `<menu>`.

### Sintaxe e estrutura

A estrutura básica do `<menu>` é a seguinte:

```html
<menu>
  <li><button onclick="novaAba()">Nova Aba</button></li>
  <li><button onclick="fecharAba()">Fechar Aba</button></li>
</menu>

```

- **`<menu>`**: O contêiner para os itens do menu.
- **`<li>`**: Usado para listar os itens do menu.
- **`<button>`**: Define os comandos ou ações que o usuário pode realizar.

### Componentes principais

- **Atributos**:
    - `type`: Define o tipo de menu. Os valores possíveis são `context` (menu de contexto) e `toolbar` (barra de ferramentas). O valor padrão é `list`.
- **Elementos**:
    - `<menuitem>`: Define um item no menu. No entanto, o suporte para este elemento é limitado.
    - `<li>`: Usado para listar os itens do menu.
    - `<button>` ou `<a>`: Usados para definir comandos ou ações.

### Uso avançado

- **Menus de contexto personalizados**: O `<menu>` pode ser usado para criar menus de contexto personalizados que aparecem quando o usuário clica com o botão direito em um elemento.

```html
<div contextmenu="meuMenu">
  Clique com o botão direito aqui.
  <menu id="meuMenu" type="context">
    <menuitem label="Copiar" onclick="copiarTexto()"></menuitem>
    <menuitem label="Colar" onclick="colarTexto()"></menuitem>
  </menu>
</div>

```

- **Barras de ferramentas**: O `<menu>` pode ser usado para criar barras de ferramentas com ações interativas.

```html
<menu type="toolbar">
  <li><button onclick="novaAba()">Nova Aba</button></li>
  <li><button onclick="fecharAba()">Fechar Aba</button></li>
</menu>

```

### 4. Exemplos práticos

### Exemplo básico

```html
<menu>
  <li><button onclick="alert('Nova Aba Criada')">Nova Aba</button></li>
  <li><button onclick="alert('Aba Fechada')">Fechar Aba</button></li>
</menu>

```

### Exemplo avançado

```html
<div contextmenu="meuMenu">
  Clique com o botão direito aqui.
  <menu id="meuMenu" type="context">
    <menuitem label="Copiar" onclick="alert('Texto Copiado')"></menuitem>
    <menuitem label="Colar" onclick="alert('Texto Colado')"></menuitem>
  </menu>
</div>

<script>
  // Polyfill para suporte a <menuitem>
  if (!('menuitem' in document.createElement('menuitem'))) {
    document.querySelectorAll('menuitem').forEach(item => {
      const button = document.createElement('button');
      button.textContent = item.getAttribute('label');
      button.onclick = item.onclick;
      item.replaceWith(button);
    });
  }
</script>

```

### 5. Informações adicionais

- **Acessibilidade**: O `<menu>` deve ser usado com cuidado para garantir que seja acessível. Use ARIA roles e atributos para melhorar a acessibilidade.
- **Compatibilidade**: O suporte para `<menu>` e `<menuitem>` é limitado em alguns navegadores. Verifique a compatibilidade e considere usar polyfills ou alternativas.
- **Estilização**: O `<menu>` pode ser estilizado com CSS para se adequar ao design da sua aplicação.
