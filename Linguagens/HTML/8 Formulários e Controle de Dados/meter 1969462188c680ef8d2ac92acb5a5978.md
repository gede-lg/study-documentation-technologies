# <meter>

A tag `<meter>` é um elemento HTML5 semântico usado para exibir um valor escalar dentro de um intervalo conhecido, como por exemplo a quantidade de bateria restante, níveis de aprovação ou indicadores de desempenho. Diferente do `<progress>`, que mostra o andamento de uma tarefa em progresso, o `<meter>` representa uma medida ou avaliação estática de um dado.

---

## 1. Introdução

O `<meter>` é utilizado para indicar visualmente informações quantitativas, quando há um intervalo definido de valores. Ele é especialmente útil em casos onde se deseja mostrar um status, como o nível de umidade, a avaliação de um produto, ou a capacidade de um recurso. Sua semântica permite que tecnologias assistivas interpretem e comuniquem corretamente o contexto do valor exibido.

## 3. Conteúdo Detalhado

### 3.1 Definição e Conceitos Fundamentais

- **O que é `<meter>`?**
    
    O `<meter>` representa um dado numérico dentro de um intervalo fixo. Ele é ideal para mostrar indicadores que possuem limites conhecidos, fornecendo ao usuário uma visualização clara de onde o valor atual se encontra em relação a esses limites.
    
- **Diferença entre `<meter>` e `<progress>`:**
    - **`<progress>`:** Indica o progresso de uma tarefa em andamento (por exemplo, um download).
    - **`<meter>`:** Representa um valor fixo ou medido dentro de um intervalo determinado (por exemplo, nível de bateria ou pontuação de avaliação).

---

### 3.2 Sintaxe e Estrutura

A estrutura básica do `<meter>` envolve o uso dos atributos que definem o intervalo e o valor a ser exibido.

**Exemplo Básico:**

```html
<meter value="0.6" min="0" max="1">60%</meter>

```

- **Atributos Principais:**
    - **`value`:** Valor atual a ser exibido.
    - **`min`:** Valor mínimo do intervalo (padrão é 0 se omitido).
    - **`max`:** Valor máximo do intervalo.
    - **`low`:** Limite inferior que indica um valor considerado "baixo".
    - **`high`:** Limite superior que indica um valor considerado "alto".
    - **`optimum`:** Valor considerado ideal ou ótimo dentro do intervalo.

---

### 3.3 Componentes e Atributos Específicos

- **`min` e `max`:**
    
    Definem os limites inferior e superior do intervalo de valores, respectivamente. Estes atributos ajudam a determinar a posição do valor atual dentro do contexto total.
    
- **`value`:**
    
    Representa a medição atual a ser mostrada. Ele deve estar compreendido entre `min` e `max`.
    
- **`low` e `high`:**
    
    Permitem definir pontos de corte para indicar valores baixos e altos. Por exemplo, se o valor está abaixo de `low` pode ser considerado insatisfatório, enquanto valores acima de `high` podem ser considerados excelentes ou críticos.
    
- **`optimum`:**
    
    Indica o valor ideal ou desejável dentro do intervalo. Essa informação pode ser utilizada para alterar visualmente o `<meter>` (por exemplo, com cores diferentes) para indicar quão próximo o valor atual está do ideal.
    

---

### 3.4 Uso Avançado e Boas Práticas

- **Customização com CSS:**
    
    Embora a aparência padrão do `<meter>` seja definida pelo navegador, você pode aplicar estilos personalizados para alterar cores, tamanhos e até efeitos de transição. A customização pode variar entre navegadores.
    
- **Integração com JavaScript:**
    
    O `<meter>` pode ser atualizado dinamicamente via JavaScript para refletir mudanças em tempo real, como em dashboards ou aplicações interativas.
    
- **Acessibilidade:**
    
    Utilizar o `<meter>` melhora a semântica do documento, permitindo que leitores de tela interpretem corretamente o significado do valor exibido. Certifique-se de que os atributos `min`, `max` e `value` estejam corretamente configurados para que a informação seja comunicada de forma eficaz.

## 4. Exemplos Práticos

### Exemplo 1: Indicador de Nível de Bateria

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Exemplo de Meter - Nível de Bateria</title>
  <style>
    meter {
      width: 100%;
      height: 20px;
    }
  </style>
</head>
<body>
  <h2>Nível de Bateria</h2>
  <meter value="0.75" min="0" max="1" low="0.2" high="0.8" optimum="1">
    75%
  </meter>
</body>
</html>

```

### Exemplo 2: Indicador de Satisfação do Cliente

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Exemplo de Meter - Satisfação do Cliente</title>
  <style>
    meter {
      width: 100%;
      height: 25px;
      margin-top: 10px;
    }
  </style>
</head>
<body>
  <h2>Satisfação do Cliente</h2>
  <meter value="8" min="0" max="10" low="4" high="7" optimum="10">
    8/10
  </meter>
</body>
</html>

```

## 6. Conclusão

A tag `<meter>` é uma ferramenta valiosa para representar visualmente dados quantitativos dentro de um intervalo definido. Por meio de seus atributos, é possível configurar e interpretar valores em contextos variados, desde a indicação de níveis de bateria até avaliações de desempenho ou satisfação. O uso adequado do `<meter>` melhora a semântica e a acessibilidade dos conteúdos, proporcionando uma experiência de usuário mais informativa e intuitiva. Dominar essa tag é essencial para desenvolvedores que desejam criar interfaces ricas e comunicativas em suas aplicações web.
