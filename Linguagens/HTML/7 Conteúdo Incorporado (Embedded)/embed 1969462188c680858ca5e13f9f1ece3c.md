# <embed>

### **1. Introdução**

A tag `<embed>` em HTML5 é utilizada para incorporar conteúdo externo em uma página web, como vídeos, áudios, PDFs ou até aplicativos baseados em plugins (ex: Flash, embora obsoleto). Sua importância reside na capacidade de integrar recursos multimídia ou interativos diretamente no HTML, sem dependência exclusiva de APIs modernas como `<video>` ou `<iframe>`. Embora menos utilizada atualmente devido à evolução das tecnologias web, ainda é relevante em contextos legados ou para conteúdos específicos que requerem plugins.

### **3. Conteúdo Detalhado**

### **Definição e conceitos fundamentais**

- **Tag `<embed>`**: Elemento vazio (sem tag de fechamento) que insere conteúdo externo via plugins do navegador.
- **Conceitos básicos**:
    - Ideal para conteúdos que não são nativamente suportados pelo HTML (ex: PDFs, SVG).
    - Requer plugins habilitados no navegador para funcionar.
- **Conceitos avançados**:
    - Pode ser controlado via JavaScript para dinamismo.
    - Menos seguro que alternativas modernas devido à dependência de plugins.

### **Sintaxe e estrutura**

```html
<embed
  src="arquivo.extensão"
  type="tipo-mime"
  width="largura"
  height="altura"
>

```

- **Exemplo básico**:
    
    ```html
    <embed src="video.mp4" type="video/mp4" width="640" height="480">
    
    ```
    

### **Componentes principais**

- **Atributos obrigatórios**: Nenhum, mas `src` e `type` são altamente recomendados.
- **Atributos específicos**:
    
    
    | Atributo | Valores | Descrição |
    | --- | --- | --- |
    | `src` | URL | Caminho do recurso externo. |
    | `type` | Tipo MIME (ex: `application/pdf`) | Define o tipo do conteúdo. |
    | `width` | Pixels ou % | Largura do elemento. |
    | `height` | Pixels ou % | Altura do elemento. |

### **Uso avançado**

- **Integração com JavaScript**:
    
    ```html
    <embed id="meuEmbed" src="doc.pdf" type="application/pdf">
    <script>
      document.getElementById('meuEmbed').src = 'novo-doc.pdf';
    </script>
    
    ```
    
- **Casos reais**:
    - Incorporação de visualizadores de PDF legados.
    - Suporte a aplicações antigas baseadas em Flash (com ressalvas de segurança).

### **4. Exemplos práticos**

### **Exemplo 1: Incorporar um PDF**

```html
<embed
  src="manual.pdf"
  type="application/pdf"
  width="800"
  height="600"
>

```

### **Exemplo 2: Vídeo com fallback (usando `<embed>` e `<object>`)**

```html
<embed
  src="video.swf"
  type="application/x-shockwave-flash"
  width="640"
  height="480"
>
<object data="video.mp4" type="video/mp4">
  <p>Seu navegador não suporta o conteúdo.</p>
</object>

```

---

### **5. Informações adicionais**

- **Browser Support**: Suportado por todos os navegadores modernos, mas a renderização depende de plugins.
- **Segurança**: Plugins como Flash são desencorajados devido a vulnerabilidades.
- **Acessibilidade**: Não oferece suporte nativo a fallback. Use `<object>` para alternativas.
