# <tr>

## 1. Introdução

A tag `<tr>` (Table Row) é um elemento fundamental para a criação de tabelas em HTML, responsável por definir uma linha dentro de uma tabela. Tabelas são estruturas essenciais para organizar dados de forma tabular, como planilhas, listas comparativas, calendários ou qualquer conteúdo que exija disposição em linhas e colunas. O `<tr>` atua como contêiner para células de dados (`<td>`) ou cabeçalhos (`<th>`), garantindo a estruturação lógica e semântica dos dados. Seu uso correto é crucial para acessibilidade, SEO e renderização consistente em diferentes dispositivos.

## 3. Conteúdo Detalhado

### Definição e Conceitos Fundamentais

- **O que é `<tr>`?**
Define uma linha em uma tabela HTML. Deve ser aninhado dentro de `<table>`, `<thead>`, `<tbody>`, ou `<tfoot>`.
- **Conceitos básicos vs. avançados**
    - **Básico:** Criação de linhas com células (`<td>`) e cabeçalhos (`<th>`).
    - **Avançado:** Integração com atributos de acessibilidade, manipulação dinâmica via JavaScript, e estilização complexa com CSS.

### Sintaxe e Estrutura

```html
<table>
  <tr>
    <th>Cabeçalho 1</th>
    <th>Cabeçalho 2</th>
  </tr>
  <tr>
    <td>Dado 1</td>
    <td>Dado 2</td>
  </tr>
</table>

```

### Componentes Principais

- **Parentes diretos:** `<table>`, `<thead>`, `<tbody>`, `<tfoot>`.
- **Filhos diretos:** `<td>` (célula de dados) e `<th>` (célula de cabeçalho).

### Atributos Específicos

- **Atributos globais:** `class`, `id`, `style`, `title`, etc.
- **Atributos obsoletos no HTML5:** `align`, `valign`, `bgcolor` (devem ser substituídos por CSS).
- **Atributos de acessibilidade:** `aria-*` (ex: `aria-label`).

### Uso Avançado

1. **Tabelas responsivas:** Combinar `<tr>` com CSS para adaptar tabelas a dispositivos móveis.
2. **Dinamismo:** Adicionar/remover linhas via JavaScript:
    
    ```jsx
    const table = document.querySelector('table');
    const newRow = table.insertRow();
    newRow.innerHTML = '<td>Novo dado</td><td>Outro dado</td>';
    
    ```
    
3. **Acessibilidade:** Associar `<th>` com `scope="row"` para indicar cabeçalhos de linha.
4. **Mesclagem de células:** Usar `rowspan` ou `colspan` em `<td>`/`th>` para células que ocupam múltiplas linhas/colunas.

## 4. Exemplos Práticos

### Exemplo Básico

```html
<table border="1">
  <thead>
    <tr>
      <th>Nome</th>
      <th>Idade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>João</td>
      <td>25</td>
    </tr>
    <tr>
      <td>Maria</td>
      <td>30</td>
    </tr>
  </tbody>
</table>

```

### Exemplo Avançado (Mesclagem de Linhas)

```html
<table border="1">
  <tr>
    <th rowspan="2">Mês</th>
    <th colspan="2">Vendas</th>
  </tr>
  <tr>
    <th>Produto A</th>
    <th>Produto B</th>
  </tr>
  <tr>
    <td>Janeiro</td>
    <td>R$ 200</td>
    <td>R$ 150</td>
  </tr>
</table>

```

---

## 5. Informações Adicionais

- **Acessibilidade:** Sempre use `<th>` para cabeçalhos e defina `scope="col"` ou `scope="row"`.
- **Performance:** Evite tabelas com milhares de `<tr>` sem paginação ou carregamento lazy.
- **Semântica:** Priorize `<thead>`, `<tbody>`, e `<tfoot>` para melhor organização.
- **CSS:** Use `:nth-child(even/odd)` para estilizar linhas alternadas.
