# <th>, <td>

**1. Introdução**

As tags `<th>` e `<td>` são elementos fundamentais para a criação de tabelas em HTML5, permitindo a organização de dados de forma estruturada e semântica. Tabelas são essenciais para exibir informações tabulares, como planilhas, calendários ou dados comparativos. A correta utilização dessas tags impacta diretamente na acessibilidade, SEO e usabilidade, pois define a relação entre cabeçalhos e células de dados, facilitando a interpretação por navegadores, leitores de tela e mecanismos de busca.

**3. Conteúdo Detalhado**

### **3.1 Definição e conceitos fundamentais**

- **`<th>` (Table Header)**: Define uma célula de cabeçalho em uma tabela. Geralmente é exibido em negrito e centralizado.
- **`<td>` (Table Data)**: Define uma célula de dados padrão. Exibido com alinhamento à esquerda e estilo normal.
- **Diferença**: `<th>` é usado para rótulos (ex.: "Nome", "Idade"), enquanto `<td>` contém os dados associados (ex.: "João", "25").

### **3.2 Sintaxe e estrutura**

Ambas as tags são usadas dentro de linhas (`<tr>`) em uma tabela:

```html
<table>
  <tr>
    <th>Cabeçalho 1</th>
    <th>Cabeçalho 2</th>
  </tr>
  <tr>
    <td>Dado 1</td>
    <td>Dado 2</td>
  </tr>
</table>

```

### **3.3 Componentes principais**

- **Contexto**: Ambas devem estar dentro de `<tr>`, que por sua vez está dentro de `<table>`.
- **Elementos relacionados**:
    - `<thead>`, `<tbody>`, `<tfoot>`: Agrupam linhas em cabeçalho, corpo e rodapé da tabela.
    - `<caption>`: Adiciona uma legenda à tabela.

### **3.4 Propriedades e atributos específicos**

### **Atributos do `<th>`**

| Atributo | Valores Possíveis | Descrição |
| --- | --- | --- |
| `scope` | `row`, `col`, `rowgroup`, `colgroup` | Define a abrangência do cabeçalho (ex.: para linha ou coluna). |
| `colspan` | Número inteiro | Quantidade de colunas que a célula ocupa. |
| `rowspan` | Número inteiro | Quantidade de linhas que a célula ocupa. |
| `headers` | IDs de outros `<th>` | Associa a célula a cabeçalhos específicos (uso avançado). |

### **Atributos do `<td>`**

| Atributo | Valores Possíveis | Descrição |
| --- | --- | --- |
| `colspan` | Número inteiro | Mesma função que em `<th>`. |
| `rowspan` | Número inteiro | Mesma função que em `<th>`. |
| `headers` | IDs de `<th>` | Relaciona a célula a cabeçalhos específicos. |

**Nota**: Ambos compartilham atributos globais (ex.: `class`, `id`, `style`).

### **3.5 Uso avançado**

- **Acessibilidade**: Use `scope` em `<th>` para indicar se o cabeçalho se refere a uma linha (`row`) ou coluna (`col`).
- **Células mescladas**: Combine `colspan` e `rowspan` para criar layouts complexos.
- **Associação explícita**: Utilize `headers` em `<td>` para vincular dados a múltiplos cabeçalhos (útil para tabelas grandes).
- **Integração com CSS/JavaScript**: Estilize células ou adicione interatividade via classes ou eventos.

**4. Exemplos práticos**

### **4.1 Uso básico**

```html
<table border="1">
  <thead>
    <tr>
      <th>Nome</th>
      <th>Idade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Maria</td>
      <td>30</td>
    </tr>
  </tbody>
</table>

```

### **4.2 Uso avançado (colspan e headers)**

```html
<table border="1">
  <tr>
    <th id="produto">Produto</th>
    <th id="preco">Preço</th>
  </tr>
  <tr>
    <td headers="produto">Notebook</td>
    <td headers="preco">R$ 2500</td>
  </tr>
  <tr>
    <th colspan="2">Total: R$ 2500</th>
  </tr>
</table>

```

---

**5. Informações adicionais**

- **Acessibilidade**: Sempre use `<th>` para cabeçalhos e evite tabelas aninhadas.
- **Semântica**: Priorize `<thead>`, `<tbody>` e `<tfoot>` para melhor organização.
- **Responsividade**: Em dispositivos móveis, considere técnicas como overflow ou CSS Grid para tabelas grandes.
