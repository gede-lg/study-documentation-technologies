# Grade

Olá, Gedê!

Com certeza. Preparei uma grade de estudos completa para você dominar o Sass, do básico ao avançado. A grade está organizada em módulos lógicos para que você possa evoluir passo a passo.

---

## Módulo 1: Primeiros Passos e Conceitos Fundamentais

Este módulo foca na instalação e nos conceitos essenciais para começar a usar o Sass.

1. **O que é Sass?**: Entenda o que é, por que usá-lo e a diferença entre Sass e SCSS.
2. **Instalação e Compilação**: Como instalar o Sass e configurar um projeto. Aprenda a usar a linha de comando para compilar arquivos `.scss` para `.css`.
3. **Variáveis (`$`)**: Aprenda a criar e utilizar variáveis para armazenar valores como cores, fontes e tamanhos.
4. **Aninhamento (`&`)**: Entenda como aninhar seletores para criar hierarquias de CSS mais limpas e legíveis, usando o `&` para referenciar o seletor pai.
5. **Importação (`@import`)**: Como organizar seu código Sass em múltiplos arquivos (por exemplo, `_variables.scss`, `_mixins.scss`) e importá-los em um único arquivo principal.

---

## Módulo 2: O Poder da Reutilização e da Lógica

Aqui, você vai aprender a reutilizar código e adicionar lógica à sua estilização.

1. **Mixins (`@mixin` e `@include`)**: Crie blocos de código reutilizáveis com `mixins`. Aprenda a passar argumentos e incluir o `mixin` em outros seletores.
2. **Partials (`_` prefixo)**: Entenda por que e como usar arquivos parciais (com o prefixo `_`) para modularizar seu código sem que eles sejam compilados para arquivos `.css` separados.
3. **Funções (`@function`)**: Crie funções personalizadas para realizar cálculos e retornar valores. Diferencie `mixins` e `funções`.
4. **Operadores Matemáticos**: Utilize operadores como `+`, , , `/` e `%` para fazer cálculos diretamente nas suas folhas de estilo.

---

## Módulo 3: Ferramentas Avançadas e Estruturas de Controle

Neste módulo, você irá explorar as funcionalidades mais avançadas do Sass para criar estilos dinâmicos e complexos.

1. **Herança (`@extend`)**: Aprenda a compartilhar um conjunto de propriedades CSS entre seletores, evitando a duplicação de código.
2. **Condicionais (`@if`, `@else if`, `@else`)**: Use estruturas de controle para aplicar estilos baseados em condições.
3. **Loops (`@for`, `@each`, `@while`)**: Automatize a criação de classes ou estilos usando loops. Por exemplo, gere classes de espaçamento (ex: `.m-5`, `.p-10`) automaticamente.
4. **Mapas (`map-get`, `map-has-key`)**: Armazene dados em pares de chave-valor e aprenda a acessá-los, o que é excelente para gerenciar temas ou configurações.

---

## Módulo 4: Melhores Práticas e Arquitetura de Projetos

Este módulo foca em como organizar e manter um projeto Sass de forma escalável.

1. **Arquitetura de Projeto**: Entenda as diferentes abordagens de arquitetura, como **7-1 Pattern**, e como organizar seus arquivos e pastas para projetos grandes.
2. **BEM (Block, Element, Modifier)**: Aprenda a usar a metodologia BEM em conjunto com o Sass para escrever seletores mais claros e modulares.
3. **Práticas de Codificação**: Crie um guia de estilo para sua equipe, definindo padrões para nomes de variáveis, `mixins`, aninhamento e mais.

### Projeto Prático

Ao final do curso, crie um projeto pessoal utilizando todos os conceitos aprendidos. Pode ser um site *single-page* ou o *front-end* de um formulário de cadastro, como o que você está construindo. Focar em um projeto prático é a melhor forma de fixar o conhecimento.

Essa grade completa vai te guiar do zero ao avançado no mundo do Sass. Lembre-se de praticar bastante em cada módulo para realmente internalizar os conceitos.
