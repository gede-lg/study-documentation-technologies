
Os operadores relacionais em Go são utilizados para comparar dois valores. Eles retornam um valor booleano (`true` ou `false`) com base na relação entre os operandos. Esses operadores são fundamentais para o controle de fluxo e tomada de decisões em programas.

## Operadores Relacionais

1. **Igual a (`==`)**
2. **Diferente de (`!=`)**
3. **Maior que (`>`)**
4. **Menor que (`<`)**
5. **Maior ou igual a (`>=`)**
6. **Menor ou igual a (`<=`)**

Vamos explorar cada um desses operadores em detalhe:

### 1. Igual a (`==`)

#### O que é e para que serve?
O operador `==` verifica se dois valores são iguais. Ele retorna `true` se os valores forem iguais e `false` caso contrário.

#### Quando utilizar?
Utiliza-se `==` quando se deseja verificar a igualdade entre dois valores. É comum em estruturas condicionais como `if` e `switch`.

#### Sintaxe de uso
```go
a == b
```

#### Exemplo
```go
package main

import "fmt"

func main() {
    a := 5
    b := 5
    c := 10
    
    fmt.Println(a == b) // true
    fmt.Println(a == c) // false
}
```

#### Restrições de uso
Os operandos devem ser do mesmo tipo para comparação.

### 2. Diferente de (`!=`)

#### O que é e para que serve?

#### Restrições de uso
Os operandos devem ser do mesmo tipo para comparação.

### 3. Maior que (`>`)

#### O que é e para que serve?
O operador `>` verifica se o valor do lado esquerdo é maior que o valor do lado direito. Ele retorna `true` se for maior e `false` caso contrário.

#### Quando utilizar?
Utiliza-se `>` quando se deseja verificar se um valor é maior que outro.

#### Sintaxe de uso
```go
a > b
```

#### Exemplo
```go
package main

    a := 5
    b := 3

    fmt.Println(b > a) // false

```

#### Restrições de uso
Os operandos devem ser de tipos comparáveis, geralmente numéricos ou strings.

### 5. Maior ou igual a (`>=`)

## Considerações Adicionais

### Comparação de Strings
Strings podem ser comparadas usando os operadores relacionais. A comparação é lexicográfica, baseada nos valores Unicode dos caracteres.

#### Exemplo
```go
package main

import "fmt"

func main() {
    s1 := "apple"
    s2 := "banana"
    
    fmt.Println(s1 == s2)  // false
    fmt.Println(s1 != s2)  // true
    fmt.Println(s1 < s2)   // true (porque "apple" vem antes de "banana" lexicograficamente)
}
```

### Comparação de Ponteiros
Ponteiros podem ser comparados para verificar se apontam para o mesmo endereço de memória.

#### Exemplo
```go
package main

import "fmt"

func main() {
    a := 5
    b := 5
    p1 := &a
    p2 := &b
    p3 := &a
    
    fmt.Println(p1 == p2) // false
    fmt.Println(p1 == p3) // true
}
```

### Comparação de Estruturas
Estruturas (structs) podem ser comparadas se todos os seus campos puderem ser comparados. A comparação é feita campo por campo.

#### Exemplo
```go
package main

import "fmt"

type Point struct {
    X int
    Y int
}

func main() {
    p1 := Point{X: 1, Y: 2}
    p2 := Point{X: 1, Y: 2}
    p3 := Point{X: 2, Y: 3}
    
    fmt.Println(p1 == p2) // true
    fmt.Println(p1 != p3) // true
}
```
