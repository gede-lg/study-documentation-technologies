# Introdução sobre Funções no Go

## O que é e para que serve?

Uma função em Go é um bloco de código que pode ser chamado para executar uma tarefa específica. Funções são fundamentais para a modularidade, reutilização de código e organização de programas. Elas permitem dividir o programa em partes menores e mais gerenciáveis, facilitando a leitura, manutenção e depuração do código.

## Tipos de Funções

### 1. Função Simples Sem Parâmetros e Sem Retorno

#### O que é?
Uma função simples que não recebe parâmetros e não retorna nenhum valor. Utilizada para executar uma tarefa específica sem a necessidade de dados de entrada ou saída.

#### Sintaxe
```go
func nomeDaFuncao() {
    // corpo da função
}
```

#### Exemplo
```go
func dizerOla() {
    fmt.Println("Olá, Mundo!")
}

func main() {
    dizerOla()
}
```

#### Restrições
Não pode retornar valores ou aceitar parâmetros.

### 2. Função Com Parâmetros

#### O que é?
Uma função que recebe um ou mais parâmetros de entrada. Utilizada quando é necessário processar ou manipular dados fornecidos na chamada da função.

#### Sintaxe
```go
func nomeDaFuncao(param1 tipo1, param2 tipo2) {
    // corpo da função
}
```

#### Exemplo
```go
func somar(a int, b int) {
    resultado := a + b
    fmt.Println(resultado)
}

func main() {
    somar(3, 4)
}
```

#### Restrições
Os tipos dos parâmetros devem ser especificados. Se os parâmetros forem do mesmo tipo, a notação pode ser simplificada:
```go
func somar(a, b int) {
    // ...
}
```

### 3. Função Com Múltiplos Retornos

#### O que é?
Uma função que retorna mais de um valor. Utilizada frequentemente para retornar um resultado e um possível erro.

#### Sintaxe
```go
func nomeDaFuncao() (tipo1, tipo2) {
    // corpo da função
    return valor1, valor2
}
```

#### Exemplo
```go
func dividir(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("divisão por zero")
    }
    return a / b, nil
}

func main() {
    resultado, err := dividir(10, 2)
    if err != nil {
        fmt.Println("Erro:", err)
    } else {
        fmt.Println("Resultado:", resultado)
    }
}
```

#### Restrições
Os tipos de retorno devem ser especificados e a função deve retornar exatamente o número de valores declarados.

### 4. Função Anônima (Literal de Função)

#### O que é?
Uma função sem nome que pode ser atribuída a uma variável ou executada imediatamente. Utilizada para criar funções rápidas e descartáveis.

#### Sintaxe
```go
func(param1 tipo1, param2 tipo2) {
    // corpo da função
}
```

#### Exemplo
```go
func main() {
    anonimFunc := func(a, b int) int {
        return a + b
    }
    fmt.Println(anonimFunc(3, 4))

    // Execução imediata
    func(msg string) {
        fmt.Println(msg)
    }("Olá, Função Anônima!")
}
```

#### Restrições
Não pode ser chamada diretamente por nome, apenas através da variável a que está atribuída ou imediatamente.

### 5. Função Variádica

#### O que é?
Uma função que pode receber um número variável de argumentos. Utilizada quando o número de parâmetros não é fixo.

#### Sintaxe
```go
func nomeDaFuncao(param1 tipo1, params ...tipo2) {
    // corpo da função
}
```

#### Exemplo
```go
func somarTudo(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

func main() {
    fmt.Println(somarTudo(1, 2, 3, 4, 5))
}
```

#### Restrições
A função pode ter apenas um parâmetro variádico e ele deve ser o último na lista de parâmetros.

### 6. Funções como Valores e Parâmetros

#### O que é?
Funções podem ser tratadas como valores e passadas como parâmetros para outras funções. Utilizada para maior flexibilidade e composição de funções.

#### Sintaxe
```go
func nomeDaFuncao(funcaoParametro func(tipo1, tipo2) tipo3) {
    // corpo da função
}
```

#### Exemplo
```go
func aplicarFuncao(f func(int, int) int, a, b int) int {
    return f(a, b)
}

func main() {
    somar := func(a, b int) int {
        return a + b
    }
    fmt.Println(aplicarFuncao(somar, 5, 3))
}
```

#### Restrições
A função passada como parâmetro deve ter a assinatura compatível com a esperada.

### 7. Funções com Retornos Nomeados

#### O que é?
Funções que têm nomes para seus valores de retorno. Utilizada para tornar o código mais claro e permitir o uso de `return` sem argumentos.

#### Sintaxe
```go
func nomeDaFuncao() (retorno1 tipo1, retorno2 tipo2) {
    // corpo da função
    return
}
```

#### Exemplo
```go
func dividir(a, b float64) (resultado float64, err error) {
    if b == 0 {
        err = errors.New("divisão por zero")
        return
    }
    resultado = a / b
    return
}

func main() {
    resultado, err := dividir(10, 2)
    if err != nil {
        fmt.Println("Erro:", err)
    } else {
        fmt.Println("Resultado:", resultado)
    }
}
```

#### Restrições
Os nomes dos retornos devem ser utilizados no corpo da função.

### 8. Declaração de Funções Dentro de Funções

#### O que é?
Declaração de uma função dentro do corpo de outra função. Utilizada para encapsular lógica específica e não reutilizável fora do escopo da função externa.

#### Sintaxe
```go
func externa() {
    interna := func(param1 tipo1, param2 tipo2) tipo3{
        // corpo da função interna
    }
    // uso da função interna
}
```

#### Exemplo
```go
func main() {
    calcular := func(a, b int) int {
        return a + b
    }
    resultado := calcular(5, 7)
    fmt.Println("Resultado:", resultado)
}
```

#### Restrições
A função interna só pode ser chamada dentro do escopo da função onde foi declarada.

## Conclusão

As funções em Go são poderosas e versáteis, oferecendo uma variedade de formas para lidar com diferentes cenários de programação. Entender como e quando usar cada tipo de função é essencial para escrever código Go eficiente, limpo e fácil de manter.
