
## O que é e para que serve?

Uma função com parâmetros em Go é uma função que recebe um ou mais valores como entrada quando é chamada. Esses valores, conhecidos como parâmetros ou argumentos, são utilizados dentro da função para realizar operações ou cálculos. Funções com parâmetros são essenciais para criar código reutilizável e modular, permitindo que a mesma função opere em diferentes dados de entrada.

### Para que serve?

- **Modularidade:** Permite dividir o código em blocos menores e reutilizáveis.
- **Flexibilidade:** A mesma função pode ser utilizada com diferentes entradas, tornando o código mais versátil.
- **Clareza:** Melhora a legibilidade do código ao separar a lógica da função dos dados específicos com os quais ela trabalha.

## Quando utilizar?

Utilize funções com parâmetros quando:

- **Há necessidade de processar diferentes dados de entrada:** Se a lógica da função permanece a mesma, mas os dados com os quais ela trabalha variam.
- **Deseja-se evitar duplicação de código:** Em vez de escrever a mesma lógica várias vezes para diferentes valores, use parâmetros para tornar a função genérica.
- **Precisa-se de flexibilidade:** Funções com parâmetros permitem passar diferentes valores, facilitando testes e manutenção do código.

## Sintaxe de uso

A sintaxe para definir uma função com parâmetros em Go é simples. Aqui está a estrutura básica:

```go
func nomeDaFuncao(param1 tipo1, param2 tipo2) {
    // corpo da função
}
```

### Detalhamento da Sintaxe

- **func:** Palavra-chave usada para definir uma função.
- **nomeDaFuncao:** Nome da função que será chamada para executá-la.
- **param1, param2:** Nomes dos parâmetros. Pode haver um ou mais parâmetros, separados por vírgulas.
- **tipo1, tipo2:** Tipos dos parâmetros. Cada parâmetro deve ter um tipo especificado.

### Exemplo

Aqui está um exemplo de uma função simples que soma dois números inteiros:

```go
func somar(a int, b int) int {
    return a + b
}

func main() {
    resultado := somar(3, 4)
    fmt.Println("Resultado:", resultado)
}
```

Neste exemplo:

- A função `somar` recebe dois parâmetros `a` e `b`, ambos do tipo `int`.
- A função retorna a soma de `a` e `b`.
- No `main`, a função `somar` é chamada com os argumentos `3` e `4`, e o resultado é impresso.

### Notação Simplificada

Se vários parâmetros consecutivos tiverem o mesmo tipo, você pode listar os nomes dos parâmetros antes do tipo, reduzindo a repetição:

```go
func subtrair(a, b int) int {
    return a - b
}
```

## Restrições de uso

- **Tipos dos Parâmetros:** Todos os parâmetros devem ter seus tipos especificados. Se não especificados corretamente, o compilador emitirá um erro.
- **Número de Parâmetros:** Go permite um número arbitrário de parâmetros, mas para funções com muitos parâmetros, considere refatorar para melhorar a legibilidade e manutenção.
- **Tipos Compatíveis:** Os valores passados para os parâmetros devem ser compatíveis com os tipos esperados. Passar valores de tipos incompatíveis resultará em erro de compilação.
- **Valor de Retorno:** Uma função pode ter parâmetros e ainda assim não ter retorno, ou pode retornar um ou mais valores, conforme necessário.

### Exemplos de Restrições

**Exemplo de erro de tipo incompatível:**

```go
func multiplicar(a int, b int) int {
    return a * b
}

func main() {
    resultado := multiplicar(3, "4") // Erro: argumento "4" é do tipo string, mas a função espera um int
    fmt.Println("Resultado:", resultado)
}
```

**Exemplo de função com muitos parâmetros (dificuldade de manutenção):**

```go
func criarUsuario(nome string, idade int, email string, endereco string, telefone string) {
    // código para criar usuário
}
```

Neste caso, considere usar uma estrutura (`struct`) para agrupar os parâmetros relacionados:

```go
type Usuario struct {
    Nome     string
    Idade    int
    Email    string
    Endereco string
    Telefone string
}

func criarUsuario(u Usuario) {
    // código para criar usuário
}

func main() {
    u := Usuario{"João", 30, "joao@example.com", "Rua A, 123", "555-1234"}
    criarUsuario(u)
}
```

## Conclusão

Funções com parâmetros são uma ferramenta poderosa em Go, permitindo a criação de código modular, flexível e reutilizável. Compreender como e quando utilizá-las é essencial para escrever programas eficientes e bem estruturados. A sintaxe clara e as restrições simples ajudam a garantir que seu código seja robusto e fácil de entender.
