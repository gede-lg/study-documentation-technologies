
## O que é e para que serve?

Uma função normal com parâmetros e um retorno em Go é uma função que recebe um ou mais valores de entrada e retorna um único valor de saída. Este tipo de função é extremamente útil para realizar cálculos, processar dados e retornar resultados específicos baseados nos dados de entrada.

### Para que serve?

- **Processamento de Dados:** Realiza operações e retorna o resultado.
- **Cálculos e Computações:** Executa cálculos matemáticos e retorna o valor calculado.
- **Transformações:** Converte dados de um formato para outro e retorna o resultado.
- **Verificações:** Avalia condições e retorna um valor booleano (verdadeiro ou falso).

## Quando utilizar?

Utilize funções normais com parâmetros e um retorno quando:

- **Necessitar de um Resultado Específico:** A função deve realizar uma operação e devolver um resultado que será usado posteriormente.
- **Encapsular Lógica de Processamento:** Para encapsular a lógica de processamento em um bloco de código reutilizável.
- **Realizar Cálculos:** Quando precisar fazer cálculos e utilizar o resultado em outras partes do código.
- **Melhorar a Legibilidade:** Para dividir o código em partes menores e mais legíveis, cada uma com uma responsabilidade clara.

## Sintaxe de uso

A sintaxe para definir uma função com parâmetros e um retorno em Go é simples e direta. Aqui está a estrutura básica:

```go
func nomeDaFuncao(param1 tipo1, param2 tipo2) tipoDeRetorno {
    // corpo da função
    return valorDeRetorno
}
```

### Detalhamento da Sintaxe

- **func:** Palavra-chave usada para definir uma função.
- **nomeDaFuncao:** Nome da função que será chamada para executá-la.
- **param1, param2:** Nomes dos parâmetros. Pode haver um ou mais parâmetros, separados por vírgulas.
- **tipo1, tipo2:** Tipos dos parâmetros. Cada parâmetro deve ter um tipo especificado.
- **tipoDeRetorno:** Tipo do valor que a função retornará.
- **valorDeRetorno:** Valor calculado ou processado que será retornado pela função.

### Exemplo

Aqui está um exemplo de uma função simples que soma dois números inteiros e retorna o resultado:

```go
func somar(a int, b int) int {
    return a + b
}

func main() {
    resultado := somar(3, 4)
    fmt.Println("Resultado:", resultado)
}
```

Neste exemplo:

- A função `somar` recebe dois parâmetros `a` e `b`, ambos do tipo `int`.
- A função retorna a soma de `a` e `b`, que também é do tipo `int`.
- No `main`, a função `somar` é chamada com os argumentos `3` e `4`, e o resultado é impresso.

### Notação Simplificada

Se vários parâmetros consecutivos tiverem o mesmo tipo, você pode listar os nomes dos parâmetros antes do tipo, reduzindo a repetição:

```go
func subtrair(a, b int) int {
    return a - b
}
```

### Exemplo Prático

Aqui está um exemplo prático de uma função que converte graus Celsius para Fahrenheit:

```go
func celsiusParaFahrenheit(celsius float64) float64 {
    return celsius*9/5 + 32
}

func main() {
    tempC := 25.0
    tempF := celsiusParaFahrenheit(tempC)
    fmt.Printf("%.2f graus Celsius são %.2f graus Fahrenheit\n", tempC, tempF)
}
```

### Restrições de uso

- **Tipos dos Parâmetros:** Todos os parâmetros devem ter seus tipos especificados. Se não especificados corretamente, o compilador emitirá um erro.
- **Tipo de Retorno:** O tipo do valor retornado deve ser especificado e deve corresponder ao tipo do valor retornado pela função.
- **Valor de Retorno:** A função deve retornar exatamente um valor do tipo especificado. Se a função não retornar um valor, ou retornar um valor de tipo diferente, o compilador emitirá um erro.

### Exemplo de Restrição

**Exemplo de erro de tipo incompatível:**

```go
func multiplicar(a int, b int) int {
    return a * b
}

func main() {
    resultado := multiplicar(3, "4") // Erro: argumento "4" é do tipo string, mas a função espera um int
    fmt.Println("Resultado:", resultado)
}
```

**Exemplo de erro por falta de retorno:**

```go
func dividir(a, b float64) float64 {
    if b == 0 {
        fmt.Println("Erro: divisão por zero")
        // Erro: a função deve retornar um valor float64
    }
    return a / b
}
```

Neste caso, a função `dividir` deve sempre retornar um valor do tipo `float64`, mesmo em casos de erro. Pode ser corrigido retornando um valor padrão, como `0`, ou retornando um valor especial junto com um indicador de erro, utilizando múltiplos retornos.

## Conclusão

Funções normais com parâmetros e um retorno são uma ferramenta fundamental em Go para criar código modular, flexível e reutilizável. Entender como e quando utilizá-las permite escrever programas eficientes e bem estruturados. A sintaxe clara e as restrições simples ajudam a garantir que seu código seja robusto e fácil de entender.
