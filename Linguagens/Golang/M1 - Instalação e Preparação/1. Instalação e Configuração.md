### Como Instalar Go no Windows

Instalar o Go no Windows é um processo simples que envolve baixar o instalador do site oficial do Go e configurar algumas variáveis de ambiente. Vamos passar por cada etapa detalhadamente.

#### 1. Baixar e Instalar Go

1. **Acesse o Site Oficial do Go:**
   - Vá para [go.dev](https://go.dev).
   - Clique em "Download" no menu superior para acessar a página de downloads.
   - Escolha a versão apropriada para Windows (normalmente um arquivo `.msi`).

2. **Executar o Instalador:**
   - Depois de baixar o instalador (`go1.x.x.windows-amd64.msi`), clique duas vezes no arquivo para executá-lo.
   - Siga as instruções do assistente de instalação, aceitando os termos de licença e confirmando as etapas até a instalação ser concluída.

#### 2. Configuração das Variáveis de Ambiente

##### 2.1. GOPATH e GOROOT

**GOROOT:**
   - **O que é?** GOROOT é o diretório onde o Go é instalado. Ele contém o código fonte da linguagem, os binários compilados e as bibliotecas padrão.
   - **Para que serve?** Serve como base para o compilador e outras ferramentas encontrarem os recursos necessários para compilar e rodar programas em Go.

**GOPATH:**
   - **O que é?** GOPATH é o diretório de trabalho do Go. É onde você colocará o código fonte dos seus projetos, bem como onde as bibliotecas e dependências de terceiros serão armazenadas.
   - **Para que serve?** Ele organiza seu ambiente de desenvolvimento Go, facilitando a gestão de pacotes e projetos.

##### 2.2. Configuração do PATH

1. **Abra o Painel de Controle:**
   - Pressione `Windows + R`, digite `control` e pressione Enter.

2. **Vá para Sistema:**
   - No Painel de Controle, vá para `Sistema e Segurança` > `Sistema`.

3. **Configurações Avançadas:**
   - Clique em `Configurações avançadas do sistema` à esquerda.
   - Na janela `Propriedades do Sistema`, clique no botão `Variáveis de Ambiente`.

4. **Editar o PATH:**
   - Na seção `Variáveis do sistema`, encontre a variável `Path` e selecione `Editar`.
   - Adicione o caminho para o diretório `bin` do Go. Por padrão, este caminho será `C:\Go\bin`.

```shell
C:\Go\bin
```

5. **Adicionar GOPATH:**
   - Na janela `Variáveis de Ambiente`, clique em `Novo` em `Variáveis do usuário`.
   - Nomeie a variável como `GOPATH` e defina seu valor como o diretório de trabalho para seus projetos Go. Por exemplo:

```shell
C:\Users\SeuUsuario\go
```

6. **Adicionar GOROOT (opcional):**
   - Se `GOROOT` não for automaticamente configurado durante a instalação, adicione-o manualmente.
   - Nomeie a variável como `GOROOT` e defina seu valor como o diretório onde o Go foi instalado. Por exemplo:

```shell
C:\Go
```

##### 2.3. Configuração do ROOT

1. **Adicionar GOROOT ao PATH (se necessário):**
   - Se `GOROOT` não foi automaticamente configurado durante a instalação, você pode adicioná-lo ao PATH.
   - Na seção `Variáveis do sistema`, edite o `Path` novamente e adicione:

```shell
%GOROOT%\bin
```

#### 3. Verificar a Instalação

1. **Abrir o Prompt de Comando:**
   - Pressione `Windows + R`, digite `cmd` e pressione Enter.

2. **Verificar a Versão do Go:**
   - No Prompt de Comando, digite:

```shell
go version
```

   - Se tudo estiver configurado corretamente, você verá a versão do Go instalada, algo como:

```shell
go version go1.x.x windows/amd64
```

3. **Verificar Configuração do GOPATH e GOROOT:**
   - Para garantir que `GOPATH` e `GOROOT` estão configurados corretamente, digite:

```shell
go env
```

   - Isso exibirá todas as variáveis de ambiente do Go, incluindo `GOPATH` e `GOROOT`.

#### 4. Criar e Executar seu Primeiro Programa Go

1. **Criar Diretório do Projeto:**
   - Navegue até seu diretório `GOPATH` e crie um novo diretório para seu projeto. Por exemplo:

```shell
mkdir %GOPATH%\src\hello
cd %GOPATH%\src\hello
```

2. **Criar um Arquivo Go:**
   - Crie um arquivo chamado `hello.go` e edite-o com o seguinte conteúdo:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

3. **Compilar e Executar o Programa:**
   - No Prompt de Comando, estando no diretório do projeto, compile e execute o programa:

```shell
go run hello.go
```

   - Você deverá ver a saída:

```shell
Hello, World!
```
