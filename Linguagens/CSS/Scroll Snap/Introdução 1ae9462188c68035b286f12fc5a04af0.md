# Introdução

## 1. Introdução

**Scroll Snap** é um conjunto de propriedades CSS que permite criar experiências de rolagem mais suaves e controladas. Com ele, os desenvolvedores podem definir pontos "fixos" (snap points) onde a rolagem será automaticamente alinhada, garantindo que os itens ou seções se posicionem de maneira precisa ao final da rolagem. Essa técnica é muito útil para carrosséis, galerias de imagens, listas horizontais e layouts que requerem um comportamento de rolagem intuitivo e organizado.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    Scroll Snap permite que o conteúdo seja "fixado" em pontos específicos durante a rolagem, melhorando a experiência do usuário e a estética do layout.
    
- **Propriedades Principais:**
    - **scroll-snap-type:** Define a política de snapping para o contêiner de rolagem, determinando a direção e o comportamento do snap.
    - **scroll-snap-align:** Define como os itens dentro do contêiner se alinham em relação aos pontos de snap.
- **Benefícios:**
    - **Experiência de Usuário Melhorada:** A rolagem se torna mais previsível e suave, ajudando os usuários a navegar pelo conteúdo sem precisar ajustar manualmente a posição.
    - **Organização Visual:** Garante que elementos ou seções se alinhem de forma consistente, criando uma interface mais limpa e estruturada.
    - **Aplicações Diversas:** Ideal para carrosséis, galerias, listas de produtos, páginas de apresentação, entre outros layouts interativos.

---

## 3. Propriedades Principais e Sintaxe

### 3.1. **scroll-snap-type**

Define como o contêiner de rolagem se comporta em relação aos pontos de snap. A sintaxe básica é:

```css
seletor {
  scroll-snap-type: [none | x | y | both] [mandatory | proximity];
}

```

- **Direção:**
    - `none`: Desativa o snapping.
    - `x`: Aplica snapping horizontal.
    - `y`: Aplica snapping vertical.
    - `both`: Aplica snapping em ambas as direções.
- **Comportamento:**
    - `mandatory`: O contêiner deve sempre parar nos pontos de snap.
    - `proximity`: O snapping é aplicado somente se o ponto de snap estiver próximo do ponto de parada (mais suave).

**Exemplo:**

```css
.carousel {
  scroll-snap-type: x mandatory;
  overflow-x: auto;
  display: flex;
  scroll-behavior: smooth;
}

```

### 3.2. **scroll-snap-align**

Define como os itens do contêiner se alinham em relação aos pontos de snap. A sintaxe é:

```css
seletor {
  scroll-snap-align: [start | center | end] [start | center | end]?;
}

```

- **Valores:**
    - `start`: O início do item será alinhado com o ponto de snap.
    - `center`: O centro do item será alinhado.
    - `end`: O final do item será alinhado.

Você pode definir um valor para o eixo principal e, opcionalmente, um para o eixo perpendicular.

**Exemplo:**

```css
.carousel-item {
  scroll-snap-align: center;
  flex: 0 0 auto; /* Evita que os itens encolham ou cresçam automaticamente */
}

```

## 4. Exemplos Práticos

### Exemplo 1: Carrossel Horizontal com Scroll Snap

```html
<div class="carousel">
  <div class="carousel-item">Item 1</div>
  <div class="carousel-item">Item 2</div>
  <div class="carousel-item">Item 3</div>
  <div class="carousel-item">Item 4</div>
</div>

```

```css
.carousel {
  display: flex;
  overflow-x: auto;
  scroll-snap-type: x mandatory;
  scroll-behavior: smooth;
  gap: 10px;
  padding: 10px;
  border: 1px solid #ddd;
}

.carousel-item {
  flex: 0 0 80%;
  scroll-snap-align: center;
  background-color: #3498db;
  color: #fff;
  padding: 20px;
  border-radius: 8px;
  text-align: center;
}

```

*Explicação:*

Este carrossel horizontal permite a rolagem suave e garante que cada item se alinhe centralmente ao final da rolagem. O `scroll-snap-type: x mandatory;` força a parada nos pontos de snap definidos por `scroll-snap-align: center;` nos itens.

---

### Exemplo 2: Layout de Página com Seções de Rolagem Suave

```html
<div class="page-container">
  <section class="page-section">Seção 1</section>
  <section class="page-section">Seção 2</section>
  <section class="page-section">Seção 3</section>
</div>

```

```css
.page-container {
  height: 100vh;
  overflow-y: auto;
  scroll-snap-type: y mandatory;
  scroll-behavior: smooth;
}

.page-section {
  height: 100vh;
  scroll-snap-align: start;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 2rem;
  background: #ecf0f1;
  border-bottom: 2px solid #bdc3c7;
}

```

*Explicação:*

Cada `<section>` ocupa 100% da altura da viewport (`100vh`) e se alinha ao início da página ao rolar. O `scroll-snap-type: y mandatory;` garante que a rolagem vertical pare exatamente no início de cada seção, proporcionando uma experiência de navegação "snap-to-section".

## 5. Informações Adicionais

- **scroll-behavior:**
    
    Essa propriedade, quando definida como `smooth`, faz com que a rolagem entre os pontos de snap seja animada, criando transições suaves.
    
- **Compatibilidade:**
    
    As media queries e os prefixed versions foram historicamente necessários, mas atualmente a maioria dos navegadores modernos oferece suporte a **scroll-snap-type** e **scroll-snap-align** sem prefixos.
    
- **Aplicações:**
    - Carrosséis e galerias de imagens.
    - Seções de páginas longas, como parallax scrolling ou apresentações de slides.
    - Interfaces interativas que se beneficiam de uma navegação fluida e controlada.
- **Customização:**
    
    É possível ajustar a experiência de rolagem combinando **scroll-snap** com outras propriedades, como **overflow** e **gap**, para criar layouts sofisticados e responsivos.
