# :root

---

## **1. Introdução**

A pseudo-classe `:root` em CSS seleciona o elemento raiz do documento, que, em HTML, é o elemento `<html>`. Ela é fundamental para declarar variáveis CSS globais (custom properties), permitindo a definição centralizada de valores reutilizáveis, como cores, tamanhos e outros estilos. Sua relevância está na facilitação da manutenção do código, consistência visual e implementação de temas dinâmicos.

**Definição e Conceitos Fundamentais**:

- **Tema Principal**: `:root` é uma pseudo-classe que representa o elemento de nível superior do documento.
- **Subtema**: Variáveis CSS (`-var`), que são frequentemente declaradas em `:root` para escopo global.
- **Propósito**: Centralizar valores reutilizáveis e permitir a criação de estilos dinâmicos e adaptáveis.

## **3. Conteúdo Detalhado**

### **Sintaxe e Estrutura**

```css
:root {
  --cor-primaria: #2ecc71;
  --espacamento: 1rem;
}

```

- **Seletor**: `:root` tem prioridade sobre o seletor `html` (maior especificidade).
- **Escopo**: Variáveis declaradas em `:root` são acessíveis em todo o documento.

### **Componentes Principais**

- **Variáveis CSS**:
Declaradas com `-nome-var: valor;` e acessadas com `var(--nome-var)`.
    
    ```css
    .botao {
      background-color: var(--cor-primaria);
      padding: var(--espacamento);
    }
    
    ```
    
- **Interação**: Variáveis podem ser sobrescritas em seletores aninhados, permitindo adaptação contextual.

### **Restrições de Uso**

- **Herança**: Variáveis em `:root` são herdadas por todos os elementos, mas não são imutáveis (podem ser redefinidas localmente).
- **Suporte**: Compatível com todos os navegadores modernos. Evitar uso em projetos que demandem suporte para IE11.

## **4. Exemplos de Código**

### **Exemplo Básico: Definição de Tema**

```css
:root {
  --fonte-principal: 'Arial', sans-serif;
  --cor-fundo: #f8f9fa;
  --cor-texto: #333;
}

body {
  font-family: var(--fonte-principal);
  background-color: var(--cor-fundo);
  color: var(--cor-texto);
}

```

### **Exemplo Avançado: Tema Escuro/Claro**

```css
:root {
  --cor-fundo: #fff;
  --cor-texto: #000;
}

@media (prefers-color-scheme: dark) {
  :root {
    --cor-fundo: #1a1a1a;
    --cor-texto: #fff;
  }
}

body {
  background-color: var(--cor-fundo);
  color: var(--cor-texto);
  transition: background-color 0.3s ease;
}

```

### **Exemplo com JavaScript (Troca de Temas)**

```html
<button onclick="alternarTema()">Alternar Tema</button>

```

```css
:root {
  --cor-fundo: #fff;
  --cor-texto: #000;
}

:root.tema-escuro {
  --cor-fundo: #1a1a1a;
  --cor-texto: #fff;
}

```

```jsx
function alternarTema() {
  document.documentElement.classList.toggle('tema-escuro');
}

```

## **5. Informações Adicionais**

- **Especificidade**: `:root` tem maior prioridade que `html`. Por exemplo, `:root { --cor: red; }` sobrescreve `html { --cor: blue; }`.
- **XML/SVG**: Em outros formatos, `:root` pode referenciar elementos como `<svg>`.
- **Performance**: Variáveis em `:root` são resolvidas durante o carregamento, otimizando a renderização.
