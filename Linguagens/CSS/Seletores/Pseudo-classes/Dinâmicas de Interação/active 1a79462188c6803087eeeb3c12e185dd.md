# :active

## 1. Introdução

A pseudo-classe `:active` em CSS é utilizada para definir o estilo de um elemento enquanto ele está sendo ativado pelo usuário, ou seja, durante o período em que ocorre uma interação (como clicar com o mouse ou tocar em dispositivos móveis). Ela é especialmente relevante para aprimorar a experiência do usuário, fornecendo feedback visual imediato durante interações, o que pode melhorar a usabilidade e a responsividade das interfaces web.

### Relevância e Importância

- **Feedback Visual:** Ajuda a indicar que um elemento foi clicado ou está sendo ativado, tornando a interface mais intuitiva.
- **Interatividade:** Complementa outras pseudo-classes, como `:hover` e `:focus`, para oferecer uma experiência rica e dinâmica.
- **Acessibilidade:** Quando bem utilizada, contribui para a acessibilidade, auxiliando usuários a entenderem que uma ação foi iniciada.

### Definição e Conceitos Fundamentais

- **Pseudo-classe `:active`:** Aplica estilos a um elemento no momento em que ele está sendo ativado (por exemplo, quando um botão é pressionado).
- **Subtemas Relacionados:**
    - **`:hover`:** Aplica estilos quando o ponteiro do mouse está sobre o elemento.
    - **`:focus`:** Aplica estilos quando o elemento recebe foco, seja via teclado ou interação de mouse.

Essas pseudo-classes trabalham em conjunto para fornecer uma experiência de usuário coesa e interativa.

## 3. Conteúdo Detalhado

### Sintaxe e Estrutura

A sintaxe básica para utilizar a pseudo-classe `:active` é bastante simples:

```css
seletor:active {
  /* Propriedades CSS aplicadas durante a ativação */
  propriedade: valor;
}

```

**Exemplo:**

```css
button:active {
  background-color: #555;
  color: #fff;
  transform: scale(0.98); /* Efeito de "pressionado" */
}

```

Neste exemplo, quando o botão é clicado, ele muda a cor de fundo, a cor do texto e aplica uma leve redução de escala para simular um efeito de pressão.

---

### Componentes Principais e Funções

1. **Seletores:**
    - Pode ser aplicado a praticamente qualquer elemento, mas é comum usá-lo em botões, links e outros elementos interativos.
2. **Propriedades Comuns Utilizadas:**
    - `background-color`: Altera a cor de fundo durante a interação.
    - `color`: Modifica a cor do texto.
    - `transform`: Pode ser usado para aplicar efeitos de escala ou rotação, dando uma sensação tátil.
    - `box-shadow`: Pode ser utilizado para adicionar ou remover sombras, acentuando o efeito visual.
3. **Interação com Outras Pseudo-classes:**
    - **`:hover`:** Estiliza o elemento quando o ponteiro está sobre ele.
    - **`:focus`:** Estiliza o elemento quando ele recebe foco (ex.: via teclado).
    - É importante definir a ordem dos seletores para evitar conflitos, seguindo a especificidade do CSS (ex.: `:link`, `:visited`, `:hover`, `:active`).

---

### Restrições de Uso

- **Transições e Animações:**
    - O estado `:active` é transitório, ocorrendo apenas durante a ação de clicar. Por isso, transições ou animações aplicadas podem não ser notadas se a duração do clique for muito breve.
- **Compatibilidade:**
    - Embora amplamente suportado em navegadores modernos, alguns comportamentos podem variar ligeiramente em dispositivos de toque versus dispositivos com mouse.
- **Especificidade:**
    - Deve-se tomar cuidado com a ordem dos seletores e a cascata de estilos para garantir que o estado `:active` seja aplicado conforme o esperado sem ser sobrescrito por outras regras.

## 4. Exemplos de Código Otimizados

### Exemplo 1: Botão com Efeito de Pressão

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Exemplo :active</title>
  <style>
    /* Estilo básico do botão */
    .botao {
      padding: 10px 20px;
      background-color: #007BFF;
      border: none;
      color: #fff;
      cursor: pointer;
      transition: transform 0.1s ease;
    }

    /* Estilo aplicado quando o botão é ativado (pressionado) */
    .botao:active {
      background-color: #0056b3;
      transform: scale(0.95); /* Efeito de "afundamento" */
    }
  </style>
</head>
<body>
  <button class="botao">Clique Aqui</button>
</body>
</html>

```

*Comentários:*

- O botão possui uma transição suave para o efeito de transformação.
- A mudança de cor e a escala reduzem o tamanho do botão, dando a impressão de que ele foi pressionado.

### Exemplo 2: Link com Feedback Visual

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Exemplo :active em Links</title>
  <style>
    a {
      text-decoration: none;
      color: #0066cc;
      transition: color 0.2s ease;
    }

    /* Estilo aplicado quando o link está sendo clicado */
    a:active {
      color: #004499;
    }
  </style>
</head>
<body>
  <a href="https://www.example.com">Visite o Example.com</a>
</body>
</html>

```

*Comentários:*

- Aqui, a mudança de cor no estado `:active` indica que o link foi ativado, oferecendo feedback visual ao usuário.

## 5. Informações Adicionais

- **Combinação com Transições:**
    - Utilizar a propriedade `transition` pode suavizar as alterações visuais no estado `:active`, proporcionando uma experiência mais agradável.
- **Dispositivos de Toque:**
    - Em dispositivos móveis, o estado `:active` pode se comportar de maneira diferente. É comum a utilização de bibliotecas JavaScript ou ajustes CSS para garantir a consistência visual.
- **Acessibilidade e Usabilidade:**
    - Considere também os estados `:hover` e `:focus` para fornecer um feedback completo, especialmente para usuários que navegam com teclado.
- **Práticas Recomendadas:**
    - Mantenha os estilos simples e consistentes.
    - Teste em múltiplos dispositivos para garantir uma experiência uniforme.
