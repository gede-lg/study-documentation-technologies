# Introdução

## 1. Introdução

**Media Queries** são uma parte fundamental do design responsivo, permitindo que o CSS seja aplicado de forma condicional com base nas características do dispositivo, como a largura da tela, orientação, resolução e outras condições. Elas possibilitam que o layout e o estilo de uma página se adaptem a diferentes tamanhos de tela e dispositivos, proporcionando uma melhor experiência de usuário.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    Media Queries permitem definir regras CSS que só serão aplicadas se certas condições forem atendidas, como uma determinada largura máxima ou mínima de tela.
    
- **Importância:**
    - **Responsividade:** Permitem que o design se ajuste dinamicamente a diferentes tamanhos de tela, como em dispositivos móveis, tablets e desktops.
    - **Melhoria de Usabilidade:** Otimizam a experiência do usuário ao garantir que a interface seja legível e funcional em qualquer dispositivo.
    - **Flexibilidade:** Possibilitam a criação de layouts complexos que se adaptam a múltiplas condições, melhorando o desempenho visual e a acessibilidade.
- **Aplicação:**
    
    As media queries são geralmente definidas no CSS usando a regra `@media` e podem combinar múltiplas condições com operadores lógicos.
    

---

## 3. Sintaxe de Media Queries

A sintaxe básica de uma media query é:

```css
@media [tipo de mídia] and (condição) {
  /* Regras CSS aplicadas quando a condição é atendida */
}

```

### Exemplos de Sintaxe:

- **Exemplo 1: Largura Máxima de 600px**
    
    ```css
    @media (max-width: 600px) {
      /* Regras CSS para telas com largura máxima de 600px */
      body {
        background-color: lightgray;
      }
      .container {
        padding: 10px;
      }
    }
    
    ```
    
    *Explicação:*
    
    As regras dentro desta media query serão aplicadas somente se a largura da tela for de 600px ou menos. Esse tipo de consulta é comumente usado para otimizar o design para dispositivos móveis.
    
- **Exemplo 2: Tela com Largura Mínima de 768px**
    
    ```css
    @media screen and (min-width: 768px) {
      /* Regras CSS para telas com largura mínima de 768px */
      .container {
        width: 750px;
        margin: 0 auto;
      }
    }
    
    ```
    
    *Explicação:*
    
    Essa media query aplica as regras apenas em dispositivos com um tipo de mídia `screen` (como monitores, tablets e smartphones) e com uma largura de tela de pelo menos 768px, ideal para tablets e desktops.
    

---

## 4. Exemplos Práticos

### Exemplo 3: Alterando a Disposição do Layout

```css
/* Estilos padrão para dispositivos móveis */
.container {
  display: flex;
  flex-direction: column;
  padding: 15px;
}

/* Regras para telas maiores que 768px */
@media screen and (min-width: 768px) {
  .container {
    flex-direction: row;
    justify-content: space-between;
    padding: 30px;
  }
}

```

*Explicação:*

- Para telas com largura menor que 768px, o contêiner usa um layout em coluna (vertical).
- Para telas maiores, a media query altera o layout para linha (horizontal), distribuindo os itens com `justify-content: space-between` e aumentando o padding.

### Exemplo 4: Ajustando Tipografia

```css
body {
  font-size: 16px;
}

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}

```

*Explicação:*

- Em telas maiores, o tamanho padrão da fonte é 16px.
- Em telas de 600px ou menos, a media query diminui o tamanho da fonte para 14px, melhorando a legibilidade em dispositivos menores.

---

## 5. Informações Adicionais

- **Tipos de Mídia:**
    
    Você pode especificar o tipo de mídia (como `screen`, `print`, `all`, etc.) para aplicar regras específicas para diferentes contextos. Por exemplo:
    
    ```css
    @media print {
      /* Regras para impressão */
      body {
        font-size: 12pt;
      }
    }
    
    ```
    
- **Combinação de Condições:**
    
    Media queries podem combinar várias condições usando operadores como `and`, `or` (através de múltiplas queries) e `not`:
    
    ```css
    @media screen and (min-width: 768px) and (orientation: landscape) {
      /* Regras para telas largas em modo paisagem */
    }
    
    ```
    
- **Uso Responsivo:**
    
    Media queries são essenciais para criar designs responsivos e são amplamente utilizadas para ajustar layouts, tipografia, espaçamentos e outros aspectos visuais com base nas características do dispositivo.

## 7. Conclusão

As Media Queries são uma ferramenta essencial para criar layouts responsivos que se adaptam a diferentes dispositivos e tamanhos de tela. Utilizando a sintaxe `@media` com condições como `(max-width: 600px)` e `(min-width: 768px)`, os desenvolvedores podem aplicar regras CSS específicas que otimizam a experiência do usuário em qualquer dispositivo. Dominar as media queries é fundamental para o design moderno e responsivo, permitindo que as interfaces se ajustem dinamicamente, mantendo uma aparência consistente e funcional. Explore os exemplos e referências indicados para aprofundar seu conhecimento e aplicar essas técnicas de forma eficaz em seus projetos de CSS.
