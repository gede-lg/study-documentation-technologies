# @import

A at-rule **@import** permite importar folhas de estilo externas para o documento atual, possibilitando a modularização e a organização do código CSS. Ela torna mais fácil manter os estilos separados em diferentes arquivos, contribuindo para um código mais limpo e reutilizável.

---

## 1. Introdução

No desenvolvimento de projetos web, é comum dividir os estilos em vários arquivos, cada um responsável por um aspecto específico do design (por exemplo, reset, tipografia, layout). A declaração `@import` possibilita que esses arquivos sejam integrados em um único ponto de entrada. Embora seja uma ferramenta útil para organizar o código, é importante usar `@import` com cautela, pois pode impactar a performance de carregamento da página.

---

## 2. Quando e Por Que Usar @import

- **Modularização:**
    
    Permite separar o CSS em vários arquivos, facilitando a manutenção e a reutilização de estilos.
    
- **Organização:**
    
    Ajuda a estruturar o projeto, por exemplo, isolando resets, estilos de componentes e layouts em arquivos distintos.
    
- **Flexibilidade:**
    
    Possibilita a aplicação condicional de folhas de estilo (com media queries integradas), o que pode ser útil para designs responsivos.
    

**Observação sobre Performance:**

Cada uso de `@import` pode levar a uma requisição HTTP separada, o que pode aumentar o tempo de carregamento da página. Em projetos de alta performance, é recomendável considerar alternativas como a concatenação dos arquivos CSS durante o processo de build.

---

## 3. Sintaxe e Exemplo

### Sintaxe Básica

```css
@import url("nome-do-arquivo.css");

```

- **url("...")**: Especifica o caminho para o arquivo CSS que será importado. Pode ser um caminho relativo ou absoluto.
- **Opções de media query:** Você também pode incluir media queries diretamente na regra de importação para carregar estilos apenas em determinadas condições.

### Exemplo Prático

```css
/* Importa uma folha de reset que padroniza estilos entre navegadores */
@import url("reset.css");

/* Importa estilos de tipografia apenas para telas com largura mínima de 768px */
@import url("typography.css") screen and (min-width: 768px);

/* Estilos principais do site */
body {
  font-family: 'Helvetica', sans-serif;
  color: #333;
}

```

> Comentário:
> 
> 
> No exemplo acima, o primeiro `@import` traz os estilos de reset para garantir uma base consistente entre navegadores. O segundo `@import` aplica os estilos de tipografia apenas em telas maiores que 768px, utilizando uma media query integrada.
> 

---

## 4. Considerações Importantes

- **Posicionamento:**
    
    A regra `@import` deve ser declarada antes de quaisquer outras regras CSS no arquivo. Qualquer conteúdo (incluindo comentários ou espaços em branco) antes de um `@import` pode fazer com que o navegador ignore essa declaração.
    
- **Impacto na Performance:**
    
    Como cada `@import` pode resultar em uma requisição HTTP separada, é importante considerar a combinação ou a concatenação dos arquivos CSS durante o processo de build, especialmente para projetos de grande escala ou com foco em performance.
    
- **Compatibilidade:**
    
    A maioria dos navegadores suporta a regra `@import`, mas seu uso em conjunto com media queries pode apresentar variações de suporte em navegadores mais antigos.
    

---

## 5. Boas Práticas

- **Organização de Arquivos:**
    
    Utilize `@import` para segmentar o CSS em arquivos lógicos (ex.: reset, componentes, layout, temas), mas evite o uso excessivo para não comprometer a performance.
    
- **Processamento de Build:**
    
    Em ambientes de produção, prefira ferramentas de build (como Webpack, Gulp ou outros) para concatenar e minificar os arquivos CSS, reduzindo as requisições HTTP.
    
- **Uso de Media Queries:**
    
    Combine `@import` com media queries para carregar estilos específicos apenas quando necessário, melhorando a eficiência do carregamento.

A at-rule **@import** é uma ferramenta poderosa para organizar e modularizar o CSS, permitindo a separação de estilos em arquivos distintos que podem ser importados conforme necessário.
