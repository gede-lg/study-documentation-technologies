# column-count

## 1. Introdução

A propriedade **column-count** faz parte do módulo de layout multicoluna do CSS. Ela define o número de colunas em que o conteúdo de um elemento será automaticamente dividido. Essa propriedade é particularmente útil para criar layouts semelhantes a jornais e revistas, onde o texto é distribuído em várias colunas, melhorando a legibilidade e a estética do conteúdo.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **column-count** especifica o número desejado de colunas em que o conteúdo de um elemento será fragmentado.
    
- **Aplicabilidade:**
    
    Ideal para layouts de textos longos, como artigos, blogs e páginas de notícias, onde a distribuição em múltiplas colunas pode facilitar a leitura e criar um design atraente.
    
- **Comportamento Automático:**
    
    Quando o número de colunas é definido, o navegador distribui o conteúdo igualmente entre as colunas, ajustando a largura de cada uma conforme o espaço disponível.
    

---

## 3. Sintaxe e Estrutura

A sintaxe básica é simples:

```css
seletor {
  column-count: número;
}

```

- **número:**
Um valor numérico inteiro que representa o número de colunas desejado.

**Exemplo:**

```css
.article {
  column-count: 3;
}

```

Neste exemplo, o conteúdo do elemento com a classe `.article` será automaticamente dividido em três colunas.

## 4. Exemplos de Código Otimizados

### Exemplo 1: Layout Básico com Três Colunas

```css
.article {
  width: 800px;
  margin: 0 auto;
  column-count: 3;       /* Divide o conteúdo em 3 colunas */
  column-gap: 20px;      /* Espaço de 20px entre as colunas */
  padding: 10px;
  background-color: #f9f9f9;
  border: 1px solid #ccc;
  font-size: 1rem;
  line-height: 1.5;
}

```

### Exemplo 2: Adaptando o Número de Colunas com Media Queries

```css
.article {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
  column-count: 3;
  column-gap: 15px;
  padding: 15px;
  font-size: 1rem;
  line-height: 1.5;
  background-color: #fff;
}

/* Em telas menores, reduzir o número de colunas para melhorar a legibilidade */
@media (max-width: 600px) {
  .article {
    column-count: 1;
  }
}

```

```html
<div class="article">
  <p>Conteúdo de exemplo para demonstrar como o número de colunas se adapta ao tamanho da tela. Em dispositivos menores, o conteúdo é apresentado em uma única coluna para facilitar a leitura.</p>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Pellentesque habitant morbi tristique senectus et netus et malesuada fames ac turpis egestas.</p>
  <p>Vestibulum ante ipsum primis in faucibus orci luctus et ultrices posuere cubilia curae; Sed at velit vitae nunc volutpat aliquam.</p>
</div>

```

*Explicação:*

Em telas maiores, o conteúdo é distribuído em três colunas. Em dispositivos com largura máxima de 600px, a media query altera o valor de **column-count** para `1`, exibindo o conteúdo em uma única coluna e melhorando a experiência do usuário em dispositivos móveis.

## 5. Informações Adicionais

- **Propriedades Relacionadas:**
    - **column-gap:** Define o espaçamento entre as colunas.
    - **column-rule:** Permite desenhar uma linha entre as colunas, especificando a largura, o estilo e a cor dessa linha.
    - **column-width:** Define a largura ideal de cada coluna. Quando combinada com **column-count**, o navegador tenta equilibrar o layout com base no espaço disponível.
- **Responsividade e Legibilidade:**
    
    O uso de **column-count** é fundamental para criar layouts de múltiplas colunas que se adaptem ao tamanho do contêiner. Ajustar o número de colunas com media queries pode melhorar significativamente a legibilidade em dispositivos com diferentes larguras de tela.
    
- **Estilo Jornalístico:**
    
    Layouts multi-coluna são ideais para sites de notícias e revistas online, onde a divisão do conteúdo em colunas facilita a leitura e a organização visual.
