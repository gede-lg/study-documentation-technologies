# columns

## 1. Introdução

A propriedade **columns** faz parte do módulo de layout multicoluna do CSS e é uma propriedade abreviada que permite definir, de forma simples e concisa, o número e a largura das colunas em que o conteúdo de um elemento será dividido. Utilizar **columns** possibilita a criação de layouts com múltiplas colunas – comuns em jornais, revistas e designs responsivos – sem a necessidade de estruturar manualmente o conteúdo em elementos separados.

---

## 2. Conceitos Fundamentais

- **CSS Multi-Column Layout:**
    
    Um conjunto de propriedades que permitem dividir o conteúdo de um elemento em várias colunas, gerenciando automaticamente a distribuição do texto, espaçamento entre colunas e regras visuais entre elas.
    
- **Propriedade Abreviada:**
    
    **columns** é uma propriedade abreviada que combina:
    
    - **column-width:** A largura ideal de cada coluna.
    - **column-count:** O número de colunas desejado.
- **Benefícios:**
    - **Automação:** Facilita a criação de layouts em múltiplas colunas sem necessidade de marcar manualmente cada coluna.
    - **Flexibilidade:** Permite que o navegador ajuste o layout conforme o espaço disponível, tornando o design responsivo.
    - **Simplicidade:** Reduz o código, pois várias configurações podem ser aplicadas com uma única declaração.

---

## 3. Sintaxe e Estrutura

A sintaxe básica da propriedade **columns** é:

```css
seletor {
  columns: <column-width> <column-count>;
}

```

- **:**
    
    Define a largura ideal para cada coluna (ex.: `200px`, `15em`). Pode ser omitido para que o navegador decida o tamanho com base no conteúdo e no espaço disponível.
    
- **:**
    
    Especifica o número de colunas desejado (ex.: `3`, `4`). Também pode ser omitido, permitindo que o navegador crie quantas colunas forem necessárias com base no tamanho do contêiner.
    

**Exemplos:**

- Definir apenas o número de colunas:
    
    ```css
    .container {
        columns: 3;
    }
    
    ```
    
- Definir apenas a largura das colunas:
    
    ```css
    .container {
        columns: 200px;
    }
    
    ```
    
- Definir ambos, largura e número:
    
    ```css
    .container {
        columns: 200px 3;
    }
    
    ```
    

*Observação:* Quando ambos os valores são definidos, o navegador tenta respeitar a largura especificada e, em seguida, cria o número de colunas desejado, ajustando conforme o espaço disponível.

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Divisão Simples em 3 Colunas

```css
.container {
    width: 800px;
    margin: 0 auto;
    columns: 3; /* Cria 3 colunas igualmente distribuídas */
    column-gap: 20px; /* Espaço entre as colunas */
    background-color: #f9f9f9;
    padding: 10px;
    border: 1px solid #ccc;
}

.container p {
    font-size: 1rem;
    line-height: 1.5;
    margin: 0 0 1em 0;
}

```

```html
<div class="container">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed non risus.</p>
  <p>Cras vel lorem. Etiam pellentesque aliquet tellus.</p>
  <p>Phasellus pharetra nulla ac diam. Quisque semper justo at risus.</p>
  <p>Donec venenatis, turpis vel hendrerit interdum, dui ligula ultricies purus.</p>
  <p>Suspendisse potenti. Sed egestas, ante et vulputate volutpat, eros pede semper est.</p>
</div>

```

*Explicação:*

Este exemplo divide o conteúdo em três colunas com um espaço de 20px entre elas, facilitando a leitura e criando um layout de estilo jornalístico.

### Exemplo 2: Colunas com Largura Ideal e Número Máximo

```css
.container {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
    columns: 250px 4; /* Largura ideal de 250px, mas até 4 colunas se houver espaço */
    column-gap: 15px;
    padding: 20px;
    background-color: #fff;
    border: 1px solid #ddd;
}

```

```html
<div class="container">
  <p>Texto de exemplo para demonstrar como as colunas se ajustam de acordo com o espaço disponível. O navegador tenta criar colunas com 250px, mas se não houver espaço suficiente, poderá reduzir o número de colunas.</p>
  <p>Mais um parágrafo de texto para preencher o conteúdo e visualizar o efeito do layout multi-coluna.</p>
  <p>Continuamos adicionando conteúdo para que o efeito fique mais evidente, especialmente em telas maiores.</p>
</div>

```

*Explicação:*

O navegador criará até 4 colunas com 250px cada, mas se o contêiner for menor, o número de colunas pode ser ajustado conforme o espaço disponível.

## 5. Informações Adicionais

- **Propriedades Complementares:**
    - **column-gap:** Define o espaço entre as colunas.
    - **column-rule:** Permite desenhar uma linha entre as colunas (cor, estilo e largura).
    - **column-span:** Permite que um item se estenda por todas as colunas, se necessário.
- **Responsividade:**
    
    Ao usar unidades relativas ou definindo apenas o número de colunas, os layouts multi-coluna se adaptam automaticamente ao tamanho do contêiner, sendo ideais para designs responsivos.
    
- **Legibilidade e Estética:**
    
    O uso adequado de **columns** e suas propriedades relacionadas pode melhorar a legibilidade e a organização do conteúdo, criando um design limpo e profissional, especialmente em sites de notícias, blogs e revistas online.
