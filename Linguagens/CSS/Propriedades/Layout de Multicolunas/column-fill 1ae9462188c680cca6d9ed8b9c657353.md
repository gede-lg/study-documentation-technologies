# column-fill

## 1. Introdução

A propriedade **column-fill** é parte do módulo de layout multicoluna do CSS e controla como o conteúdo é distribuído entre as colunas de um contêiner. Em layouts com múltiplas colunas, essa propriedade determina se o conteúdo deve ser distribuído uniformemente (balançado) entre as colunas ou se deve preencher cada coluna até que não haja mais conteúdo, resultando em colunas de alturas diferentes.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **column-fill** define a estratégia de preenchimento de conteúdo em um contêiner de múltiplas colunas.
    
- **Valores Principais:**
    - **balance:**
        
        Tenta distribuir o conteúdo de forma que todas as colunas tenham alturas aproximadamente iguais, promovendo um layout mais harmonioso e equilibrado.
        
    - **auto:**
        
        Faz com que o conteúdo preencha a primeira coluna até o final, antes de passar para a próxima, podendo resultar em colunas com alturas desiguais.
        
- **Importância:**
    - **Legibilidade e Estética:** A distribuição balanceada é geralmente preferida em designs editoriais para melhorar a legibilidade e o apelo visual.
    - **Performance:** O modo `auto` pode ser mais rápido para renderizar, pois não exige o cálculo de balanceamento entre colunas, embora possa resultar em um layout menos uniforme.

---

## 3. Sintaxe e Estrutura

A sintaxe básica é:

```css
seletor {
    column-fill: valor;
}

```

- **valor:** Pode ser `balance` ou `auto`.

**Exemplo:**

```css
.article {
    column-count: 3;
    column-gap: 20px;
    column-fill: balance;
}

```

*Neste exemplo, o conteúdo do elemento `.article` será distribuído entre 3 colunas tentando manter alturas equilibradas.*

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Distribuição Balanceada das Colunas

```css
.article-balance {
    width: 800px;
    margin: 0 auto;
    column-count: 3;
    column-gap: 20px;
    column-fill: balance;  /* Tenta equilibrar a altura das colunas */
    padding: 10px;
    background-color: #f9f9f9;
    border: 1px solid #ccc;
}

.article-balance p {
    font-size: 1rem;
    line-height: 1.5;
    margin-bottom: 1em;
}

```

```html
<div class="article-balance">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Curabitur vel dolor eget justo ultricies fermentum.</p>
  <p>Fusce interdum libero nec massa dapibus, id pretium erat mattis. Duis vitae mi nec sapien placerat fermentum.</p>
  <p>Vestibulum ante ipsum primis in faucibus orci luctus et ultrices posuere cubilia curae; Sed at turpis velit.</p>
  <p>Morbi tristique nulla eu lorem porta, a eleifend metus bibendum. Integer tincidunt nulla vitae mi fermentum, at luctus magna porttitor.</p>
</div>

```

*Explicação:*

Com `column-fill: balance;`, o navegador tenta distribuir o conteúdo de forma que as alturas das três colunas fiquem aproximadamente iguais, melhorando a estética e a legibilidade.

### Exemplo 2: Preenchimento Automático (auto)

```css
.article-auto {
    width: 800px;
    margin: 0 auto;
    column-count: 3;
    column-gap: 20px;
    column-fill: auto;  /* Preenche a primeira coluna até o final antes de ir para a próxima */
    padding: 10px;
    background-color: #ecf0f1;
    border: 1px solid #aaa;
}

.article-auto p {
    font-size: 1rem;
    line-height: 1.5;
    margin-bottom: 1em;
}

```

```html
<div class="article-auto">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  <p>Proin interdum libero nec massa dapibus, id pretium erat mattis.</p>
  <p>Vestibulum ante ipsum primis in faucibus orci luctus et ultrices posuere cubilia curae.</p>
  <p>Sed at turpis velit. Integer tincidunt nulla vitae mi fermentum, at luctus magna porttitor.</p>
  <p>Ut semper, nisi ut tristique fermentum, massa odio efficitur quam, ac varius ligula urna non neque.</p>
</div>

```

*Explicação:*

Com `column-fill: auto;`, o navegador preenche a primeira coluna até que todo o conteúdo dessa coluna esteja "completo" antes de iniciar a próxima. Isso pode resultar em colunas com alturas desiguais, especialmente se o conteúdo não se distribuir uniformemente.

---

## 5. Informações Adicionais

- **Considerações de Performance:**
    
    O valor `auto` pode ser mais performático em alguns cenários, pois não exige cálculos de balanceamento entre as colunas. Contudo, para layouts editoriais onde a estética é crucial, `balance` é geralmente preferido.
    
- **Compatibilidade:**
    
    A propriedade **column-fill** é suportada pela maioria dos navegadores modernos. Entretanto, sempre verifique a compatibilidade para garantir que seu layout funcione corretamente em todos os ambientes-alvo.
    
- **Aplicações:**
    - **Design Editorial:** Em sites de notícias ou revistas online, onde uma distribuição uniforme das colunas melhora a leitura.
    - **Layouts de Galeria:** Pode ser utilizada para ajustar a disposição de imagens e outros conteúdos quando dividido em múltiplas colunas.

## 7. Conclusão

A propriedade **column-fill** é uma ferramenta essencial para definir como o conteúdo é distribuído entre as colunas em um layout multicoluna. Ao escolher entre `balance` e `auto`, os desenvolvedores podem controlar se as colunas terão alturas uniformes ou se serão preenchidas sequencialmente, permitindo maior flexibilidade no design. Essa propriedade é especialmente útil para layouts editoriais, onde a legibilidade e a estética do conteúdo são cruciais.
