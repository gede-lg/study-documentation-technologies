# cursor

## 1. Introdução

A propriedade **cursor** em CSS define o ícone do cursor que aparece quando o usuário posiciona o mouse sobre um elemento. Essa propriedade é fundamental para melhorar a experiência do usuário, fornecendo indicações visuais sobre a interatividade dos elementos na página, como botões, links e áreas arrastáveis.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **cursor** especifica o tipo de cursor que deve ser exibido quando o ponteiro do mouse passa sobre um elemento.
    
- **Importância:**
    - **Feedback Visual:** Ajuda a comunicar aos usuários que uma ação pode ser executada (por exemplo, um link que muda o cursor para uma mão).
    - **Melhoria da Usabilidade:** Indicadores de cursor apropriados podem tornar a interface mais intuitiva.
    - **Personalização:** Permite a utilização de cursores personalizados através de URLs, aumentando a identidade visual do site.

---

## 3. Sintaxe e Valores

A sintaxe básica é:

```css
seletor {
    cursor: valor;
}

```

### Valores Comuns:

- **default:**
    
    O cursor padrão, geralmente uma seta.
    
    ```css
    cursor: default;
    
    ```
    
- **pointer:**
    
    Geralmente utilizado para links ou botões, representa uma mão indicando que o elemento é clicável.
    
    ```css
    cursor: pointer;
    
    ```
    
- **text:**
    
    Usado em áreas de texto, indica que o conteúdo pode ser selecionado.
    
    ```css
    cursor: text;
    
    ```
    
- **move:**
    
    Indica que o elemento pode ser movido.
    
    ```css
    cursor: move;
    
    ```
    
- **wait:**
    
    Exibe um cursor de espera (geralmente um relógio ou uma ampulheta) indicando que uma ação está em andamento.
    
    ```css
    cursor: wait;
    
    ```
    
- **help:**
    
    Apresenta um cursor que sugere que ajuda está disponível.
    
    ```css
    cursor: help;
    
    ```
    
- **not-allowed:**
    
    Indica que a ação não é permitida.
    
    ```css
    cursor: not-allowed;
    
    ```
    
- **crosshair:**
    
    Mostra um cursor em forma de cruz, frequentemente utilizado em aplicativos de desenho ou edição.
    
    ```css
    cursor: crosshair;
    
    ```
    
- **Custom Cursor:**
    
    É possível utilizar um cursor personalizado especificando uma URL para uma imagem:
    
    ```css
    cursor: url('cursor-personalizado.png'), auto;
    
    ```
    
    *Observação:* Sempre inclua um valor de fallback (por exemplo, `auto`) para navegadores que não suportam cursores personalizados.

## 4. Exemplos de Código Otimizados

### Exemplo 1: Cursor para Elementos Clicáveis

```css
.button {
    padding: 10px 20px;
    background-color: #3498db;
    color: #fff;
    border: none;
    border-radius: 4px;
    cursor: pointer; /* Indica que o botão é clicável */
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: #2980b9;
}

```

```html
<button class="button">Clique Aqui</button>

```

*Explicação:*

Ao definir `cursor: pointer;`, o botão indica claramente que é interativo, melhorando a experiência do usuário.

### Exemplo 2: Utilizando um Cursor Personalizado

```css
.custom-cursor {
    cursor: url('imagens/cursor-personalizado.png'), auto;
    padding: 20px;
    background-color: #e74c3c;
    color: #fff;
    border-radius: 4px;
}

```

```html
<div class="custom-cursor">
  Passe o mouse aqui para ver o cursor personalizado.
</div>

```

*Explicação:*

A URL define um cursor personalizado. Se o navegador não conseguir carregar a imagem, usará o valor de fallback `auto`.

### Exemplo 3: Indicando Ação Proibida

```css
.disabled {
    opacity: 0.5;
    cursor: not-allowed;
}

```

```html
<button class="disabled" disabled>Botão Desabilitado</button>

```

*Explicação:*

`cursor: not-allowed;` indica que o botão não pode ser clicado, fornecendo um feedback visual claro ao usuário.

## 5. Informações Adicionais

- **Feedback e Usabilidade:**
    
    O uso apropriado da propriedade **cursor** ajuda a orientar os usuários, melhorando a experiência interativa e a navegação.
    
- **Suporte a Cursores Personalizados:**
    
    Ao usar cursores personalizados, é importante fornecer um valor de fallback para garantir que o cursor seja exibido corretamente em todos os navegadores.
    
- **Integração com Transições:**
    
    A propriedade **cursor** pode ser combinada com outras interações CSS (como `:hover` e `:active`) para criar efeitos de feedback dinâmicos.
    
- **Boas Práticas:**
    
    Utilize cursores que correspondam à ação esperada pelo usuário. Por exemplo, use `pointer` para links e botões, `text` para áreas de seleção de texto, e `not-allowed` para elementos desabilitados.
