# quotes

## 1. Introdução

A propriedade **quotes** é utilizada no CSS para definir os caracteres que serão usados como aspas de abertura e fechamento quando se utiliza conteúdo gerado por meio dos valores `open-quote` e `close-quote` em pseudo-elementos, geralmente `::before` e `::after`. Essa propriedade é especialmente útil para criar estilos tipográficos para citações e diálogos, permitindo que você controle os caracteres de aspas de maneira semântica e consistente em todo o documento.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    A propriedade **quotes** especifica uma lista de pares de aspas para serem usadas por `open-quote` e `close-quote`. Cada par consiste em um valor para a abertura e um para o fechamento.
    
- **Objetivo:**
    - Controlar os caracteres de aspas geradas automaticamente pelo CSS.
    - Permitir a personalização de citações em diferentes idiomas ou estilos tipográficos.
- **Utilização em Pseudo-elementos:**
    
    Ao definir `content: open-quote;` ou `content: close-quote;` em um pseudo-elemento, o navegador utiliza os valores definidos em **quotes** para exibir as aspas.
    

---

## 3. Sintaxe e Valores

A sintaxe básica para **quotes** é:

```css
seletor {
    quotes: "abertura1" "fechamento1" "abertura2" "fechamento2" ...;
}

```

- Cada par de valores define as aspas de abertura e fechamento correspondentes. Se houver mais de um par, o navegador usará o primeiro par para a primeira citação, o segundo para citações aninhadas, e assim por diante.

**Exemplos:**

- **Exemplo 1: Aspas padrão para citações em inglês:**
    
    ```css
    blockquote {
        quotes: "“" "”" "‘" "’";
    }
    
    ```
    
    Neste exemplo, as citações de nível superior utilizarão “ e ”, enquanto as citações aninhadas usarão ‘ e ’.
    
- **Exemplo 2: Aspas simples para um estilo minimalista:**
    
    ```css
    q {
        quotes: "'" "'";
    }
    
    ```
    
    Aqui, as aspas de abertura e fechamento serão ambas aspas simples.

## 4. Exemplos de Código Otimizados

### Exemplo 1: Citações com Aspas Personalizadas em Blockquotes

```css
/* Define as aspas para blockquote */
blockquote {
    quotes: "«" "»" "‹" "›";
    font-style: italic;
    color: #555;
    margin: 20px;
    padding: 10px 20px;
    border-left: 4px solid #ccc;
}

/* Insere aspas automaticamente com pseudo-elementos */
blockquote::before {
    content: open-quote;
    font-size: 2rem;
    line-height: 0;
    vertical-align: -0.4em;
    color: #ccc;
}

blockquote::after {
    content: close-quote;
    font-size: 2rem;
    line-height: 0;
    vertical-align: -0.4em;
    color: #ccc;
}

```

```html
<blockquote>
  A verdadeira educação não é a mera preparação para a vida, mas a própria vida.
</blockquote>

```

*Explicação:*

O `blockquote` é estilizado para incluir aspas de abertura e fechamento personalizadas (« e » para a citação principal, e ‹ e › para citações aninhadas, se houver). Os pseudo-elementos `::before` e `::after` inserem automaticamente as aspas definidas pela propriedade **quotes**.

### Exemplo 2: Citações Inline com Aspas Simples

```css
q {
    quotes: "“" "”";
}

q::before {
    content: open-quote;
    color: #2c3e50;
}

q::after {
    content: close-quote;
    color: #2c3e50;
}

```

```html
<p>Ela disse, <q>o conhecimento é poder</q>, e inspirou muitos ao seu redor.</p>

```

*Explicação:*

A tag `<q>` é usada para citações inline, e os pseudo-elementos `::before` e `::after` adicionam aspas estilizadas (“ e ”) em torno do conteúdo da citação.

## 5. Informações Adicionais

- **Herança e Escopo:**
    
    A propriedade **quotes** é herdada. Se você definir as aspas em um elemento pai, os elementos filhos também usarão essas definições, a menos que sejam sobrescritas.
    
- **Fallback:**
    
    Se a propriedade **quotes** não for definida, o navegador usará os valores padrão do sistema para as aspas.
    
- **Compatibilidade:**
    
    A propriedade **quotes** é amplamente suportada em navegadores modernos, mas seu comportamento pode variar em implementações mais antigas. Sempre teste a renderização das aspas geradas, especialmente se estiver criando designs tipográficos sofisticados.
    
- **Utilização em Diferentes Idiomas:**
    
    Você pode personalizar os pares de aspas conforme as convenções de diferentes idiomas ou estilos de design, tornando os seus documentos mais adequados ao público-alvo.

## 7. Conclusão

A propriedade **quotes** é uma ferramenta valiosa para controlar o estilo das aspas geradas pelo CSS, permitindo que você defina e personalize os caracteres de abertura e fechamento para citações. Combinada com os pseudo-elementos `::before` e `::after`, ela oferece uma maneira elegante de inserir conteúdo gerado dinamicamente, enriquecendo a tipografia e a semântica dos documentos. Dominar o uso de **quotes** pode melhorar significativamente a apresentação de citações e elementos de destaque, contribuindo para interfaces mais refinadas e profissionalmente apresentadas.
