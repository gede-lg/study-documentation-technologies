# object-fit

## 1. Introdução

A propriedade **object-fit** é usada para especificar como o conteúdo de um elemento substituto (como uma imagem ou vídeo) deve se ajustar ao seu contêiner. Ela permite controlar a forma como o elemento é redimensionado para preencher (ou não) a caixa do contêiner, sem distorcer sua proporção ou perder partes importantes da imagem ou do vídeo.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **object-fit** determina como o conteúdo (imagem, vídeo, etc.) será redimensionado para se adaptar às dimensões do seu contêiner. Essa propriedade é útil especialmente quando o conteúdo tem uma proporção diferente da caixa em que ele se encontra.
    
- **Importância:**
    - **Preservação da Proporção:** Permite que o conteúdo seja escalado mantendo suas proporções originais.
    - **Flexibilidade Visual:** Oferece diversas opções para lidar com o ajuste de imagens ou vídeos em contêineres de tamanhos variados, sem distorções.
    - **Aplicações Práticas:** Essencial para layouts responsivos, galerias de imagens, vídeos e outros elementos visuais.

---

## 3. Sintaxe e Valores

A sintaxe básica para **object-fit** é:

```css
seletor {
  object-fit: valor;
}

```

### Valores Comuns:

- **fill (valor padrão):**
    
    O conteúdo é redimensionado para preencher completamente o contêiner. Isso pode resultar em distorção se as proporções do conteúdo não coincidirem com as do contêiner.
    
    ```css
    img {
      object-fit: fill;
    }
    
    ```
    
- **contain:**
    
    O conteúdo é redimensionado para caber dentro do contêiner, mantendo suas proporções. Pode haver espaços vazios (letterboxing ou pillarboxing) se as proporções não coincidirem.
    
    ```css
    img {
      object-fit: contain;
    }
    
    ```
    
- **cover:**
    
    O conteúdo é redimensionado para preencher o contêiner, mantendo suas proporções. O conteúdo pode ser cortado (crop) se suas proporções não coincidirem com as do contêiner.
    
    ```css
    img {
      object-fit: cover;
    }
    
    ```
    
- **none:**
    
    O conteúdo não é redimensionado. Se o tamanho do conteúdo for maior que o contêiner, ele será cortado; se for menor, não preencherá o contêiner.
    
    ```css
    img {
      object-fit: none;
    }
    
    ```
    
- **scale-down:**
    
    O conteúdo é redimensionado para o menor valor entre `none` e `contain`. Ou seja, ele se comporta como `none` ou `contain`, dependendo do qual resulta em um tamanho menor.
    
    ```css
    img {
      object-fit: scale-down;
    }
    
    ```

## 4. Exemplos de Código Otimizados

### Exemplo 1: Utilizando **object-fit: cover**

```css
.image-cover {
  width: 300px;
  height: 200px;
  border: 1px solid #ccc;
  object-fit: cover;
}

```

```html
<img src="exemplo.jpg" alt="Exemplo de imagem" class="image-cover">

```

*Explicação:*

A imagem é redimensionada para preencher a área de 300x200 pixels, mantendo suas proporções. Partes da imagem podem ser cortadas para que não haja espaços vazios.

---

### Exemplo 2: Utilizando **object-fit: contain**

```css
.image-contain {
  width: 300px;
  height: 200px;
  border: 1px solid #ccc;
  object-fit: contain;
  background: #eee;
}

```

```html
<img src="exemplo.jpg" alt="Exemplo de imagem" class="image-contain">

```

*Explicação:*

A imagem é redimensionada para caber completamente dentro da área de 300x200 pixels, mantendo suas proporções. Isso pode resultar em espaços vazios se as proporções da imagem não coincidirem com as do contêiner.

---

### Exemplo 3: Utilizando **object-fit: none**

```css
.image-none {
  width: 300px;
  height: 200px;
  border: 1px solid #ccc;
  object-fit: none;
  overflow: hidden;
}

```

```html
<img src="exemplo.jpg" alt="Exemplo de imagem" class="image-none">

```

*Explicação:*

A imagem não é redimensionada. Se a imagem for maior que 300x200 pixels, ela será cortada para se ajustar ao contêiner.

---

### Exemplo 4: Utilizando **object-fit: scale-down**

```css
.image-scale-down {
  width: 300px;
  height: 200px;
  border: 1px solid #ccc;
  object-fit: scale-down;
}

```

```html
<img src="exemplo.jpg" alt="Exemplo de imagem" class="image-scale-down">

```

*Explicação:*

A imagem será redimensionada para o menor valor entre manter seu tamanho original (`none`) e caber dentro do contêiner (`contain`), resultando em um ajuste que pode ser útil para evitar redimensionamento desnecessário.

## 5. Informações Adicionais

- **Preservação de Proporção:**
    
    Os valores `contain` e `cover` são os mais utilizados quando se deseja manter as proporções originais da imagem ou vídeo.
    
- **Aplicação em Vídeos:**
    
    A propriedade **object-fit** pode ser aplicada a vídeos para controlar como eles se ajustam ao seu contêiner, ideal para criar interfaces responsivas.
    
- **Combinação com background-size:**
    
    Enquanto **object-fit** se aplica a elementos substitutos (como `<img>` ou `<video>`), a propriedade `background-size` realiza uma função similar para imagens de fundo.
    
- **Contexto de Uso:**
    
    É amplamente utilizada em layouts responsivos, galerias de imagens, apresentações de produtos e interfaces onde o controle visual sobre o conteúdo substituto é necessário.
