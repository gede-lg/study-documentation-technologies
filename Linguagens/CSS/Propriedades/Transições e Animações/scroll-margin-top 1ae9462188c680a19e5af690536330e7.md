# scroll-margin-top

## 1. Introdução

A propriedade **scroll-margin-top** é parte do conjunto de propriedades de controle de rolagem do CSS que permite adicionar um espaço extra (margem) no topo de um elemento durante a rolagem. Em outras palavras, quando um elemento é "snapado" ou focalizado através de âncoras, a margem definida por **scroll-margin-top** garante que haja um espaço entre o início do elemento e o topo da viewport. Essa funcionalidade é especialmente útil em layouts responsivos e em interfaces onde há cabeçalhos fixos, para evitar que o conteúdo fique oculto atrás desses elementos.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **scroll-margin-top** define a margem extra que deve ser aplicada no lado superior de um elemento durante a rolagem. Esse valor é levado em conta quando o navegador posiciona o elemento, por exemplo, quando um link âncora é clicado ou quando um método JavaScript (como `scrollIntoView()`) é utilizado para mover o elemento para a viewport.
    
- **Objetivo e Benefícios:**
    - **Melhoria na Experiência de Navegação:** Garante que o elemento não fique encostado diretamente na borda superior da viewport, melhorando a legibilidade.
    - **Compensação de Cabeçalhos Fixos:** Muito útil quando há cabeçalhos fixos ou outros elementos fixos na parte superior da tela, para que o conteúdo não seja parcialmente ocultado.
    - **Ajuste Fino em Layouts Responsivos:** Permite personalizar a posição dos elementos durante a rolagem, adaptando a experiência do usuário a diferentes tamanhos de tela.
- **Relação com Outras Propriedades:**
    
    **scroll-margin-top** faz parte da família **scroll-margin**, que também inclui:
    
    - **scroll-margin-right**
    - **scroll-margin-bottom**
    - **scroll-margin-left**
    
    Esses valores podem ser aplicados individualmente ou de forma abreviada usando a propriedade **scroll-margin**.
    

---

## 3. Sintaxe e Valores

A sintaxe básica para **scroll-margin-top** é:

```css
seletor {
  scroll-margin-top: valor;
}

```

- **valor:**
Pode ser especificado em unidades absolutas (por exemplo, `px`, `cm`) ou relativas (por exemplo, `em`, `%`). O valor definido representa o espaço extra que será aplicado na parte superior do elemento durante a rolagem.

**Exemplo:**

```css
.section {
  scroll-margin-top: 80px;
}

```

*Neste exemplo, quando a seção é rolada para a viewport, ela ficará 80px abaixo do topo da área de visualização, permitindo que um cabeçalho fixo, por exemplo, não o sobreponha.*

---

## 4. Exemplos Práticos

### Exemplo 1: Compensando um Cabeçalho Fixo em uma Página com Âncoras

Imagine uma página com um cabeçalho fixo de 80px de altura. Para garantir que o conteúdo não fique oculto atrás do cabeçalho ao clicar em links âncora, você pode usar **scroll-margin-top**:

```css
header {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  height: 80px;
  background: #333;
  color: #fff;
  display: flex;
  align-items: center;
  padding: 0 20px;
  z-index: 1000;
}

section {
  scroll-margin-top: 80px; /* Espaço extra igual à altura do cabeçalho */
  padding: 20px;
  border-bottom: 1px solid #ddd;
  margin-top: 100px; /* Espaço adicional para visualização */
}

```

```html
<header>
  <nav>
    <a href="#section1">Seção 1</a>
    <a href="#section2">Seção 2</a>
    <a href="#section3">Seção 3</a>
  </nav>
</header>

<main>
  <section id="section1">
    <h2>Seção 1</h2>
    <p>Conteúdo da Seção 1...</p>
  </section>
  <section id="section2">
    <h2>Seção 2</h2>
    <p>Conteúdo da Seção 2...</p>
  </section>
  <section id="section3">
    <h2>Seção 3</h2>
    <p>Conteúdo da Seção 3...</p>
  </section>
</main>

```

*Explicação:*

Cada `<section>` utiliza `scroll-margin-top: 80px;` para garantir que, ao ser rolada para a viewport, a parte superior da seção fique 80px abaixo do topo, evitando que o cabeçalho fixo sobreponha o conteúdo.

---

### Exemplo 2: Aplicação em um Carrossel Horizontal

Para um carrossel horizontal, **scroll-margin-top** pode ser útil quando combinado com outras direções de margem de rolagem para criar espaçamento consistente ao posicionar os itens:

```css
.carousel {
  display: flex;
  overflow-x: auto;
  scroll-snap-type: x mandatory;
  scroll-behavior: smooth;
  gap: 10px;
  padding: 10px;
  border: 1px solid #ddd;
}

.carousel-item {
  flex: 0 0 80%;
  scroll-snap-align: center;
  scroll-margin-top: 20px; /* Espaço extra ao posicionar o item */
  background-color: #3498db;
  color: #fff;
  padding: 20px;
  border-radius: 8px;
  text-align: center;
}

```

```html
<div class="carousel">
  <div class="carousel-item">Item 1</div>
  <div class="carousel-item">Item 2</div>
  <div class="carousel-item">Item 3</div>
</div>

```

*Explicação:*

Cada item do carrossel terá uma margem extra de 20px no topo quando for alinhado na viewport, contribuindo para uma experiência visual mais equilibrada e ajudando a evitar que os itens fiquem colados nas bordas superior ou inferior.

---

### Exemplo 3: Configuração de Margens de Rolagem para Elementos Focados

Você pode aplicar **scroll-margin-top** em elementos que serão focados via métodos programáticos, como `scrollIntoView()`, para garantir que eles não fiquem presos no topo da viewport:

```css
.article-section {
  scroll-margin-top: 50px;
  padding: 15px;
  border: 1px solid #ccc;
}

```

```html
<div class="article-section" id="target">
  <h2>Seção de Destino</h2>
  <p>Este é o conteúdo da seção que será focada quando um usuário interagir com um link ou ação específica.</p>
</div>

```

*Explicação:*

Ao usar `element.scrollIntoView()`, o elemento com a classe `.article-section` será posicionado de modo que haja um espaço de 50px entre sua parte superior e o topo da viewport, melhorando a legibilidade e a visibilidade.

## 5. Informações Adicionais

- **Integração com Scroll Snap:**
    
    **scroll-margin-top** é frequentemente combinada com outras propriedades de scroll snap (como `scroll-snap-align`) para criar layouts onde os elementos se posicionam com margens consistentes, proporcionando uma experiência de rolagem precisa e agradável.
    
- **Unidades de Medida:**
    
    Valores podem ser especificados em unidades absolutas (`px`, `cm`) ou relativas (`em`, `%`). Unidades relativas garantem que o espaço se ajuste proporcionalmente ao tamanho do elemento ou da fonte, sendo úteis em layouts responsivos.
    
- **Compatibilidade:**
    
    A propriedade **scroll-margin-top** é amplamente suportada em navegadores modernos. No entanto, é sempre bom testar seu comportamento em diferentes ambientes, especialmente em layouts complexos ou em dispositivos móveis.
