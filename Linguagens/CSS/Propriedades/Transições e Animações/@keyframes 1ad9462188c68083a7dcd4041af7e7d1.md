# @keyframes

## 1. Introdução

A at-rule **@keyframes** é um dos pilares das animações em CSS. Ela permite definir a sequência de estados que uma animação percorre, especificando como as propriedades dos elementos devem mudar ao longo do tempo. Em conjunto com a propriedade `animation` e suas subpropriedades, **@keyframes** possibilita criar efeitos dinâmicos e interativos sem a necessidade de JavaScript. Essa abordagem declarativa torna o processo de animação mais intuitivo e eficiente, facilitando a criação de transições suaves, movimentos e transformações complexas.

## 2. Conceitos Fundamentais

- **Definição:**
    
    **@keyframes** define uma "linha do tempo" para a animação, descrevendo os estágios intermediários (chamados de "keyframes") que determinam os valores das propriedades animadas.
    
- **Importância:**
    - Permite especificar como um elemento deve se transformar de um estado inicial a um estado final.
    - Oferece flexibilidade para criar animações personalizadas com múltiplos pontos de controle.
    - É essencial para criar efeitos como fade, slide, rotação, escalonamento, entre outros, que enriquecem a experiência do usuário.

## 3. Sintaxe e Estrutura

A sintaxe básica da at-rule **@keyframes** é a seguinte:

```css
@keyframes nome-da-animacao {
  /* Declaração de keyframes */
  0% {
    /* Estado inicial da animação */
    propriedade: valor;
  }
  50% {
    /* Estado intermediário */
    propriedade: valor;
  }
  100% {
    /* Estado final da animação */
    propriedade: valor;
  }
}

```

- **nome-da-animacao:**
    
    Um identificador único para a animação, que será referenciado pela propriedade `animation-name`.
    
- **Keyframes:**
    
    São definidos em porcentagens (0% a 100%), embora também seja possível usar as palavras-chave `from` (equivalente a 0%) e `to` (equivalente a 100%). Dentro de cada keyframe, você declara as propriedades CSS que serão alteradas.
    

**Exemplo simples:**

```css
@keyframes fadeIn {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

```

Neste exemplo, a animação `fadeIn` faz com que o elemento passe de totalmente transparente (opacity: 0) a completamente opaco (opacity: 1).

## 4. Componentes e Funcionamento

### 4.1. Definindo Estados Intermediários

Você pode definir quantos keyframes forem necessários para controlar a animação. Por exemplo:

```css
@keyframes slideAndFade {
  0% {
    transform: translateX(-100%);
    opacity: 0;
  }
  50% {
    transform: translateX(0);
    opacity: 0.5;
  }
  100% {
    transform: translateX(0);
    opacity: 1;
  }
}

```

- **0% (from):** Estado inicial – o elemento começa fora da tela à esquerda e é invisível.
- **50%:** Estado intermediário – o elemento está parcialmente visível e já se posicionou na posição final horizontalmente.
- **100% (to):** Estado final – o elemento está completamente visível na posição final.

### 4.2. Utilizando Palavras-Chave: `from` e `to`

Em vez de usar porcentagens, você pode usar as palavras-chave:

```css
@keyframes grow {
  from {
    transform: scale(0);
  }
  to {
    transform: scale(1);
  }
}

```

Este exemplo define uma animação onde o elemento cresce de um tamanho nulo para seu tamanho original.

### 4.3. Propriedades Animáveis

Nem todas as propriedades CSS são animáveis. **@keyframes** funciona com propriedades que possuem valores numéricos ou que podem ser interpolados, como:

- `opacity`
- `transform` (rotate, scale, translate, etc.)
- `color` (por meio de interpolação de cores)
- `width`, `height` e outras propriedades dimensionais

## 5. Exemplos de Implementação

### 5.1. Exemplo Completo: Fade In e Slide

```css
@keyframes fadeSlideIn {
  0% {
    opacity: 0;
    transform: translateY(20px);
  }
  100% {
    opacity: 1;
    transform: translateY(0);
  }
}

.elemento-animado {
  animation-name: fadeSlideIn;
  animation-duration: 1s;
  animation-timing-function: ease-out;
  animation-fill-mode: forwards;
}

```

```html
<div class="elemento-animado">
  Este elemento aparece suavemente, deslizando para cima.
</div>

```

### 5.2. Exemplo com Múltiplos Keyframes

```css
@keyframes colorChange {
  0% {
    background-color: #ff0000;
  }
  33% {
    background-color: #00ff00;
  }
  66% {
    background-color: #0000ff;
  }
  100% {
    background-color: #ff0000;
  }
}

.caixa-colorida {
  width: 200px;
  height: 200px;
  animation: colorChange 4s infinite;
}

```

```html
<div class="caixa-colorida">
  A cor desta caixa muda gradualmente.
</div>

```

## 6. Informações Adicionais

- **Integração com Propriedades de Animação:**
    
    Para utilizar uma animação definida com **@keyframes**, é necessário referenciá-la com a propriedade `animation-name` e configurar outras subpropriedades (como `animation-duration`, `animation-timing-function`, etc.) para controlar o comportamento da animação.
    
- **Controle Fino:**
    
    É possível combinar **@keyframes** com pseudo-classes (como `:hover` ou `:focus`) para iniciar animações em resposta a interações do usuário.
    
- **Performance:**
    
    Animações CSS são geralmente eficientes, mas é importante evitar animações muito complexas ou que afetem muitos elementos simultaneamente, para manter um bom desempenho, especialmente em dispositivos móveis.
    
- **Acessibilidade:**
    
    Sempre considere a possibilidade de fornecer alternativas ou desativar animações para usuários que possam ter dificuldades com movimentos excessivos. Utilize a mídia query `prefers-reduced-motion` para ajustar as animações conforme as preferências do usuário.

## 8. Conclusão

A at-rule **@keyframes** é uma ferramenta essencial para criar animações ricas e dinâmicas em CSS. Ela permite que você defina detalhadamente como um elemento deve evoluir de um estado a outro ao longo do tempo, dando aos desenvolvedores controle total sobre as transições visuais. Ao combinar **@keyframes** com as propriedades de animação (como `animation-duration`, `animation-timing-function`, etc.), é possível criar efeitos sofisticados que melhoram a experiência do usuário e enriquecem a interface. Explore os exemplos e recursos indicados para aprofundar seu conhecimento e aplicar essas técnicas de animação de forma eficaz em seus projetos.
