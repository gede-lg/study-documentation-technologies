# stroke

## 1. Introdução

A propriedade **stroke** é usada em SVG para definir a cor do contorno (linha) de um elemento gráfico. Essa propriedade permite que você estilize os traçados de formas vetoriais (como `<path>`, `<circle>`, `<rect>`, `<line>`, etc.), contribuindo para a aparência visual dos gráficos. Além de **stroke**, existem outras propriedades relacionadas que permitem controlar a largura, o estilo e outros aspectos do contorno.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **stroke** especifica a cor do contorno de um elemento SVG. Ela é aplicada aos elementos gráficos para definir como os seus traçados serão renderizados.
    
- **Importância:**
    - **Estética Visual:** Controla a cor do contorno, possibilitando designs vibrantes, sutis ou contrastantes conforme o necessário.
    - **Diferenciação:** Permite destacar ou separar elementos dentro de um gráfico.
    - **Interatividade e Dinamismo:** Pode ser animada ou alterada via CSS ou SMIL para criar efeitos visuais interativos.
- **Relação com Outras Propriedades:**
    
    **stroke** é frequentemente usada em conjunto com:
    
    - **stroke-width:** Define a espessura do contorno.
    - **stroke-linecap:** Controla o formato dos finais dos traçados (ex.: butt, round, square).
    - **stroke-linejoin:** Define como os cantos do traçado são renderizados (ex.: miter, round, bevel).
    - **stroke-dasharray:** Cria um padrão de traços e espaços para o contorno.
    - **stroke-dashoffset:** Ajusta a posição inicial do padrão de traços.

---

## 3. Sintaxe e Estrutura

A sintaxe básica para **stroke** é simples:

```css
seletor {
  stroke: valor;
}

```

- **valor:**
Pode ser definido por nomes de cores, valores hexadecimais, RGB, RGBA, HSL, HSLA, entre outros.

**Exemplos:**

```css
svg path {
  stroke: #3498db;
}

```

```css
svg circle {
  stroke: rgb(231, 76, 60);
}

```

## 4. Exemplos de Código Otimizados

### Exemplo 1: Aplicando Cor ao Contorno de um Elemento SVG

```html
<svg width="200" height="200">
  <rect x="10" y="10" width="180" height="180" fill="none" stroke="#2ecc71" stroke-width="4"/>
</svg>

```

*Explicação:*

Neste exemplo, um retângulo é desenhado com contorno verde (`#2ecc71`) com uma espessura de 4px e sem preenchimento (graças a `fill="none"`).

---

### Exemplo 2: Utilizando Propriedades de Traçado Avançadas

```html
<svg width="300" height="100">
  <line x1="10" y1="50" x2="290" y2="50" fill="none"
        stroke="orange"
        stroke-width="5"
        stroke-linecap="round"
        stroke-dasharray="10,5"
        stroke-dashoffset="0"/>
</svg>

```

*Explicação:*

Aqui, uma linha é desenhada com os seguintes estilos:

- **stroke:** A linha tem a cor laranja.
- **stroke-width:** A espessura é de 5px.
- **stroke-linecap:** Os finais da linha são arredondados.
- **stroke-dasharray:** Cria um padrão de traços de 10px seguidos por 5px de espaço.
- **stroke-dashoffset:** Inicia o padrão de traços no começo da linha.

---

### Exemplo 3: Animação do Contorno com CSS

```html
<svg width="200" height="200">
  <circle cx="100" cy="100" r="80" fill="none"
          stroke="#e74c3c"
          stroke-width="6"
          stroke-dasharray="502"
          stroke-dashoffset="502"
          class="animated-circle"/>
</svg>

```

```css
.animated-circle {
  animation: draw 3s forwards;
}

@keyframes draw {
  to {
    stroke-dashoffset: 0;
  }
}

```

*Explicação:*

Nesse exemplo, um círculo SVG tem um contorno animado que "desenha" o traçado. Inicialmente, o contorno está oculto (`stroke-dashoffset: 502`), e a animação gradualmente reduz esse valor para 0, revelando o contorno.

## 5. Informações Adicionais

- **Interação com Outras Propriedades SVG:**
    
    **stroke** pode ser combinada com outras propriedades SVG para criar efeitos complexos e interativos, como animações de traçado e variações de cor.
    
- **Uso de Gradientes e Padrões:**
    
    Em vez de uma cor sólida, **stroke** pode referenciar gradientes ou padrões definidos dentro do próprio SVG usando a função `url()`.
    
    ```css
    svg path {
      stroke: url(#meuGradiente);
    }
    
    ```
    
- **Aplicações Comuns:**
    - Ilustrações vetoriais.
    - Ícones e logotipos.
    - Gráficos interativos e animações.
    - Interfaces que requerem personalização visual dos elementos gráficos.

## 7. Conclusão

A propriedade **stroke** é fundamental para definir a cor e o estilo do contorno dos elementos SVG, permitindo a criação de gráficos e ilustrações vetoriais ricos e interativos. Ao combiná-la com propriedades relacionadas como **stroke-width**, **stroke-linecap**, **stroke-linejoin**, **stroke-dasharray** e **stroke-dashoffset**, os desenvolvedores podem criar uma ampla variedade de efeitos visuais, desde contornos simples até animações complexas. Dominar **stroke** e suas subpropriedades é crucial para qualquer projeto que envolva SVG, contribuindo para interfaces modernas e visualmente impressionantes.
