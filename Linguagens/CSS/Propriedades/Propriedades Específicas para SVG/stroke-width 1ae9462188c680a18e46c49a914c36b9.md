# stroke-width

## 1. Introdução

A propriedade **stroke-width** é usada em elementos SVG para definir a espessura do traçado (contorno) dos elementos gráficos, como `<path>`, `<circle>`, `<rect>`, `<line>`, entre outros. Ela determina quão grossa ou fina a linha que contorna o elemento será renderizada. Essa propriedade é fundamental para o controle visual de gráficos vetoriais, permitindo ajustes finos que influenciam a estética e a legibilidade das ilustrações.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **stroke-width** especifica a largura do contorno do elemento SVG. Ela pode ser definida em diversas unidades (como pixels, em, rem, etc.) para obter o efeito desejado.
    
- **Importância:**
    - **Aparência Visual:** Controla a espessura do traçado, influenciando o destaque e a clareza dos elementos gráficos.
    - **Design Responsivo:** Permite ajustar a espessura das linhas para diferentes contextos e dispositivos.
    - **Interatividade:** Em animações e interações, a modulação da largura do traçado pode criar efeitos visuais dinâmicos.
- **Relação com Outras Propriedades de Traçado:**
    
    **stroke-width** é frequentemente utilizada em conjunto com:
    
    - **stroke:** Para definir a cor do contorno.
    - **stroke-linecap:** Para definir o formato dos finais dos traçados (butt, round, square).
    - **stroke-linejoin:** Para controlar como os cantos do traçado são renderizados (miter, round, bevel).
    - **stroke-dasharray:** Para criar padrões de traços e espaços.
    - **stroke-dashoffset:** Para ajustar a posição do padrão de traços.

---

## 3. Sintaxe e Estrutura

A sintaxe básica para a propriedade **stroke-width** é:

```css
seletor {
  stroke-width: <valor>;
}

```

- **:**
Pode ser um valor com unidade, como `2px`, `0.5em`, ou qualquer outra unidade CSS válida.

**Exemplo:**

```css
svg path {
  stroke-width: 3px;
}

```

*Este exemplo define a espessura do traçado de um `<path>` em 3 pixels.*

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Traçado Fixo para um Elemento SVG

```html
<svg width="200" height="200">
  <circle cx="100" cy="100" r="80" fill="none" stroke="#3498db" stroke-width="4" />
</svg>

```

*Explicação:*

Um círculo SVG é desenhado sem preenchimento (`fill="none"`), com um contorno azul de 4px de espessura.

---

### Exemplo 2: Animação do Traçado com Variação de Largura

```html
<svg width="200" height="200">
  <rect x="20" y="20" width="160" height="160" fill="none" stroke="#e74c3c" stroke-width="2">
    <animate attributeName="stroke-width" from="2" to="8" dur="1s" repeatCount="indefinite" />
  </rect>
</svg>

```

*Explicação:*

Neste exemplo, um retângulo SVG tem um contorno cuja largura é animada, variando de 2px a 8px continuamente. Isso cria um efeito dinâmico que pode chamar a atenção ou indicar uma ação.

---

### Exemplo 3: Traçado Personalizado com Propriedades Complementares

```html
<svg width="300" height="100">
  <line x1="10" y1="50" x2="290" y2="50" fill="none"
        stroke="#2ecc71"
        stroke-width="5"
        stroke-linecap="round" />
</svg>

```

*Explicação:*

Uma linha SVG é desenhada com um contorno verde de 5px, onde os finais da linha são arredondados devido à propriedade `stroke-linecap: round`.

## 5. Informações Adicionais

- **Unidades e Escalabilidade:**
    
    Utilizar unidades relativas como `em` ou `rem` pode ajudar a manter a consistência do design em layouts responsivos, adaptando a espessura do traçado conforme o tamanho do texto ou do contêiner.
    
- **Interação com Outras Propriedades SVG:**
    
    O **stroke-width** é apenas um dos aspectos que definem o estilo do traçado. Para um controle completo, combine-o com propriedades como **stroke**, **stroke-dasharray** e **stroke-linejoin**.
    
- **Aplicações Comuns:**
    - Ilustrações vetoriais e ícones.
    - Gráficos e diagramas.
    - Interfaces interativas que utilizam animações de SVG.
    - Qualquer projeto que requeira controle preciso sobre a aparência dos traçados.
