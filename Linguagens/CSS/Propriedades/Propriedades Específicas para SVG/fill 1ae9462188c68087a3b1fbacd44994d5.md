# fill

## 1. Introdução

A propriedade **fill** é utilizada principalmente em gráficos vetoriais SVG para definir a cor, padrão ou gradiente aplicado ao interior de elementos gráficos, como `<rect>`, `<circle>`, `<path>`, entre outros. Essa propriedade determina como o "preenchimento" do elemento será exibido, influenciando a aparência visual dos gráficos.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **fill** especifica o "pintura" utilizada para preencher o interior de um elemento SVG. Ela pode ser definida com cores sólidas, valores RGBA, gradientes e padrões.
    
- **Importância:**
    - **Estética dos Gráficos:** Controla a cor e o visual do conteúdo interno de formas SVG.
    - **Flexibilidade:** Permite utilizar valores estáticos ou dinâmicos, incluindo gradientes e padrões, para enriquecer a aparência dos gráficos.
    - **Interatividade:** Pode ser animada ou alterada em resposta a interações, oferecendo efeitos visuais dinâmicos em aplicações web.
- **Aplicabilidade:**
    
    A propriedade **fill** é específica para elementos SVG e não afeta elementos HTML padrão. Para HTML, outras propriedades (como `background-color`) são utilizadas.
    

---

## 3. Sintaxe e Estrutura

A sintaxe básica é:

```css
seletor {
  fill: valor;
}

```

### Exemplos de Valores:

- **Cores Sólidas:**
    
    Pode ser definida com nomes de cores, valores hexadecimais, RGB, RGBA, HSL, HSLA, etc.
    
    ```css
    svg rect {
      fill: #3498db; /* Azul */
    }
    
    ```
    
- **Nenhum Preenchimento:**
    
    Para não preencher o elemento, pode-se usar `none`.
    
    ```css
    svg path {
      fill: none;
    }
    
    ```
    
- **Gradientes e Padrões:**
    
    Pode referenciar gradientes ou padrões definidos dentro do próprio SVG com `url()`.
    
    ```css
    svg circle {
      fill: url(#meuGradiente);
    }
    
    ```
    

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Aplicando uma Cor Sólida

```html
<svg width="150" height="150">
  <rect width="100%" height="100%" fill="#e74c3c" />
</svg>

```

*Explicação:*

Um retângulo SVG é preenchido com a cor vermelha (#e74c3c).

---

### Exemplo 2: Usando `fill: none`

```html
<svg width="150" height="150">
  <circle cx="75" cy="75" r="50" stroke="#2ecc71" stroke-width="4" fill="none" />
</svg>

```

*Explicação:*

Um círculo SVG é desenhado com uma borda (stroke), mas seu interior não é preenchido, graças a `fill: none`.

---

### Exemplo 3: Aplicando um Gradiente com fill

```html
<svg width="200" height="200">
  <defs>
    <linearGradient id="grad1" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" style="stop-color:#f1c40f;stop-opacity:1" />
      <stop offset="100%" style="stop-color:#e67e22;stop-opacity:1" />
    </linearGradient>
  </defs>
  <rect width="200" height="200" fill="url(#grad1)" />
</svg>

```

*Explicação:*

Um gradiente linear é definido dentro do elemento `<defs>` e aplicado como preenchimento do retângulo usando `fill: url(#grad1)`.

---

## 5. Informações Adicionais

- **Herança:**
    
    Em SVG, a propriedade **fill** é herdada pelos elementos filhos, a menos que seja sobrescrita.
    
- **Interação com Outras Propriedades:**
    
    A propriedade **fill** pode ser combinada com **stroke** (que define a cor do contorno) para criar gráficos com preenchimento e contorno distintos.
    
- **Transições e Animações:**
    
    **fill** pode ser animada com CSS ou SMIL para criar efeitos de mudança de cor e gradientes dinâmicos.
    
- **Contexto:**
    
    Lembre-se de que **fill** é aplicável somente a elementos SVG. Para elementos HTML, utilize propriedades como `background-color`.
