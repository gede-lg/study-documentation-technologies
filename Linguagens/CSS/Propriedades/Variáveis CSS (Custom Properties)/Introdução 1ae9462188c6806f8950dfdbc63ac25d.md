# Introdução

## 1. Introdução

As variáveis CSS, também conhecidas como propriedades customizadas, permitem armazenar valores reutilizáveis no CSS. Isso facilita a manutenção e a consistência do design, pois você pode definir cores, tamanhos e outros valores em um único lugar e usá-los em várias partes do seu código. As variáveis CSS são especialmente úteis para criar temas dinâmicos e gerenciar alterações de estilo de forma centralizada.

---

## 2. Declaração e Uso

### 2.1. Declaração

As variáveis CSS são geralmente declaradas no seletor `:root`, que representa o elemento raiz do documento (normalmente `<html>`). Declarar variáveis no `:root` as torna disponíveis globalmente em todo o documento.

```css
:root {
  --main-color: #f00;  /* Define uma variável chamada --main-color com o valor vermelho (#f00) */
  --padding-base: 16px;
  --font-size-large: 1.5rem;
}

```

### 2.2. Uso

Para utilizar uma variável, use a função `var()`, passando o nome da variável definida.

```css
body {
  color: var(--main-color);  /* Aplica a cor definida na variável --main-color ao texto */
  padding: var(--padding-base);
  font-size: var(--font-size-large);
}

```

Você também pode fornecer um valor de fallback caso a variável não esteja definida:

```css
h1 {
  color: var(--header-color, #333);  /* Se --header-color não estiver definida, usa #333 como fallback */
}

```

---

## 3. Benefícios das Variáveis CSS

- **Reutilização:**
    
    Permite reutilizar os mesmos valores em vários locais do código, garantindo consistência e facilitando atualizações.
    
- **Manutenção Simplificada:**
    
    Alterar o valor de uma variável no `:root` reflete automaticamente em todos os lugares onde ela é utilizada, agilizando a manutenção do design.
    
- **Personalização Dinâmica:**
    
    Facilita a criação de temas e estilos dinâmicos, possibilitando a alteração de cores e outros estilos de forma centralizada.
    
- **Legibilidade:**
    
    Nomes descritivos de variáveis tornam o código mais legível e autoexplicativo.
    

---

## 4. Exemplos de Uso

### Exemplo 1: Tema Básico

```css
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --font-size-base: 16px;
}

body {
  font-size: var(--font-size-base);
  color: var(--primary-color);
}

.button {
  background-color: var(--secondary-color);
  color: #fff;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
}

```

### Exemplo 2: Valor de Fallback

```css
:root {
  --accent-color: #e74c3c;
}

h1 {
  color: var(--header-color, var(--accent-color)); /* Usa --accent-color se --header-color não estiver definida */
}

```

---

## 5. Considerações Adicionais

- **Escopo e Herança:**
    
    Variáveis definidas no `:root` são globais, mas você pode declarar variáveis em outros seletores para alterar seu valor apenas para aquele contexto específico.
    
- **Cascata:**
    
    As variáveis CSS seguem as regras de cascata do CSS. Se uma variável for redefinida em um escopo mais específico, ela substituirá o valor global dentro desse contexto.
    
- **Compatibilidade:**
    
    Variáveis CSS são suportadas pela maioria dos navegadores modernos. No entanto, para garantir compatibilidade com navegadores antigos, pode ser necessário utilizar métodos alternativos ou polyfills.

## 7. Conclusão

As variáveis CSS oferecem uma maneira poderosa de criar estilos reutilizáveis e consistentes em seu projeto. Ao declarar variáveis no `:root`, você garante que valores importantes, como cores e tamanhos, estejam centralizados e facilmente ajustáveis. O uso da função `var()` torna o código mais limpo e flexível, permitindo personalização dinâmica e facilitando a manutenção do design. Dominar o uso de variáveis CSS é fundamental para desenvolver interfaces modernas e responsivas. Explore os exemplos e referências indicadas para aprofundar seu conhecimento e aplicar essas técnicas em seus projetos.
