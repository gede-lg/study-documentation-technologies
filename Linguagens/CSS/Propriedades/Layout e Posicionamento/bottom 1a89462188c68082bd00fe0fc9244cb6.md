# bottom

## 1. Introdução

A propriedade **bottom** em CSS é utilizada para definir o deslocamento vertical de um elemento em relação à borda inferior do seu contêiner de referência. Assim como outras propriedades de posicionamento, ela é essencial para criar layouts precisos e responsivos, especialmente quando utilizada em conjunto com a propriedade **position**. Desenvolvedores empregam **bottom** para posicionar elementos de forma dinâmica, seja para alinhar conteúdos, criar rodapés fixos ou implementar designs complexos.

### Conceitos Fundamentais

- **Tema Principal (Propriedade bottom):**
    
    Determina a distância entre a borda inferior do elemento e a borda inferior de seu contêiner posicionado.
    
- **Subtemas:**
    - **Posicionamento (position):**
    A propriedade **bottom** só terá efeito se o elemento tiver um valor de **position** diferente de *static* (por exemplo, *relative*, *absolute*, *fixed* ou *sticky*).
    - **Unidades de Medida:**
    Aceita valores em pixels (px), porcentagem (%), em (em) e outras unidades, possibilitando uma ampla flexibilidade para definir o deslocamento vertical.

## 3. Conteúdo Detalhado

### 3.1. Sintaxe e Estrutura

A sintaxe básica para utilizar a propriedade **bottom** é simples e direta:

```css
.seletor {
    bottom: valor;
}

```

**Exemplos:**

- **Valor em pixels:**
    
    ```css
    .caixa {
        position: absolute;
        bottom: 20px;
    }
    
    ```
    
- **Valor em porcentagem:**
    
    ```css
    .caixa {
        position: relative;
        bottom: 10%;
    }
    
    ```
    
- **Valor em em:**
    
    ```css
    .caixa {
        position: fixed;
        bottom: 2em;
    }
    
    ```
    

### 3.2. Componentes Principais

### Relação com a Propriedade **position**

A propriedade **bottom** atua em conjunto com **position**, e seu efeito varia conforme o valor de **position** definido para o elemento:

- **relative:**
    
    O elemento é posicionado de forma relativa à sua posição original. O uso de **bottom** desloca o elemento para cima, se o valor for positivo, sem removê-lo do fluxo normal do documento.
    
- **absolute:**
    
    Remove o elemento do fluxo do documento e o posiciona em relação ao contêiner posicionado mais próximo. Aqui, **bottom** especifica a distância entre a borda inferior do elemento e a borda inferior do contêiner.
    
- **fixed:**
    
    Fixa o elemento em relação à janela de visualização. A propriedade **bottom** garante que o elemento mantenha uma distância constante da parte inferior da tela, independentemente da rolagem.
    
- **sticky:**
    
    Permite que o elemento se comporte de forma relativa até atingir um ponto de rolagem, onde então se torna fixo. **bottom** funciona de maneira semelhante, definindo a distância a partir da parte inferior do contêiner.
    

### Restrições de Uso

- **Elemento com position: static:**
    
    Se o elemento estiver com o valor padrão de **position: static**, a propriedade **bottom** não terá efeito.
    
- **Contexto de Posicionamento:**
    
    O valor de **bottom** é calculado com base no contêiner posicionado mais próximo. Se este não existir, o *body* é utilizado como referência para o posicionamento.
    

## 4. Exemplos de Código Otimizados

### Exemplo Básico: Posicionamento Absoluto

```css
/* Exemplo básico com position: absolute */
.container {
    position: relative; /* Define o contexto de posicionamento */
    width: 300px;
    height: 200px;
    background-color: #f0f0f0;
}

.elemento {
    position: absolute;
    bottom: 20px;  /* Desloca o elemento 20px acima da borda inferior do container */
    left: 30px;    /* Também podemos definir o deslocamento horizontal */
    background-color: #FF5722;
    padding: 10px;
    color: #fff;
}

```

```html
<div class="container">
    <div class="elemento">
        Posicionado 20px acima do fundo
    </div>
</div>

```

### Exemplo Avançado: Rodapé Fixo com Posicionamento Fixed

```css
/* Exemplo avançado utilizando position: fixed para um rodapé */
.rodape {
    position: fixed;
    bottom: 0;       /* Alinha o rodapé à borda inferior da janela */
    left: 0;
    width: 100%;
    background-color: #333;
    color: #fff;
    text-align: center;
    padding: 15px;
    z-index: 1000;   /* Garante que o rodapé fique sobre outros elementos */
}

```

```html
<footer class="rodape">
    Rodapé fixo na parte inferior da tela
</footer>

```

## 5. Informações Adicionais

- **Interação com Outras Propriedades:**
    
    A propriedade **bottom** trabalha em conjunto com **top**, **left** e **right** para definir a posição completa de um elemento. Compreender essa interação é vital para evitar sobreposições indesejadas e garantir que o layout se comporte conforme o esperado.
    
- **Melhores Práticas:**
    - Sempre defina um contêiner com **position: relative** ao utilizar elementos com **position: absolute** para um posicionamento previsível.
    - Utilize unidades relativas (como %, em) para desenvolver layouts responsivos.
    - Verifique a compatibilidade entre navegadores, principalmente ao usar **position: sticky**, pois seu comportamento pode variar.
- **Nuances e Considerações Avançadas:**
    - Em layouts complexos, a ordem de empilhamento (z-index) pode influenciar a visibilidade de elementos posicionados com **bottom**.
    - A propriedade **bottom** pode ser combinada com media queries para ajustar o posicionamento conforme o tamanho da tela, garantindo uma melhor experiência do usuário em dispositivos variados.
