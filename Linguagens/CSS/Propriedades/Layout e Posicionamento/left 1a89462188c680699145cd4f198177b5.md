# left

## 1. Introdução

A propriedade **left** é utilizada em CSS para definir o deslocamento horizontal de um elemento em relação à borda esquerda de seu contêiner de referência. Assim como as demais propriedades de posicionamento (top, right, bottom), **left** torna-se essencial para criar layouts precisos e responsivos. Ela permite que desenvolvedores controlem a posição dos elementos na tela, especialmente quando utilizada em conjunto com a propriedade **position**.

### Conceitos Fundamentais

- **Tema Principal (Propriedade left):**
    
    Especifica a distância entre a borda esquerda do elemento e a borda esquerda do seu contêiner posicionado.
    
- **Subtemas:**
    - **Posicionamento (position):**
    A propriedade **left** só terá efeito se o elemento possuir um valor de **position** diferente de *static* (por exemplo, *relative*, *absolute*, *fixed* ou *sticky*).
    - **Unidades de Medida:**
    Aceita valores em pixels (px), porcentagem (%), em (em) e outras unidades, permitindo flexibilidade na definição do deslocamento horizontal.

## 3. Conteúdo Detalhado

### 3.1. Sintaxe e Estrutura

A sintaxe básica para utilizar a propriedade **left** é simples:

```css
.seletor {
    left: valor;
}

```

**Exemplos:**

- **Valor em pixels:**
    
    ```css
    .caixa {
        position: absolute;
        left: 20px;
    }
    
    ```
    
- **Valor em porcentagem:**
    
    ```css
    .caixa {
        position: relative;
        left: 10%;
    }
    
    ```
    
- **Valor em em:**
    
    ```css
    .caixa {
        position: fixed;
        left: 2em;
    }
    
    ```
    

### 3.2. Componentes Principais

### Relação com a Propriedade **position**

A propriedade **left** atua em conjunto com a propriedade **position**, e seu comportamento varia conforme o valor de **position** definido:

- **relative:**
    
    Posiciona o elemento relativamente à sua posição original. O uso de **left** desloca o elemento para a direita se o valor for positivo, sem removê-lo do fluxo normal do documento.
    
- **absolute:**
    
    Remove o elemento do fluxo normal e o posiciona em relação ao contêiner posicionado mais próximo. Aqui, **left** define a distância entre a borda esquerda do elemento e a borda esquerda do contêiner.
    
- **fixed:**
    
    Fixa o elemento em uma posição relativa à janela de visualização, garantindo que o deslocamento horizontal permaneça constante mesmo com a rolagem.
    
- **sticky:**
    
    Permite que o elemento se comporte de forma relativa até atingir um determinado ponto, onde então se torna fixo. **left** funciona da mesma forma, definindo a distância a partir da esquerda do contêiner.
    

### Restrições de Uso

- **Elemento com position: static:**
    
    Se o elemento estiver com **position: static** (valor padrão), a propriedade **left** não terá efeito.
    
- **Contexto de Posicionamento:**
    
    O valor definido para **left** é calculado em relação ao contêiner posicionado mais próximo. Caso não exista, o *body* é considerado como referência.
    

## 4. Exemplos de Código Otimizados

### Exemplo Básico: Posicionamento Absoluto

```css
/* Exemplo de uso básico com position: absolute */
.container {
    position: relative; /* Define o contexto de posicionamento */
    width: 300px;
    height: 200px;
    background-color: #f0f0f0;
}

.elemento {
    position: absolute;
    left: 20px;  /* Desloca o elemento 20px a partir da borda esquerda do container */
    top: 30px;   /* Pode ser combinado com 'top' para posicionamento vertical */
    background-color: #28A745;
    padding: 10px;
    color: #fff;
}

```

```html
<div class="container">
    <div class="elemento">
        Posicionado 20px da esquerda
    </div>
</div>

```

### Exemplo Avançado: Barra de Navegação Fixa à Esquerda

```css
/* Exemplo avançado utilizando position: fixed para uma barra de navegação */
.sidebar {
    position: fixed;
    left: 0;       /* Alinha a barra à borda esquerda da janela */
    top: 0;
    width: 200px;
    height: 100%;
    background-color: #343A40;
    color: #fff;
    padding: 20px;
    overflow-y: auto;
}

```

```html
<div class="sidebar">
    <h2>Navegação</h2>
    <ul>
        <li><a href="#" style="color: #fff;">Início</a></li>
        <li><a href="#" style="color: #fff;">Sobre</a></li>
        <li><a href="#" style="color: #fff;">Contato</a></li>
    </ul>
</div>

```

## 5. Informações Adicionais

- **Interação com Outras Propriedades:**
    
    A propriedade **left** interage com **right**, **top** e **bottom** para definir a posição completa de um elemento. Conhecer essa interação é crucial para evitar conflitos e sobreposições no layout.
    
- **Melhores Práticas:**
    - Defina um contêiner com **position: relative** ao utilizar elementos com **position: absolute** para garantir que o posicionamento seja previsível.
    - Utilize unidades relativas (como %, em) para criar layouts responsivos que se adaptem a diferentes dispositivos.
    - Teste a renderização em múltiplos navegadores para assegurar a consistência do comportamento.
- **Nuances e Considerações Avançadas:**
    - Em layouts complexos, o gerenciamento do empilhamento de elementos (utilizando z-index) pode ser essencial para a visibilidade e interatividade dos elementos posicionados com **left**.
    - A utilização de media queries pode ser importante para ajustar os valores de **left** em diferentes resoluções e garantir uma experiência de usuário otimizada.
