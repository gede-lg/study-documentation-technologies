# border-spacing

## 1. Introdução

A propriedade **border-spacing** é utilizada para definir o espaço entre as bordas das células em uma tabela quando a propriedade **border-collapse** está definida como `separate`. Essa propriedade permite ajustar o espaçamento horizontal e vertical entre as células, contribuindo para uma aparência mais organizada e visualmente agradável dos dados apresentados.

### Conceitos Fundamentais

- **Tema Principal (Propriedade border-spacing):**
    
    Define o espaço entre as células de uma tabela, influenciando o espaçamento entre as bordas quando estas não estão colapsadas.
    
- **Requisito:**
    
    Funciona somente quando **border-collapse** está definido como `separate`.

## 3. Conteúdo Detalhado

### 3.1. Sintaxe e Estrutura

A sintaxe básica para a propriedade **border-spacing** é:

```css
table {
    border-spacing: valor;
}

```

Você pode especificar um único valor para aplicar o mesmo espaçamento horizontal e vertical, ou dois valores para definir separadamente:

```css
/* Mesmo espaçamento para ambos os eixos */
table {
    border-spacing: 10px;
}

/* 10px para horizontal e 15px para vertical */
table {
    border-spacing: 10px 15px;
}

```

### 3.2. Componentes Principais

- **Valor Único:**
    
    Define um mesmo espaço tanto para o espaçamento horizontal quanto para o vertical.
    
- **Valor Duplo:**
    
    O primeiro valor define o espaçamento horizontal, e o segundo valor define o espaçamento vertical.
    
- **Efeito no Layout:**
    
    A propriedade **border-spacing** afeta o espaço entre as bordas das células da tabela, criando um visual mais espaçado e, em alguns casos, facilitando a leitura dos dados.
    

## 4. Exemplos de Código Otimizados

### Exemplo Básico: Mesmos Espaçamentos Horizontal e Vertical

```css
table.exemplo-basico {
    border-collapse: separate;  /* Necessário para que border-spacing tenha efeito */
    border-spacing: 10px;
    width: 100%;
}

table.exemplo-basico th,
table.exemplo-basico td {
    border: 1px solid #333;
    padding: 8px;
}

```

```html

  <tbody>

    <tr>
      <td>Bruno</td>
      <td>34</td>
      <td>Rio de Janeiro</td>
    </tr>
  </tbody>
</table>

```

```css
table.exemplo-avancado {

table.exemplo-avancado th,
table.exemplo-avancado td {

```

## 5. Informações Adicionais

- **Uso Conjunto:**
Lembre-se de que **border-spacing** só terá efeito se **border-collapse** estiver configurado como `separate`.
- **Consistência Visual:**
Ajustar o espaçamento entre células pode melhorar a clareza e a estética das tabelas, mas mantenha consistência no design para uma experiência visual harmoniosa.
- **Responsividade:**
Considere a responsividade ao definir valores de espaçamento para que as tabelas se adaptem bem em diferentes dispositivos.
