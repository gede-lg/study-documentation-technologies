# empty-cells

## 1. Introdução

A propriedade **empty-cells** é utilizada em tabelas para controlar a exibição de bordas e fundos em células que não possuem conteúdo. Em outras palavras, ela determina se as células vazias (sem conteúdo, como texto ou elementos) devem ser exibidas com suas bordas e/ou fundos ou se devem ser ocultadas visualmente. Essa propriedade pode ser útil para ajustar o design de tabelas e melhorar a aparência visual quando se deseja evitar a exibição de espaços desnecessários.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **empty-cells** controla se as células vazias de uma tabela devem mostrar ou ocultar bordas e fundos. Ela afeta apenas células que não contêm nenhum conteúdo (texto, imagem, ou outros elementos).
    
- **Objetivo e Importância:**
    - **Estética da Tabela:** Permite refinar a aparência de tabelas, evitando que células vazias exibam bordas ou fundos que possam desviar a atenção ou causar uma aparência desorganizada.
    - **Design Consistente:** Em layouts onde algumas células podem estar vazias, essa propriedade ajuda a manter a consistência visual da tabela.
    - **Legibilidade:** A ocultação de células vazias pode contribuir para uma melhor leitura, especialmente em tabelas com muitos dados onde espaços vazios podem causar confusão visual.
- **Contexto de Uso:**
    
    Essa propriedade é aplicada exclusivamente a elementos de tabela, ou seja, células de tabelas (`<td>` e `<th>`) que não contêm conteúdo.
    

---

## 3. Sintaxe e Valores

A sintaxe básica para **empty-cells** é:

```css
seletor {
  empty-cells: valor;
}

```

### Valores Comuns:

- **show:**
    
    Exibe as bordas e o fundo das células, mesmo que estejam vazias.
    
    ```css
    td, th {
      empty-cells: show;
    }
    
    ```
    
- **hide:**
    
    Oculta as bordas e o fundo das células que estão vazias.
    
    ```css
    td, th {
      empty-cells: hide;
    }
    
    ```
    

*Observação:*

O valor padrão (initial) depende do navegador, mas geralmente é `show`.

---

## 4. Exemplos Práticos

### Exemplo 1: Exibindo Bordas em Células Vazias

```css
table {
  width: 100%;
  border-collapse: collapse;
}

td, th {
  border: 1px solid #333;
  padding: 8px;
  empty-cells: show;
}

```

```html
<table>
  <tr>
    <th>Nome</th>
    <th>Email</th>
  </tr>
  <tr>
    <td>João</td>
    <td>joao@example.com</td>
  </tr>
  <tr>
    <td>Maria</td>
    <td></td>
  </tr>
</table>

```

*Explicação:*

Mesmo a célula vazia na linha de Maria exibirá a borda, garantindo que a estrutura da tabela seja visualmente consistente.

---

### Exemplo 2: Ocultando Bordas em Células Vazias

```css
table {
  width: 100%;
  border-collapse: collapse;
}

td, th {
  border: 1px solid #333;
  padding: 8px;
  empty-cells: hide;
}

```

```html
<table>
  <tr>
    <th>Produto</th>
    <th>Preço</th>
  </tr>
  <tr>
    <td>Caneta</td>
    <td>$1.50</td>
  </tr>
  <tr>
    <td>Lápis</td>
    <td></td>
  </tr>
</table>

```

*Explicação:*

Neste exemplo, a célula vazia na linha do "Lápis" não exibirá a borda, resultando em uma aparência mais limpa quando o conteúdo não estiver disponível.

---

## 5. Informações Adicionais

- **Aplicação Exclusiva:**
    
    A propriedade **empty-cells** afeta apenas células de tabela (`<td>` e `<th>`) e não influencia outros elementos.
    
- **Combinação com `border-collapse`:**
    
    O comportamento de **empty-cells** pode variar ligeiramente dependendo do valor de `border-collapse` aplicado à tabela. Em geral, quando `border-collapse: collapse;` é utilizado, o efeito de **empty-cells** é mais evidente.
    
- **Compatibilidade:**
    
    A maioria dos navegadores modernos suporta **empty-cells**, mas seu comportamento pode ser diferente em versões antigas. É aconselhável testar em múltiplos navegadores para garantir que o resultado seja consistente.
    
- **Uso Prático:**
    - **Tabelas de Dados:** Em relatórios e tabelas de dados onde células vazias podem aparecer, o ajuste da propriedade pode ajudar a evitar a exibição de linhas desnecessárias.
    - **Design Minimalista:** Em designs onde se deseja uma aparência limpa e sem distrações, ocultar células vazias pode melhorar a estética geral da tabela.

## 7. Conclusão

A propriedade **empty-cells** é uma ferramenta simples, mas poderosa, para controlar a exibição de bordas e fundos em células de tabelas que estão vazias. Ao definir **empty-cells: show** ou **empty-cells: hide**, os desenvolvedores podem ajustar a aparência das tabelas para melhor atender aos requisitos do design e da legibilidade. Essa propriedade é especialmente útil em contextos de dados, onde a consistência visual é fundamental, e em layouts minimalistas, onde células vazias podem ser indesejadas. Dominar **empty-cells** é essencial para criar tabelas elegantes e funcionais.
