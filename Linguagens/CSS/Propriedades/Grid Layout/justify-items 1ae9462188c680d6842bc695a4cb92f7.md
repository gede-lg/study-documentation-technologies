# justify-items

## 1. Introdução

A propriedade **justify-items** é parte do CSS Grid Layout (e também pode ser aplicada em contêineres Flex quando se utiliza a propriedade `gap`). Ela define como os itens individuais dentro de um contêiner grid (ou, em alguns casos, flex) são alinhados ao longo do eixo inline dentro de cada célula. Em termos simples, **justify-items** controla o alinhamento horizontal dos itens dentro de suas respectivas células, facilitando a criação de layouts precisos e visualmente equilibrados.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **justify-items** especifica como os itens de um contêiner grid são alinhados no eixo inline (horizontal, em layouts LTR) dentro de suas células individuais.
    
- **Importância:**
    - **Controle Preciso:** Permite alinhar os itens de forma uniforme ou de acordo com as necessidades específicas do design.
    - **Harmonia Visual:** Contribui para a organização e equilíbrio visual dos elementos, melhorando a legibilidade e a estética geral.
    - **Responsividade:** Facilita a adaptação do alinhamento dos itens em diferentes dispositivos sem a necessidade de margens individuais.

---

## 3. Sintaxe e Valores

A sintaxe básica é:

```css
seletor {
    justify-items: valor;
}

```

### Valores Comuns:

- **start / flex-start:**
    
    Alinha os itens ao início da célula (à esquerda em layouts LTR).
    
    ```css
    justify-items: start;
    
    ```
    
- **end / flex-end:**
    
    Alinha os itens ao final da célula (à direita em layouts LTR).
    
    ```css
    justify-items: end;
    
    ```
    
- **center:**
    
    Centraliza os itens horizontalmente dentro da célula.
    
    ```css
    justify-items: center;
    
    ```
    
- **stretch (valor padrão):**
    
    Estica os itens para preencher a célula completamente, se não houver largura definida.
    
    ```css
    justify-items: stretch;
    
    ```

## 4. Exemplos de Código Otimizados

### Exemplo 1: Alinhamento Centralizado

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
    border: 1px solid #ccc;
    padding: 10px;
    justify-items: center; /* Centraliza os itens horizontalmente */
}

.grid-item {
    background-color: #3498db;
    color: #fff;
    padding: 20px;
    width: 80%;  /* Define largura para demonstrar o alinhamento */
    text-align: center;
}

```

```html
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>

```

*Explicação:*

Os itens dentro do contêiner grid são centralizados horizontalmente em suas células.

### Exemplo 2: Alinhamento no Início (start) e Final (end)

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    grid-gap: 15px;
    border: 1px solid #ccc;
    padding: 10px;
}

/* Primeiro item alinhado à esquerda (start) */
.item-start {
    justify-self: start;
    background-color: #2ecc71;
    color: #fff;
    padding: 20px;
}

/* Segundo item alinhado à direita (end) */
.item-end {
    justify-self: end;
    background-color: #e74c3c;
    color: #fff;
    padding: 20px;
}

```

```html
<div class="grid-container">
  <div class="item-start">Alinhado à esquerda</div>
  <div class="item-end">Alinhado à direita</div>
</div>

```

*Explicação:*

Aqui, usamos **justify-self** para sobrescrever o valor global de **justify-items** e alinhar individualmente cada item ao início ou final da célula.

---

## 5. Informações Adicionais

- **Distinção entre justify-items e justify-self:**
    - **justify-items** define o alinhamento padrão para todos os itens dentro do contêiner.
    - **justify-self** permite o alinhamento individual de um item, caso seja necessário sobrescrever o valor herdado do contêiner.
- **Eixo Inline:**
    
    A propriedade alinha os itens ao longo do eixo inline. Em layouts LTR, isso significa alinhamento horizontal; em layouts RTL, o comportamento é invertido automaticamente.
    
- **Integração com Grid Layout:**
    
    **justify-items** pode ser combinada com **align-items** (para o eixo block) e outras propriedades de grid, como **grid-template-columns** e **grid-template-rows**, para criar layouts bidimensionais completos e responsivos.
    
- **Compatibilidade:**
    
    **justify-items** é amplamente suportada em navegadores modernos que implementam CSS Grid. Para Flexbox, a propriedade equivalente para alinhamento individual é **justify-self**, e o alinhamento global pode ser controlado com **justify-content**.

## 7. Conclusão

A propriedade **justify-items** é crucial para o controle do alinhamento horizontal dos itens dentro de um contêiner CSS Grid, permitindo uma distribuição organizada e harmoniosa do conteúdo. Ao utilizar valores como `center`, `start`, `end` ou `stretch`, você pode ajustar a apresentação dos itens de acordo com as necessidades do design, contribuindo para interfaces responsivas e visualmente agradáveis. Dominar **justify-items** — juntamente com propriedades complementares como **justify-self** e **justify-content** — é essencial para criar layouts modernos e flexíveis.
