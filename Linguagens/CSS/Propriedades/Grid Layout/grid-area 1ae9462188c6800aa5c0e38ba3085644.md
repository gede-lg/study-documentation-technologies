# grid-area

## 1. Introdução

A propriedade **grid-area** é uma ferramenta versátil no CSS Grid Layout que permite posicionar e dimensionar um item dentro de um contêiner grid de duas maneiras principais:

1. **Abreviação para Posicionamento Bidimensional:**
Ela combina as propriedades **grid-row-start**, **grid-column-start**, **grid-row-end** e **grid-column-end** em uma única declaração para posicionar um item em um local específico do grid.
2. **Nomeação de Áreas:**
Quando usada em conjunto com a propriedade **grid-template-areas** no contêiner, **grid-area** permite atribuir um nome semântico a um item, que será posicionado automaticamente na área correspondente definida na grade.

---

## 2. Conceitos Fundamentais

- **Posicionamento Explícito:**
    
    Usando **grid-area** como atalho para definir onde o item inicia e termina no grid, você especifica as linhas e colunas que o item ocupará.
    
    Exemplo: `grid-area: 2 / 3 / 4 / 5;` significa que o item inicia na linha 2 e coluna 3, e termina antes da linha 4 e coluna 5.
    
- **Nomeação de Áreas:**
    
    Quando o contêiner utiliza **grid-template-areas**, os itens podem ser posicionados de acordo com os nomes das áreas.
    
    Exemplo: `grid-area: header;` posiciona o item na área nomeada "header".
    

---

## 3. Sintaxe e Estrutura

### 3.1. Abreviação para Posicionamento

A sintaxe geral para definir o posicionamento bidimensional de um item é:

```css
seletor {
    grid-area: <linha-de-início> / <coluna-de-início> / <linha-de-término> / <coluna-de-término>;
}

```

- **Exemplo:**
    
    ```css
    .item {
        grid-area: 2 / 3 / 4 / 5;
    }
    
    ```
    
    *Este item começará na linha 2 e coluna 3, e se estenderá até (mas não incluindo) a linha 4 e coluna 5.*
    

### 3.2. Nomeação de Áreas

Quando o contêiner define áreas nomeadas com **grid-template-areas**, você pode simplesmente usar:

```css
seletor {
    grid-area: nome-da-area;
}

```

- **Exemplo:**
    
    ```css
    .header {
        grid-area: header;
    }
    
    ```
    
    *Isso posiciona o elemento na área nomeada "header" definida no grid container.*

## 4. Exemplos de Código Otimizados

### Exemplo 1: Posicionamento Explícito com Valores Numéricos

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: 80px auto 60px;
    gap: 10px;
    border: 1px solid #ccc;
    padding: 10px;
}

.item-posicionado {
    background-color: #3498db;
    color: #fff;
    padding: 15px;
    text-align: center;
    /* O item ocupa da linha 2 até a linha 3 e da coluna 2 até a 4 */
    grid-area: 2 / 2 / 3 / 5;
}

```

```html
<div class="grid-container">
  <div class="item-posicionado">Item Posicionado</div>
  <!-- Outros itens serão posicionados automaticamente -->
  <div>Item A</div>
  <div>Item B</div>
  <div>Item C</div>
</div>

```

*Explicação:*

O **item-posicionado** inicia na segunda linha e segunda coluna, e se estende até (mas não inclui) a terceira linha e quinta coluna.

### Exemplo 2: Nomeação de Áreas com grid-template-areas

```css
.grid-container {
    display: grid;
    grid-template-areas:
      "header header header"
      "sidebar main main"
      "footer footer footer";
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: 80px auto 60px;
    gap: 10px;
    padding: 10px;
}

.header {
    grid-area: header;
    background-color: #e74c3c;
    color: #fff;
    text-align: center;
}

.sidebar {
    grid-area: sidebar;
    background-color: #3498db;
    color: #fff;
}

.main {
    grid-area: main;
    background-color: #2ecc71;
    color: #fff;
}

.footer {
    grid-area: footer;
    background-color: #9b59b6;
    color: #fff;
    text-align: center;
}

```

```html
<div class="grid-container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

*Explicação:*

As áreas "header", "sidebar", "main" e "footer" são definidas no contêiner usando **grid-template-areas**. Cada item recebe um nome correspondente com **grid-area**, posicionando-o automaticamente na área definida.

## 5. Informações Adicionais

- **Flexibilidade:**
    
    A propriedade **grid-area** permite posicionamento preciso sem a necessidade de alterar o HTML, o que facilita a manutenção e o reordenamento do layout.
    
- **Combinação com Outras Propriedades:**
    
    Pode ser usada em conjunto com **grid-column** e **grid-row** quando um posicionamento ainda mais detalhado é necessário. No entanto, quando **grid-area** é usada com nomes de áreas definidas no contêiner, ela simplifica e torna o código mais semântico.
    
- **Responsividade:**
    
    Ao usar **grid-area** com **grid-template-areas**, o layout pode ser facilmente reorganizado para se adaptar a diferentes tamanhos de tela, mantendo uma estrutura clara e organizada.

## 7. Conclusão

A propriedade **grid-area** é uma ferramenta poderosa e versátil no CSS Grid, permitindo posicionar itens de forma precisa, seja através de coordenadas numéricas ou nomeando áreas específicas do layout.
