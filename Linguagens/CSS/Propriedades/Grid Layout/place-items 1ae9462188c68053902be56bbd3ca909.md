# place-items

## 1. Introdução

A propriedade **place-items** é uma forma abreviada de definir o alinhamento de itens dentro de um contêiner CSS Grid (e, em alguns casos, em contêineres Flex modernos). Ela combina as propriedades **align-items** e **justify-items** em uma única declaração, permitindo que você controle o alinhamento dos itens tanto no eixo block (vertical, em layouts LTR) quanto no eixo inline (horizontal) de maneira concisa e eficiente.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **place-items** ajusta o alinhamento dos itens dentro de cada célula do grid, definindo como eles serão distribuídos ao longo dos eixos transversal e principal.
    
    - **Align-items:** Alinha os itens no eixo block (vertical em layouts LTR).
    - **Justify-items:** Alinha os itens no eixo inline (horizontal em layouts LTR).
- **Benefícios:**
    - Simplifica o código ao combinar duas propriedades em uma única linha.
    - Facilita a criação de layouts consistentes e responsivos sem a necessidade de configurar separadamente o alinhamento vertical e horizontal de cada item.

---

## 3. Sintaxe e Valores

A sintaxe básica é:

```css
seletor {
    place-items: <valor>;
}

```

Ou, para definir valores diferentes para cada eixo:

```css
seletor {
    place-items: <align-items> <justify-items>;
}

```

### Exemplos de Valores:

- **Valor Único:**
    
    ```css
    .grid-container {
        place-items: center;
    }
    
    ```
    
    *Equivalente a:*
    
    ```css
    align-items: center;
    justify-items: center;
    
    ```
    
- **Valores Diferenciados:**
    
    ```css
    .grid-container {
        place-items: flex-start end;
    }
    
    ```
    
    *Isso define `align-items: flex-start;` e `justify-items: end;`, alinhando os itens ao topo e à direita, respectivamente.*
    
- **Valores Comuns:**
    - `start` ou `flex-start`
    - `end` ou `flex-end`
    - `center`
    - `stretch` (valor padrão, onde os itens se esticam para preencher a célula)

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Alinhamento Centralizado com Valor Único

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
    gap: 10px;
    place-items: center; /* Centraliza os itens tanto horizontal quanto verticalmente */
    padding: 10px;
    background-color: #f5f5f5;
}

.grid-item {
    background-color: #3498db;
    color: #fff;
    padding: 20px;
    text-align: center;
}

```

```html
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>

```

*Explicação:*

Todos os itens dentro do contêiner grid são centralizados tanto no eixo vertical quanto no horizontal.

### Exemplo 2: Alinhamento Diferenciado com Valores Específicos

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    grid-template-rows: 150px 150px;
    gap: 15px;
    /* Alinha verticalmente ao início e horizontalmente ao final */
    place-items: flex-start flex-end;
    padding: 10px;
    background-color: #ecf0f1;
}

.grid-item {
    background-color: #e74c3c;
    color: #fff;
    padding: 20px;
    text-align: center;
}

```

```html
<div class="grid-container">
  <div class="grid-item">A</div>
  <div class="grid-item">B</div>
  <div class="grid-item">C</div>
  <div class="grid-item">D</div>
</div>

```

*Explicação:*

Os itens são alinhados ao início (topo) da célula no eixo block e ao final (à direita) no eixo inline, criando uma disposição assimétrica.

---

## 5. Informações Adicionais

- **Compatibilidade:**
    
    A propriedade **place-items** é amplamente suportada em navegadores modernos que implementam o CSS Grid Layout. Para Flexbox, a propriedade equivalente para alinhar itens individualmente é **justify-self** (para o eixo inline) e **align-self** (para o eixo block).
    
- **Responsividade:**
    
    Usar **place-items** ajuda a manter um alinhamento consistente dos itens dentro do grid, independentemente do tamanho da tela, facilitando a criação de layouts responsivos.
    
- **Simplicidade no Código:**
    
    A principal vantagem de **place-items** é a sua capacidade de substituir duas declarações (align-items e justify-items) por uma única linha, simplificando o CSS e melhorando a legibilidade.

A propriedade **place-items** é uma ferramenta poderosa e concisa para alinhar os itens dentro de um contêiner de grid, simplificando o controle sobre o alinhamento no eixo transversal e inline. Dominar **place-items** é fundamental para desenvolvedores que buscam eficiência e flexibilidade em seus projetos de CSS.
