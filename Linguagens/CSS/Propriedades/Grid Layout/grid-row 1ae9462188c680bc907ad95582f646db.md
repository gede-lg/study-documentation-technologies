# grid-row

## 1. Introdução

A propriedade **grid-row** é uma abreviação que define o posicionamento vertical de um item em um contêiner CSS Grid, combinando as propriedades **grid-row-start** e **grid-row-end**. Com ela, você pode especificar em qual linha o item deve começar e em qual linha ele deve terminar, facilitando a organização e o alinhamento dos itens dentro do grid. Essa propriedade é essencial para criar layouts bidimensionais precisos e responsivos.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **grid-row** é uma propriedade abreviada que permite definir, em uma única declaração, a posição inicial e final de um item no eixo vertical do grid. Ela é equivalente a especificar individualmente as propriedades **grid-row-start** e **grid-row-end**.
    
- **Importância:**
    - **Organização do Layout:** Permite controlar a extensão vertical dos itens dentro do grid.
    - **Flexibilidade:** Facilita o posicionamento de itens em áreas que podem abranger várias linhas.
    - **Responsividade:** Auxilia na criação de layouts complexos que se adaptam a diferentes tamanhos de tela.

---

## 3. Sintaxe e Estrutura

A sintaxe básica da propriedade **grid-row** é:

```css
seletor {
    grid-row: <grid-row-start> / <grid-row-end>;
}

```

- **:** Define a linha onde o item começa. Pode ser um número, um nome de linha ou a palavra-chave `span` seguida de um número.
- **:** Define a linha onde o item termina. Também pode ser um número, um nome de linha ou `span` seguido de um número.

### Exemplos:

- **Posicionamento Básico:**
    
    ```css
    .item {
        grid-row: 2 / 4;
    }
    
    ```
    
    *Este exemplo posiciona o item para iniciar na linha 2 e terminar antes da linha 4, ocupando, assim, duas linhas do grid.*
    
- **Utilizando `span`:**
    
    ```css
    .item {
        grid-row: 1 / span 3;
    }
    
    ```
    
    *O item começará na linha 1 e ocupará três linhas (span 3).*
    
- **Com Nomes de Linhas:**
    
    Supondo que você tenha definido linhas nomeadas no grid:
    
    ```css
    .item {
        grid-row: header-start / header-end;
    }
    
    ```

## 4. Exemplos de Código Otimizados

### Exemplo 1: Layout Simples com Posicionamento Vertical

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 100px 200px 100px;
    gap: 10px;
    border: 1px solid #ccc;
    padding: 10px;
}

.item {
    background-color: #3498db;
    color: #fff;
    padding: 10px;
    text-align: center;
}

/* O item ocupa da linha 2 até a linha 4 (ou seja, linhas 2 e 3) */
.item-extend {
    grid-row: 2 / 4;
}

```

```html
<div class="grid-container">
  <div class="item">Item 1</div>
  <div class="item item-extend">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>

```

*Explicação:*

No exemplo acima, o **Item 2** foi configurado para iniciar na linha 2 e terminar na linha 4, fazendo com que ocupe duas linhas do grid.

### Exemplo 2: Utilizando `span` para Ocupação de Múltiplas Linhas

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: auto;
    gap: 15px;
    padding: 15px;
}

.item {
    background-color: #e74c3c;
    color: #fff;
    padding: 20px;
    text-align: center;
}

/* O item ocupará 3 linhas a partir da linha onde inicia */
.item-span {
    grid-row: 1 / span 3;
}

```

```html
<div class="grid-container">
  <div class="item item-span">Item Span</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
</div>

```

*Explicação:*

O **Item Span** começa na primeira linha e, com `span 3`, ocupa três linhas consecutivas.

## 5. Informações Adicionais

- **Valores Automáticos:**
    
    Se não for especificado, os itens são posicionados automaticamente pelo navegador. Usar **grid-row** permite um controle explícito sobre onde cada item deve aparecer.
    
- **Interação com Outras Propriedades de Grid:**
    
    **grid-row** funciona em conjunto com **grid-column** (ou suas abreviações) para posicionar itens de forma bidimensional. Também pode ser usado com **grid-area** se áreas nomeadas forem definidas.
    
- **Layout Responsivo:**
    
    Ao combinar **grid-row** com media queries, é possível alterar dinamicamente o posicionamento vertical dos itens para se adaptar a diferentes tamanhos de tela.

A propriedade **grid-row** é uma ferramenta essencial para controlar a disposição vertical dos itens dentro de um contêiner de grid.
