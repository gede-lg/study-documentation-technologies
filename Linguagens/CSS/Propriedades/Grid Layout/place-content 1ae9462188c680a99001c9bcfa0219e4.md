# place-content

## 1. Introdução

A propriedade **place-content** é uma ferramenta poderosa no CSS Grid Layout (e, em alguns contextos, também aplicável em Flexbox) que permite alinhar e distribuir o espaço extra entre as faixas (linhas e colunas) dentro de um contêiner. Ela funciona como uma abreviação que combina duas propriedades essenciais: **align-content** (que alinha as faixas no eixo transversal) e **justify-content** (que alinha as faixas no eixo principal). Com **place-content**, você pode definir de forma concisa como as linhas e colunas se distribuem, tornando o código mais limpo e fácil de manter.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **place-content** é uma propriedade abreviada que controla a distribuição de espaço entre as faixas em um contêiner de grid, combinando os efeitos de **justify-content** e **align-content**.
    
- **Importância:**
    - **Distribuição de Espaço:** Define como o espaço extra é distribuído entre as linhas e colunas do grid.
    - **Alinhamento Global:** Permite alinhar as faixas em relação ao contêiner como um todo, centralizando, espalhando ou agrupando-as conforme necessário.
    - **Sintaxe Concisa:** Simplifica o código, substituindo duas declarações separadas (justify-content e align-content) por uma única linha.
- **Aplicabilidade:**
    
    Embora seja mais comum em layouts de grid, **place-content** pode ser utilizada em contextos onde a distribuição de faixas é necessária, melhorando a responsividade e a organização visual do layout.
    

---

## 3. Sintaxe e Valores

### 3.1. Sintaxe Básica

A sintaxe da propriedade **place-content** é simples:

```css
seletor {
    place-content: <align-content> <justify-content>;
}

```

- Se for utilizado um único valor, ele é aplicado tanto para **align-content** quanto para **justify-content**.

**Exemplos:**

- **Valor Único:**
    
    ```css
    .grid-container {
        place-content: center;
    }
    
    ```
    
    *Equivalente a:*
    
    ```css
    align-content: center;
    justify-content: center;
    
    ```
    
- **Valores Diferenciados:**
    
    ```css
    .grid-container {
        place-content: space-between center;
    }
    
    ```
    
    *Isso define `align-content: space-between;` e `justify-content: center;`.*
    

### 3.2. Valores Comuns

Os valores que podem ser usados em **place-content** são os mesmos aceitos por **align-content** e **justify-content**:

- **flex-start / start:** Alinha as faixas ao início do contêiner.
- **flex-end / end:** Alinha as faixas ao final do contêiner.
- **center:** Centraliza as faixas.
- **space-between:** Distribui o espaço extra igualmente entre as faixas, colocando o primeiro e o último nas extremidades.
- **space-around:** Distribui espaços iguais ao redor de cada faixa (o espaço nas extremidades será metade do valor entre as faixas).
- **space-evenly:** Garante que todo o espaço entre as faixas e as bordas do contêiner seja igual.
- **stretch (valor padrão):** Estica as faixas para preencher o contêiner, se possível.

## 4. Exemplos de Código Otimizados

### Exemplo 1: Alinhamento Central Uniforme

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 100px auto 100px;
    gap: 10px;
    place-content: center; /* Centraliza as faixas tanto verticalmente quanto horizontalmente */
    padding: 10px;
    background-color: #f5f5f5;
}

.grid-item {
    background-color: #3498db;
    color: #fff;
    padding: 20px;
    text-align: center;
}

```

```html
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>

```

*Explicação:*

Todas as faixas (linhas e colunas) são centralizadas dentro do contêiner.

### Exemplo 2: Distribuição Diferenciada com Espaços

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: 80px 120px;
    gap: 15px;
    /* Alinha verticalmente as faixas com space-between e horizontalmente com space-evenly */
    place-content: space-between space-evenly;
    padding: 10px;
    background-color: #ecf0f1;
}

.grid-item {
    background-color: #e74c3c;
    color: #fff;
    padding: 15px;
    text-align: center;
}

```

```html
<div class="grid-container">
  <div class="grid-item">A</div>
  <div class="grid-item">B</div>
  <div class="grid-item">C</div>
  <div class="grid-item">D</div>
  <div class="grid-item">E</div>
  <div class="grid-item">F</div>
  <div class="grid-item">G</div>
  <div class="grid-item">H</div>
</div>

```

*Explicação:*

Neste exemplo, o espaço extra entre as linhas é distribuído com `space-between` (primeira linha no topo, última no fundo, espaços iguais entre as linhas), e o espaço extra entre as colunas é distribuído uniformemente com `space-evenly`.

## 5. Informações Adicionais

- **Combinação com Outras Propriedades:**
    
    **place-content** é uma maneira abreviada de definir tanto **align-content** quanto **justify-content** em um contêiner grid.
    
    - **align-content** controla o alinhamento das faixas no eixo block (vertical em um layout LTR).
    - **justify-content** controla o alinhamento das faixas no eixo inline (horizontal em um layout LTR).
- **Contexto de Uso:**
    
    Essa propriedade só tem efeito se houver múltiplas faixas (linhas ou colunas) no contêiner. Se houver apenas uma faixa, **place-content** não terá impacto.
    
- **Responsividade:**
    
    **place-content** pode ser facilmente ajustada via media queries para adaptar a distribuição do espaço entre as faixas conforme o tamanho do contêiner muda, facilitando a criação de layouts responsivos.
