# align-content

## 1. Introdução

A propriedade **align-content** é utilizada no CSS Grid e Flexbox para controlar a distribuição do espaço extra entre as linhas (ou faixas) do contêiner, ao longo do eixo transversal. Enquanto **align-items** alinha os itens individualmente dentro de cada linha, **align-content** atua no agrupamento e espaçamento das próprias linhas quando há múltiplas linhas presentes.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **align-content** determina como o espaço extra entre as linhas (ou faixas) é distribuído dentro do contêiner. Esse espaço é aquele que sobra após os itens terem sido posicionados e dimensionados.
    
- **Importância:**
    - **Organização Vertical:** Em um contêiner flexível com múltiplas linhas (ou em um grid), ela define se as linhas serão empurradas para as extremidades, centralizadas ou distribuídas uniformemente.
    - **Responsividade:** Permite adaptar a distribuição de espaço entre linhas conforme o tamanho do contêiner muda.
    - **Complementaridade:** Funciona em conjunto com **justify-content** (para o eixo principal) e **align-items** (para os itens individuais dentro de cada linha).

---

## 3. Sintaxe e Valores

A sintaxe básica para **align-content** é:

```css
seletor {
  align-content: valor;
}

```

### Valores Comuns:

- **stretch (valor padrão):**
    
    As linhas se esticam para ocupar todo o espaço extra disponível no contêiner.
    
- **flex-start:**
    
    Agrupa as linhas no início do contêiner.
    
- **flex-end:**
    
    Agrupa as linhas no final do contêiner.
    
- **center:**
    
    Centraliza as linhas dentro do contêiner.
    
- **space-between:**
    
    Distribui as linhas de modo que o primeiro e o último fiquem encostados nas bordas do contêiner, com espaços iguais entre as demais.
    
- **space-around:**
    
    Distribui as linhas com espaço igual ao redor de cada linha, resultando em metade do espaço nas extremidades.
    
- **space-evenly:**
    
    Garante que todo o espaço entre as linhas e as bordas do contêiner seja igual.

## 4. Exemplos de Código Otimizados

### Exemplo 1: Distribuição Uniforme com `stretch` (Padrão)

```css
.container {
  display: flex;
  flex-wrap: wrap;         /* Necessário para múltiplas linhas */
  height: 400px;
  border: 1px solid #ccc;
  align-content: stretch;   /* Linhas se esticam para preencher o contêiner */
  padding: 10px;
}

.item {
  flex: 1 1 150px;
  background-color: #3498db;
  color: #fff;
  margin: 5px;
  padding: 10px;
  text-align: center;
}

```

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>

```

*Explicação:*

Com `align-content: stretch;`, as linhas se esticam para preencher verticalmente o espaço extra do contêiner.

### Exemplo 2: Distribuição com `space-between`

```css
.container {
  display: flex;
  flex-wrap: wrap;
  height: 400px;
  border: 1px solid #ccc;
  align-content: space-between; /* Espaço igual entre as linhas; extremidades encostam nas bordas */
  padding: 10px;
}

.item {
  flex: 1 1 150px;
  background-color: #2ecc71;
  color: #fff;
  margin: 5px;
  padding: 10px;
  text-align: center;
}

```

```html
<div class="container">
  <div class="item">A</div>
  <div class="item">B</div>
  <div class="item">C</div>
  <div class="item">D</div>
  <div class="item">E</div>
  <div class="item">F</div>
</div>

```

*Explicação:*

Com `space-between`, o espaço extra é distribuído igualmente entre as linhas, com a primeira linha alinhada ao topo e a última ao fundo.

### Exemplo 3: Centralizando Linhas com `center`

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 10px;
  height: 400px;
  border: 1px solid #ccc;
  align-content: center; /* Centraliza todas as linhas verticalmente */
  padding: 10px;
}

.item {
  background-color: #e74c3c;
  color: #fff;
  padding: 20px;
  text-align: center;
}

```

```html
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>

```

*Explicação:*

No grid, `align-content: center;` centraliza as linhas dentro do contêiner, proporcionando um visual equilibrado.

---

## 5. Informações Adicionais

- **Aplicabilidade:**
    
    **align-content** tem efeito apenas quando há múltiplas linhas ou faixas no contêiner. Se houver apenas uma linha, essa propriedade não altera a disposição dos itens.
    
- **Integração com Outras Propriedades:**
    
    É frequentemente combinada com **justify-content** (para o eixo principal) e **align-items** (para os itens dentro de cada linha) para fornecer um controle completo do layout.
    
- **Responsividade:**
    
    Em layouts responsivos, ajustar **align-content** permite que o espaçamento entre linhas se adapte de forma fluida, mantendo uma estrutura visual consistente em diferentes dispositivos.

## 7. Conclusão

A propriedade **align-content** é essencial para definir a distribuição do espaço extra entre as linhas de um contêiner Flexbox ou Grid, permitindo um alinhamento vertical (ou horizontal, dependendo do contexto) que melhora a organização e a estética do layout. Ao utilizar valores como `center`, `space-between`, `stretch` ou outros, você pode controlar de forma precisa como as linhas se posicionam, contribuindo para interfaces modernas e responsivas. Dominar **align-content**, em conjunto com outras propriedades de alinhamento, é crucial para criar layouts coesos e visualmente equilibrados.
