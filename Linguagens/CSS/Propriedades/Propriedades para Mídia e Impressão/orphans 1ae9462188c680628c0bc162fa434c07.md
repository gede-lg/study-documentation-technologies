# orphans

## 1. Introdução

A propriedade **orphans** faz parte das propriedades de paginação do CSS, que são utilizadas para controlar a forma como o conteúdo é dividido em páginas quando impresso ou renderizado em formatos paginados (como PDFs). Essa propriedade é usada para evitar que se crie um "órfão" — isto é, uma linha isolada do início de um parágrafo que fica sozinha no final de uma página.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    A propriedade **orphans** especifica o número mínimo de linhas de um bloco de conteúdo que devem aparecer na parte inferior de uma página antes que uma quebra de página seja forçada. Em outras palavras, ela ajuda a evitar que apenas uma ou poucas linhas sejam deixadas isoladas no final de uma página.
    
- **Objetivo:**
    
    Melhorar a legibilidade e a estética do conteúdo impresso, garantindo que parágrafos não sejam divididos de forma abrupta e que as páginas tenham um fluxo visual mais equilibrado.
    
- **Contexto de Uso:**
    
    **orphans** é relevante principalmente em mídias de impressão e em documentos que exigem paginação. Essa propriedade é utilizada em conjunto com **widows**, que controla o número mínimo de linhas que devem ser mantidas na parte superior de uma nova página.
    

---

## 3. Sintaxe e Valores

A sintaxe básica é:

```css
seletor {
  orphans: número;
}

```

- **número:**
Um valor numérico inteiro que indica o número mínimo de linhas que devem permanecer juntas na parte inferior de uma página.
    - Exemplo: `orphans: 3;` significa que, se possível, pelo menos 3 linhas do conteúdo devem ser impressas na parte inferior da página.

*Valor padrão:* Geralmente, o valor padrão para **orphans** é `2` (embora isso possa variar conforme o agente de impressão).

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Aplicando orphans para um Parágrafo Longo

```css
@media print {
  p {
    orphans: 3;
  }
}

```

```html
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed ut risus nec mauris facilisis fermentum.
  Phasellus tincidunt, quam a faucibus placerat, nisi nunc vehicula metus, at vehicula lacus justo in magna.
  Vivamus porttitor eros eu nulla malesuada, et aliquet magna fermentum.
  Curabitur ut sapien eu risus facilisis varius.
  Nulla facilisi. Duis et ligula ac ligula finibus tincidunt.
</p>

```

*Explicação:*

Ao aplicar `orphans: 3;` dentro de uma media query para impressão, o navegador tentará evitar que menos de 3 linhas do parágrafo fiquem isoladas na parte inferior de uma página.

---

## 5. Informações Adicionais

- **Interação com Widows:**
    
    Geralmente, **orphans** é usada em conjunto com a propriedade **widows**, que especifica o número mínimo de linhas que devem ser mantidas no início de uma nova página. Essa combinação garante que o parágrafo seja dividido de forma adequada tanto na parte inferior quanto na parte superior das páginas.
    
- **Contexto de Impressão:**
    
    Essas propriedades são específicas para a mídia de impressão e não afetam a renderização em tela. Para aplicá-las, é comum usar uma media query `@media print`.
    
- **Considerações de Legibilidade:**
    
    Ajustar os valores de **orphans** pode melhorar significativamente a legibilidade dos documentos impressos, evitando que parágrafos sejam divididos de maneira desproporcional ou quebras abruptas.
    
- **Compatibilidade:**
    
    A propriedade **orphans** é suportada na maioria dos navegadores modernos para impressão, mas o comportamento exato pode variar. Testes em diferentes ambientes são recomendados para garantir a consistência do layout impresso.
