# page-break-inside

## 1. Introdução

A propriedade **page-break-inside** é utilizada para controlar a ocorrência de quebras de página dentro de um elemento quando o conteúdo é impresso. Essa propriedade ajuda a evitar quebras indesejadas de páginas dentro de blocos de conteúdo, garantindo que seções importantes permaneçam juntas no mesmo fragmento impresso.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **page-break-inside** determina se uma quebra de página pode ocorrer dentro de um elemento. Essa propriedade é aplicada principalmente em contextos de impressão (mídia `print`) e não afeta a visualização na tela.
    
- **Importância:**
    - **Organização de Conteúdo:** Permite que conteúdos relacionados, como parágrafos, imagens ou tabelas, sejam mantidos juntos em uma mesma página, evitando divisões abruptas que prejudiquem a legibilidade.
    - **Controle de Layout:** Essencial para preparar documentos para impressão, garantindo que a formatação e a estrutura visual sejam mantidas conforme o esperado.

---

## 3. Sintaxe e Valores

A sintaxe básica para **page-break-inside** é:

```css
seletor {
  page-break-inside: valor;
}

```

### Valores Comuns:

- **auto (valor padrão):**
    
    O navegador decide automaticamente onde inserir quebras de página dentro do elemento.
    
- **avoid:**
    
    Indica que o navegador deve evitar inserir uma quebra de página dentro do elemento, se possível.
    

**Exemplo:**

```css
.article {
  page-break-inside: avoid;
}

```

*Este exemplo tenta manter o conteúdo do elemento `.article` junto, sem quebras de página internas.*

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Evitar Quebra de Página em um Parágrafo Longo

```css
.long-paragraph {
  page-break-inside: avoid;  /* Evita que o parágrafo seja dividido entre páginas */
  font-size: 1rem;
  line-height: 1.5;
  margin-bottom: 1em;
}

```

```html
<div>
  <p class="long-paragraph">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus luctus urna sed urna ultricies ac tempor dui sagittis. In condimentum facilisis porta. Sed nec diam eu diam mattis viverra. Nulla fringilla, orci ac euismod semper, magna diam porttitor mauris, quis sollicitudin sapien justo in libero.
  </p>
</div>

```

*Explicação:*

O parágrafo longo será impresso em uma única página, evitando que o conteúdo seja fragmentado por uma quebra de página.

### Exemplo 2: Aplicando em um Layout de Impressão

```css
@media print {
  .section {
    page-break-inside: avoid;
    margin-bottom: 20px;
  }
}

```

```html
<div class="section">
  <h2>Seção 1</h2>
  <p>Conteúdo da seção 1 que deve permanecer junto na mesma página para melhor legibilidade.</p>
</div>
<div class="section">
  <h2>Seção 2</h2>
  <p>Conteúdo da seção 2 que também deve ser mantido em uma única página.</p>
</div>

```

*Explicação:*

Dentro da media query `@media print`, cada seção com a classe `.section` tentará evitar quebras de página internas, mantendo o conteúdo coeso.

---

## 5. Informações Adicionais

- **Contexto de Impressão:**
    
    A propriedade **page-break-inside** só tem efeito em mídias de impressão ou quando o conteúdo é renderizado em formatos que requerem paginação (como PDFs ou e-books).
    
- **Compatibilidade:**
    
    Embora seja amplamente suportada em navegadores modernos para impressão, é importante testar o comportamento em diferentes ambientes e dispositivos para garantir que o layout impresso atenda às expectativas.
    
- **Uso Consciente:**
    
    O valor `avoid` é útil para manter a integridade do conteúdo, mas seu uso excessivo pode resultar em páginas com muito espaço em branco se muitos elementos tentarem evitar a quebra.

## 7. Conclusão

A propriedade **page-break-inside** é uma ferramenta importante para o controle de paginação em layouts de impressão, permitindo que você evite quebras de página indesejadas dentro de um elemento. Ao usar `avoid`, o conteúdo é mantido unido, o que pode melhorar significativamente a legibilidade e a estética de documentos impressos. Essa propriedade, quando combinada com outras regras de impressão e media queries, ajuda a criar layouts de impressão profissionais e organizados.
