# widows

## 1. Introdução

A propriedade **widows** faz parte do conjunto de propriedades de paginação do CSS, que são usadas para controlar como o conteúdo é dividido em páginas durante a impressão ou em outros contextos de mídia paginada. Enquanto a propriedade **orphans** lida com as linhas isoladas no final de uma página, **widows** especifica o número mínimo de linhas que devem permanecer juntas no início de uma nova página. Essa propriedade é fundamental para manter a integridade visual e a legibilidade dos parágrafos quando o conteúdo é fragmentado em páginas.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **widows** determina o número mínimo de linhas de um bloco de conteúdo (geralmente um parágrafo) que devem aparecer no topo de uma página quando uma quebra de página ocorre. Se o número de linhas no início de uma nova página for menor que o valor especificado, o navegador tentará ajustar a quebra para evitar que uma linha isolada (widow) apareça.
    
- **Importância:**
    - **Legibilidade:** Garante que os parágrafos não sejam divididos de maneira abrupta, o que pode prejudicar a compreensão do conteúdo.
    - **Estética:** Mantém a consistência visual dos documentos impressos, evitando que apenas uma ou poucas linhas fiquem isoladas no topo de uma página.
    - **Controle de Paginação:** Permite aos designers especificar regras que melhorem a disposição do texto em documentos que serão impressos ou convertidos para outros formatos paginados.
- **Relacionamento com Orphans:**
    - **orphans:** Define o número mínimo de linhas que devem aparecer no final de uma página.
    - **widows:** Define o número mínimo de linhas que devem permanecer no início de uma nova página.

---

## 3. Sintaxe e Estrutura

A sintaxe básica para a propriedade **widows** é:

```css
seletor {
  widows: número;
}

```

- **número:**
Um valor inteiro que representa o número mínimo de linhas que devem ser mantidas juntas no início de uma página.

**Exemplo:**

```css
p {
  widows: 3;
}

```

Neste exemplo, ao imprimir, se um parágrafo for dividido, o navegador tentará garantir que pelo menos 3 linhas apareçam na parte superior da nova página.

---

## 4. Exemplos de Código Otimizados

### Exemplo 1: Aplicando widows em um Parágrafo Longo

```css
@media print {
  p {
    widows: 3; /* Pelo menos 3 linhas devem aparecer no início de uma nova página */
  }
}

```

```html
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed non risus. Suspendisse lectus tortor, dignissim sit amet, adipiscing nec, ultricies sed, dolor. Cras elementum ultrices diam. Maecenas ligula massa, varius a, semper congue, euismod non, mi. Proin porttitor, orci nec nonummy molestie, enim est eleifend mi, non fermentum diam nisl sit amet erat. Duis semper. Duis arcu massa, scelerisque vitae, consequat in, pretium a, enim. Pellentesque congue. Ut in risus volutpat libero pharetra tempor. Cras vestibulum bibendum augue. Praesent egestas leo in pede.
</p>

```

*Explicação:*

Ao definir `widows: 3;`, o navegador garantirá que, se o parágrafo for dividido entre páginas, pelo menos 3 linhas apareçam na parte superior da nova página, evitando que uma única linha isolada (widow) seja exibida.

### Exemplo 2: Utilizando widows em Conjunto com orphans

```css
@media print {
  p {
    widows: 3;   /* Mínimo de 3 linhas na parte superior da nova página */
    orphans: 2;  /* Mínimo de 2 linhas no final da página anterior */
  }
}

```

```html
<p>
  Vivamus magna justo, lacinia eget consectetur sed, convallis at tellus. Curabitur arcu erat, accumsan id imperdiet et, porttitor at sem. Nulla porttitor accumsan tincidunt. Donec sollicitudin molestie malesuada. Curabitur non nulla sit amet nisl tempus convallis quis ac lectus.
</p>

```

*Explicação:*

Nesta configuração, **widows** e **orphans** trabalham juntos para assegurar que os parágrafos sejam divididos de forma harmoniosa em páginas impressas, evitando tanto a ocorrência de linhas isoladas no início quanto no final das páginas.

---

## 5. Informações Adicionais

- **Contexto de Impressão:**
    
    As propriedades **widows** e **orphans** são aplicadas apenas em contextos de impressão ou em qualquer mídia que suporte paginação. Elas não afetam a renderização na tela.
    
- **Compatibilidade:**
    
    A maioria dos navegadores modernos suporta as propriedades de paginação, mas é importante testá-las em diferentes ambientes de impressão para garantir que o layout atenda às expectativas.
    
- **Uso Estratégico:**
    
    Ajustar o número de widows e orphans pode melhorar significativamente a legibilidade e a apresentação de documentos longos, como relatórios, artigos ou e-books, proporcionando um fluxo visual mais natural.
