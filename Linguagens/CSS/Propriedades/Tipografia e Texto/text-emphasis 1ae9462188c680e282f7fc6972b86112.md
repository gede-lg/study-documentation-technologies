# text-emphasis

## 1. Introdução

A propriedade **text-emphasis** é parte do módulo de tipografia do CSS e foi projetada para adicionar ênfase visual ao texto, inserindo marcas decorativas (como pontos ou outros símbolos) ao redor do conteúdo. Essa propriedade é especialmente útil em idiomas onde a ênfase tipográfica desempenha um papel importante (como no japonês), mas também pode ser usada de forma criativa para destacar trechos de texto em qualquer contexto. Com **text-emphasis**, você pode definir tanto o símbolo de ênfase quanto sua posição, estilo e, em alguns casos, cor, proporcionando maior flexibilidade na apresentação do texto.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **text-emphasis** é uma propriedade que permite aplicar marcas de ênfase ao texto. Essas marcas são desenhadas fora do fluxo normal do conteúdo, sem alterar o layout do texto, e podem ser personalizadas para se adequar ao design do site.
    
- **Objetivo e Aplicabilidade:**
    - **Realce Visual:** Destaca palavras ou frases importantes sem alterar a estrutura do conteúdo.
    - **Estilização Cultural:** Muito utilizada em tipografias japonesas (por exemplo, para enfatizar textos de estilo tradicional), mas também aplicável em designs ocidentais para efeitos artísticos.
    - **Semântica e Acessibilidade:** A ênfase pode ser uma forma adicional de guiar o leitor para informações importantes, complementando o uso de estilos como negrito ou itálico.
- **Componentes:**
    
    **text-emphasis** pode ser visto como uma propriedade abreviada que engloba:
    
    - **text-emphasis-style:** Define o estilo da marca de ênfase (por exemplo, o símbolo ou padrão).
    - **text-emphasis-position:** Define a posição da marca em relação ao texto (por exemplo, acima ou ao lado).
    
    Em versões mais modernas do CSS, a propriedade pode ser utilizada com os dois componentes integrados ou definidos separadamente.
    

---

## 3. Sintaxe e Valores

### 3.1. Propriedade Abreviada

A sintaxe básica para **text-emphasis** é:

```css
seletor {
  text-emphasis: <text-emphasis-style> <text-emphasis-position>;
}

```

- **text-emphasis-style:**
    
    Pode ser um valor específico (como um ponto, uma gota, uma forma personalizada, ou a palavra-chave `none` para desativar a ênfase).
    
- **text-emphasis-position:**
    
    Define onde a marca de ênfase será posicionada em relação ao texto. Os valores comuns são:
    
    - `under` (ou `after`): coloca a ênfase abaixo do texto.
    - `over` (ou `before`): coloca a ênfase acima do texto.

### 3.2. Exemplos de Valores

- **Exemplo com Ponto de Ênfase:**
    
    ```css
    .emphasis {
      text-emphasis: filled dot over;
    }
    
    ```
    
    *Neste exemplo, a marca de ênfase é um ponto preenchido colocado acima do texto.*
    
- **Desativando a Ênfase:**
    
    ```css
    .no-emphasis {
      text-emphasis: none;
    }
    
    ```
    
    *Utilize `none` para remover qualquer marca de ênfase aplicada ao texto.*
    
- **Outros Estilos:**
    
    Dependendo do suporte do navegador, você pode experimentar outros estilos e símbolos:
    
    ```css
    .fancy-emphasis {
      text-emphasis: open circle under;
    }
    
    ```

## 4. Exemplos Práticos

### Exemplo 1: Ênfase em um Texto de Destaque

```css
.highlight {
  font-size: 1.2rem;
  color: #333;
  text-emphasis: filled circle over;
  text-emphasis-color: #e74c3c;
}

```

```html
<p class="highlight">
  Este texto possui uma marca de ênfase acima, destacando-o de forma sutil e elegante.
</p>

```

*Explicação:*

Neste exemplo, o parágrafo recebe uma marca de ênfase na forma de um círculo preenchido, posicionado acima do texto. A cor da marca de ênfase é definida como vermelho (#e74c3c), criando um contraste visual eficaz.

---

### Exemplo 2: Aplicando Ênfase a Palavras Específicas

```css
.important::before {
  content: "";
  text-emphasis: filled triangle over;
  text-emphasis-color: #2ecc71;
  margin-right: 0.3em;
}

```

```html
<p>
  Preste atenção na <span class="important">palavra-chave</span> neste parágrafo.
</p>

```

*Explicação:*

Usando um pseudo-elemento `::before`, adiciona-se uma marca de ênfase (um triângulo preenchido) antes da palavra “palavra-chave”. Essa abordagem permite destacar palavras específicas sem modificar o conteúdo original.

---

### Exemplo 3: Combinação com Outras Propriedades de Decoração

```css
.notice {
  font-size: 1.1rem;
  color: #555;
  text-emphasis: open dot under;
  text-emphasis-color: #f39c12;
  text-emphasis-position: under;
  line-height: 1.6;
}

```

```html
<p class="notice">
  Notas importantes podem ser destacadas com marcas de ênfase, ajudando a chamar a atenção do leitor.
</p>

```

*Explicação:*

O uso de `open dot` com ênfase posicionada abaixo do texto (`under`) oferece um efeito visual sutil que complementa o estilo geral do texto sem ser excessivamente intrusivo.

## 5. Informações Adicionais

- **Compatibilidade:**
    
    **text-emphasis** é amplamente suportada em navegadores modernos, especialmente para textos em idiomas que tradicionalmente usam marcas de ênfase, como o japonês. No entanto, o suporte pode variar para alguns valores e palavras-chave específicas.
    
- **Integração com Outras Propriedades de Texto:**
    
    É comum usar **text-emphasis** junto com outras propriedades tipográficas, como `text-decoration`, `font-weight`, e `color`, para alcançar um efeito visual coeso.
    
- **Customização:**
    
    Além dos estilos padrão (como `dot`, `circle`, `triangle`), alguns navegadores podem oferecer suporte a outros tipos de marca, ou permitir a utilização de imagens ou fontes personalizadas para criar marcas de ênfase únicas.
    
- **Uso em Diferentes Idiomas:**
    
    A ênfase tipográfica é especialmente relevante para idiomas que usam marcas específicas para indicar ênfase, como no japonês. Para outros idiomas, ela pode ser usada de forma criativa para destacar informações importantes.

## 7. Conclusão

A propriedade **text-emphasis** oferece aos desenvolvedores uma maneira elegante de adicionar marcas de ênfase ao texto, enriquecendo a tipografia e destacando informações importantes. Ao definir o estilo, a posição e a cor das marcas de ênfase, você pode criar efeitos visuais que se alinham com a identidade do design e melhoram a legibilidade. Seja para destacar palavras-chave em textos técnicos ou para criar um estilo tipográfico único em idiomas que utilizam marcas tradicionais de ênfase, dominar **text-emphasis** é essencial para aprimorar a experiência do usuário.
