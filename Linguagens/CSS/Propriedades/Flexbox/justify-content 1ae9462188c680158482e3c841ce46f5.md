# justify-content

## 1. Introdução

A propriedade **justify-content** é uma parte fundamental do modelo de layout Flexbox e também é utilizada em Grid Layout. Ela determina como o espaço extra ao longo do eixo principal de um contêiner flexível (ou grid) é distribuído entre os itens. Em outras palavras, **justify-content** alinha os itens quando há espaço sobrando, permitindo criar layouts equilibrados e responsivos.

---

## 2. Conceitos Fundamentais

- **Eixo Principal:**
    
    No Flexbox, o eixo principal é definido pela propriedade **flex-direction**. Por exemplo, se `flex-direction` for `row`, o eixo principal é horizontal; se for `column`, o eixo principal é vertical.
    
- **Distribuição de Espaço:**
    
    **justify-content** controla como o espaço extra é distribuído entre os itens ao longo do eixo principal. Ela pode centralizar, distribuir igualmente, ou alinhar os itens em uma extremidade.
    
- **Flexbox e Grid:**
    
    Embora seja mais comumente utilizada no Flexbox, a propriedade também pode ser aplicada em contêineres grid para controlar o alinhamento dos itens dentro da célula.
    

---

## 3. Sintaxe e Valores

A sintaxe básica da propriedade é:

```css
seletor {
  justify-content: valor;
}

```

### Valores Comuns:

- **flex-start:**
    
    Alinha os itens ao início do contêiner (valor padrão).
    
    ```css
    justify-content: flex-start;
    
    ```
    
- **flex-end:**
    
    Alinha os itens ao final do contêiner.
    
    ```css
    justify-content: flex-end;
    
    ```
    
- **center:**
    
    Centraliza os itens no contêiner.
    
    ```css
    justify-content: center;
    
    ```
    
- **space-between:**
    
    Distribui os itens igualmente, com o primeiro item alinhado ao início e o último ao final, e os demais distribuídos com espaços iguais entre si.
    
    ```css
    justify-content: space-between;
    
    ```
    
- **space-around:**
    
    Distribui os itens de forma que cada item receba um espaço igual ao redor, resultando em espaços menores nas extremidades.
    
    ```css
    justify-content: space-around;
    
    ```
    
- **space-evenly:**
    
    Distribui os itens de forma que o espaço entre todos os itens, bem como entre os itens e as bordas do contêiner, seja exatamente igual.
    
    ```css
    justify-content: space-evenly;
    
    ```

## 4. Exemplos de Código Otimizados

### Exemplo 1: Alinhamento Padrão (flex-start)

```css
.container {
  display: flex;
  justify-content: flex-start; /* Itens iniciam no começo do contêiner */
  border: 1px solid #ccc;
  padding: 10px;
}

.item {
  background-color: #3498db;
  color: #fff;
  padding: 20px;
  margin: 5px;
}

```

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

```

### Exemplo 2: Centralizando Itens

```css
.container {
  display: flex;
  justify-content: center; /* Centraliza os itens */
  border: 1px solid #ccc;
  padding: 10px;
}

.item {
  background-color: #e74c3c;
  color: #fff;
  padding: 20px;
  margin: 5px;
}

```

```html
<div class="container">
  <div class="item">Item A</div>
  <div class="item">Item B</div>
  <div class="item">Item C</div>
</div>

```

### Exemplo 3: Distribuição com Espaço Entre (space-between)

```css
.container {
  display: flex;
  justify-content: space-between; /* Primeiro item no início, último no fim, espaço igual entre os demais */
  border: 1px solid #ccc;
  padding: 10px;
}

.item {
  background-color: #2ecc71;
  color: #fff;
  padding: 20px;
  margin: 5px;
}

```

```html
<div class="container">
  <div class="item">Item X</div>
  <div class="item">Item Y</div>
  <div class="item">Item Z</div>
</div>

```

### Exemplo 4: Espaçamento Uniforme (space-evenly)

```css
.container {
  display: flex;
  justify-content: space-evenly; /* Espaços iguais entre itens e bordas */
  border: 1px solid #ccc;
  padding: 10px;
}

.item {
  background-color: #9b59b6;
  color: #fff;
  padding: 20px;
  margin: 5px;
}

```

```html
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>

```

## 5. Informações Adicionais

- **Flexbox Context:**
    
    **justify-content** atua apenas no eixo principal. Para alinhar os itens no eixo transversal, utilize a propriedade **align-items**.
    
- **Comportamento Responsivo:**
    
    Essa propriedade é útil para layouts responsivos, permitindo ajustar a distribuição dos itens conforme o espaço do contêiner muda.
    
- **Compatibilidade:**
    
    **justify-content** é amplamente suportada em navegadores modernos. Para versões mais antigas, verifique a compatibilidade e utilize prefixos se necessário.

## 7. Conclusão

A propriedade **justify-content** é essencial para distribuir o espaço extra e alinhar os itens dentro de um contêiner flexível de forma eficaz. Seja para centralizar itens, distribuir espaço uniformemente, ou criar layouts com espaços diferenciados, **justify-content** permite um controle refinado do layout. Ao combinar essa propriedade com outras do Flexbox, como **align-items** e **flex-direction**, você pode criar interfaces responsivas e visualmente coerentes.
