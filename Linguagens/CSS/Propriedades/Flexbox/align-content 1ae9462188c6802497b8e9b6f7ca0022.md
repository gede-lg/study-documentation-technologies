# align-content

## 1. Introdução

A propriedade **align-content** é utilizada em CSS para definir como o espaço extra entre as linhas (ou "faixas") dentro de um contêiner flexível ou grid é distribuído ao longo do eixo transversal. Ela é particularmente útil quando há múltiplas linhas de itens e você deseja controlar o espaçamento vertical (ou horizontal, em um contêiner com direção de coluna) entre essas linhas. Diferente de **align-items**, que alinha os itens individuais dentro de cada linha, **align-content** atua sobre a distribuição do espaço entre as linhas como um todo.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **align-content** controla o alinhamento e a distribuição do espaço extra entre as linhas de um contêiner, quando o contêiner possui mais de uma linha (no Flexbox) ou múltiplas faixas (no Grid).
    
- **Importância:**
    - Permite um controle refinado do layout quando os itens se agrupam em várias linhas.
    - Ajuda a manter um design equilibrado, distribuindo o espaço entre as linhas de forma uniforme ou de acordo com a necessidade.
    - Complementa outras propriedades de alinhamento como **justify-content**, **align-items** e **align-self** para oferecer controle total sobre a disposição dos elementos.

---

## 3. Sintaxe e Valores

A sintaxe básica é:

```css
seletor {
    align-content: valor;
}

```

### Valores Comuns:

- **stretch (valor padrão):**
    
    As linhas são esticadas para preencher o espaço extra disponível no contêiner.
    
- **flex-start:**
    
    As linhas são agrupadas no início do contêiner, sem espaço extra distribuído entre elas.
    
- **flex-end:**
    
    As linhas são agrupadas no final do contêiner.
    
- **center:**
    
    As linhas são centralizadas no contêiner, com espaço igual acima e abaixo.
    
- **space-between:**
    
    As linhas são distribuídas de forma que o espaço entre elas seja igual, sem espaço adicional nas extremidades.
    
- **space-around:**
    
    As linhas são distribuídas com espaços iguais ao redor de cada linha, o que significa que as extremidades terão metade do espaço.
    
- **space-evenly:**
    
    O espaço entre as linhas e as bordas do contêiner é exatamente igual.
    

---

## 4. Funcionamento e Impacto no Layout

- **Eixo Transversal:**
    
    Em um contêiner Flexbox com `flex-direction: row`, o eixo transversal é vertical, logo **align-content** controla o espaçamento entre as linhas verticalmente. Para `flex-direction: column`, o eixo transversal é horizontal e a propriedade age na distribuição horizontal das faixas.
    
- **Múltiplas Linhas:**
    
    **align-content** só tem efeito quando há múltiplas linhas de itens. Se houver apenas uma linha, a propriedade não terá impacto.
    
- **Complementaridade:**
    
    Enquanto **align-items** alinha os itens individuais dentro de cada linha, **align-content** distribui o espaço entre as linhas. Em layouts Grid, ela controla o espaçamento entre faixas de linhas ou colunas.

## 5. Exemplos de Código Otimizados

### Exemplo 1: Distribuição de Espaço com "space-between"

```css
.container {
    display: flex;
    flex-wrap: wrap;           /* Necessário para criar múltiplas linhas */
    height: 400px;
    border: 1px solid #ccc;
    align-content: space-between;
}

.item {
    flex: 1 1 150px;
    background-color: #3498db;
    color: #fff;
    margin: 5px;
    padding: 10px;
}

```

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>

```

*Explicação:*

Com `align-content: space-between;`, o espaço extra é distribuído igualmente entre as linhas, com o primeiro e o último grupo de itens encostados nas extremidades do contêiner.

### Exemplo 2: Centralizando Linhas com "center"

```css
.container {
    display: flex;
    flex-wrap: wrap;
    height: 400px;
    border: 1px solid #ccc;
    align-content: center;
}

.item {
    flex: 1 1 150px;
    background-color: #e74c3c;
    color: #fff;
    margin: 5px;
    padding: 10px;
}

```

```html
<div class="container">
  <div class="item">A</div>
  <div class="item">B</div>
  <div class="item">C</div>
  <div class="item">D</div>
  <div class="item">E</div>
  <div class="item">F</div>
</div>

```

*Explicação:*

`align-content: center;` alinha as linhas de itens no centro do contêiner, deixando espaço igual acima e abaixo das linhas.

### Exemplo 3: Utilizando "stretch" (Padrão)

```css
.container {
    display: flex;
    flex-wrap: wrap;
    height: 400px;
    border: 1px solid #ccc;
    align-content: stretch;
}

.item {
    flex: 1 1 150px;
    background-color: #2ecc71;
    color: #fff;
    margin: 5px;
    padding: 10px;
}

```

*Explicação:*

Com `align-content: stretch;`, as linhas são esticadas para preencher o espaço disponível no contêiner, criando uma aparência de distribuição uniforme verticalmente (ou horizontalmente, dependendo da direção).

---

## 6. Informações Adicionais

- **Efeito Visual e Responsividade:**
    
    A propriedade **align-content** é particularmente útil para ajustar layouts em dispositivos com diferentes tamanhos de tela, garantindo que o espaçamento entre linhas se adapte conforme o contêiner aumenta ou diminui.
    
- **Complemento ao Flexbox:**
    
    Para um controle completo, **align-content** deve ser usada juntamente com **justify-content** (para o eixo principal) e **align-items** (para os itens individuais dentro de cada linha).
    
- **Aplicações em Grid Layout:**
    
    No Grid, **align-content** funciona de maneira semelhante, controlando o espaçamento entre as faixas de linhas.

## 8. Conclusão

A propriedade **align-content** é fundamental para a criação de layouts flexíveis e responsivos, permitindo que o espaço entre as linhas de um contêiner seja distribuído de maneira harmoniosa. Ao utilizar valores como `center`, `space-between`, `stretch`, entre outros, os desenvolvedores podem ajustar a apresentação dos itens e melhorar a organização visual do conteúdo. Essa propriedade complementa outras ferramentas do Flexbox e Grid, oferecendo um controle refinado sobre o alinhamento e a distribuição do espaço extra.
