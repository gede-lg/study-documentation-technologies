# flex-shrink

## 1. Introdução

A propriedade **flex-shrink** é uma parte fundamental do modelo de layout Flexbox em CSS. Ela especifica a capacidade de um item flexível de encolher quando o espaço disponível no contêiner é insuficiente para acomodar todos os itens no seu tamanho base. Essa propriedade permite que os itens se ajustem proporcionalmente, evitando que o layout seja distorcido quando a janela de visualização é reduzida.

---

## 2. Conceitos Fundamentais

- **Flex Shrink:**
    
    Define quanto um item flexível deve encolher em relação aos outros itens quando não há espaço suficiente no contêiner.
    
- **Valor Padrão:**
    
    O valor padrão para **flex-shrink** é `1`, o que significa que, por padrão, os itens podem encolher proporcionalmente quando necessário.
    
- **Contexto Flexbox:**
    
    A propriedade **flex-shrink** só tem efeito quando o item está dentro de um contêiner com `display: flex` (ou `inline-flex`). Ela trabalha em conjunto com **flex-grow** e **flex-basis** para determinar o tamanho final dos itens.
    

---

## 3. Sintaxe e Estrutura

A sintaxe básica para definir **flex-shrink** é:

```css
seletor {
    flex-shrink: número;
}

```

- **número:**
Um valor numérico que representa a proporção com que o item encolherá. Por exemplo:
    - `0`: O item não encolherá, mesmo se não houver espaço suficiente.
    - `1` (valor padrão): O item encolherá proporcionalmente ao espaço extra necessário.
    - Valores maiores que 1: O item encolherá mais rapidamente em relação aos outros.

---

## 4. Funcionamento e Exemplos

### 4.1. Funcionamento

- **Sem Encolhimento (flex-shrink: 0):**
    
    O item manterá seu tamanho base, independentemente do espaço disponível no contêiner.
    
    ```css
    .item-fixo {
        flex-shrink: 0;
    }
    
    ```
    
- **Encolhimento Proporcional (flex-shrink: 1):**
    
    Os itens com esse valor encolhem de forma igual se o contêiner não tiver espaço suficiente.
    
    ```css
    .item {
        flex-shrink: 1;
    }
    
    ```
    
- **Encolhimento Acelerado (flex-shrink: 2 ou maior):**
    
    O item encolherá em uma taxa maior comparado a outros itens com valores menores, ajudando a ajustar o layout quando o espaço é muito restrito.
    
    ```css
    .item-rapido {
        flex-shrink: 2;
    }
    
    ```
    

### 4.2. Exemplo Prático: Layout Responsivo com Encolhimento

Imagine um contêiner flexível com dois itens onde queremos que um item encolha mais do que o outro quando o espaço se torna limitado:

```css
.container {
    display: flex;
    width: 600px;
    border: 1px solid #ccc;
}

.item-um {
    flex-basis: 300px;
    flex-shrink: 1;  /* Encolhe de forma padrão */
    background-color: #3498db;
    color: #fff;
    padding: 10px;
}

.item-dois {
    flex-basis: 300px;
    flex-shrink: 2;  /* Encolhe mais rapidamente se necessário */
    background-color: #e74c3c;
    color: #fff;
    padding: 10px;
}

```

```html
<div class="container">
  <div class="item-um">Item 1</div>
  <div class="item-dois">Item 2</div>
</div>

```

**Explicação:**

- Ambos os itens começam com um tamanho base de 300px.
- Se o contêiner reduzir sua largura abaixo do total necessário, o **Item 2** (com `flex-shrink: 2`) encolherá duas vezes mais rápido do que o **Item 1** (com `flex-shrink: 1`), permitindo um ajuste mais equilibrado do layout.

## 5. Informações Adicionais

- **Interação com Outras Propriedades Flex:**
    
    **flex-shrink** atua em conjunto com **flex-grow** e **flex-basis** para determinar o tamanho final do item flexível.
    
    - **flex-basis** define o tamanho inicial do item.
    - **flex-grow** controla o quanto ele pode crescer se houver espaço extra.
    - **flex-shrink** especifica a capacidade de encolher quando o espaço é insuficiente.
- **Uso em Layouts Responsivos:**
    
    Ajustar **flex-shrink** permite que os itens se adaptem de maneira fluida ao espaço disponível, essencial para a criação de layouts responsivos que se comportam bem em diferentes tamanhos de tela.
    
- **Valores Fracionários:**
    
    É possível usar valores fracionários (como `0.5` ou `1.5`) para um controle mais sutil do encolhimento, embora na prática os valores inteiros sejam os mais comuns.
    
- **Considerações de Design:**
    
    Definir **flex-shrink** corretamente ajuda a manter a integridade visual do layout, evitando que itens importantes se tornem muito pequenos ou desapareçam em telas menores.
