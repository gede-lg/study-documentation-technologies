# align-self

## 1. Introdução

A propriedade **align-self** faz parte do modelo Flexbox (e também do Grid Layout) e permite ajustar o alinhamento individual de um item dentro de um contêiner flexível ou de grade, de forma independente do alinhamento definido para os demais itens. Enquanto **align-items** alinha todos os itens de maneira uniforme, **align-self** oferece um controle refinado para sobrescrever esse alinhamento em itens específicos, possibilitando layouts mais personalizados e responsivos.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **align-self** especifica o alinhamento de um único item ao longo do eixo transversal (perpendicular ao eixo principal) dentro de um contêiner flexível ou grid.
    
- **Contexto Flexbox e Grid:**
    
    Em um contêiner Flexbox, o eixo transversal é definido perpendicularmente ao eixo principal (determinado por **flex-direction**). No Grid Layout, **align-self** controla o alinhamento do item dentro da célula.
    
- **Sobrescrevendo o Alinhamento Global:**
    
    Se o contêiner possui **align-items** definido, **align-self** pode ser usado para alterar o alinhamento apenas de um item específico, sem afetar os demais.
    

---

## 3. Sintaxe e Valores

A sintaxe básica para **align-self** é:

```css
seletor {
    align-self: valor;
}

```

### Valores Possíveis:

- **auto (valor padrão):**
    
    O item herda o valor da propriedade **align-items** do contêiner.
    
    ```css
    .item {
        align-self: auto;
    }
    
    ```
    
- **flex-start:**
    
    Alinha o item no início do eixo transversal.
    
    ```css
    .item {
        align-self: flex-start;
    }
    
    ```
    
- **flex-end:**
    
    Alinha o item no final do eixo transversal.
    
    ```css
    .item {
        align-self: flex-end;
    }
    
    ```
    
- **center:**
    
    Centraliza o item ao longo do eixo transversal.
    
    ```css
    .item {
        align-self: center;
    }
    
    ```
    
- **baseline:**
    
    Alinha o item com a linha de base de seu conteúdo. Isso é útil para itens de texto com diferentes tamanhos de fonte.
    
    ```css
    .item {
        align-self: baseline;
    }
    
    ```
    
- **stretch:**
    
    Faz com que o item se estique para preencher o contêiner, mantendo o alinhamento dos outros itens. Esse valor funciona quando o item não tem um tamanho fixo definido.
    
    ```css
    .item {
        align-self: stretch;
    }
    
    ```

## 4. Exemplos de Código Otimizados

### Exemplo 1: Alinhamento Padrão e Sobrescrito

Imagine um contêiner Flex com todos os itens alinhados ao centro, mas você deseja que um item específico se alinhe ao início do eixo transversal:

```css
.container {
    display: flex;
    height: 200px;
    align-items: center; /* Alinha todos os itens ao centro */
    border: 1px solid #ccc;
    padding: 10px;
}

.item {
    background-color: #3498db;
    color: #fff;
    padding: 15px;
    margin: 5px;
}

.item-especial {
    align-self: flex-start; /* Sobrescreve o alinhamento para este item específico */
    background-color: #e74c3c;
}

```

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item item-especial">Item Especial</div>
  <div class="item">Item 3</div>
</div>

```

*Explicação:*

- Todos os itens estão alinhados verticalmente ao centro, exceto o "Item Especial", que é alinhado ao início do contêiner, graças a **align-self: flex-start;**.

### Exemplo 2: Alinhamento com Diferentes Valores em um Layout de Grid

Em um contêiner Grid, **align-self** pode controlar o alinhamento de cada item dentro da célula:

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
    height: 300px;
    border: 1px solid #ccc;
}

.grid-item {
    background-color: #2ecc71;
    color: #fff;
    padding: 15px;
}

.grid-item:nth-child(2) {
    align-self: flex-end;
}

.grid-item:nth-child(3) {
    align-self: center;
}

```

```html
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2 (flex-end)</div>
  <div class="grid-item">Item 3 (center)</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>

```

*Explicação:*

- No contêiner grid, os itens geralmente se esticam para preencher a célula (com o valor padrão de **stretch**).
- O segundo item é alinhado ao final da célula e o terceiro é centralizado verticalmente, conforme definido por **align-self**.

## 5. Informações Adicionais

- **Herança e Sobrescrita:**
    
    O valor `auto` em **align-self** faz com que o item herde o alinhamento do contêiner (definido por **align-items**), permitindo que apenas itens que necessitam de um alinhamento diferente tenham valores explícitos.
    
- **Eixo Transversal:**
    
    A interpretação de **align-self** depende do eixo transversal do contêiner, que varia conforme o valor de **flex-direction** (para Flexbox) ou a configuração de linhas (para Grid Layout).
    
- **Integração com Layout Responsivo:**
    
    Utilizar **align-self** em conjunto com media queries pode ajudar a adaptar o alinhamento de itens individuais para melhorar a experiência em diferentes dispositivos e tamanhos de tela.

## 7. Conclusão

A propriedade **align-self** é uma ferramenta poderosa para personalizar o alinhamento de itens individuais dentro de contêineres Flexbox ou Grid. Ela permite que você sobrescreva o alinhamento global definido pelo contêiner, possibilitando um controle preciso sobre a disposição dos elementos. Ao dominar **align-self**, você pode criar layouts mais flexíveis, responsivos e visualmente agradáveis, adaptando o posicionamento dos itens às necessidades específicas do design.
