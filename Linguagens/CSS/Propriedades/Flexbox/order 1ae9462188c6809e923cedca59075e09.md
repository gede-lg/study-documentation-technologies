# order

## 1. Introdução

A propriedade **order** é parte integrante do modelo Flexbox em CSS e permite definir a ordem visual dos itens dentro de um contêiner flexível. Por padrão, todos os itens possuem um valor de ordem igual a `0`, e eles são exibidos na sequência em que aparecem no HTML. Ao atribuir valores diferentes para a propriedade **order**, você pode reorganizar os itens visualmente sem alterar a ordem no documento HTML.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **order** especifica a ordem de exibição de um item flexível em relação aos outros itens dentro do mesmo contêiner flex. Essa propriedade é particularmente útil quando se deseja alterar a sequência visual dos elementos sem modificar o código HTML, facilitando a criação de layouts dinâmicos e responsivos.
    
- **Valor Padrão:**
    
    Todos os itens possuem, por padrão, `order: 0;`.
    
    - Itens com valores numéricos menores são posicionados antes dos itens com valores maiores.

---

## 3. Sintaxe e Estrutura

A sintaxe básica para **order** é:

```css
.seletor {
    order: número;
}

```

- **número:**
Um valor numérico (positivo, negativo ou zero) que determina a posição do item em relação aos demais.

**Exemplo:**

```css
.item {
    order: 1;
}

```

---

## 4. Funcionamento e Impacto no Layout

- **Reorganização Visual:**
    
    A propriedade **order** permite alterar a sequência dos itens no contêiner flexível sem alterar a ordem do HTML.
    
    - Exemplo: Se um item tiver `order: -1` e os demais `order: 0`, esse item aparecerá primeiro visualmente.
- **Hierarquia Visual:**
    
    Permite criar hierarquias ou destacar itens específicos alterando sua posição relativa aos outros elementos, o que é muito útil para layouts responsivos ou reordenamento de conteúdo conforme a necessidade.
    
- **Valores Negativos e Positivos:**
    - Valores negativos posicionam o item antes dos itens com valores iguais ou maiores.
    - Valores positivos posicionam o item depois dos itens com valores iguais ou menores.

---

## 5. Exemplos de Código Otimizados

### Exemplo 1: Reorganizando Itens de uma Lista

```css
.container {
    display: flex;
    gap: 10px;
    border: 1px solid #ccc;
    padding: 10px;
}

.item {
    background-color: #3498db;
    color: #fff;
    padding: 20px;
    flex: 1;
}

/* Alterando a ordem de exibição dos itens */
.item-1 {
    order: 2;
}

.item-2 {
    order: -1;
}

.item-3 {
    order: 1;
}

```

```html
<div class="container">
  <div class="item item-1">Item 1</div>
  <div class="item item-2">Item 2</div>
  <div class="item item-3">Item 3</div>
</div>

```

*Explicação:*

- **Item 2** possui `order: -1`, logo aparece primeiro.
- **Item 3** tem `order: 1` e **Item 1** tem `order: 2`, determinando a sequência visual: Item 2, Item 3, Item 1.

### Exemplo 2: Reordenação em um Layout Responsivo

Em um layout responsivo, você pode ajustar a ordem dos itens com media queries para melhorar a experiência do usuário em diferentes dispositivos.

```css
.container {
    display: flex;
    flex-direction: row;
}

.item {
    flex: 1;
    padding: 10px;
    background-color: #2ecc71;
    color: #fff;
}

/* Ordem padrão para telas maiores */
.item-1 { order: 1; }
.item-2 { order: 2; }
.item-3 { order: 3; }

/* Em telas menores, alterar a ordem visual */
@media (max-width: 600px) {
  .container {
      flex-direction: column;
  }
  .item-1 { order: 3; }
  .item-2 { order: 1; }
  .item-3 { order: 2; }
}

```

```html
<div class="container">
  <div class="item item-1">Item 1</div>
  <div class="item item-2">Item 2</div>
  <div class="item item-3">Item 3</div>
</div>

```

*Explicação:*

- Em telas maiores, a ordem visual é Item 1, Item 2, Item 3.
- Em telas menores, a ordem muda para Item 2, Item 3, Item 1, adaptando a experiência visual conforme o dispositivo.

---

## 6. Informações Adicionais

- **Não Afeta a Ordem no HTML:**
    
    A propriedade **order** altera apenas a apresentação visual dos itens. A ordem no documento HTML permanece inalterada, o que é importante para acessibilidade e SEO.
    
- **Interação com Outras Propriedades Flex:**
    
    **order** pode ser utilizada em conjunto com outras propriedades Flexbox, como **flex-direction**, para criar layouts complexos e adaptáveis.
    
- **Aplicações Comuns:**
    - Reordenação de itens em menus e galerias.
    - Priorização de conteúdo em layouts responsivos.
    - Criação de efeitos visuais dinâmicos sem alterar o HTML.
