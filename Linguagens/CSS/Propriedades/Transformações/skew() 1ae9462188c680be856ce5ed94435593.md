# skew()

## 1. Introdução

A função **skew()** é parte da propriedade **transform** em CSS e é utilizada para inclinar (ou "distorcer") um elemento ao longo dos eixos X e Y. Em outras palavras, **skew()** altera a forma do elemento, inclinando suas bordas e criando um efeito visual de distorção sem alterar as dimensões do layout base. Essa transformação é especialmente útil para criar efeitos dinâmicos e estilizados, desde layouts assimétricos e modernos até animações interativas.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **skew()** aplica uma inclinação a um elemento, alterando os ângulos dos seus lados. Ela pode ser aplicada tanto no eixo horizontal (skewX) quanto no eixo vertical (skewY) ou em ambos simultaneamente.
    
- **Objetivos e Benefícios:**
    - **Estilização Criativa:** Permite criar designs dinâmicos e não convencionais, adicionando uma sensação de movimento ou distorção aos elementos.
    - **Efeitos Visuais:** Pode ser usada para realçar elementos de forma sutil ou dramática, dependendo do ângulo de inclinação.
    - **Integração com Outras Transformações:** **skew()** pode ser combinada com funções como `rotate()`, `translate()`, e `scale()` para compor transformações complexas e criar animações elaboradas.
- **Aplicações Comuns:**
    - Criação de layouts assimétricos ou inclinados.
    - Efeitos de destaque em banners, botões ou seções de um site.
    - Animações e transições que enfatizam o movimento e a fluidez do design.

---

## 3. Sintaxe e Valores

A função **skew()** pode ser aplicada de duas maneiras principais:

### 3.1. **skew()** (Combinação de Eixos)

Você pode utilizar um único comando **skew()** para inclinar o elemento tanto no eixo X quanto no eixo Y. A sintaxe é:

```css
transform: skew(<ânguloX>, <ânguloY>);

```

- **<ânguloX>:** Ângulo de inclinação no eixo horizontal.
- **<ânguloY>:** Ângulo de inclinação no eixo vertical.

**Exemplo:**

```css
.element {
  transform: skew(20deg, 10deg);
}

```

*Explicação:*

O elemento é inclinado 20 graus no eixo X e 10 graus no eixo Y, resultando em uma distorção que altera os ângulos originais do retângulo.

### 3.2. **skewX()** e **skewY()** (Separadamente)

Caso deseje inclinar o elemento apenas em um dos eixos, você pode usar:

- **skewX():**
    
    ```css
    transform: skewX(20deg);
    
    ```
    
    *Inclina o elemento apenas horizontalmente (eixo X).*
    
- **skewY():**
    
    ```css
    transform: skewY(15deg);
    
    ```
    
    *Inclina o elemento apenas verticalmente (eixo Y).*
    

*Observação:*

Os valores de inclinação podem ser positivos ou negativos, determinando a direção da inclinação.

## 4. Exemplos Práticos

### Exemplo 1: Inclinação Simples com skew()

```css
.card {
  width: 300px;
  height: 200px;
  background-color: #3498db;
  color: #fff;
  padding: 20px;
  transition: transform 0.3s ease;
}

.card:hover {
  transform: skew(10deg, 5deg);
}

```

```html
<div class="card">
  <h2>Cartão Inclinável</h2>
  <p>Este cartão se inclina ao passar o mouse, criando um efeito dinâmico e moderno.</p>
</div>

```

*Explicação:*

Quando o usuário passa o mouse sobre o cartão, ele é inclinado 10 graus horizontalmente e 5 graus verticalmente, gerando um efeito visual que destaca o elemento de forma sutil.

---

### Exemplo 2: Uso de skewX() para um Efeito Horizontal

```css
.banner {
  width: 100%;
  height: 150px;
  background-image: url('banner.jpg');
  background-size: cover;
  transition: transform 0.4s ease;
}

.banner:hover {
  transform: skewX(15deg);
}

```

```html
<div class="banner"></div>

```

*Explicação:*

Ao passar o mouse sobre o banner, ele é inclinado 15 graus no eixo X, criando um efeito de distorção horizontal que pode ser usado para destacar a seção ou adicionar dinamismo ao design.

---

### Exemplo 3: Combinação com Outras Transformações

```css
.button {
  padding: 10px 20px;
  background-color: #e74c3c;
  color: #fff;
  border: none;
  cursor: pointer;
  transition: transform 0.3s ease;
}

.button:hover {
  transform: translateY(-5px) skewY(10deg);
}

```

```html
<button class="button">Clique Aqui</button>

```

*Explicação:*

Neste exemplo, o botão se move 5px para cima e é inclinado 10 graus no eixo Y quando o usuário passa o mouse sobre ele, criando um efeito interativo que combina deslocamento e distorção para chamar a atenção.

---

## 5. Informações Adicionais

- **Combinação com Transições e Animações:**
    
    Utilizar **skew()** em conjunto com a propriedade `transition` permite criar animações suaves que melhoram a interatividade da interface.
    
- **Impacto Visual:**
    
    A inclinação pode ser sutil ou marcante, dependendo dos valores escolhidos. É importante testar diferentes ângulos para encontrar o equilíbrio ideal que complemente o design sem comprometer a legibilidade do conteúdo.
    
- **Contexto de Uso:**
    
    **skew()** é ideal para designs modernos e interativos, onde efeitos de distorção podem adicionar dinamismo e personalidade. No entanto, deve ser usado com moderação para não prejudicar a compreensão do conteúdo.
    
- **Performance:**
    
    Como as transformações são realizadas na camada de composição (GPU-accelerated), o uso de **skew()** geralmente não afeta o desempenho, mesmo em animações complexas.

A função **skew()** é uma ferramenta versátil dentro da propriedade **transform** que permite criar efeitos de inclinação dinâmicos e visualmente interessantes.
