# rotate()

## 1. Introdução

A função **rotate()** é parte da propriedade **transform** em CSS e permite girar um elemento em torno de um ponto fixo. Essa transformação é aplicada sem alterar o fluxo do documento, pois atua na camada de composição, garantindo animações suaves e desempenho otimizado. O **rotate()** é amplamente utilizado para criar efeitos interativos, animar elementos ou simplesmente ajustar a orientação visual de componentes da interface.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **rotate()** gira um elemento em um ângulo especificado, medido em graus (°), radianos (rad), gradians (grad) ou turns (turn). Por padrão, o elemento gira em torno de seu centro, mas esse ponto de origem pode ser alterado com a propriedade **transform-origin**.
    
- **Objetivos e Benefícios:**
    - **Criação de Animações Dinâmicas:** Permite a rotação suave de elementos, ideal para interações como hover, cliques ou transições programadas.
    - **Estética e Design:** Ajuda a ajustar a orientação de ícones, botões e outros componentes, contribuindo para layouts modernos e criativos.
    - **Desempenho:** Como a transformação ocorre na camada de composição (GPU-accelerated), ela não causa reflows e repinturas onerosas, resultando em animações fluidas.
- **Relação com Outras Funções de Transformação:**
    
    **rotate()** pode ser combinada com outras funções de transformação, como `translate()`, `scale()` e `skew()`, permitindo a criação de composições visuais complexas.
    

---

## 3. Sintaxe e Valores

A sintaxe básica para **rotate()** é:

```css
transform: rotate(<ângulo>);

```

- **<ângulo>:**
Especifica o grau de rotação. O valor pode ser medido em:
    - **Graus (°):** Exemplo: `rotate(45deg)` gira o elemento 45 graus no sentido horário.
    - **Radianos (rad):** Exemplo: `rotate(1rad)`.
    - **Gradians (grad):** Exemplo: `rotate(100grad)`.
    - **Turns (turn):** Exemplo: `rotate(0.5turn)` (equivalente a 180°).

**Exemplo Simples:**

```css
.element {
  transform: rotate(30deg);
}

```

*Explicação:*

O elemento `.element` será girado 30 graus no sentido horário em relação ao seu centro (ponto de origem padrão).

---

## 4. Exemplos Práticos

### Exemplo 1: Rotação Simples em um Elemento

```css
.box {
  width: 150px;
  height: 150px;
  background-color: #3498db;
  transition: transform 0.4s ease;
}

.box:hover {
  transform: rotate(45deg);
}

```

```html
<div class="box"></div>

```

*Explicação:*

Quando o usuário passa o mouse sobre a caixa, ela gira 45 graus, criando um efeito interativo e dinâmico sem alterar a posição ou o fluxo do documento.

---

### Exemplo 2: Rotação com Ponto de Origem Alterado

```css
.card {
  width: 200px;
  height: 100px;
  background-color: #e74c3c;
  transition: transform 0.4s ease;
  transform-origin: left center; /* Define o ponto de rotação no lado esquerdo */
}

.card:hover {
  transform: rotate(-30deg);
}

```

```html
<div class="card">Cartão Rotacionado</div>

```

*Explicação:*

Neste exemplo, o cartão gira -30 graus (no sentido anti-horário) com o ponto de origem definido no lado esquerdo, resultando em um efeito de rotação que parece "abrir" a partir de sua borda.

---

### Exemplo 3: Rotação 3D com rotateX() e rotateY()

Embora **rotate()** seja para rotação 2D, o CSS também oferece funções 3D para uma experiência mais imersiva:

```css
.box3d {
  width: 150px;
  height: 150px;
  background-color: #2ecc71;
  transition: transform 0.6s ease;
  perspective: 500px; /* Necessário para efeitos 3D */
}

.box3d:hover {
  transform: rotateX(20deg) rotateY(30deg);
}

```

```html
<div class="box3d"></div>

```

*Explicação:*

Ao passar o mouse sobre a caixa 3D, ela gira 20 graus em torno do eixo X e 30 graus em torno do eixo Y, criando uma sensação de profundidade e dinamismo.

---

### Exemplo 4: Combinação com Outras Transformações

Você pode combinar **rotate()** com outras funções de transformação para criar efeitos complexos:

```css
.button {
  padding: 10px 20px;
  background-color: #9b59b6;
  color: #fff;
  border: none;
  transition: transform 0.3s ease;
}

.button:hover {
  transform: translateY(-5px) rotate(10deg) scale(1.05);
}

```

```html
<button class="button">Clique Aqui</button>

```

*Explicação:*

O botão se move ligeiramente para cima (`translateY(-5px)`), gira 10 graus e aumenta de tamanho em 5% (`scale(1.05)`) quando o usuário passa o mouse sobre ele, criando um efeito interativo e visualmente atraente.

---

## 5. Informações Adicionais

- **Ponto de Origem:**
    
    O comportamento de **rotate()** depende do ponto de origem do elemento, que por padrão é o centro. Use a propriedade **transform-origin** para ajustar esse ponto e alterar o efeito da rotação.
    
- **Desempenho:**
    
    As transformações com **rotate()** são geralmente aceleradas pela GPU, resultando em animações suaves e eficientes sem causar reflows.
    
- **Unidades e Responsividade:**
    
    Os ângulos são unidades independentes (graus, radianos, etc.) e, embora não se ajustem automaticamente a tamanhos de tela, podem ser combinados com media queries para alterar os efeitos de rotação conforme necessário.
    
- **Combinação com Transições e Animações:**
    
    Integrar **rotate()** com `transition` ou `animation` permite criar efeitos dinâmicos e interativos, melhorando a experiência do usuário.
