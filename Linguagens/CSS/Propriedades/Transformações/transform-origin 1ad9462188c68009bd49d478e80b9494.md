# transform-origin

## 1. Introdução

A propriedade **transform-origin** define o ponto de origem para as transformações aplicadas a um elemento. Esse ponto é o "centro" em torno do qual as transformações — como rotação, escala ou inclinação — são realizadas. Ao especificar a origem da transformação, você tem controle preciso sobre como um elemento se comporta quando sujeito a mudanças visuais, podendo alterar dramaticamente o efeito final. Essa propriedade é fundamental para criar animações e efeitos 3D realistas e para ajustar a estética de transformações sem alterar a posição real do elemento no fluxo do documento.

---

## 2. Conceitos Fundamentais

- **Definição:**
    
    **transform-origin** determina o ponto fixo de referência, dentro de um elemento, em relação ao qual as transformações definidas pela propriedade **transform** serão aplicadas.
    
- **Importância:**
    - Permite personalizar a rotação, escala e outras transformações.
    - É essencial para animações 3D, onde a profundidade e o eixo de rotação precisam ser definidos com precisão.
    - Alinha visualmente o efeito desejado com a intenção do design, influenciando como o elemento “gira” ou “cresce” em relação a um ponto específico.

---

## 3. Sintaxe e Estrutura

A sintaxe básica da propriedade é:

```css
seletor {
    transform-origin: valorX valorY [valorZ];
}

```

- **valorX:** Define a posição horizontal do ponto de origem. Pode ser especificado com valores percentuais (ex.: `50%`), pixels (ex.: `100px`) ou palavras-chave (`left`, `center`, `right`).
- **valorY:** Define a posição vertical do ponto de origem. Aceita valores percentuais, pixels ou palavras-chave (`top`, `center`, `bottom`).
- **valorZ (opcional):** Utilizado em transformações 3D para definir a posição no eixo Z, geralmente em pixels (ex.: `0px`).

**Exemplos:**

- Ponto de origem central (valor padrão):
    
    ```css
    .elemento {
        transform-origin: center;
    }
    
    ```
    
- Ponto de origem no canto superior esquerdo:
    
    ```css
    .elemento {
        transform-origin: top left;
    }
    
    ```
    
- Ponto de origem com coordenadas personalizadas e valor Z para transformações 3D:
    
    ```css
    .elemento-3d {
        transform-origin: 30% 70% 50px;
    }
    
    ```
    

---

## 4. Funcionamento e Impacto Visual

### 4.1. Ponto de Origem e Transformações

- **Rotação:**
    
    Ao aplicar uma rotação com `transform: rotate(45deg);`, o elemento gira em torno do ponto especificado por **transform-origin**. Se o ponto estiver no centro (`50% 50%` ou `center`), a rotação ocorrerá em torno do centro do elemento. Alterar esse ponto pode fazer com que o elemento gire em torno de uma borda ou de qualquer outra posição interna.
    
- **Escala:**
    
    Em transformações de escala (`transform: scale(1.5);`), o elemento se amplia ou contrai em relação ao ponto de origem. Por exemplo, definir `transform-origin: top left;` fará com que a ampliação ocorra a partir do canto superior esquerdo, mantendo esse canto fixo.
    
- **Inclinação (skew):**
    
    O efeito de inclinação também é afetado pelo ponto de origem, que determina a direção e a intensidade da distorção visual.
    

### 4.2. Ponto de Origem em Transformações 3D

- Para efeitos 3D, o valor do eixo Z em **transform-origin** adiciona uma dimensão extra. Isso permite que você defina não apenas onde a transformação ocorre no plano, mas também a profundidade do ponto de referência.
- Exemplo:

Nesse caso, a rotação em torno do eixo Y será feita a partir de um ponto localizado no centro do elemento, mas 100px afastado do plano da tela.
    
    ```css
    .elemento-3d {
        transform-origin: center center 100px;
        transform: rotateY(45deg);
    }
    
    ```

## 5. Exemplos de Código Otimizados

### Exemplo 1: Rotação com Ponto de Origem Central

```css
.rotacionar {
    width: 150px;
    height: 150px;
    background-color: #3498db;
    transform: rotate(45deg);
    transform-origin: center; /* Equivalente a 50% 50% */
    transition: transform 0.5s ease;
}

```

```html
<div class="rotacionar">
    Rotacionado a partir do centro
</div>

```

### Exemplo 2: Escala a partir do Canto Superior Esquerdo

```css
.escalar {
    width: 150px;
    height: 150px;
    background-color: #e74c3c;
    transform: scale(1.2);
    transform-origin: top left;
    transition: transform 0.5s ease;
}

```

```html
<div class="escalar">
    Escala a partir do canto superior esquerdo
</div>

```

### Exemplo 3: Transformação 3D com Ponto de Origem Personalizado

```css
.elemento-3d {
    width: 200px;
    height: 200px;
    background-color: #2ecc71;
    transform: perspective(600px) rotateX(30deg);
    transform-origin: 40% 60% 50px;
    transition: transform 0.7s ease-in-out;
}

```

```html
<div class="elemento-3d">
    Efeito 3D com ponto de origem customizado
</div>

```

## 6. Informações Adicionais

- **Transform-Origin Padrão:**
    
    Se não especificado, o padrão é `50% 50%` (ou `center center`), o que significa que as transformações ocorrerão a partir do centro do elemento.
    
- **Interação com Outras Propriedades:**
    
    A propriedade **transform-origin** funciona em conjunto com **transform** e pode ser combinada com transições e animações para criar efeitos mais dinâmicos.
    
    Exemplo:
    
    ```css
    .animado {
        transform: rotate(0deg);
        transform-origin: bottom right;
        transition: transform 0.5s ease;
    }
    
    .animado:hover {
        transform: rotate(90deg);
    }
    
    ```
    
- **Unidades de Medida:**
    
    Você pode usar porcentagens, pixels e outras unidades para definir os valores de **transform-origin**. Percentagens são relativas ao tamanho do elemento, enquanto pixels fornecem um ponto fixo.
    
- **Importância nas Animações:**
    
    Alterar o ponto de origem durante uma animação pode criar efeitos de transição interessantes, como um botão que "salta" ou gira em torno de um canto específico.

## 8. Conclusão

A propriedade **transform-origin** é essencial para controlar o ponto a partir do qual as transformações ocorrem, oferecendo um nível avançado de personalização visual em CSS. Ao definir cuidadosamente o ponto de origem, você pode alterar significativamente o efeito de rotações, escalas, inclinações e transformações 3D, garantindo que a animação ou o efeito visual se alinhe perfeitamente com a intenção de design. Dominar **transform-origin** — juntamente com a propriedade **transform** — permite criar interfaces dinâmicas e interativas que melhoram a experiência do usuário e elevam o padrão estético dos projetos web.
