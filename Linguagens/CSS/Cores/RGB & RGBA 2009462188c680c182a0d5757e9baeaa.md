# RGB & RGBA

---

## 1. Introdução

As cores em CSS também podem ser especificadas diretamente pelos seus componentes vermelho, verde e azul, usando as funções `rgb()` e `rgba()`.

- **Relevância**: são intuitivas para quem pensa em termos de valores decimais ou percentuais de cada canal, e permitem controle preciso da opacidade via `rgba()`.
- **Definições e Conceitos Fundamentais**:
    - **Tema principal**: “Notação `rgb()` e `rgba()` no CSS3”.
    - **Subtemas**:
        1. Sintaxe básica de `rgb()`
        2. Sintaxe de `rgba()` (com canal alfa)
        3. Interpretação de valores decimais e percentuais
        4. Construção de cores específicas
    - **Para que servem**: definir cores visuais e transparências sem recorrer a conversões hexadecimais, facilitando controles dinâmicos e programáticos.

## 3. Conteúdo Detalhado

### 3.1. Sintaxe e Estrutura

- **`rgb()`**
    
    ```css
    rgb(<red>, <green>, <blue>)
    
    ```
    
    - Cada parâmetro pode ser:
        - **Número inteiro** de 0 a 255
        - **Percentual** de 0% a 100%
    - Exemplo:
        
        ```css
        color: rgb(255, 128, 0);   /* laranja puro */
        color: rgb(100%, 50%, 0%); /* mesmo laranja, em % */
        
        ```
        
- **`rgba()`**
    
    ```css
    rgba(<red>, <green>, <blue>, <alpha>)
    
    ```
    
    - Os três primeiros parâmetros seguem `rgb()`.
    - `<alpha>` é a opacidade, **número** de 0.0 (totalmente transparente) a 1.0 (totalmente opaco).
    - Exemplo:
        
        ```css
        background: rgba(255, 128, 0, 0.6); /* laranja com 60% de opacidade */
        
        ```
        

### 3.2. Componentes Principais

- **Canais de Cor (R, G, B)**
    1. **Red**: valor de 0 a 255 ou 0%–100%
    2. **Green**: idem
    3. **Blue**: idem
- **Canal Alfa** (`rgba()` somente)
    - Valor de 0.0 (transparente) a 1.0 (opaco)
- **Porcentagens vs Inteiros**
    - Percentual é convertido internamente para escala 0–255.
        - Ex.: `50%` → 127.5 → arredondado para 128.

### 3.3. Restrições de Uso

- **Compatibilidade**: suportado em todos os navegadores modernos e antigos.
- **Mistura de unidades**: é válido misturar inteiros e percentuais num mesmo `rgb()`.
- **Alpha de ponto flutuante**: prefira até duas casas decimais (e.g., `0.25`, `0.75`).
- **Case-insensitive**: funções podem ser escritas em maiúsculas ou minúsculas (`RGB()`, `rgba()`).

### 3.4. Interpretação de Valores e Construção de Cores

1. **Escolha do tom**: defina qual intensidade de cada canal você deseja.
2. **Decidir unidade**:
    - **Inteiro**: quando souber o valor exato (e.g., do Photoshop).
    - **Percentual**: quando pensar em frações da intensidade total.
3. **Converter percentual → inteiro** (se necessário):
    
    ```
    valor_decimal = percentual/100 × 255
    exemplo: 75% → 0.75 × 255 ≈ 191
    
    ```
    
4. **Para `rgba()`**: defina o nível de transparência desejado como um decimal entre 0 e 1.
5. **Montagem**:
    
    ```css
    /* RGB puro */
    color: rgb(191, 64, 128);
    /* RGB com alfa */
    background: rgba(191, 64, 128, 0.4);
    
    ```

## 4. Exemplos de Código Otimizados

```css
/* 1. rgb() com inteiros */
.title {
  color: rgb(34, 49, 63); /* tom de cinza-azulado */
}

/* 2. rgb() com percentuais */
.subtitle {
  color: rgb(10%, 20%, 30%); /* equivalente a rgb(25, 51, 76) */
}

/* 3. rgba() para overlay */
.overlay {
  background: rgba(0, 0, 0, 0.5); /* preto semi-transparente */
  position: fixed;
  inset: 0;
}

/* 4. Mistura de unidades */
.card {
  border: 2px solid rgba(255, 0, 0, 0.8);
  background-color: rgb(100%, 90%, 80%);
}

/* 5. Uso com variáveis CSS */
:root {
  --error-rgb: 220, 20, 60; /* crimson em valores inteiros */
}
.alert {
  background: rgba(var(--error-rgb), 0.3); /* usa o mesmo RGB com 30% alfa */
  color: rgb(var(--error-rgb));
  padding: 1rem;
}

```

## 5. Informações Adicionais

- **Animações**: ao animar entre `rgb()` e `rgba()`, browsers interpolam suavemente cada canal.
- **Ferramentas**: muitos editores (VS Code, Sublime) oferecem autocompletes que sugerem valores `rgb()` após digitar `rgb(`.
- **Acessibilidade**: para contraste de texto, use ferramentas que validem relação de contraste entre cores RGB.
