# Linear Gradient

---

## 1. Introdução

O `linear-gradient()` do CSS é uma função de imagem de fundo que gera transições suaves entre duas ou mais cores ao longo de uma linha. É amplamente usado para criar efeitos visuais dinâmicos sem depender de imagens externas.

- **Relevância**:
    - Substitui imagens estáticas, reduzindo requisições HTTP e melhorando performance.
    - Oferece flexibilidade para criar múltiplos pontos de cor e controlar exatamente direção, ângulo e posições.
- **Definições e Conceitos Fundamentais**:
    - **Tema principal**: “Uso de `linear-gradient()` no CSS3”.
    - **Subtemas**:
        1. Sintaxe básica e direção
        2. Color stops (pontos de cor) e posições
        3. Gradientes repetidos
        4. Aplicações em propriedades diversas
    - **Para que serve**: gerar fundos, bordas e máscaras com transições de cor controladas.

## 3. Conteúdo Detalhado

### 3.1. Sintaxe e Estrutura

```css
background-image: linear-gradient(
  [ <angle> | to <side-or-corner> ]?,
  <color-stop1> [ <position1> ]?,
  <color-stop2> [ <position2> ]?,
  …
);

```

- Parâmetros principais:
    - **Direção** (opcional):
        - `<angle>`: em graus (`0deg` aponta para cima, `90deg` para a direita, etc.).
        - `to <side-or-corner>`: palavras-chave (`to right`, `to bottom left`, etc.).
    - **Color stops**: lista de cores com posições opcionais.

### 3.2. Color Stops e Posições

- **Color stop**: ponto onde uma cor específica é aplicada.
    
    ```css
    linear-gradient(red 0%, yellow 50%, blue 100%);
    
    ```
    
- **Posição**: valor em `%` ou unidades (`px`, `em`), determina onde a cor aparece na linha do degradê.
    - Sem posição, o navegador distribui os stops igualmente.

### 3.3. Direção e Ângulo

- **Ângulo**: define a orientação do degradê.
    - `0deg` → de baixo para cima
    - `90deg` → da esquerda para a direita
    - Exemplo:
        
        ```css
        linear-gradient(45deg, #8e44ad, #f1c40f);
        
        ```
        
- **Palavras-chave**:
    - `to right` (esquerda→direita)
    - `to bottom` (topo→fundo)
    - `to top left`, `to bottom right`, etc.

### 3.4. Gradientes Repetidos

- Variante que repete o padrão indefinidamente:
    
    ```css
    background-image: repeating-linear-gradient(
      to right,
      #fff 0,
      #fff 10px,
      #000 10px,
      #000 20px
    );
    
    ```
    

### 3.5. Aplicações em Outras Propriedades

- **`border-image`**:
    
    ```css
    border: 8px solid;
    border-image: linear-gradient(to right, red, blue) 1;
    
    ```
    
- **Máscaras** (com `mask-image`):
    
    ```css
    mask-image: linear-gradient(to bottom, transparent, black);
    
    ```
    
- **Gradientes em texto** (com `background-clip` e `text-fill-color`):
    
    ```css
    .gradient-text {
      background: linear-gradient(90deg, #e66465, #9198e5);
      -webkit-background-clip: text;
      -webkit-text-fill-color: transparent;
    }
    
    ```

## 4. Exemplos de Código Otimizados

```css
/* 1. Degrade simples diagonal */
.banner {
  background-image: linear-gradient(135deg, #3498db, #2ecc71);
  color: white;
  padding: 2rem;
}

/* 2. Vários pontos de cor com posições */
.card {
  background: linear-gradient(
    to bottom right,
    #f1c40f 0%,    /* amarelo começa no topo */
    #e67e22 40%,   /* laranja no meio */
    #d35400 70%,   /* laranja escuro */
    #c0392b 100%   /* vermelho no fim */
  );
  border-radius: 8px;
  padding: 1.5rem;
  color: #fff;
}

/* 3. Gradiente repetido para padrão listrado */
.striped {
  background-image: repeating-linear-gradient(
    to right,
    #fff 0,
    #fff 5px,
    #ccc 5px,
    #ccc 10px
  );
  height: 50px;
}

/* 4. Uso em border-image */
.btn {
  border: 4px solid;
  border-image: linear-gradient(to right, #8e44ad, #3498db) 1;
}

/* 5. Texto com degradê */
.heading {
  font-size: 3rem;
  background: linear-gradient(90deg, #e66465, #9198e5);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

```

## 5. Informações Adicionais

- **Animação**: é possível animar os parâmetros do degradê (ângulo, stops) para efeitos dinâmicos.
- **Performance**: degradês são renderizados pela GPU; complexidades moderadas não impactam significativamente o desempenho.
- **Acessibilidade**: use contrastes adequados sobre degradês ou sobreponha camadas semi-opacas para garantir legibilidade.
