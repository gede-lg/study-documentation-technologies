# Conic Gradient

---

## 1. Introdução

O `conic-gradient()` do CSS cria um degradê angular, distribuindo as cores em torno de um ponto central, como fatias de uma pizza. Diferente dos gradientes lineares e radiais, o cônico percorre um círculo completo (360°), oferecendo efeitos de transição giratória.

- **Relevância**:
    - Ideal para representar dados circulares (gráficos de pizza), indicadores de progresso e efeitos visuais criativos.
    - Substitui imagens estáticas por uma solução 100% em CSS, com controle preciso de ângulo e paradas de cor.
- **Definições e Conceitos Fundamentais**:
    - **Tema principal**: “Uso de `conic-gradient()` no CSS3”.
    - **Subtemas**:
        1. Sintaxe básica e ângulo inicial
        2. Posições de cor (color stops)
        3. Centro do degradê
        4. Versão repetida (`repeating-conic-gradient()`)
    - **Para que serve**: criar transições angulares de cor ao redor de um ponto central.

## 3. Conteúdo Detalhado

### 3.1. Sintaxe e Estrutura

```css
background-image: conic-gradient(
  [ from <angle> ]?    /* ângulo inicial */
  [ at <position> ]?,  /* posição do centro */
  <color-stop1> [ <position1> ]?,
  <color-stop2> [ <position2> ]?,
  …
);

```

- `from <angle>`: ponto de início do degradê (padrão: 0deg, topo).
- `at <position>`: define o centro (`center`, `top left`, coordenadas etc.).

### 3.2. Color Stops e Ângulo Inicial

- **Color stop**: cor + posição angular opcional (em graus ou percentuais).
    
    ```css
    conic-gradient(
      red 0deg,
      yellow 90deg,
      green 180deg,
      blue 270deg
    );
    
    ```
    
- **Ângulo inicial**: desloca todo o degradê em relação ao topo.
    
    ```css
    conic-gradient(
      from 45deg,
      red, yellow, green, blue
    );
    
    ```
    

### 3.3. Definição do Centro

- O degradê gira em torno de um ponto:
    - `at center` (padrão)
    - `at top left`, `at 30% 70%`, etc.
    
    ```css
    conic-gradient(
      at bottom right,
      #e66465, #9198e5
    );
    
    ```
    

### 3.4. Gradiente Repetido

- Cria um padrão circular repetido:
    
    ```css
    background-image: repeating-conic-gradient(
      from 0deg at center,
      #fff 0deg 30deg,
      #000 30deg 60deg
    );
    
    ```
    
- Útil para tramas, marcadores de seção ou indicadores.

### 3.5. Restrições de Uso

- **Compatibilidade**: suportado em Chrome ≥ 69, Firefox ≥ 83, Edge ≥ 79, Safari ≥ 12.1.
- **Unidades válidas**: `deg`, `%` (proporção de 360°), `turn`.
- **Performance**: adequado para animações suaves em torno do centro.

## 4. Exemplos de Código Otimizados

```css
/* 1. Gráfico de pizza simples */
.pie-chart {
  width: 200px;
  height: 200px;
  border-radius: 50%;
  background-image: conic-gradient(
    red 0deg 25%,
    yellow 25% 50%,
    green 50% 75%,
    blue 75% 100%
  );
}

/* 2. Indicador de progresso circular */
.progress-ring {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  background-image: conic-gradient(
    #3498db 0deg var(--progress),
    #eee var(--progress) 360deg
  );
}

/* 3. Gradiente cônico deslocado e centralizado */
.banner {
  background-image: conic-gradient(
    from 45deg at 30% 70%,
    #8e44ad, #f1c40f, #e74c3c
  );
  padding: 2rem;
  color: #fff;
}

/* 4. Padrão repetido para efeito de rotação */
.spinner {
  width: 100px;
  height: 100px;
  border-radius: 50%;
  background-image: repeating-conic-gradient(
    #000 0deg 15deg,
    transparent 15deg 30deg
  );
  animation: spin 1s linear infinite;
}

@keyframes spin {
  to { transform: rotate(360deg); }
}

```

---

## 5. Informações Adicionais

- **Animações**: variando `<angle>` de `from` dinamicamente, cria-se efeito de rotação do degradê.
- **Interação com variáveis CSS**: usar `var(--progress)` facilita controles via JavaScript.
- **Uso em máscaras**: combinado com `mask-image` para revelar elementos de forma circular.
