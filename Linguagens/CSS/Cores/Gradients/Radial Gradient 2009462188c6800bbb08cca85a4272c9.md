# Radial Gradient

Você quer entender sobre `radial-gradient()` no CSS, assim como vimos o `linear-gradient()`. É uma ótima escolha, porque ele permite criar efeitos visuais bem diferentes, com cores que se espalham a partir de um centro.

---

## **1. Introdução**

O `radial-gradient()` do CSS é uma função de imagem de fundo que gera transições suaves entre duas ou mais cores, mas desta vez, em um **formato circular ou elíptico**, partindo de um ponto central. Ele é perfeito para criar efeitos de brilho, focos de luz, esferas ou até mesmo padrões repetidos, tudo sem precisar de imagens externas.

### **Relevância:**

- **Flexibilidade Criativa:** Diferente do `linear-gradient()`, o `radial-gradient()` oferece uma nova dimensão para seus designs, permitindo criar efeitos que simulam profundidade ou iluminação.
- **Performance Otimizada:** Assim como outros gradientes CSS, ele reduz o número de requisições HTTP, já que não usa arquivos de imagem, otimizando o carregamento da página.
- **Controle Preciso:** Você pode controlar o formato (círculo ou elipse), a posição do centro, o tamanho e a distribuição das cores.

### **Definições e Conceitos Fundamentais:**

- **Tema principal:** "Uso de `radial-gradient()` no CSS3".
- **Subtemas:**

    - `Color stops` (pontos de cor) e posições

    - Gradientes radiais repetidos
    - Aplicações em propriedades diversas
- **Para que serve:** Gerar fundos, bordas e máscaras com transições de cor que irradiam de um ponto.

## **3. Conteúdo Detalhado**

### **3.1. Sintaxe e Estrutura**

A sintaxe básica do `radial-gradient()` é um pouco diferente do linear:

**CSS**

`background-image: radial-gradient(
  [ <shape> || <size> ]? [ at <position> ]?,
  <color-stop1> [ <position1> ]?,
  <color-stop2> [ <position2> ]?,
  …
);`

### **Parâmetros principais:**

- **`<shape>` (opcional):** Define a forma do gradiente. Pode ser `circle` (círculo perfeito) ou `ellipse` (elipse, o padrão se omitido).
- **`<size>` (opcional):** Define o tamanho do gradiente radial. Palavras-chave como `closest-side`, `farthest-side`, `closest-corner`, `farthest-corner` controlam até onde o gradiente se estende. O padrão é `farthest-corner`.
- **`<position>` (opcional):** Define a posição do centro do gradiente (onde a primeira cor começa). Por padrão, é `center`. Você pode usar palavras-chave (ex: `top left`) ou valores numéricos (ex: `50% 50%`, `20px 30px`).
- **`<color-stop>`:** A lista de cores com posições opcionais, exatamente como no `linear-gradient()`.

### **3.2. `Color Stops` e Posições**

Funciona da mesma forma que no `linear-gradient()`. Cada `color stop` é um ponto onde uma cor específica é aplicada.

**CSS**

`radial-gradient(circle, red 0%, yellow 50%, blue 100%);`

- A **posição** é um valor em `%` ou unidades (px, em) que determina onde a cor aparece na linha imaginária do degradê radial.
- Se nenhuma posição for especificada, o navegador distribui os `stops` igualmente.

### **3.3. Forma e Extensão**

### **Forma (`<shape>`):**

- **`circle`:** Cria um gradiente circular. O raio é o mesmo em todas as direções.
    
    **CSS**
    
    `radial-gradient(circle, #f06, #3cc);`
    
- **`ellipse`:** Cria um gradiente elíptico (o padrão). Permite diferentes raios horizontal e vertical.
    
    **CSS**
    
    `radial-gradient(ellipse, #f06, #3cc);`
    

### **Extensão (`<size>`):**

As palavras-chave de tamanho definem o raio do gradiente em relação à caixa do elemento:

- **`closest-side`:** O gradiente termina na borda mais próxima do centro.
- **`farthest-side` (padrão):** O gradiente termina na borda mais distante do centro.
- **`closest-corner`:** O gradiente termina no canto mais próximo do centro.
- **`farthest-corner`:** O gradiente termina no canto mais distante do centro.

Você também pode especificar valores numéricos para o raio (ex: `100px 50px` para uma elipse com 100px de raio horizontal e 50px de raio vertical).

### **3.4. Gradientes Radiais Repetidos**

A variante `repeating-radial-gradient()` repete o padrão do gradiente infinitamente. É excelente para criar fundos com texturas ou padrões pontilhados/circulares.

**CSS**

`background-image: repeating-radial-gradient(
  circle,
  #fff 0,
  #fff 10px,
  #000 10px,
  #000 20px
);`

Este exemplo criaria um padrão de círculos concêntricos brancos e pretos.

### **3.5. Aplicações em Outras Propriedades**

Assim como o `linear-gradient()`, o `radial-gradient()` pode ser usado em:

- **`border-image`:** Para bordas com um efeito de irradiação.
    
    **CSS**
    
    `border: 8px solid;
    border-image: radial-gradient(circle, red, blue) 1;`
    
- **Máscaras (`mask-image`):** Para revelar ou esconder partes de um elemento de forma circular/elíptica.
    
    **CSS**
    
    `mask-image: radial-gradient(circle at center, black 0%, transparent 100%);`
    
- **Gradientes em texto (com `background-clip` e `webkit-text-fill-color`):** Para preencher o texto com um efeito radial.
    
    **CSS**
    
    `.gradient-text-radial {
      background: radial-gradient(circle, #ffe000, #ff0000); /* Ex: amarelo para vermelho */
      -webkit-background-clip: text;
      -webkit-text-fill-color: transparent;
    }`

## **4. Exemplos de Código Otimizados**

**CSS**

`/* 1. Degradê circular simples no centro */
.circle-bg {
  background-image: radial-gradient(circle, #3498db, #2ecc71);
  width: 200px;
  height: 200px;
  border-radius: 50%; /* Para realmente ver um círculo */
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* 2. Elipse com centro deslocado e múltiplos pontos de cor */
.focused-card {
  background: radial-gradient(
    ellipse at top left, /* O gradiente começa no canto superior esquerdo */
    #f1c40f 0%,         /* Amarelo forte no centro */
    #e67e22 50%,         /* Laranja no meio */
    #c0392b 100%         /* Vermelho na borda */
  );
  border-radius: 8px;
  padding: 1.5rem;
  color: #fff;
}

/* 3. Gradiente radial repetido para um padrão de bolinhas */
.dotted-pattern {
  background-image: repeating-radial-gradient(
    circle at center,
    #eee 0,           /* Cor clara */
    #eee 5px,         /* Termina a bolinha clara */
    #aaa 5px,         /* Começa a bolinha escura */
    #aaa 10px         /* Termina a bolinha escura */
  );
  height: 100px;
}

/* 4. Uso em border-image com foco central */
.framed-box {
  border: 6px solid;
  border-image: radial-gradient(circle at center, #8e44ad, #3498db) 1;
  padding: 1rem;
}

/* 5. Texto com degradê radial */
.heading-radial {
  font-size: 3.5rem;
  background: radial-gradient(circle at center, #ff8c00, #ff0000); /* Laranja para vermelho */
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  font-weight: bold;
}`

## **5. Informações Adicionais**

- **Animação:** Assim como o `linear-gradient()`, você pode animar a posição do centro ou o tamanho do `radial-gradient()` para criar efeitos dinâmicos e interativos.
- **Performance:** Gradientes CSS são renderizados pela GPU, então, para complexidades moderadas, o impacto no desempenho é mínimo. Use-os sem medo!
- **Acessibilidade:** Ao usar gradientes, sempre se certifique de que o **contraste** entre o texto e o fundo (ou quaisquer elementos interativos) seja suficiente para garantir a legibilidade. Ferramentas de análise de contraste podem ajudar nisso.
