### O que é e para que serve?

No Dart, o tipo `int` representa números inteiros, que são números sem casas decimais. Eles são usados para armazenar e manipular valores numéricos inteiros em operações matemáticas, contagem, índices de arrays, e muitas outras situações onde números inteiros são necessários.

### Sintaxe de uso

A declaração de uma variável do tipo `int` é simples e direta. Aqui está um exemplo básico de como declarar e inicializar uma variável inteira em Dart:

```dart
int idade = 30;
```

Você pode também declarar uma variável do tipo `int` sem inicializá-la imediatamente:

```dart
int numero;
numero = 10;
```

### Restrições de uso

#### Limitações de valor

Em Dart, o valor de uma variável do tipo `int` pode variar dependendo da plataforma:

- **Plataformas de 64 bits**: Em uma plataforma de 64 bits, um `int` pode representar valores entre `-2^63` a `2^63 - 1`.
- **Plataformas de 32 bits**: Em uma plataforma de 32 bits, um `int` pode representar valores entre `-2^31` a `2^31 - 1`.

Essa diferença é importante em situações que envolvem a portabilidade do código entre diferentes arquiteturas.

#### Restrições sintáticas

O tipo `int` só pode armazenar números inteiros. Tentar armazenar um número com casas decimais resultará em um erro de compilação. Por exemplo:

```dart
int numero = 3.14; // Isso resultará em um erro
```

### Conversões

#### Para outros tipos

##### Para `double`

Você pode converter um `int` para `double` facilmente usando a função `.toDouble()`:

```dart
int numero = 42;
double numeroDecimal = numero.toDouble();
print(numeroDecimal); // Saída: 42.0
```

##### Para `String`

Para converter um `int` para `String`, use a função `.toString()`:

```dart
int numero = 123;
String texto = numero.toString();
print(texto); // Saída: "123"
```

#### De outros tipos

##### De `String` para `int`

Para converter uma `String` que representa um número inteiro para um `int`, use a função `int.parse()`:

```dart
String texto = "456";
int numero = int.parse(texto);
print(numero); // Saída: 456
```

##### De `double` para `int`

Para converter um `double` para `int`, você pode usar o casting explícito ou a função `.toInt()`, que truncará a parte decimal:

```dart
double numeroDecimal = 78.56;
int numero = numeroDecimal.toInt();
print(numero); // Saída: 78
```

### Quando utilizar?

Utilize o tipo `int` quando precisar representar valores numéricos inteiros. Alguns exemplos incluem:

- Contagem de itens (por exemplo, número de elementos em uma lista).
- Índices de arrays.
- Representação de valores monetários inteiros (por exemplo, centavos).
- Realização de operações matemáticas que envolvem números inteiros.

### Informações adicionais

- **Operações Aritméticas**: O tipo `int` suporta operações aritméticas básicas, como adição, subtração, multiplicação, divisão (inteira), e módulo.

  ```dart
  int a = 10;
  int b = 3;
  
  print(a + b); // Saída: 13
  print(a - b); // Saída: 7
  print(a * b); // Saída: 30
  print(a ~/ b); // Saída: 3 (divisão inteira)
  print(a % b); // Saída: 1 (resto da divisão)
  ```

- **Incremento e Decremento**: O tipo `int` suporta operações de incremento e decremento.

  ```dart
  int count = 0;
  count++;
  print(count); // Saída: 1
  count--;
  print(count); // Saída: 0
  ```

- **Verificação de Tipos**: Você pode verificar se uma variável é do tipo `int` usando o operador `is`.

  ```dart
  var valor = 10;
  print(valor is int); // Saída: true
  ```

### Conclusão

O tipo `int` no Dart é uma ferramenta essencial para trabalhar com números inteiros. Com uma sintaxe simples e capacidades robustas para conversões e operações matemáticas, o `int` é adequado para uma ampla gama de aplicações. Entender suas restrições e possibilidades é crucial para utilizar o Dart de maneira eficaz e eficiente.
