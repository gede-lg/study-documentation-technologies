### O que é e para que serve?
Um `Set` no Dart é uma coleção de itens únicos, desordenados. Ele é usado para armazenar elementos onde a unicidade é crucial, evitando duplicatas. Diferente das listas, a ordem dos elementos em um `Set` não é garantida, e não é possível acessar elementos por um índice. `Sets` são úteis para operações matemáticas como união, interseção e diferença.

### Sintaxe de Uso

Um `Set` pode ser declarado de várias maneiras em Dart:

```dart
// Declaração de um Set vazio
Set<String> names = {};

// Declaração com elementos iniciais
Set<int> numbers = {1, 2, 3, 4, 5};

// Declaração de um Set a partir de uma lista
List<int> list = [1, 2, 3, 4, 5];
Set<int> numberSet = Set.from(list);
```

### Iteração sobre Sets
Para iterar sobre os elementos de um `Set`, você pode usar um loop `for`:

```dart
Set<String> fruits = {'apple', 'banana', 'mango'};
for (String fruit in fruits) {
  print(fruit);
}
```

### 15 Principais Métodos e Operações de Sets

1. **add(E value)**: Adiciona um elemento ao `Set`.
   ```dart
   numbers.add(6);
   ```

2. **`addAll(Iterable<E> elements)`**: Adiciona todos os elementos de uma coleção ao `Set`.
   ```dart
   numbers.addAll({7, 8, 9});
   ```

3. **remove(Object value)**: Remove um elemento específico do `Set`.
   ```dart
   numbers.remove(3);
   ```

4. **contains(Object element)**: Verifica se o `Set` contém um elemento específico.
   ```dart
   bool hasThree = numbers.contains(3);
   ```

5. **length**: Obtém o número de elementos no `Set`.
   ```dart
   int size = numbers.length;
   ```

6. **clear()**: Remove todos os elementos do `Set`.
   ```dart
   numbers.clear();
   ```

7. **`difference(Set<E> other)`**: Retorna a diferença entre dois `Sets`.
   ```dart
   Set<int> a = {1, 2, 3};
   Set<int> b = {3, 4, 5};
   Set<int> diff = a.difference(b);
   ```

8. **`union(Set<E> other)`**: Retorna a união de dois `Sets`.
   ```dart
   Set<int> unionSet = a.union(b);
   ```

9. **`intersection(Set<E> other)`**: Retorna a interseção de dois `Sets`.
   ```dart
   Set<int> intersectionSet = a.intersection(b);
   ```

10. **isEmpty**: Verifica se o `Set` está vazio.
    ```dart
    bool empty = numbers.isEmpty;
    ```

11. **isNotEmpty**: Verifica se o `Set` não está vazio.
    ```dart
    bool notEmpty = numbers.isNotEmpty;
    ```

12. **first**: Acessa o primeiro elemento do `Set`.
    ```dart
    int firstElement = numbers.first;
    ```

13. **last**: Acessa o último elemento do `Set`.
    ```dart
    int lastElement = numbers.last;
    ```

14. **forEach(void Function(E element) f)**: Aplica uma função a cada elemento do `Set`.
    ```dart
    numbers.forEach((number) => print(number));
    ```

15. **`map<T>(T Function(E e) f)`**: Retorna um novo `Iterable` transformando cada elemento do `Set`.
    ```dart
    Iterable<String> stringNumbers = numbers.map((number) => 'Number $number');
    ```

### Informações Adicionais
- `Sets` não aceitam valores duplicados. Adicionar um valor já existente não terá efeito.
- Os métodos `union`, `intersection` e `difference` são particularmente úteis para operações matemáticas com conjuntos.
- Dart suporta `Sets` dinâmicos e tipados. Sempre que possível, é uma boa prática definir o tipo de elementos que o `Set` deve conter.
- Ao trabalhar com grandes coleções, operações de `Set` podem ser mais eficientes do que operações equivalentes em listas, especialmente para testes de pertencimento e eliminação de duplicatas.
