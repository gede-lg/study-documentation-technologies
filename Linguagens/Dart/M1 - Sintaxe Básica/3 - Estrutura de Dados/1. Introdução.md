# Collections no Dart

No Dart, as Collections são estruturas de dados fundamentais usadas para armazenar e manipular conjuntos de dados. Elas são cruciais para qualquer tipo de programação em Dart, incluindo o desenvolvimento com Flutter. Vamos explorar os tipos principais de collections em Dart, seu propósito, estrutura hierárquica e exemplos de código.

## Tipos de Collections

### 1. List
- **Descrição**: Uma List é uma coleção ordenada e indexada de objetos. Em Dart, as Listas podem ser de tamanho fixo ou dinâmico.
- **Utilização**: Usada para armazenar uma coleção de objetos que são acessados frequentemente por índice.
- **Exemplo**:
  ```dart
  var fruits = ['apple', 'banana', 'mango'];
  print(fruits[0]); // Saída: apple
  ```

### 2. Set
- **Descrição**: Um Set é uma coleção de objetos únicos e não ordenados. É ideal para operações que exigem elementos únicos, como verificações de existência.
- **Utilização**: Usado quando é necessário garantir que não haja duplicatas na coleção.
- **Exemplo**:
  ```dart
  var numbers = Set();
  numbers.addAll([1, 2, 3, 4, 5, 1, 2]);
  print(numbers); // Saída: {1, 2, 3, 4, 5}
  ```

### 3. Map
- **Descrição**: Um Map é uma coleção de pares chave-valor. As chaves são únicas, e cada chave mapeia para exatamente um valor.
- **Utilização**: Usado para associar chaves a valores, útil para buscar dados baseados em identificadores.
- **Exemplo**:
  ```dart
  var capitals = {
    'USA': 'Washington D.C.',
    'India': 'New Delhi',
    'Japan': 'Tokyo'
  };
  print(capitals['USA']); // Saída: Washington D.C.
  ```

## Estrutura Hierárquica das Collections

Em Dart, todas as collections são derivadas da classe `Iterable`. Abaixo está uma representação simplificada de sua hierarquia em forma de texto:

```
Iterable
|
|-- List
|
|-- Set
|
|-- Map
```

- **Iterable**: A classe base para todas as collections. Oferece métodos para iterar sobre os elementos da coleção.

## Tópicos Adicionais

### Imutabilidade
- Algumas collections podem ser imutáveis, o que significa que seus elementos não podem ser alterados após a criação. Por exemplo, a função `List.unmodifiable` cria uma lista imutável.

### Métodos Comuns
- Collections em Dart vêm com uma variedade de métodos úteis, como `map`, `where`, `forEach`, que permitem manipular e acessar os dados de maneira eficiente.

### Spread Operator
- Introduzido no Dart 2.3, o operador spread (`...`) permite inserir todos os elementos de uma coleção em outra.
  ```dart
  var list1 = [1, 2, 3];
  var list2 = [0, ...list1];
  print(list2); // Saída: [0, 1, 2, 3]
  ```

### Collection if e for
- Dart permite a utilização de `if` e `for` dentro de collection literals, o que facilita a criação de collections dinâmicas.
  ```dart
  var promoActive = true;
  var nav = [
    'Home',
    'Furniture',
    'Plants',
    if (promoActive) 'Sale'
  ];
  ```

### Conclusão
As collections no Dart são ferramentas poderosas e flexíveis, essenciais para qualquer desenvolvedor Dart. Compreender suas características e saber quando e como usá-las é crucial para criar códigos eficientes e limpos.
