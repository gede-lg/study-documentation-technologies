
O Fluxo de Repetição, ou loops, são usados para executar um bloco de código repetidamente enquanto uma condição for verdadeira.

> [!NOTE]
> # For

- **Uso**: Executa um bloco de código um número definido de vezes.
- **Sintaxe**:
  ```dart
  for (inicialização; condição; incremento) {
    // código a ser repetido
  }
  ```
- **Exemplo**:
  ```dart
  for (int i = 0; i < 5; i++) {
    print("Número $i");
  }
  ```

> [!NOTE]
> # While

- **Uso**: Repete um bloco de código enquanto a condição especificada é verdadeira.
- **Sintaxe**:
  ```dart
  while (condição) {
    // código a ser repetido
  }
  ```
- **Exemplo**:
  ```dart
  int i = 0;
  while (i < 5) {
    print("Número $i");
    i++;
  }
  ```

> [!NOTE]
> # Do-While

- **Uso**: Semelhante ao `while`, mas garante que o bloco de código seja executado pelo menos uma vez.
- **Sintaxe**:
  ```dart
  do {
    // código a ser repetido
  } while (condição);
  ```
- **Exemplo**:
  ```dart
  int i = 0;
  do {
    print("Número $i");
    i++;
  } while (i < 5);
  ```

### Tópicos Adicionais
#### Uso de `break` e `continue`
- **`break`**: Usado para sair de um loop antes que ele complete sua condição normal.
- **`continue`**: Pula para a próxima iteração do loop.
- **Exemplo de `break`**:
  ```dart
  for (int i = 0; i < 10; i++) {
    if (i == 5) {
      break; // Sai do loop quando i é 5
    }
    print(i);
  }
  ```
- **Exemplo de `continue`**:
  ```dart
  for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
      continue; // Pula a iteração atual se i for par
    }
    print(i);
  }
  ```
#### Condições Encadeadas e Aninhadas
- **Encadeadas**: Sequência de `if`, `else if`, e `else`.
- **Aninhadas**: `if` dentro de outro `if`, útil para verificar múltiplas condições.
