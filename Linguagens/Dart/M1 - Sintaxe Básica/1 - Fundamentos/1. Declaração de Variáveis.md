
No Dart, a declaração de variáveis é essencial para o gerenciamento de dados em um programa. Dart oferece várias maneiras de declarar variáveis, cada uma com suas próprias características e uso específico. Abaixo, explicamos detalhadamente cada um dos tipos de declaração de variáveis em Dart.

> [!NOTE] 
> # var

**O que é e para que serve?**

A palavra-chave `var` é usada para declarar variáveis cuja tipagem é inferida pelo compilador com base no valor atribuído a ela. Serve para declarações onde o tipo da variável é claro e pode ser inferido.

**Sintaxe de uso:**

```dart
var nome = 'Luiz';
var idade = 25;
var altura = 1.75;
```

**Restrições de uso:**

- O tipo é inferido no momento da inicialização e não pode ser alterado posteriormente.
- Uma vez atribuído um valor, a variável assume o tipo desse valor e não pode ser alterada para outro tipo.

**Quando utilizar?**

Utilize `var` quando o tipo da variável for óbvio a partir do contexto e não houver necessidade de especificar explicitamente o tipo.

```dart
void main() {
  var mensagem = 'Olá, mundo!';
  print(mensagem);
}
```

> [!NOTE] 
> # const

**O que é e para que serve?**

A palavra-chave `const` é usada para declarar variáveis com valores constantes e imutáveis em tempo de compilação. Serve para definir valores que são conhecidos e não mudam durante a execução do programa.

**Sintaxe de uso:**

```dart
const PI = 3.14;
const nome = 'Athena';
```

**Restrições de uso:**

- O valor deve ser conhecido em tempo de compilação.
- Não pode ser alterado após a declaração.
- Deve ser inicializado no momento da declaração.

**Quando utilizar?**

Utilize `const` para valores que são constantes durante todo o ciclo de vida do programa e podem ser determinados em tempo de compilação.

```dart
void main() {
  const numeroMaximoDeTentativas = 3;
  print(numeroMaximoDeTentativas);
}
```

> [!NOTE]
> # final

**O que é e para que serve?**

A palavra-chave `final` é usada para declarar variáveis cujo valor é constante, mas a inicialização pode ocorrer em tempo de execução. Serve para definir valores que não mudam após a primeira atribuição.

**Sintaxe de uso:**

```dart
final dataAtual = DateTime.now();
final idade;
idade = 25;
```

**Restrições de uso:**

- O valor pode ser inicializado em tempo de execução, mas não pode ser alterado após a inicialização.
- Não pode ser reatribuído.

**Quando utilizar?**

Utilize `final` para variáveis que devem ser constantes após a inicialização, mas cujo valor não é conhecido até a execução do programa.

```dart
void main() {
  final hoje = DateTime.now();
  print(hoje);
}
```

> [!NOTE]
> # dynamic

**O que é e para que serve?**

A palavra-chave `dynamic` é usada para declarar variáveis que podem conter valores de qualquer tipo. Serve para situações onde o tipo da variável pode mudar durante a execução do programa.

**Sintaxe de uso:**

```dart
dynamic variavel = 'Texto';
variavel = 10;
variavel = true;
```

**Restrições de uso:**

- O tipo pode ser alterado a qualquer momento.
- Pode levar a erros de tempo de execução se não for usado com cuidado.

**Quando utilizar?**

Utilize `dynamic` quando for necessário armazenar valores de diferentes tipos na mesma variável. No entanto, deve ser usado com cautela para evitar erros.

```dart
void main() {
  dynamic valor = 'Texto';
  print(valor);
  valor = 10;
  print(valor);
  valor = true;
  print(valor);
}
```

> [!NOTE]
> # late

**O que é e para que serve?**

A palavra-chave `late` é usada para declarar variáveis que serão inicializadas posteriormente. Serve para evitar inicialização imediata e adiar a atribuição de valor até que seja realmente necessário.

**Sintaxe de uso:**

```dart
late String descricao;
descricao = 'Descrição detalhada';
```

**Restrições de uso:**

- Deve ser inicializada antes do primeiro uso.
- Pode ser utilizada com `final` para variáveis que são definitivamente inicializadas posteriormente.

**Quando utilizar?**

Utilize `late` quando a inicialização imediata não for possível ou desejável e você tiver certeza de que a variável será inicializada antes do uso.

```dart
void main() {
  late String mensagem;
  mensagem = 'Olá, mundo!';
  print(mensagem);
}
```

> [!NOTE]
> # static

**O que é e para que serve?**

A palavra-chave `static` é usada para declarar membros de classe que pertencem à própria classe e não a instâncias específicas da classe. Serve para valores ou métodos que são compartilhados entre todas as instâncias de uma classe.

**Sintaxe de uso:**

```dart
class Contador {
  static int contadorGlobal = 0;

  static void incrementar() {
    contadorGlobal++;
  }
}
```

**Restrições de uso:**

- Pode ser acessado sem criar uma instância da classe.
- Não pode acessar membros de instância diretamente.

**Quando utilizar?**

Utilize `static` para variáveis ou métodos que precisam ser compartilhados entre todas as instâncias de uma classe ou quando você precisa de uma maneira de acessar membros de classe sem instanciar a classe.

```dart
void main() {
  Contador.incrementar();
  print(Contador.contadorGlobal);
}
```

## Conclusão

No Dart, a escolha correta da declaração de variáveis depende do contexto e das necessidades específicas do seu programa. `var` é útil para tipos inferidos, `const` e `final` para constantes, `dynamic` para tipos dinâmicos, `late` para inicialização tardia, e `static` para membros de classe. Compreender as diferenças e restrições de cada um permite um código mais claro, eficiente e seguro.
