# Operadores em Dart

## Introdução

Os **operadores** em Dart são símbolos especiais que realizam operações sobre **valores**, **variáveis** e **expressões**. Eles são fundamentais para a construção de programas robustos, permitindo manipular dados, controlar o fluxo de execução e realizar cálculos de forma eficiente e concisa. Sem operadores, seria impossível implementar lógica e funcionalidade em qualquer aplicação.

## Tipos de Operadores

A linguagem Dart oferece uma ampla gama de operadores, cada um projetado para uma finalidade específica. A seguir, apresentamos uma classificação detalhada dos principais tipos de operadores, acompanhados de exemplos práticos para facilitar o entendimento.

### 1. Operadores Aritméticos

Os operadores aritméticos realizam operações matemáticas básicas entre valores numéricos.

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`+`|Adição|Soma dois valores|`a + b`|`2 + 3 = 5`|
|`-`|Subtração|Subtrai o segundo valor do primeiro|`a - b`|`5 - 2 = 3`|
|`*`|Multiplicação|Multiplica dois valores|`a * b`|`2 * 3 = 6`|
|`/`|Divisão|Divide o primeiro valor pelo segundo|`a / b`|`6 / 2 = 3.0`|
|`%`|Módulo|Resto da divisão|`a % b`|`7 % 3 = 1`|
|`~/`|Divisão Inteira|Resultado inteiro da divisão|`a ~/ b`|`7 ~/ 3 = 2`|
|`++`|Incremento|Incrementa o valor em 1|`a++`|`a = 5; a++; // a é 6`|
|`--`|Decremento|Decrementa o valor em 1|`a--`|`a = 5; a--; // a é 4`|

### 2. Operadores Relacionais

Os operadores relacionais comparam dois valores e retornam um valor booleano (`true` ou `false`).

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`==`|Igualdade|Verifica se os valores são iguais|`a == b`|`2 == 2` (true)|
|`!=`|Diferença|Verifica se os valores são diferentes|`a != b`|`2 != 3` (true)|
|`<`|Menor que|Verifica se o primeiro valor é menor que o segundo|`a < b`|`2 < 3` (true)|
|`<=`|Menor ou Igual a|Verifica se o primeiro valor é menor ou igual ao segundo|`a <= b`|`2 <= 3` (true)|
|`>`|Maior que|Verifica se o primeiro valor é maior que o segundo|`a > b`|`3 > 2` (true)|
|`>=`|Maior ou Igual a|Verifica se o primeiro valor é maior ou igual ao segundo|`a >= b`|`3 >= 2` (true)|

### 3. Operadores Lógicos

Os operadores lógicos são utilizados para combinar expressões booleanas.

| Operador | Nome           | Descrição                                               | Sintaxe   | Exemplo                                                 |
| -------- | -------------- | ------------------------------------------------------- | --------- | ------------------------------------------------------- |
| `&&`     | E lógico       | Retorna `true` se ambas as expressões forem verdadeiras | `a && b`  | `(2 > 0) && (2 < 4)` (true)                             |
| \| \|    | OU lógico      | \| \|                                                   | a \| \| b | Retorna `true` se ao menos uma expressão for verdadeira |
| `!`      | Negação lógica | Inverte o valor booleano da expressão                   | `!a`      | `!(2 == 3)` (true)                                      |

### 4. Operadores Condicionais

Permitem a avaliação condicional de expressões.

| Operador | Nome              | Descrição                                                                  | Sintaxe     | Exemplo                                  |
| -------- | ----------------- | -------------------------------------------------------------------------- | ----------- | ---------------------------------------- |
| `? :`    | Operador Ternário | Retorna um valor com base em uma condição                                  | `a ? b : c` | `(2 > 0) ? "Positivo" : "Negativo"`      |
| `??`     | Coalescência Nula | Retorna o valor à esquerda se não for `null`, caso contrário, o da direita | `a ?? b`    | `var nome = nomeUsuario ?? "Visitante";` |

### 5. Operadores de Atribuição

Utilizados para atribuir valores às variáveis de forma direta ou combinada com operações.

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`=`|Atribuição Simples|Atribui o valor à variável|`a = b`|`x = 5`|
|`+=`|Atribuição com Adição|Soma e atribui o valor|`a += b`|`x += 2` (x = x + 2)|
|`-=`|Atribuição com Subtração|Subtrai e atribui o valor|`a -= b`|`x -= 3` (x = x - 3)|
|`*=`|Atribuição com Multiplicação|Multiplica e atribui o valor|`a *= b`|`x *= 2` (x = x * 2)|
|`/=`|Atribuição com Divisão|Divide e atribui o valor|`a /= b`|`x /= 2` (x = x / 2)|
|`%=`|Atribuição com Módulo|Calcula o resto da divisão e atribui|`a %= b`|`x %= 2` (x = x % 2)|

### 6. Operadores de Cascata (Cascade Notation)

Os operadores de cascata (`..` e `?..`) permitem encadear múltiplas operações em um mesmo objeto sem a necessidade de reatribuir a variável a cada etapa.

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`..`|Cascata|Executa métodos ou acessa propriedades e retorna o objeto|`obj..m1()..m2()`|`var calc = Calc()..add(10)..sub(5)..mul(2);`|
|`?..`|Cascata Nula|Como `..`, mas só se o objeto não for `null`|`obj?..m1()`|`usuario?..atualizarNome("Ana")`|

**Exemplo de Uso:**

```dart
class Calc {
  int valor = 0;

  void add(int n) => valor += n;
  void sub(int n) => valor -= n;
  void mul(int n) => valor *= n;
}

void main() {
  var calc = Calc()
    ..add(10)
    ..sub(5)
    ..mul(2);

  print(calc.valor); // Saída: 10
}
```

### 7. Operadores de Acesso

Permitem acessar elementos de coleções ou propriedades/métodos de objetos.

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`[]`|Indexador|Acessa elementos de listas ou mapas|`a[b]`|`lista[0]`|
|`.`|Acesso Direto|Acessa propriedades ou métodos de um objeto|`a.b`|`pessoa.nome`|
|`?.`|Acesso Condicional|Acessa membros se o objeto não for `null`|`a?.b`|`pessoa?.nome`|
|`!`|Acesso Não Nulo|Afirma que o objeto não é `null` para acessar seus membros|`a!.b`|`pessoa!.idade`|

### 8. Operadores de Nulidade

Facilitam o tratamento de valores `null`, aumentando a segurança e robustez do código.

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`??`|Coalescência Nula|Retorna o valor à esquerda se não for `null`; caso contrário, o da direita|`a ?? b`|`var nome = nomeUsuario ?? "Visitante";`|
|`?.`|Acesso Condicional|Acessa um membro se o objeto não for `null`|`a?.b`|`pessoa?.nome`|
|`!`|Operador de Não Nulidade|Afirma que o valor não é `null`|`a!`|`String nome = nomeUsuario!;`|
|`??=`|Atribuição Coalescente|Atribui o valor à variável somente se ela for `null`|`a ??= b`|`x ??= 10;`|

**Exemplo de Uso:**

```dart
void main() {
  String? nome;

  // Usando o operador de coalescência nula para fornecer um valor padrão
  var exibirNome = nome ?? "Desconhecido";
  print(exibirNome); // Saída: Desconhecido

  // Atribuição com coalescência nula
  nome ??= "Luiz";
  print(nome); // Saída: Luiz

  // Acesso condicional
  int? idade;
  print(idade?.isNegative); // Saída: null

  // Acesso com confiança
  idade = 30;
  print(idade!.isNegative); // Saída: false
}
```

### 9. Operador Spread (`...` e `...?`)

O operador spread permite inserir todos os elementos de uma coleção em outra coleção, de forma concisa.

|Operador|Nome|Descrição|Sintaxe|Exemplo|
|---|---|---|---|---|
|`...`|Spread|Insere elementos de uma coleção dentro de outra|`[...]`|`var lista = [1, 2, ...outraLista];`|
|`...?`|Spread Null-Aware|Como `...`, mas só se a coleção não for `null`|`[...]`|`var lista = [1, 2, ...?possivelNula];`|

**Exemplo de Uso:**

```dart
void main() {
  var lista1 = [1, 2, 3];
  var lista2 = [0, ...lista1, 4];
  print(lista2); // Saída: [0, 1, 2, 3, 4]

  List<int>? listaPossivelmenteNula;
  var lista3 = [5, ...?listaPossivelmenteNula];
  print(lista3); // Saída: [5]
}
```

**Vantagens:**

- **Combinação Fácil de Coleções**: Permite combinar listas, conjuntos ou mapas de forma simples.
- **Código Limpo e Conciso**: Evita loops explícitos para adicionar elementos de uma coleção em outra.
- **Tratamento de Nulidade**: Com `...?`, evita exceções se a coleção for `null`.
