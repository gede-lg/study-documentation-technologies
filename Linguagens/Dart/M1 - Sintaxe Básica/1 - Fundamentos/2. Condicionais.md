
Condicionais são estruturas de decisão que direcionam o fluxo de execução com base em condições. No Dart, os principais condicionais são `if`, `else`, e `else if`.

> [!NOTE]
> # IF Conditional

- **Uso**: Usados para executar diferentes blocos de código com base em condições booleanas.
- **Sintaxe**:
  ```dart
  if (condição) {
    // código a ser executado se a condição for verdadeira
  } else if (outra_condição) {
    // código a ser executado se a 'outra_condição' for verdadeira
  } else {
    // código a ser executado se nenhuma das condições anteriores for verdadeira
  }
  ```
- **Exemplo**:
  ```dart
  int idade = 20;
  if (idade < 18) {
    print("Menor de idade");
  } else if (idade < 65) {
    print("Adulto");
  } else {
    print("Idoso");
  }
  ```

> [!NOTE]
> # Switch case

- **Uso**: O `switch` é utilizado para comparar uma variável com diferentes valores (casos). É uma alternativa mais limpa e organizada ao uso de múltiplos `if`-`else if`-`else` quando todas as comparações são feitas com a mesma variável.
- **Sintaxe**:

```dart
switch (variável) {
  case valor1:
    // código a ser executado se a variável é igual a valor1
    break;
  case valor2:
    // código a ser executado se a variável é igual a valor2
    break;
  // mais casos podem ser adicionados conforme necessário
  default:
    // código a ser executado se a variável não corresponde a nenhum caso
}
```
