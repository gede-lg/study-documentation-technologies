### O que é uma Função e para que serve?

Em Dart, uma função é um bloco de código organizado e reutilizável que é projetado para realizar uma tarefa específica. Funções são fundamentais na programação porque ajudam a segmentar o código em blocos lógicos, facilitando a leitura, manutenção e reutilização do código. Além disso, permitem a parametrização, o que significa que podemos alterar o comportamento de uma função passando diferentes valores (argumentos) para ela.

### Sintaxe de Uso

A sintaxe básica para declarar uma função em Dart é a seguinte:

```dart
tipoRetorno nomeDaFuncao(parametro1, parametro2, ...) {
  // Corpo da função
  return valorDeRetorno;
}
```

- `tipoRetorno` é o tipo de dado que a função irá retornar. Se a função não retornar nenhum valor, o tipo `void` é usado.
- `nomeDaFuncao` é como você chama sua função. Deve ser único dentro de seu escopo.
- `parametro1, parametro2, ...` são os parâmetros que a função aceita. São opcionais; uma função pode não ter nenhum parâmetro.
- `valorDeRetorno` é o valor que a função retorna. Deve corresponder ao `tipoRetorno`.

**Exemplo:**

```dart
int soma(int a, int b) {
  return a + b;
}

void main() {
  print(soma(5, 3)); // Saída: 8
}
```

### Funções Anônimas

Funções anônimas, como o nome sugere, são funções sem nome. São úteis para implementações rápidas de funções que serão passadas como argumentos para outras funções, especialmente quando usadas com funções de alta ordem ou em callbacks.

**Sintaxe:**

```dart
(variavel1, variavel2, ...) {
  // Corpo da função
};
```

**Exemplo:**

```dart
var lista = ['maçã', 'banana', 'laranja'];
lista.forEach((item) {
  print(item);
});
```

Neste exemplo, a função passada para `forEach` é uma função anônima que imprime cada item da lista.

### Funções Externas

Funções externas (`external functions`) são declarações de funções sem corpo que informam ao Dart que a implementação da função será fornecida por outro meio, geralmente em código nativo (C ou C++, por exemplo) através da interface de ligação de linguagem (FFI - Foreign Function Interface). Isso é útil para usar recursos específicos da plataforma ou para melhorar o desempenho de operações críticas.

**Sintaxe:**

```dart
external tipoRetorno nomeDaFuncao(parametro1, parametro2, ...);
```

Para usar funções externas, geralmente você precisa declarar a função externa em Dart e depois vinculá-la à sua implementação em código nativo usando FFI.

### Tópicos Adicionais

- **Parâmetros Opcionais:** Dart suporta parâmetros opcionais nomeados e posicionais, o que permite flexibilidade ao chamar funções.
  
- **Valores Padrão para Parâmetros:** Você pode definir valores padrão para parâmetros, tanto para parâmetros nomeados quanto posicionais, o que é útil quando um valor comum é frequentemente usado.

- **Funções de Alta Ordem:** São funções que aceitam funções como parâmetros ou retornam uma função. Isso proporciona uma grande flexibilidade e reutilização de código.

- **Arrow Functions:** Dart suporta uma sintaxe concisa para funções que contêm uma única expressão, usando `=>`, que é uma forma abreviada de escrever funções que retornam diretamente um valor.

**Exemplo com parâmetros opcionais e valores padrão:**

```dart
void saudar(String nome, {String mensagem = 'Olá'}) {
  print('$mensagem, $nome!');
}

void main() {
  saudar('Mundo'); // Saída: Olá, Mundo!
  saudar('Mundo', mensagem: 'Bem-vindo'); // Saída: Bem-vindo, Mundo!
}
```

Este exemplo mostra como você pode definir funções com parâmetros opcionais e valores padrão, oferecendo flexibilidade no modo como

 as funções são chamadas.

As funções em Dart são versáteis e poderosas, permitindo aos desenvolvedores escrever código conciso, reutilizável e fácil de manter. Com a compreensão desses conceitos, você pode aproveitar ao máximo as capacidades da linguagem para criar aplicativos robustos e eficientes.


Em Dart, existem vários tipos de funções que você pode utilizar para diferentes propósitos. Aqui está uma lista detalhada dos tipos de funções disponíveis:

## 6. Funções Assíncronas (Asynchronous Functions)

São funções que utilizam `async` e `await` para lidar com operações assíncronas, como chamadas de rede ou operações de I/O.

```dart
Future<void> buscarDados() async {
  var dados = await Future.delayed(Duration(seconds: 2), () => 'Dados carregados');
  print(dados);
}

void main() {
  buscarDados();
}
```

## 7. Funções Geradoras (Generator Functions)

Existem dois tipos de funções geradoras em Dart: `sync*` e `async*`. Elas retornam iteradores síncronos e assíncronos, respectivamente.

### Sincronous Generator (sync*)

```dart
Iterable<int> numerosPares(int n) sync* {
  for (int i = 0; i <= n; i += 2) {
    yield i;
  }
}

void main() {
  for (var numero in numerosPares(10)) {
    print(numero);
  }
}
```

### Asynchronous Generator (async*)

```dart
Stream<int> numerosPares(int n) async* {
  for (int i = 0; i <= n; i += 2) {
    await Future.delayed(Duration(seconds: 1));
    yield i;
  }
}

void main() async {
  await for (var numero in numerosPares(10)) {
    print(numero);
  }
}
```
