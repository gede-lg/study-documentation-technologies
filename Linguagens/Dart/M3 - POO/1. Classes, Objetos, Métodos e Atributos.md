## Classes
Uma classe em Dart, assim como em muitas outras linguagens de programação orientadas a objetos, é um plano ou um modelo para criar objetos. Ela define um tipo de dado, encapsulando dados (atributos) e métodos (funções ou procedimentos) que operam nesses dados. A classe é fundamental para a implementação de conceitos como herança, encapsulamento e polimorfismo.

**Para que serve?**
Serve para modelar entidades ou conceitos do mundo real ou de um domínio específico de aplicação, definindo suas características e comportamentos.

**Sintaxe de uso:**
```dart
class NomeDaClasse {
  // Atributos
  // Métodos
}
```

**Exemplo:**
```dart
class Carro {
  String marca;
  int ano;

  // Método que descreve a ação de dirigir
  void dirigir() {
    print("O carro $marca está dirigindo.");
  }
}
```

---
## Objetos

Objetos são instâncias de uma classe. Quando você cria um objeto, está criando uma entidade concreta baseada no plano definido pela classe. Cada objeto tem seu próprio conjunto de valores de atributos (estado) e compartilha os métodos definidos na classe.

**Para que serve?**
Os objetos servem para representar entidades ou conceitos específicos com base nas definições de sua classe, permitindo interagir com eles através de seus métodos e atributos.

**Sintaxe de uso:**
```dart
NomeDaClasse nomeDoObjeto = NomeDaClasse();
```

**Exemplo:**
```dart
Carro meuCarro = Carro();
meuCarro.marca = "Toyota";
meuCarro.ano = 2020;
meuCarro.dirigir();  // Saída: O carro Toyota está dirigindo.
```

---
## Métodos

Métodos são funções definidas dentro de uma classe. Eles descrevem os comportamentos ou ações que os objetos da classe podem realizar. Métodos podem acessar e modificar os atributos do objeto e também podem retornar valores.

**Para que serve?**
Servem para definir as ações ou operações que os objetos de uma classe podem realizar, encapsulando a lógica específica dentro da classe.

**Sintaxe de uso:**
```dart
retorno nomeDoMetodo(parametros) {
  // Corpo do método
}
```

**Exemplo:**
```dart
class Pessoa {
  String nome;
  int idade;

  // Método para celebrar aniversário
  void aniversario() {
    idade++;
    print("Feliz aniversário, $nome! Agora você tem $idade anos.");
  }
}
```

---
## Atributos
Atributos, também conhecidos como campos ou propriedades, são as variáveis definidas dentro de uma classe. Eles representam as características ou estados que os objetos da classe podem ter.

**Para que serve?**
Servem para armazenar os dados ou o estado de um objeto, permitindo que os métodos da classe interajam com esses dados.

**Sintaxe de uso:**
```dart
class NomeDaClasse {
  tipo nomeDoAtributo;
}
```

**Exemplo:**
```dart
class ContaBancaria {
  int numeroDaConta;
  double saldo;

  // Métodos para manipular o saldo
}
```


### Observações Importantes

- **Propriedades Computadas**: Além de simplesmente retornar ou definir o valor de uma propriedade, getters e setters podem ser usados para calcular valores dinamicamente.
- **Privacidade**: Em Dart, o prefixo `_` (underscore) antes do nome de uma variável ou método indica que é privado à sua biblioteca. Getters e setters são frequentemente usados para acessar essas propriedades privadas.
- **Uso Consciente**: Embora úteis, getters e setters não devem ser usados indiscriminadamente. Se uma propriedade precisa ser acessada diretamente sem lógica adicional, pode não ser necessário um getter/setter.

Getters e Setters são ferramentas poderosas em Dart para criar classes seguras e bem encapsuladas, contribuindo para a robustez e manutenibilidade do código.
