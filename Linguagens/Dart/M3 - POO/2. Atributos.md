### Atributos no Dart

#### O que são e para que servem?

No Dart, atributos são variáveis associadas a uma classe que representam as propriedades ou características dos objetos criados a partir dessa classe. Eles são utilizados para armazenar o estado de um objeto. Por exemplo, em uma classe `Carro`, atributos podem incluir `cor`, `modelo` e `ano`. Atributos permitem que cada instância de uma classe mantenha seus próprios valores de dados.

#### Sintaxe de uso

A definição de atributos em Dart é bastante direta. Eles são declarados dentro do corpo de uma classe e podem ser inicializados com valores padrão ou através de um construtor.

```dart
class Carro {
  // Declaração de atributos
  String cor;
  String modelo;
  int ano;

  // Construtor da classe
  Carro(this.cor, this.modelo, this.ano);
}

void main() {
  // Criação de uma instância da classe Carro
  Carro meuCarro = Carro('Vermelho', 'Civic', 2020);

  // Acesso aos atributos
  print('Meu carro é um ${meuCarro.modelo} de cor ${meuCarro.cor} e ano ${meuCarro.ano}.');
}
```

#### Restrições de uso

1. **Modificadores de Acesso**: Dart não possui modificadores de acesso tradicionais como `public`, `private` e `protected`. Em vez disso, qualquer identificador que comece com um sublinhado (`_`) é considerado privado para a biblioteca em que é declarado.

    ```dart
    class Pessoa {
      String _nome;
      
      Pessoa(this._nome);

      // Método getter para acessar o atributo privado
      String get nome => _nome;

      // Método setter para modificar o atributo privado
      set nome(String novoNome) => _nome = novoNome;
    }
    ```

2. **Tipo Dinâmico**: Em Dart, os tipos de atributos podem ser explícitos ou dinâmicos. Se um tipo não for especificado, o atributo é considerado dinâmico.

    ```dart
    class Exemplo {
      var dinamico; // Tipo dinâmico
      int inteiro;  // Tipo explícito
    }
    ```

3. **Atributos Estáticos**: Atributos podem ser declarados como estáticos, compartilhados por todas as instâncias da classe.

    ```dart
    class Contador {
      static int contador = 0;

      Contador() {
        contador++;
      }
    }

    void main() {
      Contador c1 = Contador();
      Contador c2 = Contador();

      print(Contador.contador); // Saída: 2
    }
    ```

#### Quando utilizar?

1. **Armazenamento de Estado**: Utilize atributos para armazenar o estado de um objeto. Cada instância de uma classe pode ter valores diferentes para seus atributos, permitindo um comportamento específico.

    ```dart
    class Produto {
      String nome;
      double preco;

      Produto(this.nome, this.preco);
    }
    ```

2. **Encapsulamento**: Utilize atributos privados para encapsular dados e fornecer métodos de acesso (getters e setters) para controle sobre como esses dados são acessados e modificados.

    ```dart
    class Banco {
      String _senha;

      Banco(this._senha);

      String get senha => _senha;
      set senha(String novaSenha) {
        if (novaSenha.length > 6) {
          _senha = novaSenha;
        }
      }
    }
    ```

3. **Atributos Estáticos**: Utilize atributos estáticos para dados que são comuns a todas as instâncias da classe. Isso é útil para contadores, configurações globais, entre outros.

    ```dart
    class Configuracao {
      static String urlBase = "https://api.exemplo.com";

      static void alterarUrlBase(String novaUrl) {
        urlBase = novaUrl;
      }
    }
    ```

#### Informações Adicionais

1. **Inicializadores de Lista**: Atributos podem ser inicializados com valores padrão usando inicializadores de lista no construtor.

    ```dart
    class Usuario {
      String nome;
      int idade;

      Usuario({this.nome = 'Anônimo', this.idade = 0});
    }

    void main() {
      Usuario user = Usuario();
      print('${user.nome}, ${user.idade} anos'); // Saída: Anônimo, 0 anos
    }
    ```

2. **Atributos Constantes**: Atributos podem ser constantes, utilizando `const`, tornando seus valores imutáveis e compartilhados entre todas as instâncias.

    ```dart
    class Circulo {
      static const double pi = 3.14159;
      double raio;

      Circulo(this.raio);

      double calcularArea() {
        return pi * raio * raio;
      }
    }
    ```

3. **Atributos Final**: Atributos final são definidos uma vez e não podem ser modificados posteriormente. Eles são inicializados no momento da criação da instância.

    ```dart
    class Livro {
      final String titulo;
      final String autor;

      Livro(this.titulo, this.autor);
    }
    ```

Os atributos no Dart são essenciais para a definição de propriedades de uma classe e a manutenção do estado dos objetos. Compreender como declará-los e utilizá-los adequadamente é fundamental para desenvolver aplicações robustas e bem estruturadas.
