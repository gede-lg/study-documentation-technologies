# Records no Dart

## O Que São e Para Que Servem
Records no Dart são uma forma de criar tipos de dados que agrupam várias valores. Eles são úteis para representar dados que estão logicamente conectados, mas que não requerem a complexidade de uma classe. Por exemplo, você pode usar um record para representar uma coordenada geográfica, combinando latitude e longitude em um único tipo de dado.

## Record Syntax
A sintaxe de um record é mais simples e direta do que a de uma classe. Um record é definido usando chaves (`{}`) e separando os campos por vírgulas. Por exemplo:

```dart
var point = {x: 10, y: 20};
```

Este código define um record com dois campos: `x` e `y`.

## Record Types
O tipo de um record é inferido pelos tipos de seus campos. No exemplo anterior, se `x` e `y` são inteiros, o Dart entende que `point` é um record de dois inteiros. É possível também especificar explicitamente os tipos:

```dart
var point: {int x, int y} = {x: 10, y: 20};
```

## Record Fields
Os campos de um record são os valores individuais que ele armazena. No exemplo do `point`, `x` e `y` são os campos. Você pode acessar estes campos diretamente:

```dart
print(point.x); // Saída: 10
print(point.y); // Saída: 20
```

## Records Equality
A igualdade em records é determinada pelos valores de seus campos. Dois records são considerados iguais se todos os seus campos correspondentes são iguais. Por exemplo:

```dart
var point1 = {x: 10, y: 20};
var point2 = {x: 10, y: 20};

print(point1 == point2); // Saída: true
```

## Multiple Returns
Uma vantagem dos records é a capacidade de retornar múltiplos valores de uma função de maneira organizada. Por exemplo, uma função que retorna coordenadas pode ser escrita assim:

```dart
{x: int, y: int} getCoordinates() {
  // ... cálculos
  return {x: 10, y: 20};
}

var coords = getCoordinates();
print(coords); // Saída: {x: 10, y: 20}
```

## Observações Adicionais
- **Imutabilidade**: Por padrão, os records são imutáveis. Isso significa que uma vez que um record é criado, seus valores não podem ser alterados. Isso é útil para garantir a integridade dos dados.
- **Desestruturação**: Dart permite desestruturar um record, ou seja, extrair seus campos diretamente em variáveis. Por exemplo:
  ```dart
  var {x, y} = point;
  print(x); // Saída: 10
  print(y); // Saída: 20
  ```
- **Uso com Flutter**: Records podem ser especialmente úteis em Flutter para passar dados entre widgets de forma eficiente e organizada.
