# Métodos

## 1. Introdução Breve

O **Document Object Model (DOM)** é uma interface fundamental no desenvolvimento web, permitindo que linguagens de programação, como JavaScript, interajam com a estrutura, estilo e conteúdo de documentos HTML e XML de maneira dinâmica. Os métodos do DOM são essenciais para selecionar, manipular, criar e remover elementos, além de gerenciar eventos e estilos na página. Compreender os diversos métodos disponíveis e suas categorias facilita a construção de interfaces ricas e interativas, aprimorando a experiência do usuário.

## 3. Conteúdo Detalhado

### 1. Seleção de Elementos

A seleção de elementos no DOM é uma das operações mais comuns e fundamentais. O JavaScript oferece diversos métodos para localizar elementos específicos na estrutura da página.

### `querySelector` e `querySelectorAll`

- **`querySelector`**
    
    Retorna o primeiro elemento que corresponde ao seletor CSS especificado.
    
    **Sintaxe:**
    
    ```jsx
    const elemento = document.querySelector(seletor);
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const botãoEnviar = document.querySelector('.btn-enviar');
    
    ```
    
- **`querySelectorAll`**
    
    Retorna uma NodeList contendo todos os elementos que correspondem ao seletor CSS especificado.
    
    **Sintaxe:**
    
    ```jsx
    const elementos = document.querySelectorAll(seletor);
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const itensMenu = document.querySelectorAll('.menu-item');
    
    ```
    

### `getElementById`

Seleciona um único elemento pelo seu ID. É um dos métodos mais rápidos para seleção de elementos, já que IDs são únicos no documento.

**Sintaxe:**

```jsx
const elemento = document.getElementById(id);

```

**Exemplo:**

```jsx
const cabeçalho = document.getElementById('header');

```

### `getElementsByClassName`

Retorna uma HTMLCollection de todos os elementos que possuem a classe especificada.

**Sintaxe:**

```jsx
const elementos = document.getElementsByClassName(nomeDaClasse);

```

**Exemplo:**

```jsx
const botões = document.getElementsByClassName('btn');

```

### `getElementsByTagName`

Retorna uma HTMLCollection de todos os elementos com a tag especificada.

**Sintaxe:**

```jsx
const elementos = document.getElementsByTagName(tagName);

```

**Exemplo:**

```jsx
const parágrafos = document.getElementsByTagName('p');

```

### `closest`

Método de um elemento que retorna o ancestral mais próximo que corresponde ao seletor especificado, incluindo o próprio elemento.

**Sintaxe:**

```jsx
const ancestral = elemento.closest(seletor);

```

**Exemplo:**

```jsx
const botão = document.querySelector('.btn');
const container = botão.closest('.container');

```

### 2. Manipulação de Elementos

Manipular elementos envolve criar, remover, substituir ou clonar nós na árvore DOM, bem como alterar seu conteúdo.

### Criação e Remoção de Nós

### `createElement`

Cria um novo elemento HTML especificado.

**Sintaxe:**

```jsx
const novoElemento = document.createElement(tagName);

```

**Exemplo:**

```jsx
const novaDiv = document.createElement('div');

```

### `createTextNode`

Cria um novo nó de texto.

**Sintaxe:**

```jsx
const texto = document.createTextNode(textoString);

```

**Exemplo:**

```jsx
const texto = document.createTextNode('Olá, Mundo!');

```

### `appendChild`

Anexa um nó como o último filho de um nó pai especificado.

**Sintaxe:**

```jsx
pai.appendChild(filho);

```

**Exemplo:**

```jsx
const lista = document.getElementById('lista');
const novoItem = document.createElement('li');
novoItem.textContent = 'Item 4';
lista.appendChild(novoItem);

```

### `removeChild`

Remove um nó filho de um nó pai.

**Sintaxe:**

```jsx
pai.removeChild(filho);

```

**Exemplo:**

```jsx
const lista = document.getElementById('lista');
const item = document.getElementById('item-3');
lista.removeChild(item);

```

### `replaceChild`

Substitui um nó filho existente por um novo nó.

**Sintaxe:**

```jsx
pai.replaceChild(novoFilho, filhoAntigo);

```

**Exemplo:**

```jsx
const lista = document.getElementById('lista');
const novoItem = document.createElement('li');
novoItem.textContent = 'Item 5';
const itemAntigo = document.getElementById('item-2');
lista.replaceChild(novoItem, itemAntigo);

```

### `cloneNode`

Clona um nó. Se o parâmetro for `true`, realiza uma clonagem profunda (incluindo todos os filhos).

**Sintaxe:**

```jsx
const clone = elemento.cloneNode(deep);

```

**Exemplo:**

```jsx
const original = document.getElementById('original');
const clone = original.cloneNode(true);
document.body.appendChild(clone);

```

### Alteração de Conteúdo

### `innerHTML`

Obtém ou define o HTML interno de um elemento.

**Sintaxe:**

```jsx
const html = elemento.innerHTML;
elemento.innerHTML = novoHTML;

```

**Exemplo:**

```jsx
const conteúdo = document.getElementById('conteudo');
conteúdo.innerHTML = '<p>Novo parágrafo</p>';

```

### `textContent`

Obtém ou define o texto interno de um elemento, ignorando qualquer HTML.

**Sintaxe:**

```jsx
const texto = elemento.textContent;
elemento.textContent = novoTexto;

```

**Exemplo:**

```jsx
const título = document.getElementById('titulo');
título.textContent = 'Título Atualizado';

```

### `outerHTML`

Obtém ou define o HTML externo de um elemento, incluindo o próprio elemento.

**Sintaxe:**

```jsx
const htmlExterno = elemento.outerHTML;
elemento.outerHTML = novoHTML;

```

**Exemplo:**

```jsx
const parágrafo = document.getElementById('parag');
parágrafo.outerHTML = '<div id="parag">Novo Div</div>';

```

### 3. Manipulação de Atributos

Gerenciar atributos de elementos é crucial para alterar comportamentos, estilos e propriedades dinâmicas.

### `getAttribute`

Obtém o valor de um atributo especificado de um elemento.

**Sintaxe:**

```jsx
const valor = elemento.getAttribute(nomeDoAtributo);

```

**Exemplo:**

```jsx
const link = document.querySelector('a');
const href = link.getAttribute('href');

```

### `setAttribute`

Define ou atualiza o valor de um atributo especificado em um elemento.

**Sintaxe:**

```jsx
elemento.setAttribute(nomeDoAtributo, valor);

```

**Exemplo:**

```jsx
const imagem = document.querySelector('img');
imagem.setAttribute('src', 'nova-imagem.png');

```

### `removeAttribute`

Remove um atributo especificado de um elemento.

**Sintaxe:**

```jsx
elemento.removeAttribute(nomeDoAtributo);

```

**Exemplo:**

```jsx
const botão = document.querySelector('button');
botão.removeAttribute('disabled');

```

### `hasAttribute`

Verifica se um elemento possui um atributo específico.

**Sintaxe:**

```jsx
const existe = elemento.hasAttribute(nomeDoAtributo);

```

**Exemplo:**

```jsx
const input = document.querySelector('input');
const possuiPlaceholder = input.hasAttribute('placeholder');

```

### 4. Manipulação de Classes e Estilos

Gerenciar classes e estilos permite alterar a aparência e o comportamento visual dos elementos de forma eficiente.

### `classList`

Propriedade que retorna uma coleção de classes do elemento, permitindo manipulação fácil.

**Métodos Principais:**

- `add(nomeClasse)`: Adiciona uma classe.
- `remove(nomeClasse)`: Remove uma classe.
- `toggle(nomeClasse)`: Alterna a presença de uma classe.
- `contains(nomeClasse)`: Verifica se a classe está presente.
- `replace(oldClass, newClass)`: Substitui uma classe por outra.

**Exemplo:**

```jsx
const caixa = document.querySelector('.caixa');

// Adicionar uma classe
caixa.classList.add('ativo');

// Remover uma classe
caixa.classList.remove('inativo');

// Alternar uma classe
caixa.classList.toggle('visível');

// Verificar se possui uma classe
if (caixa.classList.contains('ativo')) {
    console.log('A caixa está ativa.');
}

// Substituir uma classe
caixa.classList.replace('ativo', 'inativo');

```

### `style`

Permite acesso direto aos estilos CSS inline de um elemento.

**Sintaxe:**

```jsx
elemento.style.propriedade = valor;

```

**Exemplo:**

```jsx
const botão = document.querySelector('button');

// Alterar cor de fundo
botão.style.backgroundColor = 'blue';

// Alterar margem
botão.style.margin = '10px';

// Remover um estilo
botão.style.backgroundColor = '';

```

### 5. Manipulação de Eventos

Gerenciar eventos é essencial para criar interações dinâmicas e responsivas na página.

### `addEventListener`

Adiciona um ou mais manipuladores de eventos a um elemento.

**Sintaxe:**

```jsx
elemento.addEventListener(evento, funçãoCallback, [opções]);

```

**Exemplo:**

```jsx
const botão = document.querySelector('button');
botão.addEventListener('click', () => {
    alert('Botão clicado!');
});

```

### `removeEventListener`

Remove um manipulador de evento previamente adicionado.

**Sintaxe:**

```jsx
elemento.removeEventListener(evento, funçãoCallback, [opções]);

```

**Exemplo:**

```jsx
function clicar() {
    alert('Botão clicado!');
}

const botão = document.querySelector('button');
botão.addEventListener('click', clicar);

// Remover o evento após algum tempo
setTimeout(() => {
    botão.removeEventListener('click', clicar);
}, 5000);

```

### `dispatchEvent`

Dispara um evento especificado em um elemento.

**Sintaxe:**

```jsx
elemento.dispatchEvent(evento);

```

**Exemplo:**

```jsx
const botão = document.querySelector('button');
const evento = new Event('click');
botão.dispatchEvent(evento);

```

### 6. Traversal (Navegação na Árvore DOM)

Navegar pela estrutura da árvore DOM é fundamental para acessar elementos relacionados.

### Propriedades

- **`parentNode`**
    
    Retorna o nó pai do elemento.
    
    **Exemplo:**
    
    ```jsx
    const filho = document.querySelector('.filho');
    const pai = filho.parentNode;
    
    ```
    
- **`childNodes`**
    
    Retorna uma NodeList de todos os nós filhos, incluindo nós de texto e comentários.
    
    **Exemplo:**
    
    ```jsx
    const lista = document.querySelector('ul');
    const filhos = lista.childNodes;
    
    ```
    
- **`children`**
    
    Retorna uma HTMLCollection de elementos filhos (exclui nós de texto e comentários).
    
    **Exemplo:**
    
    ```jsx
    const lista = document.querySelector('ul');
    const elementosFilhos = lista.children;
    
    ```
    
- **`firstChild` e `lastChild`**
    
    Retornam o primeiro e o último nó filho, respectivamente.
    
    **Exemplo:**
    
    ```jsx
    const primeiroFilho = lista.firstChild;
    const ultimoFilho = lista.lastChild;
    
    ```
    
- **`nextSibling` e `previousSibling`**
    
    Retornam o próximo e o nó irmão anterior, respectivamente.
    
    **Exemplo:**
    
    ```jsx
    const elemento = document.querySelector('.elemento');
    const próximo = elemento.nextSibling;
    const anterior = elemento.previousSibling;
    
    ```
    

### Métodos

- **`contains`**
    
    Verifica se um nó é descendente de outro.
    
    **Sintaxe:**
    
    ```jsx
    pai.contains(filho);
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const container = document.querySelector('.container');
    const elemento = document.querySelector('.elemento');
    console.log(container.contains(elemento)); // true ou false
    
    ```
    
- **`compareDocumentPosition`**
    
    Compara a posição de dois nós na árvore DOM.
    
    **Sintaxe:**
    
    ```jsx
    const posição = nó1.compareDocumentPosition(nó2);
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const nó1 = document.getElementById('nó1');
    const nó2 = document.getElementById('nó2');
    const posição = nó1.compareDocumentPosition(nó2);
    
    if (posição & Node.DOCUMENT_POSITION_FOLLOWING) {
        console.log('nó2 está após nó1');
    }
    
    ```
    

### 7. APIs Avançadas

As APIs avançadas do DOM proporcionam funcionalidades mais sofisticadas, como encapsulamento de componentes e observação de mudanças na estrutura.

### Shadow DOM

Permite encapsular a estrutura e o estilo de componentes web, evitando conflitos com o restante do documento.

**Métodos Principais:**

- **`attachShadow`**
    
    Anexa um shadow root a um elemento.
    
    **Sintaxe:**
    
    ```jsx
    const shadow = elemento.attachShadow({ mode: 'open' });
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const componente = document.querySelector('.componente');
    const shadow = componente.attachShadow({ mode: 'open' });
    shadow.innerHTML = `<p>Conteúdo encapsulado</p>`;
    
    ```
    
- **`getRootNode`**
    
    Retorna o nó raiz do contexto atual, que pode ser o documento ou o shadow root.
    
    **Sintaxe:**
    
    ```jsx
    const raiz = elemento.getRootNode();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const raiz = componente.getRootNode();
    
    ```
    

### MutationObserver

Permite observar mudanças na árvore DOM e reagir a elas de forma assíncrona.

**Métodos Principais:**

- **`observe`**
    
    Inicia a observação de um nó especificado com as opções fornecidas.
    
    **Sintaxe:**
    
    ```jsx
    observer.observe(nó, opções);
    
    ```
    
- **`disconnect`**
    
    Para de observar mudanças.
    
    **Sintaxe:**
    
    ```jsx
    observer.disconnect();
    
    ```
    
- **`takeRecords`**
    
    Retorna e limpa a fila de registros de mutação.
    
    **Sintaxe:**
    
    ```jsx
    const registros = observer.takeRecords();
    
    ```
    

**Exemplo:**

```jsx
const alvo = document.getElementById('alvo');

const observador = new MutationObserver((mutationsList, observer) => {
    for (let mutação of mutationsList) {
        if (mutação.type === 'childList') {
            console.log('Mudança na lista de filhos:', mutação);
        }
    }
});

observador.observe(alvo, { childList: true, subtree: true });

// Para desconectar o observador
// observador.disconnect();

```

### 8. Manipulação de Formulários

Gerenciar formulários é essencial para coletar e validar dados do usuário.

### Métodos Principais

- **`checkValidity`**
    
    Verifica se os campos do formulário são válidos conforme as restrições definidas.
    
    **Sintaxe:**
    
    ```jsx
    const válido = formulário.checkValidity();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const formulário = document.querySelector('form');
    if (formulário.checkValidity()) {
        console.log('Formulário válido');
    } else {
        console.log('Formulário inválido');
    }
    
    ```
    
- **`reportValidity`**
    
    Verifica a validade do formulário e exibe mensagens de erro se necessário.
    
    **Sintaxe:**
    
    ```jsx
    const éVálido = formulário.reportValidity();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const formulário = document.querySelector('form');
    formulário.addEventListener('submit', (e) => {
        if (!formulário.reportValidity()) {
            e.preventDefault();
        }
    });
    
    ```
    
- **`reset`**
    
    Reseta o formulário, limpando todos os campos.
    
    **Sintaxe:**
    
    ```jsx
    formulário.reset();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const botãoReset = document.querySelector('#resetar-form');
    botãoReset.addEventListener('click', () => {
        formulário.reset();
    });
    
    ```
    
- **`submit`**
    
    Submete o formulário programaticamente.
    
    **Sintaxe:**
    
    ```jsx
    formulário.submit();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const botãoSubmit = document.querySelector('#enviar-form');
    botãoSubmit.addEventListener('click', () => {
        formulário.submit();
    });
    
    ```
    

### 9. Navegação e Histórico

Gerenciar a navegação e o histórico do navegador é importante para criar experiências de usuário mais fluidas e controladas.

### Métodos Principais

- **`history.pushState`**
    
    Adiciona um novo estado ao histórico do navegador sem recarregar a página.
    
    **Sintaxe:**
    
    ```jsx
    history.pushState(dadosEstado, título, url);
    
    ```
    
    **Exemplo:**
    
    ```jsx
    history.pushState({ página: 2 }, 'Página 2', '/pagina-2');
    
    ```
    
- **`history.replaceState`**
    
    Substitui o estado atual no histórico do navegador.
    
    **Sintaxe:**
    
    ```jsx
    history.replaceState(dadosEstado, título, url);
    
    ```
    
    **Exemplo:**
    
    ```jsx
    history.replaceState({ página: 3 }, 'Página 3', '/pagina-3');
    
    ```
    
- **`history.back`**
    
    Navega para a página anterior no histórico.
    
    **Sintaxe:**
    
    ```jsx
    history.back();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const botãoVoltar = document.querySelector('#voltar');
    botãoVoltar.addEventListener('click', () => {
        history.back();
    });
    
    ```
    
- **`history.forward`**
    
    Navega para a próxima página no histórico.
    
    **Sintaxe:**
    
    ```jsx
    history.forward();
    
    ```
    
    **Exemplo:**
    
    ```jsx
    const botãoAvançar = document.querySelector('#avançar');
    botãoAvançar.addEventListener('click', () => {
        history.forward();
    });
    
    ```

## 5. Informações Adicionais

### Performance e Otimização

- **Minimize Reflows e Repaints:**
    - Alterações no DOM podem causar reflows e repaints, impactando a performance. Agrupe mudanças e aplique-as de uma vez sempre que possível.
- **Uso de Fragmentos de Documento (`DocumentFragment`):**
    - Utilize `DocumentFragment` para construir partes do DOM fora da árvore principal, reduzindo operações de reflow.
    
    **Exemplo:**
    
    ```jsx
    const fragmento = document.createDocumentFragment();
    for (let i = 0; i < 100; i++) {
        const li = document.createElement('li');
        li.textContent = `Item ${i}`;
        fragmento.appendChild(li);
    }
    document.getElementById('lista').appendChild(fragmento);
    
    ```
    
- **Delegação de Eventos:**
    - Em vez de adicionar múltiplos listeners a elementos semelhantes, adicione um único listener ao ancestral comum.
    
    **Exemplo:**
    
    ```jsx
    const lista = document.getElementById('lista');
    lista.addEventListener('click', (e) => {
        if (e.target && e.target.nodeName === 'LI') {
            console.log('Item clicado:', e.target.textContent);
        }
    });
    
    ```
    

### Compatibilidade entre Navegadores

Embora a maioria dos métodos do DOM seja amplamente suportada nos navegadores modernos, é essencial considerar a compatibilidade com versões mais antigas se o suporte a navegadores legados for necessário.

- **Ferramentas de Transpilação e Polyfills:**
    - Utilize ferramentas como Babel para transpilar código moderno para versões compatíveis.
    - Implemente polyfills para métodos não suportados em alguns navegadores.
    
    **Exemplo de Polyfill para `closest`:**
    
    ```jsx
    if (!Element.prototype.closest) {
        Element.prototype.closest = function(s) {
            let el = this;
            do {
                if (el.matches(s)) return el;
                el = el.parentElement || el.parentNode;
            } while (el !== null && el.nodeType === 1);
            return null;
        };
    }
    
    ```
    

### Boas Práticas

- **Evite Seleções Repetidas:**
    - Armazene referências a elementos que serão reutilizados para evitar chamadas repetidas ao DOM.
    
    **Exemplo:**
    
    ```jsx
    // Ineficiente
    document.querySelector('#botao').addEventListener('click', () => {
        document.querySelector('#caixa').classList.toggle('ativo');
    });
    
    // Otimizado
    const botão = document.querySelector('#botao');
    const caixa = document.querySelector('#caixa');
    botão.addEventListener('click', () => {
        caixa.classList.toggle('ativo');
    });
    
    ```
    
- **Utilize Delegação de Eventos:**
    - Para elementos dinâmicos, a delegação de eventos é mais eficiente e mantém o código limpo.
- **Mantenha o Código Limpo e Organizado:**
    - Separe funcionalidades em funções específicas e mantenha uma estrutura lógica.
- **Gerencie Memória e Recursos:**
    - Remova event listeners desnecessários e desconecte observadores quando não forem mais necessários para evitar vazamentos de memória.
