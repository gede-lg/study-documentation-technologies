# Nós

## 1. Introdução Breve

O **Document Object Model (DOM)** é uma representação estruturada de documentos HTML e XML, onde cada parte do documento é modelada como um objeto chamado **nó** (Node). Os **nós do DOM** são os blocos de construção fundamentais que permitem aos desenvolvedores interagir, manipular e modificar dinamicamente o conteúdo, a estrutura e o estilo das páginas web. Compreender os diferentes tipos de nós, suas propriedades e métodos é essencial para criar interfaces web ricas, interativas e responsivas.

## 3. Conteúdo Detalhado

### 1. Tipos de Nós no DOM

O DOM define uma variedade de tipos de nós, cada um representando diferentes partes ou estruturas de um documento. Abaixo estão os principais tipos de nós categorizados de acordo com suas funcionalidades e características.

### 1.1. `Node`

- **Descrição:**
    
    `Node` é a interface base de todos os tipos de nós no DOM. Ele define propriedades e métodos comuns a todos os nós, como navegação na árvore DOM e manipulação de filhos.
    
- **Propriedades Principais:**
    - `nodeName`
    - `nodeValue`
    - `nodeType`
    - `parentNode`
    - `childNodes`
    - `firstChild`
    - `lastChild`
    - `previousSibling`
    - `nextSibling`
    - `textContent`
- **Métodos Principais:**
    - `appendChild()`
    - `removeChild()`
    - `replaceChild()`
    - `cloneNode()`
    - `normalize()`
    - `hasChildNodes()`
- **Exemplo:**
    
    ```jsx
    const node = document.createElement('div');
    console.log(node.nodeName); // "DIV"
    console.log(node.nodeType); // 1 (Element)
    
    ```
    

### 1.2. `Document`

- **Descrição:**
    
    Representa todo o documento HTML ou XML carregado na janela do navegador. É a raiz da árvore DOM e permite o acesso a todos os elementos do documento.
    
- **Propriedades Principais:**
    - `documentElement`
    - `body`
    - `head`
    - `title`
    - `URL`
- **Métodos Principais:**
    - `createElement()`
    - `createTextNode()`
    - `getElementById()`
    - `getElementsByClassName()`
    - `getElementsByTagName()`
    - `querySelector()`
    - `querySelectorAll()`
- **Exemplo:**
    
    ```jsx
    console.log(document.title); // Exibe o título da página
    const novoParagrafo = document.createElement('p');
    novoParagrafo.textContent = 'Novo parágrafo';
    document.body.appendChild(novoParagrafo);
    
    ```
    

### 1.3. `DocumentType`

- **Descrição:**
    
    Representa a declaração de tipo de documento (doctype) associada ao documento. Geralmente utilizada para definir o tipo de HTML ou XML.
    
- **Propriedades Principais:**
    - `name`
    - `publicId`
    - `systemId`
- **Exemplo:**
    
    ```jsx
    const doctype = document.doctype;
    console.log(doctype.name); // "html"
    console.log(doctype.publicId); // ""
    console.log(doctype.systemId); // ""
    
    ```
    

### 1.4. `Element`

- **Descrição:**
    
    Representa elementos HTML ou XML no documento. É uma subclasse de `Node` e fornece propriedades e métodos específicos para elementos, como manipulação de atributos e estilos.
    
- **Propriedades Principais:**
    - `tagName`
    - `id`
    - `className`
    - `attributes`
    - `dataset`
    - `style`
- **Métodos Principais:**
    - `getAttribute()`
    - `setAttribute()`
    - `removeAttribute()`
    - `getElementsByTagName()`
    - `getElementsByClassName()`
    - `querySelector()`
    - `querySelectorAll()`
    - `closest()`
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.id = 'minha-div';
    div.className = 'container';
    div.setAttribute('data-info', 'exemplo');
    document.body.appendChild(div);
    
    console.log(div.dataset.info); // "exemplo"
    
    ```
    

### 1.5. `Text`

- **Descrição:**
    
    Representa o conteúdo de texto dentro de elementos ou atributos. É uma subclasse de `Node` e não pode ter filhos.
    
- **Propriedades Principais:**
    - `textContent`
- **Métodos Principais:**
    - `splitText()`
    - `replaceWholeText()`
- **Exemplo:**
    
    ```jsx
    const texto = document.createTextNode('Texto inicial');
    const paragrafo = document.createElement('p');
    paragrafo.appendChild(texto);
    document.body.appendChild(paragrafo);
    
    console.log(paragrafo.textContent); // "Texto inicial"
    texto.nodeValue = 'Texto atualizado';
    console.log(paragrafo.textContent); // "Texto atualizado"
    
    ```
    

### 1.6. `Comment`

- **Descrição:**
    
    Representa comentários no documento. Utilizado para inserir notas ou explicações no código que não são exibidas ao usuário.
    
- **Propriedades Principais:**
    - `nodeValue`
- **Exemplo:**
    
    ```jsx
    const comentario = document.createComment('Este é um comentário');
    document.body.appendChild(comentario);
    
    console.log(comentario.nodeValue); // "Este é um comentário"
    
    ```
    

### 1.7. `ProcessingInstruction`

- **Descrição:**
    
    Representa instruções de processamento em documentos XML. Não é comumente utilizado em documentos HTML.
    
- **Propriedades Principais:**
    - `target`
    - `data`
- **Exemplo:**
    
    ```jsx
    const pi = document.createProcessingInstruction('xml-stylesheet', 'type="text/xsl" href="style.xsl"');
    document.insertBefore(pi, document.firstChild);
    
    console.log(pi.target); // "xml-stylesheet"
    console.log(pi.data); // 'type="text/xsl" href="style.xsl"'
    
    ```
    

### 1.8. `CDATASection`

- **Descrição:**
    
    Representa uma seção de dados de caracteres em documentos XML, onde o conteúdo é tratado como texto puro, sem interpretar tags ou entidades. Não é aplicável a documentos HTML.
    
- **Propriedades Principais:**
    - `data`
- **Exemplo:**
    
    ```jsx
    // CDATASection não é suportado em documentos HTML, mas em XML:
    const cdata = document.createCDATASection('Conteúdo de CDATA');
    const elemento = document.createElement('example');
    elemento.appendChild(cdata);
    document.body.appendChild(elemento);
    
    console.log(elemento.textContent); // "Conteúdo de CDATA"
    
    ```
    

### 1.9. `DocumentFragment`

- **Descrição:**
    
    Representa uma parte leve da árvore DOM que pode ser usada para construir um subconjunto de nós antes de inseri-los no documento principal. Utilizado para otimizar a manipulação do DOM e minimizar reflows.
    
- **Propriedades Principais:**
    - `childNodes`
- **Métodos Principais:**
    - Herda métodos de `Node`, como `appendChild()`, `removeChild()`, etc.
- **Exemplo:**
    
    ```jsx
    const fragmento = document.createDocumentFragment();
    for (let i = 1; i <= 5; i++) {
        const li = document.createElement('li');
        li.textContent = `Item ${i}`;
        fragmento.appendChild(li);
    }
    const lista = document.createElement('ul');
    lista.appendChild(fragmento);
    document.body.appendChild(lista);
    
    ```
    

### 1.10. `Attr`

- **Descrição:**
    
    Representa um atributo de um elemento. Embora seja tecnicamente um nó, não é parte da árvore de elementos, mas sim um componente de um elemento.
    
- **Propriedades Principais:**
    - `name`
    - `value`
- **Métodos Principais:**
    - `specified` (obsoleto)
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    const attr = document.createAttribute('data-info');
    attr.value = 'valor personalizado';
    div.setAttributeNode(attr);
    
    console.log(div.getAttribute('data-info')); // "valor personalizado"
    
    ```
    

### 2. Propriedades Comuns dos Nós

Embora diferentes tipos de nós possuam propriedades e métodos específicos, há algumas propriedades comuns que todos os nós compartilham, herdadas da interface `Node`.

### `nodeName`

- **Descrição:**
    
    Retorna o nome do nó. Para elementos, corresponde ao nome da tag em maiúsculas.
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    console.log(div.nodeName); // "DIV"
    
    const texto = document.createTextNode('Texto');
    console.log(texto.nodeName); // "#text"
    
    ```
    

### `nodeValue`

- **Descrição:**
    
    Obtém ou define o valor do nó. Para elementos, geralmente retorna `null`. Para nós de texto, retorna o conteúdo do texto.
    
- **Sintaxe:**
    
    ```jsx
    const valor = nodo.nodeValue;
    nodo.nodeValue = 'Novo valor';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const texto = document.createTextNode('Texto original');
    console.log(texto.nodeValue); // "Texto original"
    texto.nodeValue = 'Texto atualizado';
    console.log(texto.nodeValue); // "Texto atualizado"
    
    ```
    

### `nodeType`

- **Descrição:**
    
    Retorna um número inteiro que representa o tipo de nó. Cada tipo de nó tem um valor distinto.
    
- **Tabela de Tipos de Nós:**
    
    
    | Tipo de Nó | Valor (`nodeType`) |
    | --- | --- |
    | Element | 1 |
    | Attr | 2 |
    | Text | 3 |
    | Comment | 8 |
    | Document | 9 |
    | DocumentType | 10 |
    | ProcessingInstruction | 7 |
    | CDATASection | 4 |
    | DocumentFragment | 11 |
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    console.log(div.nodeType); // 1
    
    const texto = document.createTextNode('Texto');
    console.log(texto.nodeType); // 3
    
    const comentario = document.createComment('Comentário');
    console.log(comentario.nodeType); // 8
    
    ```
    

### `parentNode`

- **Descrição:**
    
    Retorna o nó pai de um elemento ou `null` se não houver pai.
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    const paragrafo = document.createElement('p');
    div.appendChild(paragrafo);
    
    console.log(paragrafo.parentNode === div); // true
    console.log(div.parentNode); // Pode ser o documento ou outro elemento
    
    ```
    

### `childNodes`

- **Descrição:**
    
    Retorna uma `NodeList` de todos os nós filhos, incluindo nós de texto e comentários.
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = 'Texto <span>com</span> <b>formatação</b>';
    
    const filhos = div.childNodes;
    filhos.forEach(nodo => console.log(nodo.nodeName));
    // Output:
    // #text
    // SPAN
    // #text
    // B
    
    ```
    

### `firstChild`

- **Descrição:**
    
    Retorna o primeiro nó filho, incluindo nós de texto e comentários.
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = '<p>Parágrafo</p>Texto';
    
    console.log(div.firstChild.nodeName); // "P"
    
    ```
    

### `lastChild`

- **Descrição:**
    
    Retorna o último nó filho, incluindo nós de texto e comentários.
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = '<p>Parágrafo</p>Texto';
    
    console.log(div.lastChild.nodeName); // "#text"
    console.log(div.lastChild.nodeValue); // "Texto"
    
    ```
    

### `previousSibling`

- **Descrição:**
    
    Retorna o nó irmão anterior, incluindo nós de texto e comentários.
    
- **Exemplo:**
    
    ```jsx
    const ul = document.createElement('ul');
    const li1 = document.createElement('li');
    li1.textContent = 'Item 1';
    const li2 = document.createElement('li');
    li2.textContent = 'Item 2';
    
    ul.appendChild(li1);
    ul.appendChild(li2);
    
    console.log(li2.previousSibling.nodeName); // "LI"
    
    ```
    

### `nextSibling`

- **Descrição:**
    
    Retorna o próximo nó irmão, incluindo nós de texto e comentários.
    
- **Exemplo:**
    
    ```jsx
    const ul = document.createElement('ul');
    const li1 = document.createElement('li');
    li1.textContent = 'Item 1';
    const li2 = document.createElement('li');
    li2.textContent = 'Item 2';
    
    ul.appendChild(li1);
    ul.appendChild(li2);
    
    console.log(li1.nextSibling.nodeName); // "LI"
    
    ```
    

### `textContent`

- **Descrição:**
    
    Obtém ou define o conteúdo de texto de um nó e seus descendentes. Diferente de `innerText`, ignora estilos CSS e inclui texto de nós ocultos.
    
- **Sintaxe:**
    
    ```jsx
    const texto = nodo.textContent;
    nodo.textContent = 'Novo texto';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = '<p>Parágrafo <span>com</span> texto</p>';
    
    console.log(div.textContent); // "Parágrafo com texto"
    
    div.textContent = 'Texto totalmente novo';
    console.log(div.innerHTML); // "Texto totalmente novo"
    
    ```
    

### 2. Propriedades de Conteúdo

Essas propriedades permitem acessar e modificar o conteúdo interno ou externo de elementos e nós no DOM.

### `innerHTML`

- **Descrição:**
    
    Obtém ou define o conteúdo HTML interno de um elemento. Permite inserir, modificar ou remover elementos e conteúdo HTML dinamicamente.
    
- **Sintaxe:**
    
    ```jsx
    const html = elemento.innerHTML;
    elemento.innerHTML = '<p>Novo conteúdo HTML</p>';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = '<h1>Título</h1><p>Parágrafo</p>';
    document.body.appendChild(div);
    
    // Atualizar o conteúdo HTML
    div.innerHTML += '<span>Span Adicionado</span>';
    
    ```
    

### `outerHTML`

- **Descrição:**
    
    Obtém ou define o conteúdo HTML externo de um elemento, incluindo o próprio elemento. Útil para substituir elementos inteiros no DOM.
    
- **Sintaxe:**
    
    ```jsx
    const htmlExterno = elemento.outerHTML;
    elemento.outerHTML = '<div>Substituição completa</div>';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const paragrafo = document.createElement('p');
    paragrafo.textContent = 'Parágrafo original';
    document.body.appendChild(paragrafo);
    
    // Substituir o parágrafo por uma div
    paragrafo.outerHTML = '<div>Div substituta</div>';
    
    ```
    

### `textContent`

- **Descrição:**
    
    Obtém ou define o conteúdo de texto de um nó e seus descendentes, ignorando qualquer HTML. É uma maneira segura de inserir texto sem risco de injeção de HTML.
    
- **Sintaxe:**
    
    ```jsx
    const texto = elemento.textContent;
    elemento.textContent = 'Novo texto puro';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = '<p>Parágrafo <span>com</span> texto</p>';
    
    console.log(div.textContent); // "Parágrafo com texto"
    
    // Atualizar o conteúdo de texto
    div.textContent = 'Texto totalmente novo';
    
    ```
    

### `innerText`

- **Descrição:**
    
    Similar a `textContent`, mas leva em consideração estilos CSS e não inclui texto de nós ocultos. Atualiza o estilo visual e pode disparar reflows.
    
- **Sintaxe:**
    
    ```jsx
    const textoVisível = elemento.innerText;
    elemento.innerText = 'Texto visível atualizado';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const div = document.createElement('div');
    div.innerHTML = '<p style="display: none;">Texto oculto</p><p>Texto visível</p>';
    
    console.log(div.innerText); // "Texto visível"
    
    // Atualizar o texto visível
    div.innerText = 'Novo texto visível';
    
    ```
    

### `nodeValue`

- **Descrição:**
    
    Obtém ou define o valor de um nó de texto ou atributo. Para elementos, geralmente retorna `null`. Utilizado principalmente com nós de texto e atributos.
    
- **Sintaxe:**
    
    ```jsx
    const valor = nodo.nodeValue;
    nodo.nodeValue = 'Novo valor';
    
    ```
    
- **Exemplo:**
    
    ```jsx

    console.log(texto.nodeValue); // "Texto original"
    
    texto.nodeValue = 'Texto atualizado';

    const div = document.createElement('div');
    console.log(div.nodeValue); // null
    
    ```

### `className` e `classList`

- **`className`**
    
    Obtém ou define a lista completa de classes de um elemento como uma string. Útil para definir ou substituir todas as classes de uma vez.
    
    - **Sintaxe:**
        
        ```jsx
        const classes = elemento.className;

        ```

        ```jsx
        const div = document.createElement('div');

        console.log(div.className); // "container principal"
        
        // Substituir todas as classes

        console.log(div.className); // "secundario"
        
        ```
        
- **`classList`**
    
    Retorna um objeto `DOMTokenList` que representa a lista de classes de um elemento, permitindo manipulação mais granular, como adicionar, remover ou alternar classes individualmente.
    
    - **Sintaxe:**
        
        ```jsx
        elemento.classList.add('nova-classe');
        elemento.classList.remove('classe-existente');
        elemento.classList.toggle('classe-toggle');

        ```
        
    - **Exemplo:**
        
        ```jsx
        const div = document.createElement('div');
        div.classList.add('container');
        div.classList.add('principal');
        console.log(div.classList); // DOMTokenList ["container", "principal"]
        
        div.classList.remove('principal');
        console.log(div.classList); // DOMTokenList ["container"]
        
        div.classList.toggle('ativo');
        console.log(div.classList); // DOMTokenList ["container", "ativo"]
        
        console.log(div.classList.contains('ativo')); // true
        
        ```
        

### `attributes`

- **Descrição:**
    
    Retorna uma `NamedNodeMap` contendo todos os atributos de um elemento. Permite iterar sobre os atributos e acessar seus nomes e valores.


    ```jsx
    const atributos = link.attributes;
    Array.from(atributos).forEach(attr => {
        console.log(`${attr.name} = ${attr.value}`);
    });
    // Output:
    // href = https://exemplo.com
    // target = _blank
    
    ```
    

### `getAttributeNode`

    

### `dataset`

- **Descrição:**

### 8. Propriedades de Posição e Dimensões

### `offsetParent`

### 10. Propriedades de Acessibilidade

Essas propriedades garantem que o conteúdo da página seja acessível a todos os usuários, incluindo aqueles que utilizam tecnologias assistivas.

### `tabIndex`

- **Descrição:**
    
    Define a ordem de tabulação para elementos interativos, permitindo a navegação por teclado. Um valor de `0` inclui o elemento na ordem natural de tabulação, enquanto `-1` remove o elemento da ordem de tabulação, tornando-o focável apenas via script.
    
- **Sintaxe:**
    
    ```jsx
    elemento.tabIndex = 0; // Tornar focável
    elemento.tabIndex = -1; // Remover da ordem de tabulação
    
    ```
    
- **Exemplo:**
    
    ```html
    <div id="caixa" tabindex="0" aria-label="Caixa Interativa"></div>
    
    ```
    
    ```jsx
    const caixa = document.getElementById('caixa');
    
    // Estilizando a caixa para visualização
    caixa.style.width = '150px';
    caixa.style.height = '150px';
    caixa.style.backgroundColor = 'lightgreen';
    caixa.style.border = '2px solid black';
    
    // Adicionando foco via teclado
    caixa.addEventListener('focus', () => {
        caixa.style.borderColor = 'blue';
    });
    
    caixa.addEventListener('blur', () => {
        caixa.style.borderColor = 'black';
    });
    
    ```
    

### `aria-label`

- **Descrição:**
    
    Fornece um rótulo acessível para elementos que não possuem texto visível. Útil para ícones, botões customizados ou elementos interativos sem texto.
    
- **Sintaxe:**
    
    ```jsx
    elemento.setAttribute('aria-label', 'Descrição do Elemento');
    
    ```
    
- **Exemplo:**
    
    ```html
    <button id="buscar">
        <img src="buscar.png" alt="">
    </button>
    
    ```
    
    ```jsx
    const botaoBuscar = document.getElementById('buscar');
    botaoBuscar.setAttribute('aria-label', 'Buscar');
    
    // Adicionando evento de clique
    botaoBuscar.onclick = function() {
        alert('Busca realizada!');
    };
    
    ```
    

### 11. Propriedades de Navegação e Histórico

Essas propriedades permitem interagir com a navegação do usuário e o histórico do navegador, facilitando a criação de experiências de navegação mais controladas e personalizadas.

### `window.location`

- **Descrição:**
    
    Representa a URL atual da janela e fornece métodos para redirecionamento e atualização da página. Permite obter informações sobre a URL e navegar para novas URLs.
    
- **Sintaxe:**
    
    ```jsx
    const urlAtual = window.location.href;
    window.location.href = 'https://novo-site.com'; // Redireciona para um novo site
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const botaoRedirecionar = document.createElement('button');
    botaoRedirecionar.textContent = 'Ir para Exemplo.com';
    document.body.appendChild(botaoRedirecionar);
    
    botaoRedirecionar.onclick = function() {
        window.location.href = 'https://exemplo.com';
    };
    
    ```
    

### `window.history`

- **Descrição:**
    
    Permite manipular o histórico de navegação do usuário, facilitando a implementação de navegação sem recarregar a página. Utilizado para criar experiências de página única (SPA).
    
- **Sintaxe:**
    
    ```jsx
    window.history.back(); // Navega para a página anterior
    window.history.forward(); // Navega para a próxima página no histórico
    window.history.pushState(dadosEstado, título, url); // Adiciona um novo estado ao histórico
    window.history.replaceState(dadosEstado, título, url); // Substitui o estado atual no histórico
    
    ```
    
- **Exemplo:**
    
    ```jsx
    const botaoAdicionarEstado = document.createElement('button');
    botaoAdicionarEstado.textContent = 'Adicionar Estado';
    document.body.appendChild(botaoAdicionarEstado);
    
    const botaoVoltar = document.createElement('button');
    botaoVoltar.textContent = 'Voltar';
    document.body.appendChild(botaoVoltar);
    
    const botaoAvancar = document.createElement('button');
    botaoAvancar.textContent = 'Avançar';
    document.body.appendChild(botaoAvancar);
    
    botaoAdicionarEstado.onclick = function() {
        const estado = { pagina: 'nova' };
        const titulo = 'Nova Página';
        const url = '/nova-pagina';
        window.history.pushState(estado, titulo, url);
        console.log('Estado adicionado:', estado);
    };
    
    botaoVoltar.onclick = function() {
        window.history.back();
    };
    
    botaoAvancar.onclick = function() {
        window.history.forward();
    };
    
    // Lidar com alterações no estado
    window.onpopstate = function(event) {
        console.log('Estado atual:', event.state);
    };
    
    ```
    

### 12. Propriedades de Manipulação de Documentos

Essas propriedades fornecem acesso a informações gerais sobre o documento e permitem modificações em nível global.

### `document.title`

- **Descrição:**
    
    Obtém ou define o título do documento, exibido na aba do navegador. Essencial para SEO e usabilidade, fornecendo contexto sobre o conteúdo da página.
    
- **Sintaxe:**
    
    ```jsx
    const titulo = document.title;
    document.title = 'Novo Título';
    
    ```
    
- **Exemplo:**
    
    ```jsx
    console.log(document.title); // Exibe o título atual da página
    
    // Atualizar o título da página
    document.title = 'Página Atualizada';
    console.log(document.title); // "Página Atualizada"
    
    ```
    

### `document.URL`

- **Descrição:**
    
    Retorna a URL completa do documento atual. Útil para obter informações sobre a localização da página ou para redirecionamentos baseados na URL.
    
- **Sintaxe:**
    
    ```jsx
    const url = document.URL;
    console.log(url);
    
    ```
    
- **Exemplo:**
    
    ```jsx
    console.log(document.URL); // "https://www.exemplo.com/pagina"
    
    ```
    

### `document.cookie`

- **Descrição:**
    
    Permite acessar e modificar os cookies associados ao documento. Utilizado para armazenar pequenas quantidades de dados persistentes no navegador do usuário.
    
- **Sintaxe:**
    
    ```jsx
    document.cookie = "usuario=joao; expires=Fri, 31 Dec 2025 23:59:59 GMT; path=/";
    const cookies = document.cookie;
    console.log(cookies);
    
    ```
    
- **Exemplo:**
    
    ```jsx
    // Definir um cookie
    document.cookie = "nome=Maria; expires=Fri, 31 Dec 2025 23:59:59 GMT; path=/";
    
    // Ler os cookies
    console.log(document.cookie); // "nome=Maria"
    
    // Atualizar um cookie existente
    document.cookie = "nome=Ana; expires=Fri, 31 Dec 2025 23:59:59 GMT; path=/";
    console.log(document.cookie); // "nome=Ana"
    
    // Excluir um cookie definindo uma data de expiração no passado
    document.cookie = "nome=Ana; expires=Thu, 01 Jan 1970 00:00:00 GMT; path=/";
    console.log(document.cookie); // ""
    
    ```

## 5. Informações Adicionais

### Performance e Otimização

- **Uso de `DocumentFragment`:**
    
    Quando for necessário inserir múltiplos elementos no DOM, utilize `DocumentFragment` para reduzir reflows e repaints, melhorando a performance.
    
    **Exemplo:**
    
    ```jsx
    const fragmento = document.createDocumentFragment();
    for (let i = 1; i <= 1000; i++) {
        const li = document.createElement('li');
        li.textContent = `Item ${i}`;
        fragmento.appendChild(li);
    }
    document.getElementById('lista').appendChild(fragmento);
    
    ```
    
- **Minimize Reflows e Repaints:**
    
    Alterações frequentes no DOM podem causar reflows (recalcular layout) e repaints (re-renderizar estilos), afetando a performance. Agrupe mudanças e aplique-as de uma vez sempre que possível.
    
    **Exemplo:**
    
    ```jsx
    const caixa = document.getElementById('caixa');
    
    // Ineficiente: múltiplas alterações que causam reflows
    caixa.style.width = '200px';
    caixa.style.height = '200px';
    caixa.style.backgroundColor = 'blue';
    
    // Otimizado: usa batch updates
    caixa.style.cssText += 'width: 200px; height: 200px; background-color: blue;';
    
    ```
    

### Compatibilidade entre Navegadores

- **Verifique Suporte de Propriedades:**
    
    Embora a maioria dos nós e suas propriedades sejam amplamente suportadas nos navegadores modernos, é importante verificar a compatibilidade com versões mais antigas, especialmente para propriedades ou métodos mais recentes.
    
- **Use Polyfills Quando Necessário:**
    
    Implemente polyfills para nós ou propriedades que não são suportados em alguns navegadores, garantindo funcionalidade consistente.
    
    **Exemplo de Polyfill para `classList`:**
    
    ```jsx
    if (!('classList' in document.documentElement)) {
        Object.defineProperty(Element.prototype, 'classList', {
            get: function() {
                var self = this;
                function add(cls) {
                    if (!self.className.includes(cls)) {
                        self.className += ' ' + cls;
                    }
                }
                function remove(cls) {
                    self.className = self.className.replace(new RegExp('(?:^|\\s)' + cls + '(?!\\S)', 'g'), '');
                }
                function toggle(cls) {
                    if (self.className.includes(cls)) {
                        remove(cls);
                    } else {
                        add(cls);
                    }
                }
                function contains(cls) {
                    return self.className.includes(cls);
                }
                return {
                    add,
                    remove,
                    toggle,
                    contains
                };
            }
        });
    }
    
    ```
    

### Boas Práticas

- **Evite Manipulações Excessivas do DOM:**
    
    Manipulações frequentes no DOM podem impactar a performance. Utilize técnicas como `DocumentFragment` e batching de atualizações para otimizar.
    
- **Use Classes para Estilos:**
    
    Prefira adicionar ou remover classes em vez de modificar estilos inline diretamente. Isso promove reutilização e manutenibilidade do código CSS.
    
    **Exemplo:**
    
    ```jsx
    // Preferível
    elemento.classList.add('ativo');
    
    // Evitar
    elemento.style.backgroundColor = 'green';
    
    ```
    
- **Mantenha o Código Limpo e Organizado:**
    
    Separe funcionalidades em funções específicas e mantenha uma estrutura lógica para facilitar a leitura e manutenção do código.
    
- **Gerencie Memória e Recursos:**
    
    Remova event listeners desnecessários e desconecte observadores (`MutationObserver`, por exemplo) quando não forem mais necessários para evitar vazamentos de memória.
    
    **Exemplo:**
    
    ```jsx
    const botão = document.getElementById('botao');
    
    function handler() {
        console.log('Botão clicado');
    }
    
    botão.addEventListener('click', handler);
    
    // Remover o listener quando não for mais necessário
    botão.removeEventListener('click', handler);
    
    ```

Os **nós do Document Object Model (DOM)** são elementos essenciais que permitem aos desenvolvedores interagir, manipular e modificar dinamicamente o conteúdo, a estrutura e o estilo das páginas web. Desde a criação e inserção de elementos, passando pela manipulação de nós de texto e comentários, até a utilização de fragmentos de documentos para otimização de performance, o domínio dos diferentes tipos de nós e suas propriedades é fundamental para o desenvolvimento front-end eficiente e eficaz.

Ao compreender a hierarquia, as propriedades comuns e específicas, bem como as melhores práticas para manipulação dos nós, os desenvolvedores podem criar interfaces web mais robustas, interativas e acessíveis. Além disso, a atenção à performance e à compatibilidade entre navegadores assegura que as aplicações funcionem de maneira consistente e responsiva em diversos ambientes.

Investir tempo no estudo e na prática das funcionalidades dos nós do DOM certamente aprimorará as habilidades em JavaScript e contribuirá para a construção de soluções web avançadas e de alta qualidade.
