### Efeito de Sombra nos Elementos

A propriedade `box-shadow` é utilizada para criar efeitos de sombra em elementos.

- **Valores:**
  1. Deslocamento horizontal: Pode ser um valor positivo (direita) ou negativo (esquerda).
  2. Deslocamento vertical: Pode ser um valor positivo (para baixo) ou negativo (para cima).
  3. Raio de desfoque: Define o quão difusa será a borda da sombra.
  4. Cor da sombra: Define a cor da sombra.

```css
/* Exemplo de sombra nos elementos */
.elemento-com-sombra {
    box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.3);
    /* Deslocamento horizontal, deslocamento vertical, raio de desfoque, cor da sombra */
}
```

### Efeito de Sombra nos Textos

A propriedade `text-shadow` é usada para aplicar sombras a textos.

- **Valores:**
  1. Deslocamento horizontal: Pode ser um valor positivo (direita) ou negativo (esquerda).
  2. Deslocamento vertical: Pode ser um valor positivo (para baixo) ou negativo (para cima).
  3. Raio de desfoque: Define o quão difusa será a borda da sombra.
  4. Cor da sombra: Define a cor da sombra.

```css
/* Exemplo de sombra em texto */
.texto-com-sombra {
    text-shadow: 2px 2px 3px rgba(255, 0, 0, 0.5);
    /* Deslocamento horizontal, deslocamento vertical, raio de desfoque, cor da sombra */
}
```

Essas propriedades são úteis para adicionar dimensão e destaque a elementos e textos em uma página, proporcionando uma estilização visualmente atraente.
