Módulo 1: Introdução ao Java
----------------------------

### 1\. História e Características do Java

*   **Origem e Evolução**: Explicação sobre o desenvolvimento do Java pela Sun Microsystems em 1995, sua evolução ao longo dos anos, e a aquisição pela Oracle.
*   **Características Principais**:
    *   **Portabilidade**: "Write Once, Run Anywhere" - Java é independente de plataforma graças à JVM (Java Virtual Machine).
    *   **Orientação a Objetos**: Tudo em Java é um objeto ou uma classe.
    *   **Sintaxe Simples**: Semelhante a C++, mas com recursos reduzidos para simplificar a programação.
    *   **Segurança**: Gerenciamento de memória automático e proteção contra várias ameaças de segurança.
    *   **Multithreading**: Suporte integrado para threads, permitindo execução de múltiplas tarefas simultaneamente.
*   **Aplicações do Java no Mundo Real**: Discussão sobre como Java é usado em desenvolvimento web, aplicativos móveis (Android), sistemas corporativos, e mais.

*   **Configuração do Ambiente**:

    *   Apresentação de IDEs populares: Eclipse, IntelliJ IDEA, NetBeans.
