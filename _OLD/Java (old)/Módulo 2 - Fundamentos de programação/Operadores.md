# Módulo 2: Fundamentos de Programação em Java

## Operadores
Os operadores em Java são símbolos especiais que realizam operações específicas em um, dois ou três operandos e retornam um resultado.

### 1. Operadores Aritméticos
Usados para realizar operações matemáticas básicas.

#### Exemplos:
- **Adição (`+`)**: `int soma = 5 + 3; // soma é 8`
- **Subtração (`-`)**: `int diferenca = 5 - 3; // diferenca é 2`
- **Multiplicação (`*`)**: `int produto = 5 * 3; // produto é 15`
- **Divisão (`/`)**: `int quociente = 5 / 3; // quociente é 1`
- **Módulo (`%`)**: `int resto = 5 % 3; // resto é 2`

### 2. Operadores Lógicos
Usados para realizar operações lógicas (geralmente em valores booleanos).

#### Exemplos:
- **E lógico (`&&`)**: `boolean resultado = (5 > 3) && (8 > 6); // resultado é true`
- **OU lógico (`||`)**: `boolean resultado = (5 < 3) || (8 > 6); // resultado é true`
- **NÃO lógico (`!`)**: `boolean resultado = !(5 > 3); // resultado é false`

### 3. Operadores Relacionais
Usados para comparar dois valores.

#### Exemplos:
- **Igual a (`==`)**: `boolean igual = (5 == 3); // igual é false`
- **Diferente de (`!=`)**: `boolean diferente = (5 != 3); // diferente é true`
- **Maior que (`>`)**: `boolean maior = (5 > 3); // maior é true`
- **Menor que (`<`)**: `boolean menor = (5 < 3); // menor é false`
- **Maior ou igual a (`>=`)**: `boolean maiorIgual = (5 >= 3); // maiorIgual é true`
- **Menor ou igual a (`<=`)**: `boolean menorIgual = (5 <= 3); // menorIgual é false`
