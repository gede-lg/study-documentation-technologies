# Arrays

## 1. Introdução

Arrays são estruturas de dados fundamentais em praticamente todas as linguagens de programação, incluindo TypeScript. Eles permitem armazenar e manipular coleções de dados de forma organizada e eficiente. Em TypeScript, os arrays podem ser **homogêneos** (todos os elementos do mesmo tipo) ou **heterogêneos** (elementos de tipos variados), possibilitando flexibilidade no desenvolvimento de aplicações. A compreensão dos métodos e técnicas de manipulação de arrays é crucial para escrever código limpo, legível e eficiente.

## 3. Conteúdo Detalhado

### Definição e Conceitos Fundamentais

### O que são Arrays?

Um **array** é uma coleção de elementos indexados, onde cada posição armazena um valor. Em TypeScript, os arrays são utilizados para armazenar sequências de dados, permitindo acesso e manipulação de elementos através de índices numéricos.

### Arrays Homogêneos vs. Arrays Heterogêneos

- **Arrays Homogêneos:** São arrays onde todos os elementos possuem o mesmo tipo. Isso facilita a inferência de tipos e permite maior segurança em tempo de compilação.
    
    **Exemplo:**
    
    ```tsx
    const numeros: number[] = [1, 2, 3, 4, 5];
    
    ```
    
- **Arrays Heterogêneos:** São arrays que podem conter elementos de tipos diferentes. Embora ofereçam flexibilidade, podem reduzir a segurança dos tipos e exigir verificações adicionais.
    
    **Exemplo:**
    
    ```tsx
    const misturado: (number | string | boolean)[] = [1, "dois", true];
    
    ```
    

### Sintaxe e Estrutura em TypeScript

### Declaração de Arrays

TypeScript oferece duas formas principais de declarar arrays:

1. **Utilizando a notação de colchetes (`[]`):**
    
    ```tsx
    const frutas: string[] = ['maçã', 'banana', 'laranja'];
    
    ```
    
2. **Utilizando a generic `Array<T>`:**
    
    ```tsx
    const numeros: Array<number> = [10, 20, 30];
    
    ```
    

### Acesso e Modificação de Elementos

- **Acesso:** Utiliza-se o índice do elemento (iniciando em 0).
    
    ```tsx
    console.log(frutas[0]); // Saída: 'maçã'
    
    ```
    
- **Modificação:** Atribui-se um novo valor a um índice específico.
    
    ```tsx
    frutas[1] = 'uva';
    console.log(frutas); // Saída: ['maçã', 'uva', 'laranja']
    
    ```
    

### Componentes Principais e Métodos de Manipulação

Arrays em TypeScript vêm com vários métodos integrados que facilitam a manipulação de seus elementos. A seguir, listamos alguns dos principais:

- **`.push()`**: Adiciona um ou mais elementos ao final do array.
    
    ```tsx
    frutas.push('abacaxi');
    
    ```
    
- **`.pop()`**: Remove e retorna o último elemento do array.
    
    ```tsx
    const ultimaFruta = frutas.pop();
    
    ```
    
- **`.shift()`**: Remove e retorna o primeiro elemento do array.
    
    ```tsx
    const primeiraFruta = frutas.shift();
    
    ```
    
- **`.unshift()`**: Adiciona um ou mais elementos no início do array.
    
    ```tsx
    frutas.unshift('morango');
    
    ```
    
- **`.splice()`**: Pode ser usado para adicionar, remover ou substituir elementos em uma posição específica.
    
    ```tsx
    // Remove 1 elemento a partir do índice 1
    const removidos = frutas.splice(1, 1);
    
    ```
    
- **`.slice()`**: Retorna uma cópia de uma parte do array.
    
    ```tsx
    const subArray = frutas.slice(0, 2);
    
    ```
    
- **`.map()`**: Cria um novo array com o resultado da aplicação de uma função a cada elemento.
    
    ```tsx
    const numerosDobrados = numeros.map(num => num * 2);
    
    ```
    
- **`.filter()`**: Cria um novo array contendo apenas os elementos que passam em uma determinada condição.
    
    ```tsx
    const numerosPares = numeros.filter(num => num % 2 === 0);
    
    ```
    
- **`.reduce()`**: Aplica uma função acumuladora contra um acumulador e cada valor do array para reduzi-lo a um único valor.
    
    ```tsx
    const soma = numeros.reduce((acumulador, valor) => acumulador + valor, 0);
    
    ```
    

### Uso Avançado

### Trabalhando com Arrays Heterogêneos

Quando se trabalha com arrays heterogêneos, é importante utilizar **type guards** ou **union types** para garantir a segurança dos tipos.

**Exemplo de Uso Avançado com Union Types e Type Guards:**

```tsx
const dados: (number | string)[] = [1, 'dois', 3, 'quatro'];

dados.forEach(item => {
  if (typeof item === 'number') {
    console.log(`Número: ${item}`);
  } else {
    console.log(`String: ${item}`);
  }
});

```

### Combinação de Métodos para Operações Complexas

É comum encadear métodos para realizar transformações e filtragens complexas.

**Exemplo:**

```tsx
const numeros: number[] = [1, 2, 3, 4, 5, 6];

// Filtra números pares, dobra seus valores e soma todos
const resultado = numeros
  .filter(num => num % 2 === 0)
  .map(num => num * 2)
  .reduce((acum, num) => acum + num, 0);

console.log(`Resultado: ${resultado}`); // Saída: Resultado: 24

```

### Destructuring com Arrays

O destructuring facilita a extração de valores de arrays, melhorando a legibilidade do código.

**Exemplo:**

```tsx
const [primeiro, segundo, ...resto] = frutas;
console.log(`Primeiro: ${primeiro}, Segundo: ${segundo}`);
console.log(`Resto: ${resto}`);

```

## 4. Exemplos de Código Otimizados

### Exemplo Básico: Manipulação de Array Homogêneo

```tsx
// Declaração de array homogêneo
const nomes: string[] = ['Alice', 'Bob', 'Carol'];

// Adicionando um novo elemento
nomes.push('David');

// Modificando um elemento
nomes[1] = 'Bruno';

// Removendo o último elemento
const removido = nomes.pop();

console.log('Array de nomes:', nomes);
console.log('Elemento removido:', removido);

```

### Exemplo Avançado: Uso de Métodos Encadeados com Arrays

```tsx
// Array de números
const valores: number[] = [10, 15, 20, 25, 30];

// Filtra valores maiores que 15, multiplica por 3 e soma o total
const total = valores
  .filter(valor => valor > 15)
  .map(valor => valor * 3)
  .reduce((acumulador, valor) => acumulador + valor, 0);

console.log('Total dos valores processados:', total);

```

### Exemplo com Arrays Heterogêneos e Type Guards

```tsx
// Array heterogêneo
const elementos: (number | string)[] = [100, 'cento', 200, 'duzentos'];

// Processa cada elemento com verificação de tipo
elementos.forEach(elemento => {
  if (typeof elemento === 'number') {
    console.log(`Número detectado: ${elemento}`);
  } else {
    console.log(`Texto detectado: ${elemento}`);
  }
});

```

## 5. Informações Adicionais

- **Imutabilidade:** Considere o uso de métodos que não modificam o array original (como `.slice()`, `.map()`, e `.filter()`) para favorecer padrões funcionais e evitar efeitos colaterais indesejados.
- **Performance:** Operações em arrays podem variar em termos de eficiência. Métodos encadeados podem ser convenientes, mas esteja atento a possíveis impactos na performance para coleções de grande tamanho.
- **TypeScript e Tipos:** Aproveite os recursos de tipagem do TypeScript para evitar erros comuns em tempo de execução, garantindo que o array esteja sendo manipulado conforme o esperado.
