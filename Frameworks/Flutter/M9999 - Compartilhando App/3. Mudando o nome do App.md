Alterar o nome de um aplicativo Flutter envolve modificar o nome exibido nos ícones do aplicativo no dispositivo do usuário. Esse processo varia ligeiramente entre as plataformas Android e iOS. A seguir, forneço um guia passo a passo detalhado para alterar o nome do seu aplicativo Flutter para ambas as plataformas.

### Passo a Passo para Alterar o Nome do App no Flutter

#### 1. Preparação

Antes de começar, certifique-se de ter o Flutter instalado e configurado corretamente em sua máquina, e de ter seu projeto Flutter aberto no seu editor de código favorito (por exemplo, Visual Studio Code ou Android Studio).

#### 2. Alterando o Nome do App no Android

1. **Abra o arquivo `AndroidManifest.xml`:**

   Navegue até o diretório `android/app/src/main` no seu projeto Flutter e abra o arquivo `AndroidManifest.xml`.

   ```bash
   android/app/src/main/AndroidManifest.xml
   ```

2. **Modifique o `android:label`:**

   No arquivo `AndroidManifest.xml`, encontre a tag `<application>` e modifique o atributo `android:label` para o novo nome do seu aplicativo.

   ```xml
   <application
       android:label="Novo Nome do App"
       android:icon="@mipmap/ic_launcher">
       ...
   </application>
   ```

3. **Atualize o nome do aplicativo para versões diferentes:**

   Caso você tenha diferentes configurações de build, como `debug`, `profile` e `release`, você pode precisar modificar os respectivos arquivos `AndroidManifest.xml` em:

   ```bash
   android/app/src/debug/AndroidManifest.xml
   android/app/src/profile/AndroidManifest.xml
   android/app/src/release/AndroidManifest.xml
   ```

   Certifique-se de que todos os arquivos `AndroidManifest.xml` tenham o nome do aplicativo atualizado.

#### 3. Alterando o Nome do App no iOS

1. **Abra o arquivo `Info.plist`:**

   Navegue até o diretório `ios/Runner` no seu projeto Flutter e abra o arquivo `Info.plist`.

   ```bash
   ios/Runner/Info.plist
   ```

2. **Modifique a chave `CFBundleName`:**

   No arquivo `Info.plist`, encontre a chave `CFBundleName` e modifique seu valor para o novo nome do aplicativo.

   ```xml
   <key>CFBundleName</key>
   <string>Novo Nome do App</string>
   ```

3. **Modifique a chave `CFBundleDisplayName`:**

   Além de `CFBundleName`, você pode querer modificar a chave `CFBundleDisplayName` (caso exista) para alterar o nome exibido no ícone do aplicativo.

   ```xml
   <key>CFBundleDisplayName</key>
   <string>Novo Nome do App</string>
   ```

#### 4. Atualizando a Configuração do Flutter

1. **Abra o arquivo `pubspec.yaml`:**

   Navegue até o arquivo `pubspec.yaml` na raiz do seu projeto Flutter.

   ```bash
   pubspec.yaml
   ```

2. **Atualize o nome do aplicativo:**

   Encontre a seção `name` e atualize o nome do seu aplicativo, se necessário.

   ```yaml
   name: novo_nome_do_app
   ```

#### 5. Limpando o Build

Para garantir que todas as mudanças sejam aplicadas corretamente, limpe os builds anteriores do seu projeto.

1. **Limpe o build:**

   Execute o comando abaixo no terminal para limpar os builds anteriores.

   ```bash
   flutter clean
   ```

2. **Recompile o projeto:**

   Execute o comando abaixo para recompilar seu projeto com o novo nome do aplicativo.

   ```bash
   flutter pub get
   flutter run
   ```

#### 6. Verificação Final

1. **Android:**

   - Abra o Android Studio.
   - Vá até `Tools > AVD Manager`.
   - Inicie um emulador Android ou conecte um dispositivo físico.
   - Execute o aplicativo e verifique se o novo nome do aplicativo está correto.

2. **iOS:**

   - Abra o Xcode.
   - Selecione `Runner` no navegador de projetos.
   - Conecte um dispositivo iOS ou inicie um simulador.
   - Execute o aplicativo e verifique se o novo nome do aplicativo está correto.

#### Resumo

Ao seguir esses passos detalhados, você conseguirá alterar o nome do seu aplicativo Flutter para ambas as plataformas, Android e iOS. É importante garantir que todas as modificações sejam realizadas corretamente e que você teste o aplicativo para verificar se as mudanças foram aplicadas conforme esperado.
