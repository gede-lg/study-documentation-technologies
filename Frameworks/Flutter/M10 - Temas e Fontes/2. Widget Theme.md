# Widget `Theme` no Flutter

## Introdução

O Flutter oferece o widget `Theme` para personalizar a aparência visual dos aplicativos. Ele é um ponto central para definir temas globais, como cores, fontes, estilos de texto e outros parâmetros visuais. O `Theme` facilita a criação de aplicativos consistentes e visualmente agradáveis, respeitando as diretrizes do Material Design ou fornecendo um estilo customizado.

## O que é e para que serve?

O `Theme` é um widget que fornece uma árvore de configurações visuais usadas por widgets descendentes. Ele permite definir temas consistentes em todo o aplicativo, como:

- Paleta de cores (primária, secundária, de fundo, etc.).
- Estilos de texto (tipografia, tamanhos de fonte, pesos).
- Estilos de botões e componentes visuais.

**Finalidade principal:**

- Centralizar a personalização visual de um aplicativo para que as mudanças afetem toda a interface sem a necessidade de modificar widgets individualmente.

---

## Como funciona?

O widget `Theme` utiliza um objeto `ThemeData` para configurar as propriedades visuais. O `Theme` é geralmente posicionado no topo da hierarquia de widgets (dentro de um `MaterialApp`) para aplicar um tema global.

Widgets filhos podem acessar o tema atual através do método `Theme.of(context)` e usar suas propriedades para obter informações sobre cores, fontes, etc.

Por exemplo:

- Botões, campos de texto e outros componentes se ajustam automaticamente ao tema definido.

---

## Sintaxe de uso

### Estrutura básica

```dart
Theme(
  data: ThemeData(
    primaryColor: Colors.blue,
    accentColor: Colors.orange,
  ),
  child: MyWidget(),
)
```

### Parâmetros do `Theme`

|Parâmetro|Tipo|Obrigatório|Descrição|
|---|---|---|---|
|`data`|`ThemeData`|Sim|O tema a ser aplicado. Define as propriedades visuais, como cores, estilos e tipografia.|
|`child`|`Widget`|Sim|O widget descendente que herda o tema definido.|

**Detalhes:**

- **`data`**: Aceita um objeto `ThemeData`, que encapsula as propriedades visuais.
- **`child`**: Define a subárvore de widgets que terá acesso ao tema configurado.

---

## Restrições de uso

1. **Acesso ao contexto**: `Theme.of(context)` só pode ser usado após o `Theme` estar na árvore de widgets.
2. **Uso com Material Components**: O `Theme` é projetado para funcionar melhor com widgets do Material Design.
3. **Interoperabilidade limitada**: Widgets que não respeitam o Material Design podem não aplicar o tema.

---

## Quando utilizar?

- **Temas globais**: Para definir um tema consistente em todo o aplicativo.
- **Seções específicas**: Para personalizar temas de partes específicas da interface.
- **Estilização dinâmica**: Para alterar temas com base na interação do usuário (e.g., modo escuro/claro).

---

## Tabela de propriedades

|Propriedade|Descrição|Sintaxe de uso|
|---|---|---|
|`data`|Define as propriedades do tema através de um objeto `ThemeData`.|`data: ThemeData(...)`|
|`child`|O widget que será estilizado pelo tema definido.|`child: MyWidget()`|

---

## Tabela de métodos

|Método|Descrição|Sintaxe de uso|
|---|---|---|
|`Theme.of(context)`|Obtém o tema atual do contexto fornecido.|`final theme = Theme.of(context);`|
|`ThemeData.copyWith`|Cria uma cópia do tema existente com alterações nas propriedades.|`theme.copyWith(primaryColor: ...)`|

## Exemplo completo

Abaixo, um exemplo detalhado de como usar o widget `Theme` para definir um tema global e um tema específico para uma seção.

### Código

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(
        primaryColor: Colors.blue,
        accentColor: Colors.orange,
        textTheme: TextTheme(
          bodyText1: TextStyle(fontSize: 18, color: Colors.black),
          headline6: TextStyle(fontSize: 24, fontWeight: FontWeight.bold),
        ),
      ),
      home: HomePage(),
    );
  }
}

class HomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final theme = Theme.of(context);
    return Scaffold(
      appBar: AppBar(
        title: Text('Exemplo de Theme'),
        backgroundColor: theme.primaryColor,
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('Texto com tema global', style: theme.textTheme.bodyText1),
            SizedBox(height: 20),
            Theme(
              data: theme.copyWith(primaryColor: Colors.green),
              child: ElevatedButton(
                onPressed: () {},
                child: Text('Botão com tema específico'),
              ),
            ),
          ],
        ),
      ),
    );
  }
}
```

### Explicação

1. **Tema global**:
    - Definido no `MaterialApp` através do parâmetro `theme`.
    - Aplica estilos consistentes para todo o aplicativo.
2. **Tema específico**:
    - Usado para estilizar o botão com uma cor primária diferente.
    - O método `Theme.copyWith` cria uma variação do tema global.

O widget `Theme` é essencial para criar aplicativos Flutter com uma aparência consistente e profissional. Ele permite centralizar o estilo visual, simplificando a manutenção e as mudanças globais no design do aplicativo.
