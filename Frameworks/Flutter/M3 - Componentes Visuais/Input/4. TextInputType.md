## Widget `TextInputType` no Flutter

### O que é e para que serve?

O `TextInputType` é uma classe no Flutter que define o tipo de teclado que deve ser exibido para o usuário ao interagir com um campo de entrada de texto (`TextField` ou `TextFormField`). Esse recurso é essencial para proporcionar uma experiência de usuário mais intuitiva, pois adapta o teclado de acordo com o tipo de dado que se espera que o usuário insira (como texto, números, e-mails, etc.).

Por exemplo, ao utilizar um campo para inserir um e-mail, o teclado exibido pode incluir caracteres especiais como "@" e ".", facilitando a inserção do endereço de e-mail.

### Sintaxe de uso

A classe `TextInputType` é utilizada dentro de widgets como `TextField` ou `TextFormField` através da propriedade `keyboardType`. Abaixo está um exemplo básico de como configurar um campo de entrada para aceitar apenas números:

```dart
TextField(
  keyboardType: TextInputType.number,
  decoration: InputDecoration(
    labelText: 'Informe um número',
  ),
),
```

### Restrições de uso

- O `TextInputType` deve ser utilizado em campos de entrada de texto (`TextField` ou `TextFormField`). Não faz sentido aplicá-lo em widgets que não recebem entrada de texto.
- A customização do teclado varia de acordo com o sistema operacional (iOS ou Android), e nem todas as variações de `TextInputType` terão o mesmo comportamento ou estarão disponíveis em todas as plataformas.

### Quando utilizar?

O `TextInputType` deve ser utilizado sempre que você deseja que o teclado exibido seja adequado ao tipo de entrada de dados esperado pelo campo de texto. Isso melhora a usabilidade do aplicativo e evita que o usuário tenha que alternar manualmente entre teclados, como o numérico e o alfabético.

### Possui widget pai ou implementa algum widget?

O `TextInputType` não é um widget em si, mas uma classe que define o tipo de entrada para widgets como `TextField` e `TextFormField`. Ele não possui um widget pai e não implementa nenhum widget diretamente. Ele é usado como uma propriedade desses widgets de entrada.

### Propriedades do `TextInputType`

Aqui está uma tabela com todas as propriedades do `TextInputType`:

| Propriedade | Descrição | Sintaxe de Uso |
|-------------|------------|----------------|
| `TextInputType.text` | Usado para entrada de texto comum. | `keyboardType: TextInputType.text` |
| `TextInputType.multiline` | Usado para entrada de múltiplas linhas de texto. | `keyboardType: TextInputType.multiline` |
| `TextInputType.number` | Usado para entrada de números. | `keyboardType: TextInputType.number` |
| `TextInputType.phone` | Usado para entrada de números de telefone. | `keyboardType: TextInputType.phone` |
| `TextInputType.datetime` | Usado para entrada de datas e horários. | `keyboardType: TextInputType.datetime` |
| `TextInputType.emailAddress` | Usado para entrada de endereços de e-mail. | `keyboardType: TextInputType.emailAddress` |
| `TextInputType.url` | Usado para entrada de URLs. | `keyboardType: TextInputType.url` |
| `TextInputType.visiblePassword` | Usado para entrada de senhas visíveis. | `keyboardType: TextInputType.visiblePassword` |
| `TextInputType.name` | Usado para entrada de nomes de pessoas. | `keyboardType: TextInputType.name` |
| `TextInputType.streetAddress` | Usado para entrada de endereços de rua. | `keyboardType: TextInputType.streetAddress` |
| `TextInputType.none` | Não exibe teclado. Usado para situações específicas onde o teclado não deve ser exibido. | `keyboardType: TextInputType.none` |
| `TextInputType.numberWithOptions` | Usado para números, permitindo opções como entrada decimal. | `keyboardType: TextInputType.numberWithOptions(decimal: true)` |

### Principais Métodos do `TextInputType`

A classe `TextInputType` em si não possui métodos específicos que são utilizados diretamente. No entanto, ela pode ser instanciada com o método `TextInputType.numberWithOptions` para configurar opções específicas. Aqui está o formato:

| Método | Descrição | Sintaxe de Uso |
|--------|-----------|----------------|
| `TextInputType.numberWithOptions` | Usado para especificar opções adicionais como entrada de números decimais. | `TextInputType.numberWithOptions(decimal: true, signed: false)` |

### Em quais categorias de widget mais se encaixa?

O `TextInputType` se encaixa nas seguintes categorias de widgets:

- **Input**: Principalmente, já que é utilizado para definir como o teclado deve aparecer em campos de entrada de texto.
- **Text**: Relaciona-se diretamente com a entrada de texto em campos de formulário.

### Em qual sub-categoria se encaixa?

O `TextInputType` se encaixa na sub-categoria de **"Teclado e Entrada de Dados"**, pois seu uso está diretamente relacionado ao controle e customização de teclados para entrada de texto.

### Exemplos adicionais

Aqui estão mais alguns exemplos de uso do `TextInputType` em diferentes contextos:

1. **Campo de e-mail**:
   ```dart
   TextField(
     keyboardType: TextInputType.emailAddress,
     decoration: InputDecoration(
       labelText: 'Email',
     ),
   ),
   ```

2. **Campo de URL**:
   ```dart
   TextField(
   ```

   ```dart
   TextField(

       labelText: 'Valor Decimal',
     ),
   ),
   ```

### Considerações finais

O `TextInputType` é uma ferramenta poderosa para garantir que a experiência do usuário seja fluida e eficiente, ajustando o teclado conforme necessário para a tarefa específica. Com ele, desenvolvedores podem criar formulários e campos de entrada que são mais intuitivos e adaptados ao conteúdo que se espera que o usuário insira.

Além disso, a customização através de `TextInputType` pode reduzir erros de entrada, facilitando a tarefa de preenchimento de formulários e outros campos de texto, o que é especialmente útil em aplicativos que exigem alta precisão, como apps financeiros ou de comunicação.
