### Widget `TextFormField` no Flutter

#### O que é e para que serve?

O `TextFormField` é um widget no Flutter utilizado para capturar entrada de texto do usuário dentro de um formulário. Ele é uma combinação do widget `TextField` com funcionalidades de validação e gerenciamento de estado fornecidas pelo `Form` e `FormState`. Isso o torna ideal para uso em cenários onde múltiplos campos de texto precisam ser validados de forma coordenada, como em formulários de login, cadastro ou outros formulários complexos.

#### Sintaxe de uso

Aqui está um exemplo básico de como utilizar o `TextFormField`:

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Exemplo TextFormField')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: MyCustomForm(),
        ),
      ),
    );
  }
}

class MyCustomForm extends StatefulWidget {
  @override
  _MyCustomFormState createState() => _MyCustomFormState();
}

class _MyCustomFormState extends State<MyCustomForm> {
  final _formKey = GlobalKey<FormState>();

  @override
  Widget build(BuildContext context) {
    return Form(
      key: _formKey,
      child: Column(
        children: <Widget>[
          TextFormField(
            decoration: InputDecoration(labelText: 'Nome'),
            validator: (value) {
              if (value == null || value.isEmpty) {
                return 'Por favor, insira seu nome';
              }
              return null;
            },
          ),
          ElevatedButton(
            onPressed: () {
              if (_formKey.currentState!.validate()) {
                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(content: Text('Processando os dados')),
                );
              }
            },
            child: Text('Enviar'),
          ),
        ],
      ),
    );
  }
}
```

#### Restrições de uso

- **Formulários**: O `TextFormField` deve ser utilizado dentro de um widget `Form`. Fora de um `Form`, o widget não poderá acessar a funcionalidade de validação de formulários.
- **Validação**: A validação é feita através de uma função de callback fornecida ao parâmetro `validator`, que deve retornar uma string com a mensagem de erro caso o valor seja inválido, ou `null` se o valor for válido.
- **Persistência de Estado**: Para garantir que os dados do campo sejam mantidos e acessíveis, é necessário usar uma chave (`GlobalKey`) para associar o formulário ao seu estado.

#### Quando utilizar?

O `TextFormField` é ideal para cenários onde é necessário capturar e validar a entrada de texto em um contexto de formulário. Alguns exemplos incluem:

- **Formulários de cadastro**: Para capturar informações como nome, email, senha, etc.
- **Formulários de login**: Para entradas como nome de usuário e senha.
- **Pesquisa e Filtros**: Em formulários que capturam critérios de pesquisa ou filtros de dados.

#### Diferença do `TextField`

A principal diferença entre `TextFormField` e `TextField` reside na capacidade de validação e gerenciamento de estado no contexto de um formulário:

- **TextFormField**:
  - Integrado ao sistema de validação de formulários do Flutter (`Form`).
  - Permite validação automática e exibição de mensagens de erro.
  - Melhor controle sobre o estado do formulário.

- **TextField**:
  - Simples campo de entrada de texto sem integração nativa com formulários.
  - Não oferece validação automática por padrão.
  - Uso mais flexível fora do contexto de formulários.

#### Tabela de Propriedades

| Propriedade           | Descrição                                                                                      | Sintaxe de Uso                       |
|-----------------------|------------------------------------------------------------------------------------------------|--------------------------------------|
| `controller`          | Controla o conteúdo do campo de texto, permitindo acesso e manipulação do texto.               | `TextEditingController()`            |
| `decoration`          | Define a decoração do campo de texto, como rótulo, borda, ícone, etc.                          | `InputDecoration(labelText: 'Nome')` |
| `keyboardType`        | Define o tipo de teclado a ser exibido (numérico, texto, email, etc.).                         | `TextInputType.text`                 |
| `validator`           | Função que valida o conteúdo do campo. Deve retornar uma string com o erro ou `null` se válido. | `(value) => value.isEmpty ? 'Erro' : null` |
| `initialValue`        | Define o valor inicial do campo de texto.                                                      | `'Valor inicial'`                    |
| `autofocus`           | Define se o campo deve ser focado automaticamente ao ser exibido.                              | `true`                               |
| `obscureText`         | Oculta o texto, útil para senhas.                                                              | `true`                               |
| `onChanged`           | Callback que é acionado sempre que o valor do campo muda.                                      | `(value) => print(value)`            |
| `onSaved`             | Função que é chamada quando o formulário é salvo.                                              | `(value) => salvar(value)`           |
| `enabled`             | Define se o campo está habilitado ou desabilitado.                                             | `true`                               |
| `maxLength`           | Define o número máximo de caracteres permitidos.                                               | `maxLength: 10`                      |
| `textInputAction`     | Define a ação do botão de retorno do teclado (como "próximo", "concluído").                    | `TextInputAction.done`               |
| `style`               | Define o estilo do texto inserido.                                                             | `TextStyle(color: Colors.blue)`      |

#### Em qual categoria de widget mais se encaixa?

- **Categoria Principal**: Input
- **Sub-Categoria**: Formulários e Captura de Dados

#### Considerações Adicionais

O `TextFormField` é uma peça fundamental na construção de formulários em aplicativos Flutter. Ele permite uma integração fácil com o sistema de validação de formulários, garantindo que os dados capturados do usuário sejam precisos e válidos antes de serem processados. Sua capacidade de ser altamente personalizável, através de diversas propriedades, torna-o um componente versátil e poderoso em aplicativos que requerem entrada de dados.

Além disso, é importante lembrar que o uso adequado de validações e feedback ao usuário, como mensagens de erro e sugestões, melhora significativamente a experiência do usuário no uso de formulários complexos.
