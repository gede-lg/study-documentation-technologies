### Widget TextField no Flutter

#### O que é e para que serve?

O `TextField` é um dos widgets mais importantes no Flutter quando se trata de entrada de texto. Ele permite ao usuário digitar e editar texto em tempo real, sendo amplamente utilizado em formulários, campos de pesquisa, caixas de diálogo, entre outros. O `TextField` captura a entrada de texto do usuário e permite manipular essa entrada de diversas formas, como validação, formatação e envio para o backend.

#### Sintaxe de Uso

A sintaxe básica para utilizar um `TextField` no Flutter é a seguinte:

```dart
TextField(
  controller: _controller,
  decoration: InputDecoration(
    labelText: 'Digite algo',
    hintText: 'Insira seu texto aqui',
  ),
  keyboardType: TextInputType.text,
  onChanged: (text) {
    print('Texto digitado: $text');
  },
)
```

No exemplo acima:

- `controller`: É um `TextEditingController` que permite controlar o conteúdo do campo de texto e realizar operações como limpar o campo ou obter o valor atual.
- `decoration`: Adiciona rótulos, dicas de entrada, ícones e estilos ao `TextField`.
- `keyboardType`: Define o tipo de teclado a ser exibido, como texto normal, numérico, e-mail, etc.
- `onChanged`: Callback acionado toda vez que o texto é alterado.

#### Restrições de Uso

Embora o `TextField` seja poderoso, ele tem algumas limitações e considerações:

1. **Foco e Teclado Virtual**: O `TextField` deve ser focado para que o teclado virtual apareça em dispositivos móveis. Isso pode ser feito manualmente usando `FocusNode` ou automaticamente ao tocar no campo.
2. **Validação**: O `TextField` por si só não possui validação embutida. Para validação, é comum o uso do `TextFormField`, que será discutido mais adiante.
3. **Interações Personalizadas**: A personalização de interações avançadas pode ser complexa e pode exigir manipulação direta dos eventos de foco, teclado e entrada de texto.

#### Quando Utilizar?

Use o `TextField` quando precisar de um campo simples para entrada de texto sem necessidade de validação complexa. Exemplos incluem:

- Campos de pesquisa
- Caixas de diálogo de texto simples
- Formulários básicos sem validação

#### Diferença do TextFormField

O `TextFormField` é uma versão mais avançada do `TextField`. Ele é projetado especificamente para ser usado dentro de um formulário (`Form`) e oferece recursos de validação embutidos. Enquanto o `TextField` é ideal para entradas de texto simples, o `TextFormField` é preferido em cenários onde a validação do campo é necessária. Um exemplo de `TextFormField`:

```dart
TextFormField(
  controller: _controller,
  decoration: InputDecoration(
    labelText: 'Digite algo',
    hintText: 'Insira seu texto aqui',
  ),
  validator: (value) {
    if (value == null || value.isEmpty) {
      return 'O campo não pode estar vazio';
    }
    return null;
  },
)
```

#### Tabela de Propriedades

Aqui está uma tabela com as principais propriedades do `TextField`:

| Propriedade          | Descrição                                                                                      | Sintaxe de Uso                            |
|----------------------|------------------------------------------------------------------------------------------------|-------------------------------------------|
| `controller`         | Controla o texto exibido no campo e permite manipular o conteúdo programaticamente             | `TextEditingController controller`        |
| `decoration`         | Permite adicionar rótulos, ícones e estilizar o campo de texto                                 | `InputDecoration decoration`              |
| `keyboardType`       | Define o tipo de teclado a ser exibido                                                         | `TextInputType keyboardType`              |
| `onChanged`          | Callback acionado sempre que o texto é alterado                                                | `Function(String) onChanged`              |
| `focusNode`          | Gerencia o foco do `TextField`                                                                 | `FocusNode focusNode`                     |
| `obscureText`        | Oculta o texto (útil para senhas)                                                              | `bool obscureText`                        |
| `maxLength`          | Define o número máximo de caracteres que podem ser inseridos                                   | `int maxLength`                           |
| `onSubmitted`        | Callback acionado quando o usuário submete o campo                                             | `Function(String) onSubmitted`            |
| `autofocus`          | Determina se o campo deve ser focado automaticamente ao ser exibido                            | `bool autofocus`                          |
| `enabled`            | Define se o campo está habilitado ou desabilitado                                              | `bool enabled`                            |
| `cursorColor`        | Define a cor do cursor                                                                         | `Color cursorColor`                       |
| `textInputAction`    | Define a ação de retorno do teclado (como 'done', 'next', 'search')                            | `TextInputAction textInputAction`         |
| `style`              | Define o estilo do texto dentro do campo                                                       | `TextStyle style`                         |
| `textAlign`          | Alinha o texto dentro do campo                                                                 | `TextAlign textAlign`                     |
| `readOnly`           | Define se o campo é somente leitura                                                            | `bool readOnly`                           |
| `textCapitalization` | Define a capitalização automática do texto (como todas maiúsculas, capitalização de sentenças) | `TextCapitalization textCapitalization`   |
| `inputFormatters`    | Define formatações específicas para o texto inserido                                           | `List<TextInputFormatter> inputFormatters`|

#### Categoria e Sub-categoria

**Categoria**: O `TextField` se encaixa na categoria `Input`, pois é diretamente relacionado à entrada de dados pelo usuário.

**Sub-categoria**: Dentro de `Input`, o `TextField` pode ser classificado como um widget de `Text Input`, que abrange todos os widgets que permitem a entrada de texto.

#### Informações Adicionais

- **Interação com Teclado**: Em dispositivos móveis, o `TextField` abre automaticamente o teclado virtual. Em desktops, o comportamento pode ser gerenciado para fornecer uma experiência similar.
- **Customização**: O `TextField` é altamente personalizável, permitindo que desenvolvedores ajustem desde a aparência do texto até o comportamento do cursor e o tipo de teclado.
- **Acessibilidade**: É importante garantir que o `TextField` seja acessível para todos os usuários, adicionando rótulos descritivos (`semantics`) para tecnologias assistivas.

#### Exemplo Completo

Aqui está um exemplo completo de um `TextField` em um formulário básico:

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Exemplo de TextField'),
        ),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: MyForm(),
        ),
      ),
    );
  }
}

class MyForm extends StatefulWidget {
  @override
  _MyFormState createState() => _MyFormState();
}

class _MyFormState extends State<MyForm> {
  final _controller = TextEditingController();

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        TextField(
          controller: _controller,
          decoration: InputDecoration(
            labelText: 'Digite seu nome',
            hintText: 'Nome completo',
          ),
          onChanged: (text) {
            print('Texto digitado: $text');
          },
        ),
        SizedBox(height: 20),
        ElevatedButton(
          onPressed: () {
            print('Nome submetido: ${_controller.text}');
          },
          child: Text('Submeter'),
        ),
      ],
    );
  }
}
```

Neste exemplo, o usuário pode digitar seu nome em um campo de texto, e ao clicar no botão, o nome é impresso no console.
